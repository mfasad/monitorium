FPS-счётчик в MSI Afterburner: как настроить вывод кадров на экран монитора

Отображение количества кадров в секунду (FPS) — обязательный инструмент для геймеров, стримеров и энтузиастов, которые хотят контролировать производительность системы. MSI Afterburner в паре с RivaTuner Statistics Server (RTSS) остаётся самым универсальным решением для вывода FPS на экран, несмотря на появление альтернатив вроде NVIDIA GeForce Experience или Steam Overlay. Главное преимущество этого метода — поддержка любых мониторов (от бюджетных 60 Гц до профессиональных 360 Гц), гибкая настройка позиции счётчика и минимальное влияние на производительность.

В этой статье мы разберём не только базовую настройку FPS-счётчика, но и нюансы, которые редко упоминают в стандартных гайдах: как избежать артефактов при записях OBS, почему счётчик может мерцать на мониторах с G-Sync/FreeSync, и как настроить вывод дополнительных метрик (загрузка GPU/CPU, температура, частота кадров). Особое внимание уделим проблемам совместимости с Windows 11 и новыми видеокартами NVIDIA RTX 40/AMD Radeon 7000.

1. Подготовка: что нужно для работы FPS-счётчика

Прежде чем настраивать отображение FPS, убедитесь, что у вас установлены все необходимые компоненты. MSI Afterburner сам по себе не умеет выводить счётчик на экран — для этого требуется RivaTuner Statistics Server (RTSS), который идёт в комплекте с программой. Вот полный список требований:

  • 🖥️ Видеокарта: Любая от NVIDIA (серии GTX/RTX) или AMD (Radeon RX 400+). Для интегрированной графики Intel могут потребоваться дополнительные настройки.
  • 📥 MSI Afterburner: Версия 4.6.5 или новее (скачивайте только с официального сайта). Старые версии могут не поддерживать новые GPU.
  • 🔧 Права администратора: RTSS требует запуска от имени администратора для корректной работы с DirectX/Vulkan.
  • 🎮 Игра: Поддерживаются практически все игры на движках Unreal Engine, Unity, Source, а также большинство AAA-проектов. Исключения — некоторые античиты (например, EAC в Fortnite или Valorant) могут блокировать оверлей.

Если вы используете многоэкранную конфигурацию (например, основной монитор 1440p 144Hz + дополнительный 1080p 60Hz), учтите: RTSS по умолчанию выводит счётчик на первичный дисплей. Чтобы перенаправить его на игровой монитор, потребуется ручная настройка в файле конфигурации (об этом расскажем в разделе про продвинутые настройки).

⚠️ Внимание: На ноутбуках с гибридной графикой (NVIDIA Optimus или AMD SmartShift) счётчик может не работать в играх, запущенных на интегрированном GPU. Чтобы исправить это, принудительно назначьте игру на дискретную видеокарту через панель управления драйвером.

2. Установка и первый запуск MSI Afterburner

Скачайте последнюю версию MSI Afterburner с официального сайта. Во время установки обязательно отметьте галочкой пункт RivaTuner Statistics Server — без него вывод FPS на экран будет невозможен. Если вы пропустили этот шаг, программу придётся переустановить.

После установки:

  1. Запустите MSI Afterburner от имени администратора (правый клик по ярлыку → Запуск от имени администратора).
  2. В главном окне программы перейдите в Настройки (шестерёнка внизу или клавиша F1).
  3. Откройте вкладку Мониторинг — здесь настраиваются все параметры отображения.

Если после запуска программы вы видите ошибку "RivaTuner Statistics Server не отвечает", выполните следующие действия:

  • 🔄 Перезагрузите компьютер.
  • 🛠️ Обновите драйверы видеокарты до последней версии.
  • 🔒 Временно отключите антивирус (например, Avast или Kaspersky может блокировать RTSS).
  • 📂 Переустановите программу, выбрав Custom Install и вручную указав путь без кириллических символов (например, C:\MSI\Afterburner\).

3. Настройка отображения FPS в играх

Теперь перейдём к самому важному — активации счётчика кадров. Вкладка Мониторинг в настройках MSI Afterburner содержит список всех доступных метрик. Нам нужны:

  1. Найдите в списке параметр Framerate (или FPS на русском).
  2. Поставьте галочку напротив него.
  3. Ниже, в разделе Свойства, выберите Показывать в ОЭД (Оверлей Экранных Данных).
  4. Нажмите ОК, чтобы сохранить изменения.

По умолчанию счётчик будет отображаться в верхнем левом углу экрана зелёным цветом. Чтобы изменить его положение или стиль:

  • 🎨 Вернитесь в Настройки → Мониторинг.
  • 📍 В разделе Оверлей Экранных Данных настройте:
    • Позиция ОЭД — выберите угол экрана.
    • Цвет текста — рекомендуем жёлтый или белый для лучшей видимости.
    • Размер шрифта — оптимально 16–24pt для мониторов Full HD.
💡

Если счётчик FPS мерцает или пропадает в некоторых играх, попробуйте включить опцию Low-level API hooking в настройках RTSS (вкладка Общие). Это поможет с играми на Vulkan или DX12, но может снизить производительность на 1–3%.

Для мониторов с высокой частотой обновления (144Hz+) полезно выводить не только FPS, но и Frame Time (время кадра в мс). Это поможет отслеживать микрозаикания, которые не видны по одному только счётчику кадров. Чтобы добавить эту метрику, повторите шаги для параметра Frametime в списке мониторинга.

4. Продвинутые настройки: RTSS и ручная конфигурация

Если стандартных настроек MSI Afterburner недостаточно, можно редактировать конфигурационный файл RivaTuner Statistics Server вручную. Это понадобится для:

  • 🖥️ Перенаправления оверлея на второй монитор.
  • 🎯 Точной настройки позиции счётчика (например, по центру экрана).
  • 🔄 Изменения формата вывода (например, добавление префикса "FPS: ").

Файл конфигурации находится по пути: 

C:\Program Files (x86)\RivaTuner Statistics Server\Profiles\

Ищите файл с названием игры (например, Cyberpunk 2077.cfg) или создайте универсальный профиль Global.cfg.

Пример редактирования для вывода FPS по центру экрана:

[OSD]

PositionX=50


PositionY=50


TextOutputFormat=%fps% FPS

Где:

  • PositionX/PositionY — координаты в процентах от краёв экрана (50 — центр).
  • TextOutputFormat — формат текста (%fps% подставляет текущее значение FPS).
⚠️ Внимание: Неправильное редактирование конфигурационных файлов может привести к сбоям в работе RTSS. Перед изменениями сделайте резервную копию файла или создайте точку восстановления системы.
Как вернуть стандартные настройки RTSS?

Если после редактирования файлов оверлей перестал работать, удалите папку Profiles в директории RTSS. Программа автоматически создаст её заново с настройками по умолчанию при следующем запуске.

5. Решение распространённых проблем

Даже после правильной настройки пользователи часто сталкиваются с проблемами. Вот самые частые из них и способы их решения:

Проблема Возможная причина Решение
Счётчик не появляется в игре Игра запущена не в полноэкранном режиме или блокируется античитом Переключитесь в Fullscreen (не Borderless Windowed). Для игр с EAC/BattlEye попробуйте Steam Overlay или NVIDIA ShadowPlay
FPS отображается с задержкой Конфликт с программой захвата видео (OBS, Streamlabs) В настройках OBS отключите Использовать захват оверлеев или снизьте приоритет RTSS в Диспетчере задач
Счётчик мерцает или пропадает Несовместимость с G-Sync/FreeSync или устаревший RTSS Обновите MSI Afterburner до последней версии. В настройках монитора отключите VSYNC в игре
Некорректные значения FPS (например, 300+ на 60Hz мониторе) Ошибка считывания данных с GPU В настройках RTSS включите Limit frame rate и установите значение на 1–2 кадра ниже максимума монитора (например, 58 для 60Hz)

Для владельцев мониторов с G-Sync или FreeSync важно понимать: счётчик FPS может отображать значения выше максимальной частоты обновления экрана (например, 180 FPS на 144Hz мониторе). Это нормально — технология адаптивной синхронизации позволяет избежать разрывов, но реальная частота обновления экрана не превышает его физический лимит.

6. Оптимизация для записей и стримов

Если вы записываете геймплей через OBS Studio или стримите в Twitch/YouTube, стандартный оверлей MSI Afterburner может попадать в запись, что не всегда желательно. Чтобы этого избежать:

  1. В OBS перейдите в Настройки → Вывод → Запись.
  2. Включите опцию Использовать многопоточную оптимизацию.
  3. В настройках источника захвата игры (Захват игры или Захват экрана) отметьте Захватить оверлеи отдельно.

Для стримеров полезно выводить FPS только на свой монитор, не показывая его зрителям. Сделать это можно двумя способами:

  • 🖼️ Способ 1: В OBS добавьте источник Текст (GDI+) и привяжите его к переменной FPS через плагин Advanced Scene Switcher.
  • 🔧 Способ 2: В настройках RTSS создайте отдельный профиль для OBS с отключённым оверлеем (OnScreenDisplay=0 в конфигурационном файле).

Критическая деталь для стримеров: если вы используете NVIDIA NVENC для кодирования видео, включите в RTSS опцию Enable low-latency mode. Это снизит задержку оверлея до 1–2 кадров, что критично для динамичных игр вроде CS2 или Valorant.

7. Альтернативные способы вывода FPS

Если MSI Afterburner по какой-то причине не подходит, рассмотрите альтернативы:

  • 🎮 Steam Overlay: Включается в настройках Steam (В игре → Включить оверлей Steam в игре). Показывает FPS, но работает только в играх из библиотеки Steam.
  • 🖥️ NVIDIA GeForce Experience: Включите В игре оверлей в настройках программы. Поддерживает вывод FPS, температуры GPU и другие метрики.
  • 🔧 Fraps: Устаревшая, но лёгкая утилита для записей и вывода FPS. Не поддерживает новые API (DX12, Vulkan).
  • 📊 CapFrameX: Продвинутый инструмент для анализа производительности с выводом графика Frame Time. Подходит для бенчмаркинга.

Сравнение альтернатив:

Программа Поддержка API Влияние на FPS Доп. функции
MSI Afterburner + RTSS DX9, DX11, DX12, Vulkan, OpenGL 1–3% Мониторинг GPU/CPU, разгон, записи
Steam Overlay DX9, DX11, Vulkan 2–5% Чат, браузер, скриншоты
NVIDIA GFE DX9, DX11, DX12 3–7% Автоматическая оптимизация, записи, стриминг
Fraps DX9, DX11, OpenGL 5–10% Запись видео, бенчмаркинг

Для большинства пользователей MSI Afterburner остаётся оптимальным выбором благодаря минимальному влиянию на производительность и широким возможностям настройки. Однако если вам нужны записи геймплея без потерь FPS, рассмотрите специализированные программы вроде OBS с плагином FPS Monitor.

💡

MSI Afterburner + RTSS — единственное решение, которое работает со всеми играми на любых API (включая новые DX12 Ultimate и Vulkan 1.3), тогда как альтернативы часто ограничены поддержкой устаревших технологий.

FAQ: Частые вопросы по настройке FPS-счётчика

Можно ли вывести FPS на второй монитор, если игра запущена на первом?

Да, но для этого потребуется ручное редактирование конфигурационного файла RTSS. В профиле игры (Имя_игры.cfg) добавьте строки:

[OSD]

DisplayAdapter=-1  ; -1 — основной монитор, 0 — первый дополнительный, 1 — второй и т.д.


PositionX=50


PositionY=50

Значение DisplayAdapter соответствует порядку мониторов в настройках Windows (Параметры → Система → Дисплей).

Почему FPS в счётчике и в игре (например, в меню CS2) отличаются?

Это связано с тем, что некоторые игры (особенно на движке Source 2) используют собственные ограничители кадров. Счётчик MSI Afterburner показывает реальный FPS, выдаваемый GPU, тогда как встроенный счётчик игры может отображать значение после применения VSYNC или FPS_limit.

Чтобы увидеть настоящий FPS, отключите все ограничители в настройках игры и драйвера видеокарты.

Как сделать так, чтобы счётчик FPS не попал в скриншот (Print Screen)?

По умолчанию оверлей RTSS попадает на скриншоты, сделанные через PrtScn или Win + Shift + S. Чтобы этого избежать:

  1. Откройте настройки RTSS (C:\Program Files (x86)\RivaTuner Statistics Server\RTSS.exe).
  2. Перейдите на вкладку Общие.
  3. Снимите галочку с Захват оверлея в скриншотах.

После этого счётчик будет виден только в реальном времени, но не на сохранённых изображениях.

Можно ли вывести FPS в играх на macOS или Linux?

MSI Afterburner и RivaTuner официально поддерживают только Windows. Для Linux альтернативы:

  • MangoHud — оверлей для игр на Vulkan/OpenGL (устанавливается через sudo pacman -S mangohud на Arch Linux).
  • vkBasalt — пост-обработка с выводом FPS для Vulkan-игр.

Для macOS вариантов почти нет — можно попробовать Xcode Frame Debugger (только для игр на Metal) или виртуальную машину с Windows.

Как сбросить все настройки MSI Afterburner к заводским?

Если вы экспериментировали с настройками и хотите вернуть всё к исходному состоянию:

  1. Закройте MSI Afterburner и RTSS.
  2. Удалите файлы конфигурации по пути: 
    C:\Users\<Ваше_имя_пользователя>\AppData\Roaming\MSI\Afterburner\
  3. Перезапустите программу — она создаст новые файлы с настройками по умолчанию.

Для полного сброса также можно использовать опцию Reset settings в меню Профиль в главном окне MSI Afterburner.
