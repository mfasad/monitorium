Как сделать скриншот одного экрана, если подключено 2 монитора: все способы

Работа с двумя мониторами упрощает многозадачность, но стандартные инструменты для создания скриншотов часто захватывают оба экрана одновременно. Это создаёт неудобства, если нужно сохранить только рабочий стол на основном дисплее или, наоборот, зафиксировать изображение с дополнительного монитора без лишних элементов. Проблема усугубляется, когда экраны имеют разное разрешение или ориентацию — итоговый снимок получается неравномерным, с чёрными полями или искажённым масштабом.

В этой статье разберём все актуальные способы сделать скриншот только одного монитора — от встроенных горячих клавиш до специализированных программ. Опишем нюансы для Windows 10/11, macOS и Linux, а также рассмотрим решения для игровых ПК, где стандартные методы могут не сработать из-за полноэкранных приложений. Отдельно остановимся на скрытой функции инструмента «Ножницы» в Windows, которая позволяет выборочно захватывать активный дисплей без стороннего ПО.

Почему стандартные скриншоты захватывают оба монитора

По умолчанию операционные системы воспринимают мультимониторную систему как единое рабочее пространство. Когда вы нажимаете PrtScn (Print Screen) или используете комбинацию Win + PrtScn, ОС фиксирует виртуальный канвас, который включает оба экрана. Это логично для большинства задач, но создаёт проблемы в специфических сценариях:

  • 📊 Отчёты и презентации — нужно показать только график на основном мониторе, без лишних окон с дополнительного.
  • 🎮 Стриминг и записи геймплея — захват второго монитора с чатом или панелью управления портит кадр.
  • 🔍 Отладка ПО — требуется зафиксировать ошибку на конкретном дисплее, где она проявляется.
  • 📝 Обучение и туториалы — инструкции становятся нечитаемыми, если на скриншоте два экрана с разным масштабированием.

Технически проблема кроется в том, как ОС обрабатывает Display Device Context. Даже если вы используете программу вроде Snipping Tool («Ножницы»), по умолчанию она предлагает захватить весь экран — а это значит оба монитора. Решения есть, но они зависят от вашей операционной системы и задач.

Способ 1: Горячие клавиши для выборочного скриншота (Windows)

В Windows 10 и Windows 11 есть скрытая возможность сделать снимок только активного монитора без сторонних утилит. Для этого:

  1. Активируйте окно или рабочий стол на том мониторе, который нужно заскринить (кликните по нему мышкой).
  2. Нажмите комбинацию: 
    Alt + PrtScn

    Эта команда копирует в буфер обмена только активное окно, но если окно развернуто на весь экран — будет захвачен весь монитор.

  3. Вставьте скриншот в Paint, Word или другой редактор (Ctrl + V).

⚠️ Внимание: Если на мониторе открыто несколько окон, Alt + PrtScn захватит только то, которое активно в данный момент. Чтобы зафиксировать весь экран, предварительно разверните любое окно на полный экран (F11 или двойной клик по заголовку).

Для Windows 11 есть альтернативный метод:

  • Нажмите Win + Shift + S — откроется панель инструмента «Вырезка».
  • Выберите режим «Экранная вырезка» (значок монитора).
  • Кликните по нужному дисплею — снимок будет скопирован в буфер.

Способ 2: Инструмент «Ножницы» (Snipping Tool) с скрытой функцией

Встроенная утилита Snipping Tool (в Windows 11 переименована в «Вырезка и аннотация») умеет захватывать отдельные мониторы, но эта опция спрятана. Вот как её активировать:

  1. Откройте «Ножницы» через поиск (Win + S → введите «ножницы»).
  2. Нажмите на стрелочку рядом с кнопкой «Создать» и выберите «Экранная вырезка».
  3. Экран потемнеет, а курсор превратится в крестик. Кликните по нужному монитору — он подсветится белой рамкой.
  4. Снимок откроется в редакторе, где его можно сохранить (Ctrl + S) или аннотировать.

🔹 Важно: Если у вас Windows 7 или старая версия Snipping Tool, опция «Экранная вырезка» может отсутствовать. В этом случае используйте Win + PrtScn (сохраняет скриншот в Этот компьютер → Изображения → Снимки экрана), а затем обрежьте лишнее в редакторе.

Способ 3: Специализированные программы для мультимониторных систем

Если встроенные инструменты не подходят (например, нужно автоматизировать процесс или делать скриншоты по таймеру), используйте сторонние утилиты. Лучшие варианты:

  • 🖥️ ShareX — бесплатная программа с поддержкой multi-monitor capture. Позволяет выбрать конкретный дисплей в меню захвата (Capture → Screen → Monitor 1/2).
  • 📸 Lightshot — после нажатия PrtScn появляется рамка, которую можно растянуть на один монитор. Удобно для быстрых снимков с аннотациями.
  • 🎛️ Snagit (платная) — профессиональный инструмент с поддержкой scrolling capture и редактором. В настройках можно присвоить горячие клавиши для каждого монитора отдельно.
  • 🎮 OBS Studio — если нужно не только скриншоты, но и запись видео с одного экрана. В источниках захвата выберите «Захват экрана» → «Конкретный монитор».

🔹 Совет для геймеров: В играх с полноэкранным режимом (Borderless Windowed или Exclusive Fullscreen) стандартные методы могут не работать. Используйте Steam Overlay (F12) или NVIDIA ShadowPlay (Alt + F1), предварительно настроив захват только основного дисплея в панели управления драйвером.

💡

В ShareX можно настроить автоматическое сохранение скриншотов в облако (Google Drive, Imgur) или отправку по email. Это удобно для удалённой работы или баг-репортов.

Способ 4: Скриншот одного монитора в macOS

На Mac процесс проще благодаря встроенным инструментам:

  1. Нажмите Cmd + Shift + 4 — курсор превратится в прицел.
  2. Нажмите Пробел — курсор станет камерой, а мониторы подсветятся синим.
  3. Кликните по нужному дисплею. Снимок сохранится на рабочий стол в формате .png.

Для записи экрана с одного монитора:

  1. Откройте QuickTime Player.
  2. Выберите Файл → Новая запись экрана.
  3. Нажмите на стрелочку рядом с кнопкой записи и выберите нужный монитор в разделе «Микрофон» (да, это странное расположение, но там же настраивается источник видео).

⚠️ Внимание: Если у вас macOS Ventura или новее, интерфейс инструмента скриншотов изменился. Теперь после нажатия Cmd + Shift + 5 появляется панель с опциями, где можно выбрать «Захват всего экрана» и указать конкретный монитор.

Способ 5: Linux и альтернативные решения

В дистрибутивах на базе GNOME (Ubuntu, Fedora) и KDE Plasma (Kubuntu) есть свои нюансы:

  • 🐧 GNOME: Нажмите Shift + PrtScn, затем обведите нужный монитор рамкой. Для захвата всего экрана используйте PrtScn, но затем обрежьте лишнее в GIMP.
  • 🐧 KDE Plasma: Утилита Spectacle позволяет выбрать конкретный монитор в меню «Режим» → «Текущий монитор».
  • 🐧 Для терминала: Используйте scrot с параметром --display: 
    scrot --display :0.0 screenshot.png  # Основной монитор

    scrot --display :0.1 screenshot.png  # Второй монитор

    Узнать идентификаторы дисплеев можно командой xrandr.

🔹 Для удалённых серверов: Если вы подключены по SSH с пробросом графики (X11 forwarding), скриншоты можно делать через Import из пакета ImageMagick: 

import -window root -display :0.0 screenshot.jpg

Как узнать идентификаторы мониторов в Linux?

Запустите в терминале команду xrandr --listmonitors. Вывод будет примерно таким:

Monitors: 2

0: +*HDMI-1 1920/527x1080/296+0+0  HDMI-1


1:  DP-1 1920/527x1080/296+1920+0  DP-1

Здесь HDMI-1 и DP-1 — идентификаторы для команд вроде scrot или ffmpeg.

Распространённые ошибки и как их избежать

Даже опытные пользователи сталкиваются с проблемами при создании скриншотов на мультимониторных системах. Вот типичные ошибки и их решения:

  • 🖼️ Снимок получается чёрным: Это происходит, если приложение (например, видеоплеер или игра) использует hardware acceleration. Решение — сделать скриншот через Steam (F12) или GeForce Experience.
  • 🔄 Мониторы захватываются в неправильном порядке: В ОС мониторы нумеруются слева направо. Если физическое расположение не совпадает с логическим, измените его в настройках дисплея (Win + P → «Расширить»).
  • 📏 Скриншот обрезается по краям: Виной разное разрешение или масштабирование (например, 125% на одном мониторе и 100% на другом). Выровняйте масштаб в Параметры → Система → Дисплей.
  • 🔒 Горячие клавиши не работают: В некоторых играх или приложениях (например, VMware) клавиши перехватываются. Используйте внешние утилиты вроде ShareX с кастомными хоткеями.

⚠️ Внимание: Если вы используете Remote Desktop (RDP) для подключения к ПК с двумя мониторами, скриншоты будут захватывать только виртуальный экран RDP-сессии, а не физические дисплеи. Чтобы зафиксировать реальные мониторы, подключитесь напрямую или используйте TeamViewer с опцией Full Screen.

💡

Перед созданием скриншота проверьте, что курсор мыши находится на целевом мониторе — многие инструменты определяют активный дисплей по положению указателя.

FAQ: Частые вопросы о скриншотах на двух мониторах

Можно ли сделать скриншот только второго монитора, если он выключен?

Нет. Операционная система не может захватить изображение с отключённого дисплея. Однако если монитор просто находится в спящем режиме (подсветка выключена, но сигнал идёт), некоторые программы (например, ShareX) могут «разбудить» его на время захвата. Проверьте настройки управления питанием в Панель управления → Электропитание.

Почему скриншот сохраняется с чёрными полями по бокам?

Это происходит, если мониторы имеют разное разрешение или ориентацию (например, один вертикальный, другой горизонтальный). Решения:

  • Обрежьте скриншот в редакторе (Paint, Photoshop).
  • Настройте одинаковое разрешение в параметрах дисплея (но это может исказить изображение).
  • Используйте ShareX с опцией «Auto-crop».

Как сделать скриншот одного монитора в полноэкранной игре?

Стандартные методы (PrtScn, Alt + PrtScn) в играх с Exclusive Fullscreen не работают. Альтернативы:

  • Используйте Steam Overlay (F12) — работает в большинстве игр из библиотеки Steam.
  • Для NVIDIA: настройте ShadowPlay в GeForce Experience на захват конкретного монитора.
  • Для AMD: используйте ReLive с настройкой области захвата.
  • Переключите игру в оконный режим (Windowed или Borderless) и используйте Win + Alt + PrtScn (для Xbox Game Bar).

Есть ли способ автоматизировать скриншоты одного монитора по расписанию?

Да. Для этого подойдут:

  • ShareX: в настройках задач (Task Settings) создайте правило с триггером по времени и действием «Capture monitor».
  • AutoHotkey: напишите скрипт с командой: 
    #IfWinActive ahk_exe game.exe  ; Пример для конкретной игры

    F9::  ; Горячая клавиша
    

    {
    

    Run, C:\Tools\ShareX\ShareX.exe -CaptureMonitor 1  ; Номер монитора
    

    }
    

    #IfWinActive

  • Task Scheduler (Windows): создайте задачу, которая запускает Snipping Tool с параметрами через командную строку.

Как скринить один монитор на ноутбуке с подключённым внешним дисплеем?

Ноутбук с внешним монитором обрабатывается ОС как мультимониторная система. Используйте те же методы:

  • Для внешнего монитора: активируйте его кликом и нажмите Win + Shift + S, затем выберите экранную вырезку.
  • Для экрана ноутбука: если внешний монитор основной, нажмите Win + P и выберите «Только экран ПК», затем сделайте скриншот (PrtScn). Верните настройки обратно.
  • В macOS: Cmd + Shift + 4 → Пробел → клик по нужному дисплею.

⚠️ Если ноутбук закрыт (режим clamshell), скриншоты экрана ноутбука сделать невозможно — ОС блокирует захват.
