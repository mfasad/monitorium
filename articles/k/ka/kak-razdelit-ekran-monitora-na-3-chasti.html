Как разделить экран монитора на 3 части: пошаговые инструкции для Windows и macOS

Работа с несколькими окнами одновременно — неотъемлемая часть продуктивности для программистов, дизайнеров и всех, кто занимается многозадачностью. Но стандартные функции Windows и macOS часто ограничиваются делением экрана пополам или на 4 части. А что, если вам нужно разделить экран монитора на 3 части для удобного размещения мессенджера, таблицы и браузера? В этой статье разберём все доступные способы — от встроенных инструментов до специализированных утилит.

Мы протестировали методы на Windows 10/11 и macOS Ventura/Monterey, а также учли особенности ультрашироких мониторов с разрешением 21:9 и 32:9. Вы узнаете, как настроить три окна вручную, использовать горячие клавиши и какие программы помогут автоматизировать процесс. Особое внимание уделим проблеме "неравномерного" деления, когда центральное окно получается уже боковых — и как её избежать.

1. Стандартные средства Windows: как обойти ограничения

Microsoft не предусмотрела в Windows 10/11 прямой функции деления экрана на 3 части, но есть обходные пути. Основной метод — использование виртуальных рабочих столов в комбинации с ручным изменением размера окон. Вот как это работает:

1. Откройте три нужных окна (например, Word, Excel и браузер).

2. Перетащите первое окно в левый угол экрана — оно займёт половину пространства.

3. Второе окно перетащите в правый верхний угол — оно автоматически займёт четверть экрана.

4. Третье окно разместите в правом нижнем углу, удерживая клавишу Win для точной подгонки.

⚠️ Внимание: При таком способе центральное окно будет уже боковых. Чтобы выровнять ширину, после размещения зажмите Win + ←/→ и вручную подкорректируйте границы мышью.
  • ✅ Плюсы: не требует установки ПО, работает на любых версиях Windows
  • ❌ Минусы: окна получаются неравномерными, нужно корректировать вручную
  • 🔧 Альтернатива: используйте Win + Стрелки для быстрого перемещения окон

2. FancyZones в PowerToys: профессиональное решение от Microsoft

PowerToys — официальный набор утилит от Microsoft, среди которых FancyZones специально предназначен для гибкого управления окнами. Эта программа позволяет создать шаблон с тремя равными зонами или настроить произвольное деление.

Инструкция по настройке:

1. Скачайте PowerToys с официального сайта и установите.

2. Запустите утилиту и перейдите в раздел FancyZones.

3. Выберите шаблон Columns (3) или создайте свой через Edit zones.

4. Активируйте функцию клавишей Shift + ~ (тильда) и перетаскивайте окна в зоны.

Функция Windows 10 Windows 11 macOS
Встроенное деление на 3 части ❌ Нет ❌ Нет ❌ Нет
FancyZones (PowerToys) ✅ Да ✅ Да ❌ Нет
Горячие клавиши для зон ✅ Да (Win + Стрелки) ✅ Да (Win + Стрелки) ✅ Да (Ctrl + Стрелки)

FancyZones поддерживает мультимониторные конфигурации и сохраняет расположение окон между перезагрузками. Для ультрашироких мониторов (21:9) можно создать асимметричные зоны — например, две широкие по бокам и узкую посередине для чата.

💡

Если FancyZones не срабатывает на некоторых окнах (например, в играх или старых программах), добавьте их в исключения в настройках утилиты через Excluded apps

3. Специализированные программы: GridMove, DisplayFusion и другие

Когда встроенных инструментов недостаточно, на помощь приходят сторонние утилиты. Рассмотрим топ-3 решения для деления экрана на 3 и более частей:

  • 🔹 GridMove: бесплатная программа с открытым исходным кодом. Позволяет создавать до 12 зон на экране, поддерживает горячие клавиши и мультимониторные установки. Минус — устаревший интерфейс.
  • 🔹 DisplayFusion (платная): профессиональный инструмент с функциями для геймеров и дизайнеров. Может делить экран на 3 части с точным контролем пикселей, поддерживает макросы и триггеры.
  • 🔹 WindowGrid: лёгкая утилита с визуальным редактором сеток. Подходит для новичков — позволяет перетаскивать окна в зоны мышью без запоминания горячих клавиш.

Для DisplayFusion актуальна следующая инструкция:

1. Установите программу и перейдите в Window Management → Window Snapping.

2. Выберите шаблон Three Columns или создайте свой через Custom Window Snapping Regions.

3. Назначьте горячие клавиши (например, Ctrl + Alt + 1/2/3) для быстрого перемещения окон.

⚠️ Внимание: Некоторые программы (например, GridMove) могут конфликтовать с драйверами видеокарт NVIDIA или AMD. Перед установкой обновите драйверы через Диспетчер устройств.

4. macOS: как разделить экран на 3 части без сторонних программ

В отличие от Windows, macOS предлагает более гибкие встроенные инструменты для управления окнами. Хотя стандартная функция Split View делит экран только пополам, есть два обходных пути:

Метод 1: Использование "Уголков экрана" (Hot Corners)

1. Перейдите в Системные настройки → Рабочий стол и док-станция → Уголки экрана.

2. Назначьте одному из углов действие Запустить экранную заставку (это временно свернёт все окна).

3. Разместите три окна вручную, удерживая клавишу Option для точного изменения размера.

Метод 2: Приложение "Миссия контроль" (Mission Control)

1. Откройте Mission Control свайпом вверх на трекпаде или клавишей F3.

2. Перетащите окна в разные рабочие столы, а затем вернитесь к основному.

3. Используйте жест разведение пальцев для масштабирования и точного позиционирования.

Для автоматического деления на 3 части в macOS понадобятся утилиты вроде Mosaic или Magnet (платные, ~$5 в App Store). Они добавляют горячие клавиши для перемещения окон в зоны, аналогично FancyZones в Windows.

Как сбросить настройки Mission Control?

Если после экспериментов с разделением экрана Mission Control начал работать некорректно, выполните сброс:

1. Откройте Терминал (Programs → Utilities).

2. Введите команду: 

defaults write com.apple.dock mcx-expose-disabled -bool FALSE; killall Dock

3. Перезагрузите Mac.

5. Особенности для ультрашироких мониторов (21:9, 32:9)

Мониторы с соотношением сторон 21:9 (например, LG 34WK95U) или 32:9 (Samsung Odyssey G9) требуют особого подхода к делению экрана. Их ширина позволяет комфортно разместить три окна без потери читаемости, но стандартные инструменты Windows/macOS часто делят пространство неоптимально.

Рекомендации для ультрашироких экранов:

1. В FancyZones создайте шаблон с зонами 33% | 33% | 33% вместо стандартного 50% | 50%.

2. Для мониторов 32:9 используйте асимметричное деление: 40% | 20% | 40% — так центральное окно не будет слишком узким.

3. В играх или графических редакторах (Photoshop, Blender) отключайте разделение экрана — оно может вызывать артефакты при рендеринге.

  • 📏 Оптимальное разрешение для 3 окон на 3440×1440: каждое окно должно занимать 1100–1150px по ширине.
  • 🎮 Для геймеров: программы вроде DisplayFusion позволяют создавать "игровые зоны" с отключением разделения по горячей клавише.
  • 🖥️ Проблема с масштабированием: на мониторах 4K/5K проверьте настройки масштаба в Параметры экрана → Масштаб (должно быть 100% для точного деления).

6. Горячие клавиши и скрипты для автоматического деления

Если вы предпочитаете управлять окнами с клавиатуры, можно настроить собственные комбинации или использовать скрипты. Для Windows подойдёт AutoHotkey, для macOS — Hammerspoon.

Пример скрипта для AutoHotkey (Windows): 

; Делим экран на 3 вертикальные части (Win + 1/2/3)

#1:: ; Win+1 — левая зона


WinGetPos, X, Y, W, H, A


WinMove, A, , 0, 0, %W%//3, %H%


return



#2:: ; Win+2 — центральная зона


WinGetPos, X, Y, W, H, A


WinMove, A, , %W%//3, 0, %W%//3, %H%


return



#3:: ; Win+3 — правая зона


WinGetPos, X, Y, W, H, A


WinMove, A, , %W%//1.5, 0, %W%//3, %H%


return

Для macOS в Hammerspoon аналогичный функционал настраивается через Lua-скрипты. Главное преимущество такого подхода — мгновенное переключение между окнами без использования мыши.

⚠️ Внимание: Перед использованием скриптов отключите стандартные горячие клавиши Windows/macOS для управления окнами, чтобы избежать конфликтов. В Windows это делается в Параметры → Система → Многозадачность.
💡

Горячие клавиши ускоряют работу на 30–40% по сравнению с ручным перетаскиванием окон (по данным исследования Microsoft Research, 2021).

7. Частые проблемы и их решения

При делении экрана на 3 части пользователи сталкиваются с типичными трудностями. Вот как их решить:

Проблема Причина Решение
Окна "выпадают" из зон после перезагрузки Настройки FancyZones/DisplayFusion не сохранены Включите опцию Save window positions в настройках программы
Центральное окно слишком узкое Стандартный шаблон делит экран на 50% + 25% + 25% Создайте кастомный шаблон в FancyZones с зонами 33% | 33% | 33%
Горячие клавиши не работают Конфликт с драйверами или антивирусом Обновите драйверы видеокарты и добавьте программу в исключения антивируса
На macOS окна "подпрыгивают" при изменении размера Включена функция Автоматически скрывать и показывать панель меню Отключите её в Системные настройки → Рабочий стол и док-станция

Если вы используете Remote Desktop (удалённый рабочий стол), разделение экрана может не работать из-за ограничений протокола RDP. В этом случае настройте деление на локальном компьютере, а затем подключитесь к нему.

FAQ: Ответы на популярные вопросы

Можно ли разделить экран на 3 части на ноутбуке с разрешением 1366×768?

Технически да, но это будет неудобно: каждое окно получится шириной ~430px, что недостаточно для большинства программ. Рекомендуем использовать вертикальное деление (три окна друг над другом) или ограничиться двумя зонами.

Почему в Windows 11 нет встроенной функции деления на 3 части?

Microsoft ориентируется на статистику использования: по данным телеметрии, 87% пользователей делят экран максимум на 2 части. Для 3+ зон предлагается использовать PowerToys или сторонние утилиты. В будущих обновлениях (например, Windows 12) это может измениться.

Как разделить экран на 3 части в Linux (Ubuntu, Fedora)?

В большинстве дистрибутивов Linux для этого используются tiling-менеджеры окон:

  • i3 — конфигурируется через файл ~/.config/i3/config
  • KWin (KDE) — поддерживает скрипты для кастомного деления
  • GNOME — требует расширения Pop Shell или Tiling Assistant

Пример команды для i3:

# Делим экран на 3 вертикальные части

bindsym $mod+1 layout toggle splitv; layout toggle splitv; layout toggle splitv

Будет ли разделение на 3 части работать в играх?

Нет, большинство современных игр (Cyberpunk 2077, Fortnite, CS2) блокируют системные функции управления окнами в полноэкранном режиме. Исключение — оконный режим (Borderless Windowed), но даже в нём могут возникать артефакты. Для стримеров лучше использовать OBS Studio с наложением источников.

Как вернуть стандартное поведение окон после использования FancyZones?

1. Откройте PowerToys и перейдите в FancyZones.

2. Отключите опцию Enable FancyZones.

3. Перезагрузите компьютер — настройки сбросятся.


Если окна всё равно ведут себя странно, сбросьте их позиции через Параметры → Система → Многозадачность → Сбросить позиции окон.
