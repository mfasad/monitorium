Как сделать два рабочих окна на одном мониторе: полное руководство

Работа с несколькими окнами на одном экране — стандартная задача для большинства пользователей ПК. Но не все знают, что современные операционные системы позволяют не просто переключаться между окнами, а фиксировать их в определённых зонах экрана, создавая удобное рабочее пространство. Это особенно актуально для владельцев широкоформатных мониторов (21:9, 32:9) или устройств с высоким разрешением (4K, 5K), где неиспользуемое пространство по бокам может достигать 30-40% площади.

В этой статье мы разберём все возможные способы разделения экрана — от встроенных функций Windows 10/11 и macOS до специализированных программ вроде FancyZones или Magnet. Вы узнаете, как:

  • 🔹 Использовать горячие клавиши для мгновенного разделения окон (Win + Стрелки)
  • 🔹 Настраивать произвольные зоны экрана под конкретные задачи (например, 70/30 для кода и терминала)
  • 🔹 Управлять окнами на Linux с помощью tiling-менеджеров (i3, Sway)
  • 🔹 Автоматизировать раскладку окон для повторяющихся сценариев (например, "Excel + браузер")

Важно: если вы используете ультрабук с маленьким экраном (13-14 дюймов), разделение на два окна может быть неудобным — в таком случае лучше рассмотреть подключение второго монитора или использование виртуальных рабочих столов.

1. Стандартное разделение экрана в Windows (без программ)

В Windows 10 и Windows 11 функция разделения экрана называется Snap Assist. Она позволяет "прикрепить" окно к одной из сторон экрана, автоматически подстраивая его размер. Вот как это работает:

  1. Откройте первое окно (например, Microsoft Word).
  2. Зажмите левую кнопку мыши на верхней панели окна и перетащите его к левому или правому краю экрана, пока не появится полупрозрачная подсветка.
  3. Отпустите кнопку — окно займёт ровно половину экрана.
  4. Выберите второе окно из появившегося меню Snap Assist (или откройте его вручную).

Для ускорения процесса используйте горячие клавиши:

  • 🔘 Win + ← — прикрепить окно к левой половине экрана
  • 🔘 Win + → — прикрепить к правой половине
  • 🔘 Win + ↑ — развернуть окно на весь экран
  • 🔘 Win + ↓ — свернуть окно или восстановить прежний размер

В Windows 11 добавлена возможность разделения на три или четыре окна — для этого после прикрепления первого окна наведите курсор на разделительную линию и выберите нужную раскладку из появившегося меню.

⚠️ Внимание: На некоторых ноутбуках с сенсорными экранами функция Snap Assist может конфликтовать с жестами масштабирования. Если окна не фиксируются, отключите сенсорный ввод в Панель управления → Оборудование и звук → Перо и сенсорный ввод.

2. Продвинутые раскладки с PowerToys (FancyZones)

Если встроенных возможностей Windows недостаточно, используйте бесплатный инструмент Microsoft PowerToys, в котором есть модуль FancyZones. Он позволяет создавать произвольные зоны на экране — например, три колонки или сетку 2×2 с разным соотношением сторон.

Как настроить:

  1. Скачайте PowerToys с официального сайта Microsoft и установите.
  2. Запустите утилиту и перейдите в раздел FancyZones.
  3. Включите модуль и выберите шаблон раскладки (например, "Grid" для сетки или "Columns" для вертикальных зон).
  4. Зажмите Shift при перетаскивании окна, чтобы активировать зоны.

Преимущества FancyZones:

  • 🎯 Сохранение раскладок для разных мониторов (полезно при использовании docking-станции)
  • 🎯 Поддержка многомониторных конфигураций с индивидуальными настройками
  • 🎯 Возможность создавать зоны с перекрытием (например, для наложения чата поверх рабочей области)

Если после обновления Windows FancyZones перестал работать, удалите файл конфигурации по пути %LocalAppData%\Microsoft\PowerToys\FancyZones\settings.json и перезапустите утилиту.

3. Разделение экрана на macOS (Split View и Magnet)

На MacBook или iMac функция разделения экрана называется Split View. В отличие от Windows, здесь окна фиксируются в полноэкранном режиме, что удобно для работы без отвлекающих элементов.

Как включить Split View:

  1. Откройте первое окно (например, Safari).
  2. Наведите курсор на зелёную кнопку свёртывания в левом верхнем углу окна.
  3. В появившемся меню выберите "Разместить окно слева" или "Разместить окно справа".
  4. Выберите второе окно из списка открытых приложений.

Для быстрого управления используйте сочетания:

  • 🔘 Control + ←/→ — переключение между окнами в Split View
  • 🔘 Esc — выход из режима разделения

Если вам нужны более гибкие настройки, установите утилиту Magnet (платно, ~$2 в App Store). Она добавляет:

  • 🧲 Перетаскивание окон в углы для разделения на 4 части
  • 🧲 Горячие клавиши для произвольных раскладок (например, Ctrl + Option + 1 для левой трети экрана)
  • 🧲 Сохранение пользовательских шаблонов
⚠️ Внимание: На macOS Ventura и новее Split View может конфликтовать с функцией Stage Manager. Если окна не фиксируются, отключите Stage Manager в Системные настройки → Рабочий стол и Dock.

4. Tiling-менеджеры для Linux (i3, Sway, Gnome)

Пользователи Linux имеют уникальное преимущество — tiling-менеджеры, которые автоматически организуют окна в мозаику без ручного перетаскивания. Самые популярные варианты:

  • 🐧 i3 — минималистичный менеджер с конфигурацией через текстовый файл
  • 🐧 Sway — аналог i3, но для протокола Wayland (совместим с NVIDIA)
  • 🐧 Gnome + Pop Shell — расширение для стандартного окружения Gnome

Пример конфигурации для i3 (файл ~/.config/i3/config):

# Разделение экрана на две колонки по умолчанию

workspace_layout tabbed



Горячие клавиши для управления окнами


bindsym $mod+h split h  # горизонтальное разделение


bindsym $mod+v split v  # вертикальное разделение


bindsym $mod+f fullscreen toggle

Преимущества tiling-менеджеров:

  • ⚡ Мгновенное переключение между окнами без мыши
  • ⚡ Автоматическое масштабирование при изменении размера экрана
  • ⚡ Минимальное потребление ресурсов (идеально для слабых ПК)
Как вернуть классичный режим окон в i3?

Чтобы временно отключить автоматическое разделение, нажмите $mod+Shift+e (выход из tiling-режима) или переключитесь на floating-окна с помощью $mod+Shift+space.

5. Программы для автоматизации раскладок (DisplayFusion, GridMove)

Если вам нужно сохранять и быстро переключать раскладки окон для разных задач (например, "дизайн", "программирование", "аналитика"), используйте специализированные программы:

Программа Платформа Особенности Цена
DisplayFusion Windows Многомониторные профили, триггеры по времени, скрипты $34 (однократно)
GridMove Windows Бесплатная, поддерживает пользовательские сетки Бесплатно
Moom macOS Интеграция с Spotlight, жесты трекпада $10
Amethyst macOS Tiling-менеджер с поддержкой Haskell-скриптов Бесплатно

Пример использования DisplayFusion для автоматизации:

  1. Создайте профиль "Работа" с раскладкой: слева — Visual Studio Code (70%), справа — Terminus (30%).
  2. Настройте триггер: применять профиль при подключении к корпоративному VPN.
  3. Сохраните профиль и назначьте ему горячую клавишу (например, Ctrl + Alt + 1).
⚠️ Внимание: Программы вроде DisplayFusion могут конфликтовать с Remote Desktop или виртуальными машинами. Перед покупкой проверьте совместимость в официальном форуме.

6. Разделение экрана в браузерах и специализированных приложениях

Некоторые программы имеют встроенные функции для разделения рабочей области. Например:

  • 🌐 Microsoft Edge и Google Chrome: расширение Split Screen для открытия двух вкладок рядом
  • 📊 Excel: режим "Новое окно" (Вид → Новое окно) для сравнения частей одной таблицы
  • 🎮 OBS Studio: разделение предварительного просмотра и программы в разных окнах
  • 📝 Notion: боковая панель для одновременной работы с двумя документами

Для Google Chrome установка расширения Split Screen займёт менее минуты:

  1. Перейдите в Chrome Web Store.
  2. Найдите расширение "Split Screen for Google Chrome".
  3. Нажмите "Установить" и подтвердите разрешения.
  4. Откройте две вкладки и нажмите иконку расширения → "Split".

Важно: В браузерах на базе Chromium (Edge, Chrome, Brave) одновременно можно разделить только вкладки одного окна. Для работы с двумя разными браузерами используйте стандартные методы ОС.

7. Оптимизация разделения для ультрашироких мониторов (21:9, 32:9)

Мониторы с соотношением сторон 21:9 или 32:9 (например, Samsung Odyssey G9 или LG 34WK95U) требуют особого подхода к организации рабочего пространства. Вот несколько советов:

  • 🖥️ Используйте три зоны: левая (30%) — мессенджеры, центр (40%) — основная работа, правая (30%) — справочные материалы.
  • 🖥️ В Windows 11 настройте виртуальные рабочие столы для разных проектов (Win + Tab → Новый рабочий стол).
  • 🖥️ Для Linux используйте i3 с конфигурацией workspace 1 output DP-1 (привязка рабочих столов к физическим мониторам).
  • 🖥️ В играх включайте режим "Ультраширокий" в настройках графики (поддерживается в The Witcher 3, Cyberpunk 2077 и др.).

Пример конфигурации для i3 на мониторе 3440×1440:

# Делим экран на три вертикальные колонки

workspace 1 output DP-1


bindsym $mod+1 move container to workspace 1


bindsym $mod+Shift+1 move container to workspace 1, split v

⚠️ Внимание: На мониторах с разрешением 5120×1440 (32:9) некоторые программы (например, Photoshop или AutoCAD) могут отображаться с артефактами. В таком случае уменьшите масштаб интерфейса в настройках приложения или используйте режим "Окно" вместо полноэкранного.
💡

Для ультрашироких мониторов оптимальное соотношение зон — 30/40/30. Это позволяет разместить текстовый редактор по центру, а справочные материалы — по бокам без прокрутки.

Частые вопросы (FAQ)

Можно ли разделить экран на два окна в Windows 7?

Да, но без Snap Assist. Используйте комбинации Win + ←/→ или вручную изменяйте размер окон, перетаскивая края. Для гибких раскладок установите GridMove или WindowGrid (бесплатные утилиты).

Почему после обновления Windows перестали работать горячие клавиши для разделения?

Сбросьте настройки Snap Assist:

  1. Откройте Параметры → Система → Многозадачность.
  2. Отключите и снова включите опцию "Закрепление окон".
  3. Перезагрузите ПК.

Если проблема осталась, проверьте конфликты с программами вроде AutoHotkey или DisplayFusion.

Как разделить экран на Mac, если Split View не работает?

Проблема может быть в настройках миссии:

  1. Откройте Системные настройки → Рабочий стол и Dock.
  2. Убедитесь, что в разделе "Миссия" включена опция "Группировать окна по приложению".
  3. Перезапустите Dock через Терминал: killall Dock.

Если используете macOS Monterey или новее, попробуйте отключить Stage Manager.

Можно ли разделить экран на Android или iPad?

Да, но с ограничениями:

  • 📱 Android: в Samsung DeX или на планшетах с Android 10+ удерживайте иконку приложения в многозадачности → выберите "Открыть в разделенном экране".
  • 🍎 iPad: в iPadOS перетащите приложение из dock на экран и отпустите на краю (поддерживается только в горизонтальной ориентации).

На смартфонах функция доступна только в режиме "Рабочий стол" (например, Samsung DeX или Huawei Desktop Mode).

Как сделать так, чтобы окна автоматически открывались в нужных зонах?

Используйте программы для управления окнами:

  • 🪟 Windows: DisplayFusion (правила для конкретных приложений) или AutoHotkey (скрипты).
  • 🍏 macOS: Moom или Rectangle (бесплатно).
  • 🐧 Linux: настройте правила в ~/.config/i3/config (для i3).

Пример правила для DisplayFusion:

[Notepad++]

Position=Left50


Size=50%×100%
