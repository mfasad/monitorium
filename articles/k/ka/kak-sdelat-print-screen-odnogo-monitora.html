Как сделать скриншот только одного монитора: все способы для Windows и Mac

Захват экрана на компьютере с несколькими мониторами часто превращается в головную боль: стандартные комбинации клавиш сохраняют изображение со ВСЕХ дисплеев сразу. А что, если нужно сфотографировать только рабочий стол на основном экране или отдельный монитор с презентацией? В этой статье разберём все рабочие методы — от встроенных инструментов до специализированных программ.

Проблема актуальна для геймеров, дизайнеров, офисных сотрудников и всех, кто работает с расширенным рабочим столом. Например, при создании гайдов по настройке UltraWide-мониторов или когда нужно продемонстрировать интерфейс программы, открытой только на втором дисплее. Мы протестировали решения для Windows 10/11 и macOS, а также рассмотрели нюансы для разных разрешений и конфигураций.

Важно: если вы используете NVIDIA Surround или AMD Eyefinity для объединения экранов в один виртуальный дисплей, стандартные методы не сработают — потребуются обходные пути, о которых расскажем ниже.

Способ 1: Горячие клавиши Windows для выборочного скриншота

В Windows нет прямой комбинации для захвата одного монитора, но можно обойти это ограничение с помощью системных инструментов. Самый быстрый метод — использование Win + Shift + S (инструмент «Ножницы»).

После нажатия экран затемнится, а в верхней части появится панель с опциями. Выберите «Прямоугольная область» (или «Окно»), затем вручную выделите нужный монитор. Скриншот сохранится в буфер обмена — его можно вставить в Paint, Word или Photoshop для сохранения. Минус способа: если мониторы имеют разное разрешение, стык между ними будет виден на скриншоте.

  • ⌨️ Win + Shift + S → выделить область мышью → вставить в редактор
  • 🖼️ Работает в Windows 10/11 без дополнительных программ
  • ⚠️ Не подходит для захвата полного экрана одного монитора (только фрагмент)
💡

Если после нажатия Win + Shift + S ничего не происходит, проверьте, не отключён ли инструмент «Ножницы» в параметрах Windows (Параметры → Специальные возможности → Клавиатура).

Для захвата полного экрана одного монитора без обрезки используйте этот лайфхак:

  1. Перетащите окно программы (например, Paint или Browser) на нужный монитор и разверните на весь экран (F11 или ⛶).
  2. Нажмите Alt + PrtScn — это сделает скриншот только активного окна, но поскольку оно развёрнуто, захватит весь монитор.

Способ 2: Программы для скриншотов с поддержкой нескольких мониторов

Если встроенные инструменты не устраивают, воспользуйтесь специализированным ПО. Мы протестировали 5 популярных программ и выделили лучшие:

Рекомендуем ShareX для продвинутых пользователей: программа позволяет настроить горячие клавиши для захвата конкретного монитора. Например, можно присвоить Ctrl + 1 для скриншота первого дисплея, Ctrl + 2 — для второго. Инструкция:

  1. Установите ShareX и откройте Task Settings → Capture → Screen recording.
  2. В разделе «Monitor» выберите нужный дисплей из выпадающего списка.
  3. Назначьте горячую клавишу в Hotkey Settings.

Способ 3: macOS — как сделать скриншот одного экрана

На Mac процесс проще благодаря встроенным инструментам. Используйте эти комбинации:

  • ⌘ + Shift + 4 → Пробел → клик по окну на нужном мониторе (захват окна)
  • ⌘ + Shift + 4 → зажать Option → выделить область на одном экране (игнорирует границы мониторов)
  • ⌘ + Shift + 5 → выбрать «Захватить выбранную часть» и вручную обвести монитор

Для захвата полного экрана одного монитора в macOS Ventura и новее:

  1. Нажмите ⌘ + Shift + 5.
  2. В панели инструментов выберите «Захватить весь экран».
  3. Наведите курсор на нужный монитор — он подсветится синим. Кликните по нему.
Что делать, если macOS захватывает чёрный экран?

Если при скриншоте одного монитора получается чёрное изображение, это связано с защитой DRM (например, при просмотре Netflix или Apple TV+). Решение:

1. Отключите режим «AirPlay Mirroring» в Системные настройки → Мониторы.

2. Попробуйте сделать скриншот через QuickTime Player (Файл → Новая запись экрана).

3. Для стриминга используйте внешний захватчик (например, Elgato).

На Mac с чипом Apple Silicon (M1/M2) скриншоты сохраняются в формате .heic по умолчанию. Чтобы изменить формат на .png или .jpg, выполните в терминале:

defaults write com.apple.screencapture type png

killall SystemUIServer

Способ 4: Обходные пути для сложных конфигураций

Если у вас нестандартная конфигурация (например, NVIDIA Surround, AMD Eyefinity или вертикально ориентированные мониторы), стандартные методы могут не сработать. Вот рабочие решения:

  • 🎮 Для NVIDIA Surround: временно отключите режим в панели управления NVIDIA (3D Settings → Configure Surround), сделайте скриншот, затем включите обратно.
  • 🔄 Для мониторов с разным масштабированием: в Windows откройте Параметры → Система → Дисплей и установите одинаковый масштаб (например, 100%) на всех экранах перед скриншотом.
  • 🖥️ Для удалённых рабочих столов (RDP): используйте комбинацию Ctrl + Alt + Minus (на клавиатуре в окне RDP), затем выберите «Full Screen» для текущего монитора.

Если вам нужно регулярно делать скриншоты одного монитора в UltraWide-конфигурации (например, 21:9 + 16:9), создайте в AutoHotkey скрипт для автоматического обрезки:

#IfWinActive ahk_exe explorer.exe  ; Работает только на рабочем столе

F1::  ; Горячая клавиша F1 для скриншота левого монитора


{


Send {PrintScreen}


Sleep 500


Run, mspaint


WinWait, Безымянный - Paint


Send ^v  ; Вставить


Sleep 300


Send ^{Left}  ; Обрезать слева (настройте пиксели под ваше разрешение)


Send {Right 10}


Sleep 100


Send {Enter}


Send ^s  ; Сохранить


Sleep 300


Send C:\Screenshots\left_screen_%A_YYYY%-%A_MM%-%A_DD%.png{Enter}


WinClose


}


return

Способ 5: Онлайн-сервисы для скриншотов (если нельзя установить ПО)

Если у вас нет прав администратора или нельзя устанавливать программы, воспользуйтесь онлайн-инструментами. Лучшие варианты:

  • 🌐 Screenshots.com: загрузка скриншота с URL (подходит для веб-страниц на одном мониторе).
  • 📱 Lightshot Online: расширение для браузера, позволяет выделять область на одном экране.
  • 🖥️ ScreenCapture.com: захват с веб-камеры или экрана через браузер (требует Java).

Ограничения онлайн-сервисов:

⚠️ Внимание: Онлайн-инструменты не могут захватывать защищённый контент (DRM), системные окна или игры в полноэкранном режиме. Также они могут сохранять скриншоты на своих серверах — не используйте их для конфиденциальной информации.

Для захвата одного монитора через браузер:

  1. Откройте на нужном экране веб-версию Lightshot или Screenshots.com.
  2. Разверните окно браузера на весь экран (F11).
  3. Сделайте скриншот через сервис — он захватит только содержимое текущего окна (т.е. одного монитора).

Распространённые ошибки и как их избежать

Even experienced users make mistakes when trying to capture a single monitor. Here are the most common pitfalls and solutions:

  • 🖱️ Проблема: Скриншот получается размытым.
    Решение: Проверьте масштабирование в настройках дисплея (Параметры → Система → Масштаб). Для чёткого скриншота установите 100%.
  • 🔍 Проблема: На скриншоте видна граница между мониторами.
    Решение: Используйте обрезку в Paint или настройте в ShareX захват конкретного монитора по координатам.
  • 🎮 Проблема: В играх скриншот делается чёрным.
    Решение: Отключите полноэкранный режим (используйте оконный) или сделайте скриншот через Steam (F12).
⚠️ Внимание: Если вы используете Windows 11 с функцией «Snap Layouts», развёрнутые окна могут автоматически перемещаться между мониторами при изменении конфигурации. Перед скриншотом отключите эту опцию в Параметры → Система → Многозадачность.

FAQ: Частые вопросы о скриншотах одного монитора

Можно ли сделать скриншот одного монитора на Linux?

Да, в большинстве дистрибутивов используйте gnome-screenshot -a -m (для GNOME) или scrot -u (для обрезки по границам окна). Для захвата конкретного монитора в KDE Plasma нажмите PrtScn, затем выберите «Текущий экран» в появившемся окне.

Почему скриншот второго монитора получается пустым?

Это типичная проблема при использовании Remote Desktop или виртуальных машин. Решения:

  1. Включите опцию «Use all monitors» в настройках RDP.
  2. Для VMware/VirtualBox установите VMware Tools/Guest Additions.
  3. Используйте mstsc /span для захвата всех мониторов в RDP.

Как автоматизировать скриншоты одного монитора по расписанию?

Используйте Task Scheduler (Windows) или cron (macOS/Linux) с командой: 

start "" "C:\Program Files\ShareX\ShareX.exe" -CaptureMonitor 2 -SaveToFile "C:\screens\monitor2_%y-%m-%d_%H-%M-%S.png"

Где 2 — номер монитора (начинается с 1). Для macOS:

screencapture -x -R$x1,y1,x2,y2 ~/Desktop/monitor1.png

Координаты x1,y1,x2,y2 можно узнать через Rectangle или Skitch.

Есть ли способ сделать скриншот одного монитора на Android-устройстве с DeX?

Да, подключите Samsung DeX к монитору, затем:

  1. На телефоне нажмите Громкость вниз + Питание (скриншот).
  2. Файл появится в папке Pictures/Screenshots на телефоне.
  3. Для захвата только экрана DeX используйте Scrcpy с параметром --crop.

💡

Для большинства задач достаточно комбинации Win + Shift + S (Windows) или ⌘ + Shift + 5 (macOS) с ручным выделением области. Специализированное ПО типа ShareX потребуется только для автоматизации или нестандартных конфигураций.
