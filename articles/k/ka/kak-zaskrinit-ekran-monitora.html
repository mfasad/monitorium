Как сделать скриншот экрана монитора: все способы для Windows, Mac и Linux

Сколько раз вы сталкивались с ситуацией, когда нужно сохранить информацию с экрана — важное сообщение, ошибку системы, смешной мем или фрагмент видео? Скриншот (снимок экрана) решает эту задачу за секунды, но не все знают, как сделать его правильно. В этой статье мы разберём все возможные способы — от стандартных горячих клавиш до специализированных программ с расширенными функциями.

Вы узнаете, как заскринить весь экран или его часть на Windows 10/11, macOS, Linux, а также с помощью онлайн-сервисов и мобильных устройств. Особое внимание уделим нюансам: где сохраняются скриншоты, как их редактировать и почему иногда клавиши не работают. Если вы работаете с несколькими мониторами или нужно захватить прокручиваемую страницу — здесь тоже найдёте решение.

Новичкам пригодятся пошаговые инструкции с картинками, а опытные пользователи обнаружат лайфхаки для ускорения процесса. Например, знали ли вы, что в Windows 11 можно сделать скриншот с задержкой или что на MacBook есть скрытая функция для записи видео с экрана? Давайте разбираться!

Стандартные способы сделать скриншот в Windows

Операционная система Windows предлагает несколько встроенных инструментов для создания снимков экрана. Их достаточно для большинства задач — от сохранения ошибки до создания презентаций. Рассмотрим каждый метод подробно.

1. Клавиша Print Screen (PrtScn)

Это самый простой и универсальный способ. Нажмите клавишу PrtScn (может называться PrntScrn или Print Screen), и снимок всего экрана будет скопирован в буфер обмена. Затем вставьте его в любой графический редактор (Paint, Photoshop) или документ (Word, Google Docs) с помощью Ctrl + V.

  • 🖥️ Весь экран — просто PrtScn
  • 🪟 Активное окно — Alt + PrtScn
  • 💾 Автоматическое сохранение — Win + PrtScn (файл появится в папке Изображения → Снимки экрана)

2. Инструмент "Ножницы" (Snipping Tool)

Более гибкий вариант, доступный в Windows 7/10/11. Позволяет выбирать область экрана, делать скриншоты с задержкой и аннотировать их. Чтобы открыть:

  1. Нажмите Win + Shift + S (быстрый вызов в Windows 10/11).
  2. Или найдите "Ножницы" через поиск в меню Пуск.
  3. Выберите тип вырезки: прямоугольник, произвольная форма, окно или весь экран.
💡

Если после нажатия Win + Shift + S ничего не происходит, проверьте, не отключены ли горячие клавиши в настройках системы (Параметры → Устройства → Ввод → Горячие клавиши).

3. Игровая панель Xbox (Game Bar)

Этот метод полезен для геймеров или тех, кто хочет записать видео с экрана. Сочетание Win + Alt + PrtScn сохраняет скриншот в папке Видео → Захват. Чтобы включить функцию:

  1. Откройте Параметры → Игры → Игровая панель Xbox.
  2. Убедитесь, что переключатель "Записывать игровые клипы и скриншоты" активен.
Где искать скриншоты в Windows?

По умолчанию файлы сохраняются:

- При Win + PrtScn: C:\Users\Имя_пользователя\Pictures\Screenshots

- При Win + Alt + PrtScn: C:\Users\Имя_пользователя\Videos\Captures

- Через "Ножницы": вы сами выбираете папку при сохранении.

Как сделать скриншот на MacBook и iMac

Устройства Apple славится интуитивным интерфейсом, и создание снимков экрана здесь не исключение. Главное — запомнить комбинации клавиш, так как они отличаются от Windows.

Базовые сочетания:

  • 🖥️ Весь экран — Command (⌘) + Shift + 3
  • 📝 Выбранная область — Command (⌘) + Shift + 4 (затем выделите зону мышью)
  • 🪟 Одно окно — Command (⌘) + Shift + 4 → Пробел (наведите курсор на окно)

По умолчанию скриншоты сохраняются на Рабочий стол в формате .png. Чтобы изменить место сохранения:

  1. Откройте Терминал (через Spotlight или папку Утилиты).
  2. Введите команду: 
    defaults write com.apple.screencapture location /путь/к/папке

    Например: /Users/Ваше_имя/Documents/Screenshots

  3. Перезагрузите системные настройки: 
    killall SystemUIServer

Дополнительные фишки:

  • 🕒 Скриншот с задержкой: используйте Терминал с командой: 
    screencapture -T 5 screenshot.png

    (цифра 5 — секунды задержки)

  • 📱 Сохранение в буфер: добавьте Control к любой комбинации (например, ⌘ + Control + Shift + 4).

Скриншоты в Linux: инструменты и команды

В мире Linux всё зависит от используемого окружения рабочего стола (GNOME, KDE, XFCE и др.). Однако есть универсальные решения, работающие почти везде.

1. Клавиша Print Screen

В большинстве дистрибутивов (например, Ubuntu, Fedora) клавиша PrtScn автоматически вызывает утилиту для создания скриншотов. Вы можете выбрать:

  • 🖥️ Весь экран
  • 🪟 Текущее окно
  • 📝 Произвольная область

2. Утилита scrot

Мощный инструмент для терминала, позволяющий гибко настраивать скриншоты. Установите его командой:

sudo apt install scrot  # Для Debian/Ubuntu

sudo dnf install scrot  # Для Fedora

Примеры использования:

scrot screenshot.png          # Снимок всего экрана

scrot -u -b 'window_%Y-%m-%d.png'  # Текущее окно с рамкой


scrot -s -d 5 'delay_%T.png'       # Область с задержкой 5 секунд

3. GNOME Screenshot и KSnapshot

Графические утилиты для популярных окружений:

  • 🐧 В GNOME (Ubuntu, Fedora): нажмите PrtScn или найдите "Screenshot" в меню приложений.
  • 💎 В KDE Plasma (Kubuntu): используйте Spectacle (Alt + Shift + PrtScn).

Программы для расширенных скриншотов

Если встроенных инструментов недостаточно, на помощь приходят сторонние программы. Они предлагают редактирование, запись видео, захват прокручиваемых страниц и даже распознавание текста (OCR).

Программа Платформа Ключевые функции Бесплатная версия
ShareX Windows Захват областей, GIF, OCR, загрузка в облако Да
Snagit Windows, Mac Редактор, шаблоны, запись видео, прокрутка Нет (триал 15 дней)
Lightshot Windows, Mac, Linux Быстрый захват, поиск похожих изображений Да
Greenshot Windows Аннотации, экспорт в Office, плагины Да

Лидер среди бесплатных — ShareX:

  • 🎯 Захват: весь экран, область, окно, прокручиваемая страница.
  • ✏️ Редактирование: стрелочки, текст, размытие, фигуры.
  • 🌐 Экспорт: автоматическая загрузка на Imgur, Google Drive, FTP.
  • 🤖 Автоматизация: горячие клавиши, задачи после захвата.

Уникальная фишка ShareX: функция OCR (распознавание текста) позволяет копировать текст с любого снимка экрана, даже если исходный файл не редактируемый (например, изображение из PDF).

💡

Для захвата длинных веб-страниц (например, ленты в соцсетях) используйте расширения браузера GoFullPage (Chrome) или FireShot (Firefox).

Скриншоты с нескольких мониторов

Если вы работаете с многомониторной системой, стандартные методы могут захватить не то, что нужно. Вот как сделать скриншот правильно:

В Windows:

  • 🖥️ PrtScn — захватит все мониторы как одно изображение.
  • 🪟 Alt + PrtScn — только активное окно (даже если оно растянуто на несколько экранов).
  • 📏 Win + Shift + S — выберите область вручную (подходит для захвата части второго монитора).

На Mac:

  • 🖥️ ⌘ + Shift + 3 — все мониторы в одном файле.
  • 📝 ⌘ + Shift + 4 → Пробел — выберите конкретный экран или окно.

Проблемы и решения:

  • ❌ Скриншот обрезается: проверьте разрешение мониторов в настройках дисплея (Параметры → Система → Дисплей). Если масштабирование отличается (например, 100% и 125%), изображение может искажаться.
  • ❌ Не видно курсор: в ShareX или Snagit включите опцию "Показать курсор" в настройках захвата.
💡

Если мониторы имеют разное разрешение, скриншот всего экрана может получиться "ступеньками". Чтобы избежать этого, временно установите одинаковое разрешение или используйте программу ShareX с настройкой "Объединить изображения".

Как сделать скриншот на телефоне и планшете

Современные смартфоны умеют создавать снимки экрана не хуже компьютеров. Способы зависят от модели и операционной системы.

На Android:

  • 📱 Стандартный метод: одновременно нажмите Кнопка питания + Уменьшение громкости (на большинстве устройств).
  • 👆 Жест пальцем: на Samsung проведите ребром ладони по экрану (нужно включить в настройках).
  • 🤖 Google Assistant: скажите "Окей, Google, сделай скриншот".

На iPhone/iPad:

  • 🍎 С клавишами:

    - На моделях с Face ID: Боковая кнопка + Увеличение громкости.

    - На моделях с Touch ID: Кнопка Home + Кнопка питания.

  • 🖼️ Предварительный просмотр: после создания скриншота нажмите на его миниатюру в левом нижнем углу для редактирования.

Где искать скриншоты:

  • 📁 Android: папка Pictures → Screenshots или DCIM → Screenshots.
  • 📁 iOS: альбом "Снимки экрана" в приложении Фото.
Как сделать скриншот прокручиваемой страницы на телефоне?

На Android используйте приложение LongShot или Scroll Capture (доступны в Google Play).

На iPhone в Safari: сделайте скриншот, затем нажмите на миниатюру → "Весь экран" (работает в iOS 13+).

Распространённые ошибки и их решения

Иногда скриншоты не получаются с первого раза. Вот типичные проблемы и способы их исправления:

1. Клавиши не работают

  • ❌ Проблема: На ноутбуке PrtScn срабатывает только с Fn (например, Fn + PrtScn).
  • ✅ Решение: Проверьте надпись на клавише — если она синего цвета, нужна Fn. Или измените поведение в BIOS (Function Key Behavior).

2. Скриншот не сохраняется

  • ❌ Проблема: При нажатии Win + PrtScn файл не появляется в папке.
  • ✅ Решение:
    1. Проверьте, хватает ли места на диске C:.
    2. Убедитесь, что папка Screenshots не удалена (создайте её вручную в Изображения).
    3. Запустите Проводник от имени администратора и повторите попытку.

3. Чёрный экран вместо скриншота

  • ❌ Проблема: При захвате видео или игры получается чёрное изображение.
  • ✅ Решение:
    • Отключите аппаратное ускорение в настройках графики (например, в Chrome: Настройки → Система → Использовать аппаратное ускорение).
    • Используйте OBS Studio для записи экрана вместо скриншотов.
💡

Если скриншоты сохраняются в низком качестве, проверьте настройки программы. Например, в ShareX перейдите в Задачи → Захват экрана → Формат изображения и выберите PNG вместо JPEG.

FAQ: Частые вопросы о скриншотах

Можно ли сделать скриншот с задержкой?

Да! В Windows используйте Ножницы (задержка до 5 секунд) или ShareX (настраиваемая задержка). На Mac введите в Терминале:

screencapture -T 10 screenshot.png

(где 10 — секунды задержки).

Как захватить контекстное меню или всплывающее окно?

Стандартные методы не всегда работают с выпадающими меню. Решения:

  • 🖥️ В Windows: используйте Win + Shift + S (задержка 1-3 секунды даст время открыть меню).
  • 🍎 На Mac: комбинация ⌘ + Shift + 4 → Пробел → клик по меню.
  • 🛠️ Универсальный способ: сделайте снимок с помощью Snagit или ShareX с задержкой.
Где хранятся скриншоты в Steam?

Игровые снимки, сделанные через F12, сохраняются по пути:

C:\Program Files (x86)\Steam\userdata\<Ваш_ID>\760\remote\<ID_игры>\screenshots

Чтобы найти свой ID, откройте Steam → Просмотр → Настройки → Интерфейс → Отображать адрес профиля в виде числа.

Как сделать скриншот в BIOS или при загрузке Windows?

Стандартные методы не работают на уровне BIOS. Альтернативы:

  • 📷 Сфотографируйте экран на телефон (главное — без бликов!).
  • 💻 Используйте второй компьютер с камерой (направьте объектив на монитор).
  • 🔧 Для UEFI BIOS некоторых материнских плат (например, ASUS, Gigabyte) есть встроенная функция сохранения скриншотов на USB-накопитель.
Можно ли сделать скриншот защищённого окна (например, Netflix)?

Сервисы с защитой от копирования (DRM), такие как Netflix, Disney+ или банковские приложения, блокируют создание снимков экрана. Обходные пути:

  • 🎥 Запишите видео экрана с помощью OBS Studio, затем сделайте скриншот из видеофайла.
  • 📱 Сфотографируйте экран на другой девайс (но помните о соблюдении авторских прав!).
    • ⚠️ Внимание: создание скриншотов или запись видео защищённого контента может нарушать пользовательское соглашение сервиса и законы об авторском праве.
