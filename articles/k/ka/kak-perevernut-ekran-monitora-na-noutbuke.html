Вы включили ноутбук, а картинка на экране вдруг оказалась вверх ногами или боком? Или вам нужно перевернуть экран для работы с графикой, чтения длинных документов в книжной ориентации, либо подключили внешний монитор с поворотным креплением? В 90% случаев проблема решается за 30 секунд — если знать правильные комбинации клавиш или путь в настройках.

Мы собрали все рабочие способы для Windows 10/11, MacOS и Linux, включая редкие случаи, когда стандартные методы не срабатывают (например, из-за конфликта драйверов или специфичных настроек BIOS). Также разберём, почему экран может самопроизвольно переворачиваться и как это предотвратить. Если вам нужно не просто вернуть изображение в нормальное положение, а настроить поворот для многомониторной системы или сенсорного экрана — здесь тоже найдёте решение.

Спойлер: в 80% случаев помогает одна клавиша. Но если она не сработала — не паникуйте, просто переходите к следующему разделу.

1. Горячие клавиши — самый быстрый способ (работает в 95% случаев)

Большинство ноутбуков поддерживают мгновенный поворот экрана через комбинации клавиш, заложенные в драйверы графики. Эти сочетания работают независимо от версии Windows и не требуют прав администратора.

Попробуйте по очереди:

  • 🔄 Ctrl + Alt + ↑ — вернуть нормальную ориентацию
  • 🔃 Ctrl + Alt + → — поворот на 90° по часовой стрелке
  • 🔁 Ctrl + Alt + ← — поворот на 90° против часовой стрелки
  • 🔻 Ctrl + Alt + ↓ — перевернуть экран вверх ногами (180°)

Если комбинации не сработали, причина может быть в:

  • 🖥️ Отключённых функциях клавиш в BIOS (редко, но бывает на некоторых Lenovo и Dell)
  • 🔧 Устаревших или сломанных драйверах видеокарты (особенно актуально для AMD Radeon)
  • 🎮 Конфликте с программным обеспечением для геймпадов/джойстиков (например, Xbox Accessories)
⚠️ Внимание: На некоторых ноутбуках HP Pavilion и Acer Swift эти комбинации по умолчанию отключены. Чтобы их активировать, нужно сначала обновить драйвер графики через Диспетчер устройств.

2. Настройки Windows: ручной поворот через параметры системы

Если горячие клавиши не сработали, воспользуйтесь встроенными настройками Windows. Этот метод универсален и работает даже на виртуальных машинах.

Инструкция для Windows 10 и Windows 11:

  1. Нажмите Win + I, чтобы открыть Параметры.
  2. Перейдите в Система → Дисплей.
  3. Найдите выпадающий список Ориентация (в Windows 11 он может скрываться под кнопкой Масштаб и макет).
  4. Выберите нужный вариант:
    • 📺 Альбомная — стандартное положение
    • 📖 Книжная — поворот на 90°
    • 🔄 Альбомная (перевёрнутая) — 180°
    • 📖 Книжная (перевёрнутая) — 270°
  • Подтвердите изменения кнопкой Сохранить (в Windows 10 система спросит: "Сохранить изменения?" — нажмите Да).

    • Если пункт Ориентация отсутствует или неактивен:

    • 🔌 Проверьте подключение монитора (на ноутбуках с hybrid graphics иногда требуется перезапуск)
    • 🔄 Перезагрузите компьютер — иногда настройки сбрасываются после обновлений
    • 🛠️ Обновите драйвер видеоадаптера (см. раздел 4)

    3. Панель управления графикой: NVIDIA, AMD, Intel

    Если стандартные настройки Windows не помогают, попробуйте использовать проприетарные утилиты от производителя вашей видеокарты. Они предлагают больше опций, включая индивидуальные настройки для каждого подключённого дисплея.

    Выберите ваш случай:

    Производитель Программа Путь к настройкам Особенности
    NVIDIA NVIDIA Control Panel Панель управления NVIDIA → Дисплей → Поворот дисплея Поддерживает поворот на 90°, 180°, 270° + зеркальное отображение. Работает даже на старых драйверах.
    AMD Radeon AMD Radeon Software Настройки → Дисплей → Дополнительные настройки → Ориентация Может конфликтовать с Windows Display Settings. При сбое требует перезагрузки.
    Intel HD Graphics Intel Graphics Command Center Система → Дисплей → Поворот Ограниченные опции на ноутбуках с hybrid graphics (NVIDIA + Intel).

    Для NVIDIA и AMD доступны горячие клавиши, но они могут отличаться от стандартных. Чтобы их настроить:

    1. Откройте панель управления вашей видеокарты.
    2. Найдите раздел Управление параметрами 3D или Горячие клавиши.
    3. Присвойте комбинации для поворота (например, Ctrl + Shift + Стрелки).
    ⚠️ Внимание: На ноутбуках с оптимусом (двумя видеокартами) панель управления NVIDIA может не показывать опции поворота, если в данный момент используется встроенная графика Intel. Чтобы это исправить, в Панель управления NVIDIA → Управление параметрами 3D выберите Высокопроизводительный процессор NVIDIA для приложения "Экран".

    4. Обновление или переустановка драйверов: если ничего не работает

    Если экран не поворачивается ни одним из способов, проблема почти наверняка в драйверах графики. Это особенно актуально после обновления Windows или если вы недавно меняли железо (например, добавили внешнюю видеокарту).

    Как проверить и исправить:

    1. Удалите текущие драйверы:
      • Нажмите Win + X → Диспетчер устройств.
      • Раскройте Видеоадаптеры, кликните правой кнопкой по вашей видеокарте → Удалить устройство.
      • Поставьте галочку Удалить программы драйверов для этого устройства (если есть).
  • Установите свежие драйверы:
    • Для NVIDIA: скачайте с официального сайта (выберите модель видеокарты вручную).
    • Для AMD: используйте утилиту AMD Auto-Detect на странице поддержки.
    • Для Intel: драйверы обычно устанавливаются через Windows Update, но для новых процессоров (12-го поколения и новее) скачивайте с Intel Driver & Support Assistant.
    • Перезагрузите ноутбук и проверьте поворот экрана.

    Если после обновления драйверов проблема осталась:

    • 🔍 Проверьте Журнал событий Windows (нажмите Win + X → Журналы Windows → Система) на наличие ошибок, связанных с display или graphics.
    • 🔧 Попробуйте откатить драйвер: в Диспетчере устройств выберите Свойства → Драйвер → Откатить.
    • 💻 Если у вас hybrid graphics, обновите драйверы обеих видеокарт (и Intel, и NVIDIA/AMD).
    💡

    Перед удалением драйверов создайте точку восстановления системы: нажмите Win + R, введите rstrui и следуйте инструкциям. Это поможет отменить изменения, если после обновления экран перестанет работать совсем.

    5. Поворот экрана на MacOS (для ноутбуков MacBook)

    На MacBook и iMac с поворотными экранами (например, Apple Studio Display) настройка ориентации отличается от Windows. В стандартной macOS нет встроенной функции поворота экрана, но её можно активировать через терминал или сторонние утилиты.

    Способ 1: Через Терминал (для опытных пользователей)

    # Поворот на 90° по часовой стрелке

    displayplacer -r90
    


    Поворот на 180°
    

    displayplacer -r180
    


    Возврат в нормальное положение
    

    displayplacer -r0

    Чтобы эти команды работали, сначала установите утилиту displayplacer через Homebrew:

    brew install displayplacer

    Способ 2: С помощью утилиты Rotate Screen

    • 📥 Скачайте программу Rotate Screen из App Store (бесплатно).
    • 🔧 Откройте приложение и выберите нужную ориентацию.
    • 🔄 Для быстрого доступа добавьте утилиту в Системные настройки → Панель управления.

    Способ 3: Для внешних мониторов

    Если вы подключили к MacBook внешний монитор с поворотным креплением:

    1. Откройте Системные настройки → Мониторы.
    2. Зажмите Option (⌥) и нажмите на кнопку Масштаб — появится скрытая опция Поворот.
    3. Выберите нужный угол (90°, 180°, 270°).
    ⚠️ Внимание: На MacBook с чипами M1/M2 некоторые утилиты для поворота экрана могут не работать из-за ограничений Apple Silicon. В этом случае используйте внешний монитор или виртуальную машину с Windows.

    6. Linux: поворот экрана через xrandr и настройки дисплея

    В дистрибутивах Linux (Ubuntu, Fedora, Mint и др.) поворот экрана настраивается через утилиту xrandr или графические интерфейсы вроде GNOME Display Settings. Способ зависит от вашего окружения рабочего стола.

    Способ 1: Через терминал (универсальный метод)

    1. Откройте терминал (Ctrl + Alt + T).
    2. Введите команду, чтобы увидеть список подключённых дисплеев: 
      xrandr -q

      В выводе найдите название вашего монитора (обычно eDP-1 для встроенного экрана ноутбука или HDMI-1/DP-1 для внешнего).

    3. Используйте одну из команд для поворота (замените eDP-1 на ваш монитор): 
      # Поворот на 90° по часовой стрелке

      xrandr --output eDP-1 --rotate right
      


      Поворот на 180°
      

      xrandr --output eDP-1 --rotate inverted
      


      Поворот на 270° (90° против часовой)
      

      xrandr --output eDP-1 --rotate left
      


      Возврат в нормальное положение
      

      xrandr --output eDP-1 --rotate normal

    Способ 2: Через графический интерфейс (GNOME/KDE)

    • 🖥️ В Ubuntu/GNOME: Настройки → Устройства → Дисплеи → выберите монитор и ориентацию.
    • 🐬 В KDE Plasma: Системные настройки → Дисплей и монитор → Ориентация.
    • 🎯 В XFCE: Параметры → Дисплей → Вращение.

    Способ 3: Для сенсорных экранов

    Если у вас ноутбук с сенсорным экраном (например, Lenovo Yoga или HP Spectre), после поворота изображение может не совпадать с touch-вводом. Чтобы исправить:

    xinput set-prop "ELAN Touchscreen" "Coordinate Transformation Matrix" 0 1 0 -1 0 1 0 0 1

    (Замените ELAN Touchscreen на название вашего тачпада — узнать его можно командой xinput list.)

    ⚠️ Внимание: На некоторых дистрибутивах (например, Arch Linux) для работы xrandr может потребоваться установить пакет xorg-xrandr:

    sudo pacman -S xorg-xrandr  # Для Arch/Manjaro

    sudo apt install x11-xserver-utils  # Для Debian/Ubuntu

    7. Почему экран самопроизвольно переворачивается и как это исправить

    Если ваш экран произвольно поворачивается без вашего участия, причины могут быть следующими:

    Причина Как проявляется Решение
    Случайное нажатие горячих клавиш Экран переворачивается при нажатии Ctrl + Alt + Стрелка (например, в играх) Отключите комбинации в настройках драйвера или переназначьте их
    Конфликт с программным обеспечением Поворот происходит при запуске определённых программ (например, AutoHotkey или эмуляторов) Проверьте автозагрузку (Win + R → msconfig) и отключите подозрительные программы
    Неисправность гироскопа (на трансформерах) Экран поворачивается при изменении положения ноутбука (актуально для Lenovo Yoga, HP x360) Отключите функцию Автоповорот в настройках Windows или обновите драйвер сенсоров
    Вирус или вредоносное ПО Поворот сопровождается другими глюками (реклама, перенаправление в браузере) Просканируйте систему Malwarebytes или Kaspersky Virus Removal Tool

    Как отключить автоповорот на трансформерах:

    1. Нажмите Win + A, чтобы открыть Центр уведомлений.
    2. Найдите плитку Режим планшета или Автоповорот и отключите её.
    3. Для полного отключения:
      • Перейдите в Параметры → Система → Дисплей.
      • В разделе Масштаб и макет найдите Автоповорот и деактивируйте его.

    • Если проблема в драйверах сенсоров:

    • Откройте Диспетчер устройств (Win + X).
    • Разверните раздел Сенсорные устройства.
    • Обновите драйвер для устройств с названиями Accelerometer, Gyroscope или Sensor Collection.
    Что делать, если экран перевернулся и мышка не работает?

    Если после поворота экрана курсор мыши исчез или перестал реагировать, попробуйте:

    1. Нажмите Ctrl + Alt + F1, чтобы переключиться в текстовый режим (для Linux/Windows с WSL).

    2. В слепую введите команду возврата экрана (например, xrandr --output eDP-1 --rotate normal для Linux).

    3. Перезагрузите ноутбук удерживая кнопку питания 10 секунд.

    4. Подключите внешнюю клавиатуру и используйте клавиши Tab/Enter, чтобы навигация по меню восстановления.

    8. Поворот экрана на ноутбуке с несколькими мониторами

    Если вы подключили внешний монитор к ноутбуку и нужно повернуть только один из экранов, используйте следующие методы:

    В Windows:

    1. Откройте Параметры → Система → Дисплей.
    2. Выберите монитор, который нужно повернуть (кликните по его схеме вверху экрана).
    3. В разделе Ориентация выберите нужный угол.
    4. Для разных ориентаций на каждом мониторе повторите шаги для каждого дисплея отдельно.

    В MacOS:

    • Откройте Системные настройки → Мониторы.
    • Зажмите Option (⌥) и нажмите на Масштаб для нужного монитора.
    • Выберите ориентацию и подтвердите.

    В Linux:

    Используйте xrandr с указанием конкретного монитора. Например, чтобы повернуть только внешний монитор HDMI-1 на 90°:

    xrandr --output HDMI-1 --rotate right --output eDP-1 --rotate normal

    Особенности многомониторных систем:

    • 🔄 Если после поворота одного экрана курсор мыши "прыгает" между мониторами, настройте их положение в параметрах дисплея (перетащите схемы мониторов так, чтобы они совпадали с физическим расположением).
    • 🎮 В играх повёрнутый экран может сбрасываться — проверьте настройки графики в лаунчере игры (например, в Steam отключите Override display mode).
    • 🖥️ На ноутбуках с hybrid graphics внешний монитор может управлять только дискретной видеокартой (например, NVIDIA), а встроенный экран — Intel. В этом случае настройки поворота могут конфликтовать.
    💡

    При подключении нескольких мониторов сначала настройте их физическое расположение в параметрах Windows (Параметры → Система → Дисплей → Перетащите мониторы), а потом поворачивайте каждый отдельно. Это предотвратит проблемы с курсором и окнами программ.

    FAQ: Частые вопросы о повороте экрана

    Можно ли повернуть экран на ноутбуке с Windows 7?

    Да, но способы немного отличаются:

    1. Кликните правой кнопкой по рабочему столу → Разрешение экрана.
    2. Выберите Ориентация и примените изменения.

    Горячие клавиши (Ctrl + Alt + Стрелка) работают, только если установлены драйверы от производителя (не стандартные от Microsoft).

    Почему после поворота экрана на 90° текст стал размытым?

    Это происходит из-за некорректного масштабирования. Чтобы исправить:

    • В Windows: перейдите в Параметры → Система → Дисплей и установите масштаб на 100%.
    • Нажмите Дополнительные параметры масштабирования и отключите Разрешить Windows исправлять размытость.
    • Для MacOS: в Системных настройках → Мониторы выберите разрешение по умолчанию (без масштабирования).

    Если размытость осталась, обновите драйвер видеокарты.

    Как зафиксировать экран в книжной ориентации для чтения?

    Для удобного чтения длинных документов (PDF, книги):

    1. Поверните экран на 90° (см. раздел 1).
    2. В программе для чтения (например, Adobe Acrobat или SumatraPDF) включите режим Полноэкранный просмотр (F11).
    3. Используйте комбинацию Ctrl + +/Ctrl + -, чтобы подогнать масштаб под экран.

    Для MacOS удобно использовать приложение Kindle или Books — они автоматически подстраиваются под ориентацию экрана.

    Можно ли повернуть экран на ноутбуке с BIOS?

    Нет, настройки BIOS/UEFI не влияют на ориентацию экрана. Поворот настраивается только на уровне операционной системы или драйверов видеокарты. Однако в некоторых случаях:

    • Если экран перевёрнут ещё на этапе загрузки (до появления логотипа Windows), проблема может быть в неисправности шлейфа матрицы или BIOS-вирусе.
    • На некоторых ноутбуках Lenovo ThinkPad в BIOS есть опция Graphics Device — её изменение может сбросить настройки поворота.

    Если подозреваете аппаратную проблему, проверьте ноутбук в сервисном центре.

    Как повернуть экран на ноутбуке с Linux без xrandr?

    Если xrandr не работает (например, на Wayland вместо X11), попробуйте:

    • 🖥️ В GNOME (Ubuntu 22.04+): используйте расширение Rotate Screen.
    • 🐧 В KDE Plasma: Настройки системы → Дисплей и монитор → Ориентация.
    • 📌 Для Wayland (Fedora, Arch): используйте wlr-randr (для композиторов Sway/Hyprland): 
      wlr-randr --output eDP-1 --transform 90
