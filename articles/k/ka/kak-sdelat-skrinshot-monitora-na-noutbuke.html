Как сделать скриншот экрана на ноутбуке: все способы для Windows, Mac и Linux

Создание скриншота (снимка экрана) на ноутбуке — одна из самых востребованных операций, которая пригодится и для работы, и для обучения, и даже для общения. Но несмотря на кажущуюся простоту, у многих пользователей возникают сложности: где сохраняется файл, как сделать снимок только активного окна или как обрезать ненужные части кадра. Особенно запутанно бывает на MacBook с их уникальными сочетаниями клавиш или на Linux-системах, где стандартные инструменты часто отсутствуют.

В этой статье мы разберём все актуальные способы создания скриншотов на ноутбуках под управлением Windows 10/11, macOS и Linux (включая Ubuntu, Fedora и Mint). Вы узнаете не только базовые клавишные комбинации, но и скрытые функции встроенных утилит, а также лучшие бесплатные программы для расширенного редактирования снимков. Отдельно остановимся на проблемах, с которыми сталкиваются владельцы ноутбуков с сенсорными экранами или гибридными устройствами (например, Microsoft Surface или Lenovo Yoga).

1. Стандартные клавишные комбинации для Windows

На большинстве ноутбуков под управлением Windows сделать скриншот можно буквально в одно нажатие — без установки дополнительного ПО. Основные комбинации работают одинаково на всех устройствах, независимо от производителя (HP, Dell, Asus, Acer и др.), но есть нюансы с сохранением файла.

Самый универсальный способ — клавиша PrtScn (Print Screen). На некоторых ноутбуках она может быть совмещена с другой функцией (например, SysRq), поэтому для её активации потребуется нажать Fn. Вот что делают разные комбинации:

  • 🖥️ PrtScn — копирует весь экран в буфер обмена (нужно вставить в Paint или другой редактор).
  • 🪟 Win + PrtScn — сохраняет скриншот сразу в папку Изображения → Снимки экрана (файл в формате PNG).
  • 🖼️ Alt + PrtScn — копирует в буфер только активное окно (без фона).
  • 🎮 Win + Alt + PrtScn — делает скриншот в игре (работает только в Windows 10/11 с включённой Игровой панелью).

Если на вашем ноутбуке клавиша PrtScn отсутствует (как на некоторых ультрабуках Lenovo IdeaPad или Huawei MateBook), попробуйте комбинацию Fn + Windows + Пробел. Это альтернативный способ вызова встроенной утилиты Ножницы (о ней поговорим ниже).

2. Встроенная утилита «Ножницы» (Snipping Tool) в Windows

Утилита Ножницы (или Snipping Tool в английской версии) — это встроенный инструмент Windows, который позволяет не только делать скриншоты, но и сразу их редактировать. Она доступна во всех версиях ОС, начиная с Windows Vista, но в Windows 11 получила значительное обновление.

Чтобы открыть Ножницы, выполните одно из действий:

  • 🔍 Введите в поиске Ножницы и выберите приложение.
  • 🖱️ Нажмите Win + Shift + S — это откроет мини-панель для выбора области экрана (даже если утилита не запущена).
  • 📁 Перейдите по пути: C:\Windows\System32\SnippingTool.exe.

После запуска выберите один из режимов:

Произвольная формаРисование контура мышью для скриншота любой областиCtrl + Shift + D
ПрямоугольникВыделение прямоугольной области (стандартный режим)Ctrl + Shift + R
ОкноСнимок активного окна (аналог Alt + PrtScn)Ctrl + Shift + W
Весь экранСнимок всего рабочего стола (аналог PrtScn)Ctrl + Shift + F

В Windows 11 утилита получила дополнительные функции: задержка съёмки (до 10 секунд), возможность добавлять аннотации (стрелки, текст, маркеры) и сохранять скриншоты в JPG, PNG или GIF. Также появилась интеграция с OneDrive — можно автоматически загружать снимки в облако.

3. Скриншоты на ноутбуках MacBook (macOS)

На ноутбуках Apple MacBook (включая модели MacBook Air и MacBook Pro) процесс создания скриншотов отличается от Windows. Здесь используются комбинации с клавишей Command (⌘), а файлы по умолчанию сохраняются на Рабочий стол в формате PNG.

Основные комбинации:

  • 📱 ⌘ + Shift + 3 — снимок всего экрана.
  • 🖥️ ⌘ + Shift + 4 — выбор области экрана (появится прицел для выделения).
  • 🪟 ⌘ + Shift + 4 + Пробел — снимок активного окна (после нажатия клавиш курсор превратится в иконку камеры).
  • 🎥 ⌘ + Shift + 5 — открывает панель с расширенными настройками (запись экрана, таймер, выбор места сохранения).

В macOS Monterey и новее (включая Ventura и Sonoma) появилась возможность изменять формат сохраняемых файлов. Для этого введите в Терминале команду:

defaults write com.apple.screencapture type jpg

killall SystemUIServer

После этого скриншоты будут сохраняться в JPG. Чтобы вернуть PNG, замените в команде jpg на png.

⚠️ Внимание: На MacBook с чипом Apple M1/M2 скриншоты, сделанные через ⌘ + Shift + 5, могут автоматически сохраняться в iCloud, если включена синхронизация Рабочего стола. Проверьте настройки в Системные настройки → Apple ID → iCloud → Параметры....
Как отменить сохранение скриншота в macOS?

Если вы сделали скриншот по ошибке, его миниатюра появится в правом нижнем углу экрана. Нажмите на неё и перетащите в корзину или нажмите Esc, чтобы отменить сохранение. Это работает только в течение 5 секунд после создания снимка.

4. Скриншоты в Linux: Ubuntu, Fedora, Mint

В дистрибутивах Linux нет универсального способа сделать скриншот — всё зависит от используемой графической оболочки (GNOME, KDE Plasma, XFCE и др.). Однако большинство систем поддерживают клавишу PrtScn, а также имеют встроенные утилиты вроде gnome-screenshot или ksnapshot.

Вот основные способы для популярных дистрибутивов:

  • 🐧 Ubuntu/GNOME: PrtScn — весь экран, Alt + PrtScn — активное окно, Shift + PrtScn — выбор области.
  • 💎 KDE Plasma (Kubuntu): PrtScn открывает Spectacle — утилиту с расширенными настройками.
  • 🍃 Linux Mint (Cinnamon): PrtScn сохраняет скриншот в ~/Pictures/Screenshots.

Если стандартные комбинации не работают, установите утилиту shutter (доступна в репозиториях большинства дистрибутивов). Она позволяет делать скриншоты с задержкой, добавлять эффекты и даже редактировать снимки. Установка в Ubuntu/Debian:

sudo apt update

sudo apt install shutter

Для пользователей Wayland (альтернатива X11 в новых версиях GNOME и KDE) стандартные сочетания клавиш могут не работать. В этом случае используйте терминальные команды:

gnome-screenshot -a -f ~/screenshot.png  # Выбор области

gnome-screenshot -w -f ~/window.png      # Активное окно

💡

Если скриншоты в Linux сохраняются в низком разрешении, проверьте настройки в утилите dconf-editor по пути org.gnome.gnome-screenshot. Параметр auto-save-directory отвечает за папку сохранения, а include-pointer — за отображение курсора.

5. Специализированные программы для скриншотов

Если встроенных инструментов недостаточно, можно воспользоваться сторонними программами. Они предлагают расширенные функции: запись видео с экрана, редактирование скриншотов, загрузку в облако и даже распознавание текста (OCR). Вот топ-5 бесплатных решений:

  • 📸 ShareX (Windows) — поддерживает GIF-анимации, автоматические действия после съёмки (загрузка на Imgur, Google Drive), а также имеет встроенный редактор.
  • 🖼️ Lightshot (Windows/macOS) — позволяет быстро выделить область и отредактировать её (добавить текст, стрелки), а затем поделиться ссылкой.
  • 🎥 Flameshot (Linux/Windows/macOS) — открытый аналог Lightshot с поддержкой аннотаций и размытия отдельных частей снимка.
  • 🔍 Snagit (Windows/macOS) — платная программа с пробным периодом, но предлагает уникальные функции вроде создания scrolling screenshots (скриншоты длинных веб-страниц).
  • ☁️ Greenshot (Windows) — лёгкая утилита с интеграцией в Confluence, Jira и другие корпоративные сервисы.

Для профессиональных задач (например, создания мануалов или презентаций) рекомендуем Snagit — он умеет захватывать прокручиваемые окна браузера, добавлять эффекты переходов и даже записывать видео с веб-камеры одновременно со скриншотом. Бесплатной альтернативой станет Flameshot, который поддерживает Wayland и имеет минималистичный интерфейс.

⚠️ Внимание: При использовании ShareX или Lightshot для загрузки скриншотов в облако проверьте настройки конфиденциальности. Некоторые сервисы (например, Imgur) могут делать снимки общедоступными по умолчанию.

6. Скриншоты на ноутбуках с сенсорным экраном и гибридных устройствах

Ноутбуки-трансформеры (например, Lenovo Yoga, HP Envy x360 или Microsoft Surface Book) и устройства с сенсорными экранами требуют особого подхода. Здесь стандартные клавишные комбинации могут не срабатывать из-за отсутствия физической клавиатуры или специфической раскладки клавиш.

Вот как сделать скриншот в таких случаях:

  • 📱 Планшетный режим (Windows): Нажмите одновременно Кнопка питания + Увеличение громкости (аналог PrtScn). Снимок сохранится в Изображения → Снимки экрана.
  • 🖥️ Microsoft Surface: Используйте Fn + Пробел (эмуляция PrtScn) или Win + Shift + S для выделения области.
  • 🍎 iPad с клавиатурой: Если подключена внешняя клавиатура, работают стандартные комбинации macOS (⌘ + Shift + 3/4).
  • ⚡ ChromeOS (на Chromebook): Нажмите Ctrl + Показать окна (клавиша с прямоугольником и двумя вертикальными линиями).

На Microsoft Surface Pro и подобных устройствах также доступна ручка Surface Pen. Дважды нажмите на её кнопку, чтобы сделать снимок активного окна и открыть его в OneNote для аннотирования. Эта функция работает только при подключённой ручке и включённом Surface App.

💡

На гибридных устройствах всегда проверяйте, включён ли режим «Ноутбук» или «Планшет» в настройках экрана. В планшетном режиме клавишные комбинации могут блокироваться.

7. Редактирование и оптимизация скриншотов

Сделанный скриншот часто требует доработки: обрезки, размытия личных данных или добавления поясняющих надписей. Для этого не обязательно использовать Photoshop — есть множество бесплатных инструментов:

  • 🎨 Paint 3D (Windows) — встроенное приложение для базовой обрезки, добавления текста и 3D-эффектов.
  • 🖌️ GIMP (Windows/macOS/Linux) — полноценный аналог Photoshop с поддержкой слоёв и фильтров.
  • 📝 Paint.NET (Windows) — лёгкий редактор с плагинами для размытия, коррекции цвета и работы с прозрачностью.
  • 🌐 Online-редакторы: Photopea, Pixlr или Canva — не требуют установки, но нужна стабильная связь с интернетом.

Если вам нужно часто делать скриншоты для документации или обучения, обратите внимание на следующие фишки:

  • 🔍 Используйте горячие клавиши в редакторах: в GIMP Shift + C вызывает инструмент обрезки, а в Paint.NET Ctrl + Shift + X обрезает холст по выделению.
  • 📏 Для точной обрезки удерживайте Shift — это зафиксирует пропорции (например, для квадратного кадра).
  • 🔒 Чтобы скрыть конфиденциальную информацию, используйте инструмент Размытие или наложите поверх неё однотонный прямоугольник.

Для оптимизации размера файла перед отправкой по почте или загрузкой на сайт используйте сервисы сжатия:

  • 📎 TinyPNG — сжимает PNG и JPG без потери качества.
  • 🖼️ ImageOptim — приложение для macOS/Windows с поддержкой пакетной обработки.

8. Частые проблемы и их решения

Даже с такими простыми операциями, как скриншоты, могут возникать сложности. Вот наиболее распространённые проблемы и способы их устранения:

ПроблемаВозможная причинаРешение
Скриншоты не сохраняютсяНе хватает прав на запись в папку или диск переполненПроверьте свободное место на диске или измените папку сохранения в настройках утилиты
Клавиша PrtScn не работаетОтключена в BIOS или конфликт с FnПопробуйте Fn + PrtScn или проверьте настройки клавиатуры в BIOS
Снимок экрана чёрныйКонфликт с драйверами видеокарты или включён HDRОбновите драйверы или отключите HDR в настройках экрана
Не работает Win + PrtScnОтключена служба Снимок экрана в WindowsЗапустите services.msc и включите службу Snipping Tool
Скриншоты в низком разрешенииМасштабирование экрана (>100%) или настройки DPIВерните масштаб на 100% или используйте Ножницы с настройкой качества

Если скриншоты делаются, но не копируются в буфер обмена, проверьте работу службы rdpclip.exe (отвечает за обмен данными между приложениями). Запустите её вручную через Диспетчер задач → Файл → Запустить новую задачу.

⚠️ Внимание: На некоторых ноутбуках с NVIDIA Optimus (например, MSI GS66 или Asus ROG Zephyrus) скриншоты в играх могут получаться чёрными. Это связано с тем, что игра запущена на дискретной видеокарте. Решение — использовать встроенную утилиту GeForce Experience для съёмки (Alt + F1).

FAQ: Ответы на частые вопросы

🔹 Как сделать скриншот только одного монитора, если подключено несколько?

В Windows нажмите Win + Shift + S, затем выберите нужный монитор в панели инструментов. В macOS используйте ⌘ + Shift + 4, затем нажмите Пробел — курсор превратится в камеру, и вы сможете кликнуть по нужному экрану.

🔹 Куда сохраняются скриншоты на MacBook?

По умолчанию — на Рабочий стол в формате Снимок экрана [дата] в [время].png. Чтобы изменить папку, используйте команду в Терминале:

defaults write com.apple.screencapture location ~/Downloads

killall SystemUIServer

После этого скриншоты будут сохраняться в Загрузки.

🔹 Можно ли сделать скриншот с задержкой?

Да, во всех современных ОС есть такая функция:

  • Windows: В утилите Ножницы нажмите Задержка и выберите время (до 10 секунд).
  • macOS: Нажмите ⌘ + Shift + 5, затем выберите Параметры → Таймер.
  • Linux: В Flameshot или Shutter есть встроенная задержка (до 60 секунд).
🔹 Как сделать скриншот меню (например, контекстного)?

Используйте задержку съёмки:

  1. Откройте утилиту для скриншотов и установите задержку 5–10 секунд.
  2. Запустите съёмку, затем быстро откройте нужное меню.
  3. Если меню исчезает при клике, попробуйте использовать клавиатурные сочетания (например, Alt + PrtScn после открытия меню).

В macOS для съёмки меню удерживайте Ctrl при нажатии ⌘ + Shift + 4 — это отключит закрытие меню при клике.

🔹 Почему скриншоты получаются размытыми?

Это связано с масштабированием экрана:

  • В Windows проверьте настройки масштаба в Параметры → Система → Экран. Если стоит значение больше 100%, снимок будет сжат.
  • В macOS размытость может появляться при съёмке экрана с разрешением Retina. Чтобы получить чёткий снимок, используйте команду:
defaults write com.apple.screencapture disable-shadow -bool true

killall SystemUIServer

Это отключит тени вокруг окон, которые иногда размывают края.
