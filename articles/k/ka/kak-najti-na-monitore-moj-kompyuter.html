Где найти «Мой компьютер» на экране монитора: все способы для Windows

Вы включили компьютер, но на рабочем столе нет привычного значка «Этот компьютер» (или «Мой компьютер» в старых версиях Windows)? Не паникуйте — это стандартная ситуация после обновления системы или сброса настроек. Значок мог быть скрыт автоматически, удалён пользователем или просто «затерялся» среди других ярлыков. В этой статье мы разберём 5 проверенных способов, как найти «Мой компьютер» на экране монитора, даже если вы не видите его на рабочем столе.

Важно понимать, что сам «Мой компьютер»** — это не физический объект на мониторе, а системная папка Windows, отображающая подключённые диски, сетевые устройства и параметры системы. Его отсутствие на рабочем столе не означает, что он «исчез» из системы. Достаточно знать, где искать — и вы получите доступ к файлам, настройкам дисков и управлению устройствами за считанные секунды.

Инструкции ниже подходят для Windows 10 и Windows 11, а также большинства сборок на их основе (например, Windows 10 LTSC). Если вы используете macOS или Linux, принципы будут другими — об этом мы кратко упомянем в конце статьи.

1. Способ: Поиск через меню «Пуск»

Самый быстрый метод — воспользоваться встроенной поисковой строкой. Он работает даже если значок скрыт или удалён.

Шаги:

  • 🔍 Нажмите на кнопку Пуск (или клавишу Win на клавиатуре).
  • 📝 В строке поиска введите Этот компьютер (для Windows 10/11) или Мой компьютер (для Windows 7/8).
  • 🖱️ В результатах поиска появится нужный элемент — кликните по нему.

Если поиск не срабатывает (например, из-за сбоя службы Windows Search), попробуйте альтернативный вариант:

  • 🔄 Нажмите Win + R, введите explorer и нажмите Enter.
  • 📁 В открывшемся проводнике слева в панели навигации найдите раздел «Этот компьютер»** — он всегда отображается там.
💡

Если поиск не находит «Мой компьютер», проверьте, не отключена ли служба Windows Search. Для этого нажмите Win + R, введите services.msc, найдите службу Windows Search и убедитесь, что она запущена.

2. Способ: Горячие клавиши для мгновенного доступа

Для опытных пользователей клавиатурные комбинации экономят время. Вот 3 полезных сочетания, которые откроют «Мой компьютер» в один клик:

Сочетание клавиш Действие Примечание
Win + E Открывает проводник с выделенным разделом «Этот компьютер»** Работает во всех версиях Windows
Win + X → E Открывает меню быстрого доступа, где есть пункт Проводник В Windows 11 меню Win + X слегка изменено
Alt + D (в открытом проводнике) Перемещает фокус на адресную строку, где можно ввести Этот компьютер Полезно, если папка не видна в боковом меню

Если горячие клавиши не срабатывают, проверьте, не отключены ли они в настройках системы. Например, в некоторых сборках Windows функции клавиш Win могут быть заблокированы администратором или сторонним ПО (например, AutoHotkey).

3. Способ: Восстановление значка на рабочем столе

Если вам удобнее иметь постоянный доступ к «Мой компьютер» прямо на экране, можно вернуть значок вручную. Это занимает меньше минуты:

Инструкция для Windows 10/11:

  1. 🖱️ Кликните правой кнопкой мыши на пустом месте рабочего стола.
  2. 📋 Выберите Персонализация (в Windows 11) или Параметры значков рабочего стола (в Windows 10).
  3. 🔧 В открывшемся окне найдите раздел «Значки рабочего стола»** и поставьте галочку напротив Компьютер.
  4. 💾 Нажмите Применить → ОК.

Если пункта Персонализация нет в контекстном меню (например, из-за кастомизированной оболочки), используйте альтернативный путь:

Панель управления → Оформление и персонализация → Персонализация → Темы → Параметры значков рабочего стола

4. Способ: Через проводник Windows

Проводник (File Explorer) — это «сердце» файловой системы Windows, и «Мой компьютер» всегда доступен там. Вот как его найти:

Шаги:

  • 📂 Откройте проводник (например, через Win + E или двойной клик по значку папки на панели задач).
  • 📌 В левой части окна найдите раздел «Этот компьютер» (в Windows 10/11) или «Компьютер» (в Windows 7/8).
  • 🔗 Кликните по нему — откроется окно с дисками (Локальный диск C:, D: и т.д.) и сетевыми устройствами.

Если раздел «Этот компьютер»** не отображается в боковом меню, попробуйте:

  • 🔄 Перезапустить проводник: откройте Диспетчер задач (Ctrl + Shift + Esc), найдите процесс Проводник, кликните Перезапустить.
  • 🔧 Сбросить настройки проводника: в проводнике перейдите в Вид → Параметры → Восстановить значения по умолчанию.
Что делать, если проводник не открывается?

Если проводник не запускается даже через Win + E, проверьте целостность системных файлов. Откройте командную строку от имени администратора (Win + X → Терминал (администратор)) и выполните команду: 

sfc /scannow

После завершения проверки перезагрузите ПК.

5. Способ: Через командную строку или PowerShell

Для продвинутых пользователей или автоматизации задач можно открыть «Мой компьютер» через командную строку. Это полезно, если графический интерфейс работает со сбоями.

Команды:

  • 📋 Для командной строки (cmd): 
    explorer /root,::{20D04FE0-3AEA-1069-A2D8-08002B30309D}
  • 💻 Для PowerShell: 
    Start-Process "explorer.exe" -ArgumentList "shell:::{20D04FE0-3AEA-1069-A2D8-08002B30309D}"

Эти команды используют CLSID (идентификатор класса) системной папки «Мой компьютер». Если команда не срабатывает, проверьте, не изменялся ли этот идентификатор в вашей версии Windows (актуально для кастомизированных сборок).

💡

Команды для открытия «Мой компьютер» универсальны и работают даже если графический интерфейс Windows повреждён или заблокирован.

6. Способ: Альтернативные методы (для macOS и Linux)

Если вы используете не Windows, а другую операционную систему, принципы доступа к файловой системе будут иными. Вот краткое руководство:

Для macOS:

  • 🍎 Аналог «Мой компьютер» — это приложение Finder. Откройте его через док-панель илиSpotlight (Cmd + Пробел → введите Finder).
  • 📁 В боковом меню Finder выберите Устройства — там отобразятся подключённые диски и сетевые ресурсы.

Для Linux (Ubuntu, Mint и др.):

  • 🐧 Откройте файловый менеджер (Nautilus, Dolphin, Thunar — в зависимости от дистрибутива).
  • 📂 В боковом меню найдите раздел Устройства или Файловая система — там будут отображены все подключённые диски.
  • 🔧 Если значков нет, проверьте, не скрыты ли они в настройках менеджера файлов (обычно в меню Вид → Показать скрытые устройства).

В Linux также можно использовать терминал для доступа к дискам. Например, команда lsblk покажет все подключённые блоковые устройства (диски и разделы), а nautilus / откроет корневой каталог в графическом режиме.

💡

В Linux для быстрого доступа к дискам можно создать ярлыки на рабочем столе. Например, в Ubuntu кликните ПКМ на рабочем столе → Создать запускатель → укажите команду nautilus /media/$USER (для внешних дисков) или nautilus / (для корневой файловой системы).

Частые проблемы и решения

Иногда доступ к «Мой компьютер» затруднён из-за системных сбоев или вирусов. Рассмотрим типичные scenarios и способы их устранения.

Проблема 1: Значок «Этот компьютер» серый и не открывается

⚠️ Внимание: Если значок стал тусклым и не реагирует на клики, это может указывать на повреждение системных файлов или конфликт с антивирусом. Перед действиями ниже проверьте компьютер на вирусы (например, через Kaspersky Virus Removal Tool).
  • 🔧 Запустите проверку системных файлов: 
    sfc /scannow
  • 🔄 Перезапустите службу Проводник через Диспетчер задач.
  • 📥 Если проблема остаётся, восстановите систему через Win + R → rstrui (выберите точку восстановления до появления сбоя).

Проблема 2: «Мой компьютер» открывается, но не показывает диски

  • 📁 Проверьте, подключены ли диски физически (особенно если речь о внешних накопителях).
  • 🔧 Откройте Управление дисками (Win + X → Управление дисками) и проверьте, отображаются ли диски там. Если диск не инициализирован или не имеет буквы, кликните по нему ПКМ и выберите Инициализировать диск или Изменить букву диска.
  • 🛠️ Обновите драйверы контроллера дисков через Диспетчер устройств.

Проблема 3: После обновления Windows значок пропал

⚠️ Внимание: Обновления Windows (особенно крупные, как переход с Windows 10 на 11) могут сбрасывать настройки рабочего стола. Если после апгрейда пропало несколько значков, восстановите их через Параметры → Персонализация → Темы → Параметры значков рабочего стола.

FAQ: Частые вопросы

Можно ли создать ярлык «Мой компьютер» на панели задач?

Да, но не напрямую. Сначала восстановите значок на рабочем столе (см. Способ 3), затем перетащите его на панель задач, удерживая Ctrl + Shift (это закрепит его как отдельный элемент, а не временную иконку).

Почему в «Мой компьютер» не отображается внешний жёсткий диск?

Причин несколько:

  • 🔌 Диск не подключён к USB 3.0 (попробуйте другой порт).
  • 📛 Диску не присвоена буква (проверьте в Управлении дисками).
  • 🛡️ Антивирус блокирует доступ (временно отключите защиту и проверьте).
  • 🔋 Не хватает питания (подключите диск через USB-хаб с внешним питанием).
Как вернуть старое название «Мой компьютер» вместо «Этот компьютер»?

Это требует редактирования реестра:

  1. 🔧 Нажмите Win + R, введите regedit.
  2. 📁 Перейдите по пути: 
    HKEY_CLASSES_ROOT\CLSID\{20D04FE0-3AEA-1069-A2D8-08002B30309D}\LocalizedString
  3. 📝 Измените значение параметра (По умолчанию) на @=%SystemRoot%\system32\shell32.dll,-9216 (для «Мой компьютер»).
  4. 🔄 Перезагрузите ПК.

⚠️ Внимание: Неправильное редактирование реестра может привести к сбоям системы. Создайте точку восстановления перед изменениями.

Можно ли открыть «Мой компьютер» на удалённом ПК через RDP?

Да. Подключитесь к удалённому рабочему столу, затем используйте любой из описанных способов (например, Win + E или поиск через меню Пуск). В окне RDP отобразятся диски удалённого компьютера, а не вашего локального.

Почему в Windows 11 нет значка «Этот компьютер» на рабочем столе по умолчанию?

Microsoft изменила политику отображения значков, чтобы уменьшить визуальный шум на рабочем столе. В Windows 11 по умолчанию показывается только корзина. Чтобы вернуть «Этот компьютер», следуйте Спосobu 3 из этой статьи.
