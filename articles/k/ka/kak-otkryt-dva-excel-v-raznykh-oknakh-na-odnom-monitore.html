Как открыть два Excel одновременно в разных окнах на одном мониторе: полное руководство

Работа с несколькими таблицами Excel одновременно — стандартная задача для аналитиков, бухгалтеров и менеджеров. Но когда оба файла открываются в одном окне с вкладками, переключаться между ними неудобно, а сравнивать данные приходится вслепую. К счастью, в Microsoft Excel есть несколько способов разделить рабочие книги по разным окнам даже на одном мониторе.

В этой статье вы найдёте 5 проверенных методов для открытия двух (и более) файлов Excel в отдельных окнах, включая горячие клавиши, настройки программы и обходные пути для старых версий. Мы также разберём типичные ошибки (например, когда окна сливаются обратно) и дадим рекомендации по оптимальной организации рабочего пространства. Все инструкции актуальны для Windows 10/11 и Office 2016–2026, включая подписку Microsoft 365.

Если вы часто работаете с большими массивами данных, этот гайд сэкономит вам часы времени. Начнём с самого простого способа — он сработает в 90% случаев.

Способ 1: Открытие файлов через проводник Windows

Самый надёжный метод — запустить каждую книгу Excel в отдельном процессе. Для этого не нужно менять настройки программы, достаточно правильно открыть файлы.

Вот пошаговая инструкция:

  1. Закройте все открытые окна Excel.
  2. Откройте Проводник Windows (Win + E).
  3. Найдите первый файл Excel, зажмите клавишу Shift и кликните по нему правой кнопкой мыши.
  4. В контекстном меню выберите Открыть в новом окне (в некоторых версиях — Открыть с помощью → Excel).
  5. Повторите шаги 3–4 для второго файла.

Теперь у вас будут два независимых окна Excel, которые можно расположить рядом на одном мониторе. Чтобы упростить задачу, используйте сочетание Win + Стрелка влево/вправо для автоматического прикрепления окон к краям экрана.

⚠️ Внимание: Если вы откроете третий файл тем же способом, он может "прилипнуть" к одному из существующих окон. В этом случае используйте Способ 3 (ниже), чтобы принудительно разделить процессы.

Способ 2: Горячие клавиши для дублирования окна

Если оба файла уже открыты в одном окне Excel на вкладках, их можно быстро разделить без закрытия программы. Для этого:

  1. Активируйте вкладку с первым файлом.
  2. Нажмите комбинацию Alt + F11, чтобы открыть редактор VBA.
  3. Закройте редактор (Alt + Q) — это принудительно создаст новое окно для текущей книги.
  4. Повторите шаги 1–3 для второго файла.

Теперь у вас будут два отдельных окна. Этот метод работает потому, что Excel воспринимает открытие редактора VBA как команду для создания нового экземпляра программы. Минус способа — временное появление окна редактора, которое может сбить с толку новичков.

Альтернативная комбинация (для Excel 2019 и новее):

  • 📋 Нажмите View → New Window (Просмотр → Новое окно) для текущей книги.
  • 🔄 Повторите для второго файла.
  • 🖥️ Разместите окна рядом с помощью Win + Стрелка.
💡

Если горячие клавиши не срабатывают, проверьте, не отключены ли они в настройках Excel. Перейдите в Файл → Параметры → Дополнительно → Параметры отображения и убедитесь, что флажок "Игнорировать другие приложения, использующие DDE" снят.

Способ 3: Запуск Excel в разных пользовательских сессиях

Для опытных пользователей есть радикальный метод — запуск Excel под разными учётными записями Windows. Это гарантированно разделит процессы, но потребует дополнительных действий.

Инструкция:

  1. Создайте второго пользователя в Windows (Параметры → Учётные записи → Семья и другие пользователи).
  2. Запустите Excel от имени первого пользователя и откройте первый файл.
  3. Нажмите Win + L, войдите под второй учётной записью и откройте второй файл.
  4. Вернитесь к первой сессии (Win + L). Теперь у вас два независимых окна.

Этот способ полезен, если вы работаете с конфиденциальными данными и хотите полностью изолировать процессы. Однако он требует переключения между пользователями, что может быть неудобно.

Способ 4: Использование параметра командной строки

Для продвинутых пользователей есть способ запустить Excel с параметром, который принудительно создаёт новый экземпляр программы. Это полезно, если стандартные методы не работают.

Как это сделать:

  1. Закройте все окна Excel.
  2. Нажмите Win + R, введите excел /x и нажмите Enter.
  3. В открывшемся окне откройте первый файл.
  4. Повторите шаги 2–3 для второго файла.

Параметр /x сбрасывает все настройки и запускает Excel в "чистом" режиме, что гарантирует разделение процессов. Этот метод особенно эффективен, если предыдущие способы приводили к слиянию окон после перезапуска программы.

Что делать, если окна всё равно объединяются?

Если после использования параметра /x окна продолжают сливаться, проверьте настройки групповой политики (для корпоративных ПК). Иногда администраторы отключают возможность многопроцессорной работы Excel через политику User Configuration → Administrative Templates → Microsoft Excel → Options → Disable multiple Excel instances.

⚠️ Внимание: Параметр /x сбрасывает некоторые пользовательские настройки (например, последние используемые папки). После работы сохраните файлы и закройте Excel корректно, чтобы избежать потери данных.

Способ 5: Альтернативные программы для работы с Excel

Если ни один из способов не помог, рассмотрите использование сторонних программ, которые поддерживают многоконное отображение по умолчанию. Вот топ-3 альтернативы:

  • 📊 LibreOffice Calc — бесплатный аналог Excel с поддержкой нескольких окон "из коробки". Достаточно открыть файлы через меню Файл → Открыть.
  • 💼 WPS Office — лёгкий офисный пакет с интуитивным интерфейсом. Для разделения окон используйте Окно → Новое окно.
  • 🌐 OnlyOffice — облачный редактор с поддержкой табов и многоконного режима. Подходит для командной работы.

Преимущество альтернативных программ — отсутствие проблем с объединением окон. Однако они могут не поддерживать некоторые функции Excel (например, Power Query или VBA-макросы). Перед переходом проверьте совместимость с вашими файлами.

💡

Если вы часто работаете с большими таблицами, комбинация Excel (для сложных расчётов) + LibreOffice Calc (для просмотра) может стать оптимальным решением.

Типичные ошибки и их решения

Даже после успешного разделения окон пользователи сталкиваются с типичными проблемами. Вот самые распространённые и способы их устранения:

  • 🔄 Окна объединяются после перезапуска Excel: Используйте Файл → Параметры → Дополнительно и снимите флажок "Показывать все окна на панели задач".
  • 🚫 Невозможно перетащить вкладку в новое окно: В Excel 2016 и старше эта функция отключена. Используйте Способ 1 или Способ 4.
  • 📉 Зависание при открытии второго файла: Закройте все процессы excel.exe в Диспетчере задач и повторите попытку с параметром /x.
  • 🔒 Файлы открываются в режиме "Только чтение": Проверьте, не заблокированы ли они в Свойства → Общие → Атрибуты.

Если проблема сохраняется, попробуйте обновить Office до последней версии. В Microsoft 365 (подписочная модель) баги с многоконным режимом исправляются быстрее, чем в автономных версиях (Excel 2019/2021).

💡

Создайте ярлык на рабочем столе с параметром "C:\Program Files\Microsoft Office\root\Office16\EXCEL.EXE" /x (путь может отличаться). Теперь вы сможете быстро запускать Excel в "чистом" режиме двойным кликом.

Оптимальная организация рабочего пространства

Теперь, когда у вас два отдельных окна Excel, важно правильно организовать их отображение. Вот несколько советов для максимальной продуктивности:

  • 📏 Разделение экрана 50/50: Используйте Win + Стрелка влево/вправо для автоматического выравнивания окон. В Windows 11 доступны дополнительные макеты (Win + Z).
  • 🔍 Синхронизация прокрутки: Если вы сравниваете данные, включите режим View → Sync Scrolling (Просмотр → Синхронная прокрутка).
  • 📌 Закрепление важных окон: В Windows 10/11 можно закрепить окна на виртуальных рабочих столах (Win + Tab → Новый рабочий стол).
  • 🖱️ Использование нескольких мониторов: Если у вас два экрана, перетащите одно окно на второй монитор для увеличения рабочей области.

Для удобства создайте пользовательский макет в Excel:

  1. Откройте оба файла в отдельных окнах.
  2. Расположите их рядом и отрегулируйте размеры.
  3. Сохраните рабочую область: View → Save Workspace (Просмотр → Сохранить рабочую область).

Теперь вы сможете восстановить это расположение одним кликом: View → Open Workspace.

FAQ: Частые вопросы

Можно ли открыть два Excel в разных окнах на Mac?

Да, но процесс отличается. В macOS по умолчанию каждый файл открывается в новом окне. Если окна объединяются, зажмите Option (⌥) при запуске Excel или используйте команду open -n /Applications/Microsoft\ Excel.app в Terminal.

Почему при открытии второго файла первый закрывается?

Это происходит, если в настройках Excel включён режим Single Document Interface (SDI). Чтобы исправить:

  1. Закройте Excel.
  2. Запустите его с параметром excel /safe.
  3. Перейдите в Файл → Параметры → Дополнительно и снимите флажок Игнорировать другие приложения, использующие DDE.
Как вернуть объединённые окна обратно?

Если окна слились после перезапуска, накройте одно окно другим и перетащите вкладку вниз. Альтернативно, закройте все окна и откройте файлы заново через Файл → Открыть (не через недавние документы).

Работает ли это в Excel Online?

Нет, веб-версия Excel Online не поддерживает многоконный режим. Вы можете открыть два браузера (например, Chrome и Edge) и загрузить файлы в каждом из них.

Можно ли сравнивать два файла Excel в одном окне?

Да, используйте инструмент View Side by Side (Просмотр → Рядом). Он синхронизирует прокрутку и выделяет различия. Для глубокого анализа используйте надстройку Compare Sheets или Power Query.

Если ни один из методов не сработал, проверьте, не блокирует ли ваш антивирус (например, Kaspersky или ESET) создание новых процессов Excel. Добавьте программу в исключения или временно отключите защиту.

Для корпоративных пользователей: некоторые политики безопасности могут ограничивать многопроцессорный режим. В этом случае обратитесь в IT-отдел за разрешением или используйте Способ 5 (альтернативные программы).
