Как сделать скриншот с монитора: все способы для Windows, macOS и Linux

Создание скриншотов (или "принтскринов") — одна из самых востребованных операций при работе с компьютером. Без них сложно обойтись, когда нужно сохранить важную информацию с экрана, продемонстрировать ошибку технической поддержке или поделиться интересным контентом в соцсетях. Но несмотря на кажущуюся простоту, у многих пользователей возникают вопросы: какие клавиши нажимать, куда сохраняется снимок экрана, и почему иногда скриншот получается чёрным.

В этой статье мы разберём все актуальные способы создания скриншотов на мониторе — от стандартных горячих клавиш до специализированных программ. Особый акцент сделаем на нюансах для разных операционных систем (Windows 10/11, macOS, Linux), а также на решении типичных проблем, например, когда скриншот не сохраняется или делается не того экрана (актуально для многомониторных систем). Если вы используете ноутбук с внешним монитором или ультрабук с сенсорным дисплеем — здесь тоже найдёте ответы.

Стандартные горячие клавиши для скриншотов в Windows

В Windows есть несколько встроенных комбинаций клавиш для создания снимков экрана. Их преимущество — не нужно устанавливать дополнительное ПО, а недостаток — ограниченные возможности редактирования. Вот основные варианты:

  • 🖥️ PrtScn (Print Screen) — копирует весь экран в буфер обмена. Чтобы сохранить изображение, вставьте его в Paint, Word или другой редактор (Ctrl + V).
  • 📁 Win + PrtScn — сохраняет снимок всего экрана в папку Изображения → Снимки экрана (формат .png).
  • 🪟 Alt + PrtScn — копирует в буфер только активное окно (например, браузер или программу).
  • 🎮 Win + Alt + PrtScn — делает скриншот игры или приложения в режиме полного экрана (работает только в Windows 10/11 с последними обновлениями).

На некоторых ноутбуках (например, Lenovo, HP или Dell) клавиша PrtScn может быть совмещена с другой функцией (например, SysRq). В этом случае придётся нажимать её с зажатой клавишей Fn — например, Fn + PrtScn. Также на устройствах с Touch Bar (как у MacBook Pro) стандартные комбинации могут не работать — потребуется альтернативный способ.

⚠️ Внимание: Если у вас подключено несколько мониторов, комбинация PrtScn сделает снимок всех экранов одновременно. Чтобы захватить только один монитор, используйте инструмент Ножницы (о нём — в следующем разделе).

Где искать сохранённые скриншоты? По умолчанию они попадают в папку:

C:\Users\<Ваше_имя_пользователя>\Pictures\Screenshots

Если папки нет — проверьте, не отключено ли автоматическое сохранение в настройках системы (Параметры → Система → Память устройства).

Инструмент "Ножницы" — гибкий захват экрана

Встроенная утилита "Ножницы" (Snipping Tool в английской версии) позволяет выбирать область экрана для скриншота, добавлять разметку и сразу сохранять файл. Это удобно, когда нужно захватить только часть монитора — например, окно чата или график в Excel.

Как открыть:

  • 🔍 Через поиск: введите в строке Ножницы и выберите приложение.
  • 📂 Через меню Пуск → Стандартные → Ножницы.
  • ⌨️ Горячие клавиши: Win + Shift + S (в Windows 10/11 это вызовет режим выделения области).

После запуска выберите один из режимов:

РежимОписаниеГорячие клавиши
Произвольная формаРисование границ от руки (удобно для нестандартных областей)—
ПрямоугольникВыделение прямоугольной области (самый популярный вариант)Shift + стрелки (для точной настройки)
ОкноЗахват активного окна целикомAlt + выделение
Весь экранАналог PrtScn, но с возможностью редактирования—

После создания скриншота откроется окно редактирования, где можно:

  • 🖍️ Добавить текст или рисунок (инструменты Перо, Маркер, Ластик).
  • 📋 Скопировать в буфер или сохранить как файл (.png, .jpg, .gif).
  • 📤 Отправить по почте или в другое приложение.
⚠️ Внимание: В Windows 11 инструмент "Ножницы" объединён с Snip & Sketch. Если у вас не открывается привычное окно — обновите систему или используйте комбинацию Win + Shift + S для быстрого захвата.

Скриншоты на Mac: особенности macOS

На компьютерах Apple (MacBook, iMac, Mac Mini) процесс создания скриншотов отличается от Windows. Здесь есть свои горячие клавиши и встроенная утилита Снимок экрана (Screenshot). Основные комбинации:

  • 📸 Command + Shift + 3 — снимок всего экрана (сохраняется на рабочий стол в формате .png).
  • 📐 Command + Shift + 4 — выбор области экрана (появится прицел для выделения).
  • 🖥️ Command + Shift + 4 + Пробел — захват отдельного окна (после нажатия кликните на нужное окно).
  • 📁 Command + Shift + 5 — открывает панель управления снимками (аналог "Ножниц" в Windows).

В macOS Monterey и новее (2021+) при использовании Command + Shift + 5 доступны дополнительные опции:

  • 🎥 Запись видео с экрана или его части.
  • 🕒 Таймер задержки (2 или 10 секунд).
  • 📌 Сохранение в буфер обмена вместо файла.

Важно: Если у вас подключён второй монитор, комбинация Command + Shift + 3 сделает скриншоты с обоих экранов, но сохранит их как отдельные файлы (например, "Снимок экрана 2026-05-20 в 14.30.01.png" и "Снимок экрана 2026-05-20 в 14.30.02.png").

Где искать скриншоты на Mac? По умолчанию они сохраняются на Рабочий стол, но путь можно изменить через терминал. Для этого введите команду:

defaults write com.apple.screencapture location ~/Downloads/ && killall SystemUIServer

Эта команда перенаправит сохранение в папку Загрузки. Чтобы вернуть стандартное расположение, замените ~/Downloads/ на ~/Desktop/.

💡

Если скриншот на Mac получается чёрным, проверьте настройки конфиденциальности в Системные настройки → Защита и безопасность → Конфиденциальность → Экранная запись. Возможно, системе запрещено захватывать изображение с определённых приложений (например, Netflix или банковских программ).

Специализированные программы для скриншотов

Если встроенных инструментов недостаточно, можно воспользоваться сторонними программами. Они предлагают расширенные функции: запись видео с экрана, редактирование скриншотов, загрузку в облако и даже распознавание текста (OCR). Вот топовые решения:

ПрограммаПлатформаОсобенностиЦена

ShareXWindowsЗахват с прокруткой, OCR, загрузка на 80+ хостинговБесплатно
SnagitWindows, MacПрофессиональное редактирование, шаблоны, запись видеоОт 50$

Например, ShareX удобен для блогеров и технических специалистов благодаря:

  • 📜 Захвату прокручиваемых страниц (например, длинной веб-страницы или документа PDF).
  • 🔍 Распознаванию текста (OCR) прямо на скриншоте.
  • 🌐 Автоматической загрузке на Imgur, Google Drive или FTP.

Для Linux (например, Ubuntu, Fedora) чаще всего используют Flameshot или стандартную утилиту gnome-screenshot. Установить Flameshot можно командой:

sudo apt install flameshot

После установки назначьте горячую клавишу (например, PrtScn) в настройках системы.

⚠️ Внимание: Некоторые программы (например, Snagit) могут конфликтовать со встроенными инструментами Windows. Если после установки перестали работать Win + PrtScn или "Ножницы", проверьте настройки программы или отключите её автозапуск.
Как сделать скриншот с задержкой?

Если нужно захватить выпадающее меню или контекстное окно, которое исчезает при нажатии клавиш, используйте задержку:

  1. В "Ножницах" (Windows) нажмите Отложить и выберите время (до 5 секунд).
  2. В macOS при захвата через Command + Shift + 5 установите таймер в панели инструментов.
  3. В ShareX или Snagit задержка настраивается в параметрах захвата (обычно до 30 секунд).

Проблемы со скриншотами и их решения

Иногда скриншоты не получаются с первого раза. Рассмотрим типичные ошибки и способы их исправления.

1. Скриншот получается чёрным

Это чаще всего происходит в двух случаях:

  • 🎮 При попытке сделать снимок игры или видео с защитой от копирования (DRM, например, в Netflix или Disney+). Решение: используйте встроенные инструменты игры (например, F12 в Steam) или специализированное ПО вроде OBS Studio.
  • 🖥️ На ноутбуках с гибридной графикой (NVIDIA Optimus или AMD Switchable) может сработать неправильный GPU. Решение: обновите драйверы видеокарты или попробуйте сделать скриншот через Paint (PrtScn → Ctrl + V).

2. Не работает клавиша PrtScn

Возможные причины:

  • 🔧 Клавиша отключена в BIOS/UEFI (редко, но встречается на некоторых материнских платах ASUS или Gigabyte).
  • 🎹 На ноутбуках может требоваться Fn + PrtScn (проверьте значки на клавише).
  • 🖱️ Конфликт с программным обеспечением (например, Logitech Options для мышек или клавиатур).

3. Скриншот сохраняется не туда

В Windows путь сохранения можно изменить через реестр:

  1. Нажмите Win + R, введите regedit.
  2. Перейдите по пути: 
    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer
  3. Создайте параметр ScreenshotIndex (тип DWORD) и установите значение 1.
  4. Перезагрузите компьютер.

В macOS путь меняется через терминал (см. раздел про Mac выше).

💡

Если скриншот не сохраняется, проверьте права доступа к папке "Снимки экрана" (в Windows) или "Рабочий стол" (в macOS). Иногда антивирусы (например, Kaspersky) блокируют запись файлов.

Скриншоты на сенсорных мониторах и планшетах

Если вы используете монитор с сенсорным экраном (например, Microsoft Surface, Wacom Cintiq или HP Spectre), стандартные клавиатурные комбинации могут быть недоступны. В этом случае:

  • 📱 На Windows-планшетах (например, Surface Pro) одновременно нажмите Кнопка питания + Уменьшение громкости.
  • 🖊️ На графических планшетах (Wacom, Huion) используйте сочетание кнопок на стилусе (если поддерживается) или подключённую клавиатуру.
  • ⌨️ Для 2-in-1 ноутбуков (например, Lenovo Yoga) переведите устройство в режим ноутбука и используйте стандартные клавиши.

На Android-планшетах с подключённым монитором (например, Samsung DeX) скриншот делается комбинацией Кнопка питания + Уменьшение громкости. Снимок сохранится в галерее устройства, а не на компьютере.

Если сенсорный монитор подключён к Mac (например, iPad в режиме Sidecar), скриншот делается стандартными клавишами macOS, но захват будет только с основного экрана. Чтобы сделать снимок с iPad, используйте его собственные инструменты (Кнопка питания + Домой или Кнопка питания + Увеличение громкости для моделей без кнопки Домой).

Как сделать скриншот с веб-страницы полностью

Стандартные инструменты захватывают только видимую часть экрана. Если нужно сохранить всю страницу (например, длинную ленту в Twitter или статью в Wikipedia), используйте одно из этих решений:

  • 🌐 Расширения для браузера:
    • GoFullPage (Chrome, Edge) — сохраняет страницу в .png или .pdf.
    • FireShot (Firefox, Chrome) — позволяет редактировать скриншот перед сохранением.
    • Awesome Screenshot — захват с аннотациями и размытием.
  • 🖥️ Программы:
    • ShareX (раздел Capture → Scrolling Window).
    • Snagit (режим Scrolling Window).
  • 📱 Мобильные решения:
    • На Android: LongShot или Stitch & Share.
    • На iOS: встроенная функция "Снимок экрана с прокруткой" (в Safari удерживайте снимок экрана и выберите "Весь экран").

Пример работы с GoFullPage:

  1. Установите расширение из Chrome Web Store.
  2. Откройте нужную страницу.
  3. Нажмите на иконку расширения в панели инструментов.
  4. Подождите несколько секунд (в зависимости от длины страницы).
  5. Скачайте результат в формате .png или .pdf.
⚠️ Внимание: Некоторые сайты (например, Facebook или Instagram) блокируют захват контента через расширения. В этом случае используйте программные решения вроде ShareX или делайте скриншоты по частям вручную.

FAQ: Частые вопросы о скриншотах

Можно ли сделать скриншот с видео в полноэкранном режиме?

Да, но не все способы сработают из-за защиты DRM. Попробуйте:

  1. Встроенные инструменты плеера (например, F12 в VLC).
  2. Захват через OBS Studio (настройте источник "Захват экрана").
  3. На Windows 10/11 используйте Win + Alt + PrtScn (для игр и некоторых приложений).

Если видео защищено (например, Netflix), скриншот будет чёрным — это ограничение правообладателей.

Как сделать скриншот на мониторе с высоким разрешением (4K, 5K)?

Стандартные инструменты сохраняют скриншот в разрешении экрана, но иногда изображение получается размытым. Чтобы избежать потери качества:

  • В Windows: используйте ShareX с настройкой "Capture client area at cursor position" (в параметрах захвата).
  • В macOS: по умолчанию скриншоты сохраняются в разрешении Retina (удвоенное количество пикселей).
  • Для веб-страниц: расширения вроде GoFullPage сохраняют в оригинальном разрешении.

Если скриншот всё равно размытый, проверьте масштабирование экрана в настройках дисплея (должно быть 100%).

Где найти скриншоты, сделанные на телефоне, но отображаемые на мониторе?

Если вы подключили смартфон к монитору через HDMI, USB-C или беспроводные технологии (Miracast, AirPlay), скриншоты делаются на устройстве, а не на ПК. Где искать:

  • Android: папка Pictures/Screenshots или DCIM/Screenshots.
  • iPhone: альбом "Снимки экрана" в приложении "Фото".

Чтобы перенести их на компьютер, используйте облачное хранилище (Google Фото, iCloud) или подключите телефон по кабелю.

Можно ли сделать скриншот с командной строки?

Да, в Windows и Linux это возможно без графического интерфейса:

  • Windows (PowerShell): 
    Add-Type -AssemblyName System.Windows.Forms

    $screen = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds
    

    $bitmap = New-Object System.Drawing.Bitmap($screen.Width, $screen.Height)
    

    $graphics = [System.Drawing.Graphics]::FromImage($bitmap)
    

    $graphics.CopyFromScreen($screen.Location, [System.Drawing.Point]::Empty, $screen.Size)
    

    $bitmap.Save("C:\screenshot.png", [System.Drawing.Imaging.ImageFormat]::Png)

  • Linux (терминал): 
    sudo apt install scrot  # Установка утилиты

    scrot screenshot.png   # Снимок всего экрана
    

    scrot -u -b screenshot.png  # Снимок активного окна

В macOS для этого используйте команду screencapture:

screencapture -i ~/Desktop/screenshot.png
Как автоматизировать создание скриншотов (например, каждый час)?

Для автоматизации подойдут:

  • Windows: AutoHotkey (скрипт с таймером) или Task Scheduler (планировщик задач) с запуском ShareX.
  • macOS: Automator (создайте задачу с командой screencapture).
  • Linux: cron + scrot (пример команды для захвата каждые 60 минут: 
    crontab -e

    0     scrot /home/user/screenshots/%Y-%m-%d_%H-%M-%S.png

Для мониторинга изменений на экране (например, отслеживания цены в магазине) используйте Pixel Watcher или ScreenWatch.
