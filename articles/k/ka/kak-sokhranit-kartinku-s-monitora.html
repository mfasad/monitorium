Запечатлеть то, что отображается на экране монитора, приходится ежедневно: от сохранения важных документов до фиксации ошибок в программах. Но не все знают, что способов сделать это больше десятка — от стандартных инструментов операционной системы до специализированного софта с расширенными возможностями. Выбор метода зависит от того, нужно ли вам сохранить весь экран, активное окно или произвольную область, а также от требуемого качества и формата выходного файла.

В этой статье мы разберём не только базовые комбинации клавиш, но и малоизвестные функции встроенных утилит, а также профессиональные программы для тех, кто работает с высоким разрешением (например, 4K или 8K). Особое внимание уделим нюансам сохранения скриншотов на Windows 11, macOS Ventura и Linux с разными графическими оболочками. А ещё вы узнаете, как обойти ограничения некоторых приложений (например, Netflix или банковских сервисов), которые блокируют создание снимков экрана.

1. Стандартные горячие клавиши: быстрее не бывает

Самый универсальный способ — использование комбинаций клавиш, которые работают без установки дополнительного софта. Они подходят для большинства задач, когда нужно быстро сохранить изображение с монитора.

На Windows основные сочетания:

  • 🖥️ PrtScn (Print Screen) — копирует весь экран в буфер обмена (нужно вставить в Paint или другой редактор).
  • 🪟 Alt + PrtScn — сохраняет только активное окно.
  • 💾 Win + PrtScn — автоматически сохраняет скриншот в папку Изображения → Снимки экрана (формат .png).
  • 🎮 Win + Alt + PrtScn — делает снимок только игры (работает в Windows 10/11 с включённой Игровой панелью Xbox).

На macOS логика другая:

  • 🌐 Cmd + Shift + 3 — сохраняет весь экран на рабочий стол (формат .png).
  • 🖼️ Cmd + Shift + 4 — позволяет выбрать область экрана мышью.
  • 📋 Cmd + Shift + 4 → Пробел — фиксирует конкретное окно (появляется камера для выбора).
  • 📁 Cmd + Ctrl + Shift + 3/4 — копирует скриншот в буфер обмена вместо сохранения на диск.
⚠️ Внимание: На ноутбуках HP, Lenovo и некоторых других брендах клавиша PrtScn может быть совмещена с Fn. Чтобы активировать её, потребуется нажать Fn + PrtScn или изменить настройки в BIOS/UEFI.

Для Linux комбинации зависят от графической оболочки:

  • 🐧 GNOME (Ubuntu, Fedora): PrtScn — весь экран, Alt + PrtScn — активное окно, Shift + PrtScn — область.
  • 🖥️ KDE Plasma: Spectacle (встроенная утилита) запускается по PrtScn.
  • 🌿 Xfce: PrtScn через утилиту xfce4-screenshooter.

Если горячие клавиши не работают, проверьте:

  1. Не отключены ли они в настройках системы (например, в Параметры → Устройства → Клавиатура на Windows).
  2. Не конфликтуют ли они с софтом для управления клавиатурой (например, Razer Synapse или Logitech Options).
  3. Не используется ли виртуальная машина (в VirtualBox или VMware может потребоваться настройка гостевых дополнений).

2. Встроенные утилиты: больше возможностей без установки программ

Если горячих клавиш недостаточно, воспользуйтесь стандартными инструментами операционной системы. Они позволяют настраивать задержку съёмки, выбирать формат сохранения и даже записывать видео с экрана.

На Windows 10/11 это утилита «Ножницы» (Snipping Tool):

  • 📏 Режимы: Произвольная форма, Прямоугольник, Окно, Весь экран.
  • ⏱️ Задержка: до 5 секунд (полезно для фиксации выпадающих меню).
  • 🎨 Редактирование: маркер, ластик, выделение текста (с Windows 11).
  • 💾 Сохранение: .png, .jpg, .gif (в новых версиях).

Чтобы открыть «Ножницы», нажмите Win + Shift + S (быстрый вызов) или найдите через поиск. В Windows 11 утилита объединена с Snip & Sketch и поддерживает распознавание текста (OCR) прямо на скриншоте.

На macOS встроенное приложение «Снимок экрана» (Screenshot) предлагает:

  • 📍 Захват с привязкой к координатам (удобно для веб-дизайнеров).
  • 🎥 Запись видео экрана (с микрофоном или без).
  • 📌 Фиксация положения курсора на скриншоте.
  • 🔄 Сохранение в iCloud или на диск.

Чтобы открыть панель инструментов, используйте комбинацию Cmd + Shift + 5. Здесь можно настроить:

  • Место сохранения (рабочий стол, документы, почта).
  • Включение/отключение тени вокруг окон.
  • Показ плавающей миниатюры после съёмки.
⚠️ Внимание: В macOS Monterey и новее скриншоты по умолчанию сохраняются в формате .heic (высокое сжатие). Если нужен .png, измените формат в Терминале командой: 
defaults write com.apple.screencapture type png

  • 🖱️ Захват области с фиксированным соотношением сторон (например, 16:9).
  • 🕒 Автоматическое именование файлов по дате/времени.

3. Профессиональные программы: для дизайнеров и геймеров

Если вам нужны расширенные функции — аннотации, запись видео, захват прокручиваемых страниц или работа с 4K/8K — стандартных инструментов будет недостаточно. Рассмотрим топовые решения для разных задач.

Программа Платформа Ключевые функции Цена
ShareX Windows Захват прокручиваемых страниц, OCR, более 50 инструментов редактирования, поддержка .webp и .avif. Бесплатно
Snagit Windows, macOS Запись видео с веб-камеры, шаблоны аннотаций, интеграция с Google Drive и Slack. От $62.99
Lightshot Windows, macOS, Linux Быстрый захват области, поиск похожих изображений в интернете, минималистичный редактор. Бесплатно
Greenshot Windows Экспорт в Excel/Word, настройка горячих клавиш для каждого действия, поддержка плагинов. Бесплатно
CleanShot X macOS Захват с прокруткой, скрытие значков рабочего стола, запись GIF с экрана. От $29

Для геймеров и стримеров актуальны:

  • 🎮 NVIDIA ShadowPlay (для видеокарт NVIDIA) — запись последних 5–20 минут геймплея по горячей клавише.
  • 🎥 OBS Studio — захват экрана с наложением веб-камеры и микрофона (подходит для стримов).
  • 🖥️ MSI Afterburner — фиксация FPS на скриншоте (полезно для бенчмарков).

Если вы работаете с многомониторными системами, обратите внимание на:

  • 🖥️🖥️ Dual Monitor Tools — захват конкретного монитора в многоэкранной конфигурации.
  • 🔄 DisplayFusion — настройка отдельных горячих клавиш для каждого дисплея.
⚠️ Внимание: Программы вроде Snagit или CleanShot X могут конфликтовать с софтом для удалённого доступа (например, TeamViewer или AnyDesk). Перед использованием проверьте настройки совместимости в Панель управления → Программы → Программы и компоненты.
💡

Если вам нужно сделать скриншот страницы целиком (включая прокручиваемую часть), используйте расширение GoFullPage для Chrome или FireShot для Firefox. Они сохраняют страницу в PDF или PNG с разрешением до 3840×2160 (4K).

4. Онлайн-сервисы: когда нельзя установить программу

Если вы работаете на чужом компьютере или не можете установить софт (например, в корпоративной сети), воспользуйтесь онлайн-инструментами. Они работают прямо в браузере и не требуют регистрации.

Популярные сервисы:

  • 🌐 Screenshot.guru — захват веб-страницы по URL (включая прокручиваемые элементы).
  • 📱 Lightshot Online — редактор с возможностью добавлять стрелки и текст.
  • 🖼️ ImgBB — загрузка скриншота с автоматической оптимизацией для соцсетей.
  • 🔍 ScreenShotLayer — создание скриншотов с геолокацией и метками времени.

Как пользоваться:

  1. Откройте сервис в браузере (Chrome, Firefox, Edge).
  2. Нажмите кнопку Захватить экран или Upload (если скриншот уже сделан).
  3. Выделите область мышью или загрузите файл.
  4. Отредактируйте (при необходимости) и скачайте результат.

Преимущества онлайн-сервисов:

  • ✅ Нет нужды в установке ПО.
  • ✅ Работают на любых устройствах (включая Chromebook).
  • ✅ Поддержка облачного хранения (ссылка на скриншот доступна сразу).

Недостатки:

  • ❌ Ограничения по размеру файла (обычно до 10–20 МБ).
  • ❌ Риск утечки данных (не используйте для конфиденциальной информации).
  • ❌ Зависимость от скорости интернета (при медленном соединении возможны артефакты).
⚠️ Внимание: Некоторые сервисы (например, Gyazo или Droplr) автоматически публикуют скриншоты в открытом доступе. Перед использованием проверьте настройки конфиденциальности в аккаунте.
Как обойти блокировку скриншотов в банковских приложениях?

Некоторые программы (например, 1С, СберБанк Онлайн или Netflix) блокируют стандартные способы создания скриншотов. В этом случае:

1. Используйте второй монитор или виртуальную машину — иногда блокировка срабатывает только на основном дисплее.

2. Попробуйте программный рендер через OBS Studio (запись экрана с последующим извлечением кадра).

3. Для Android/iOS подключите устройство к ПК и сделайте скриншот через ADB или QuickTime Player (на macOS).

5. Скриншоты на телевизорах и внешних мониторах

Если вам нужно сохранить изображение с Smart TV или внешнего монитора, подключённого к ПК, стандартные методы могут не сработать. Здесь потребуются специальные подходы.

Для телевизоров с Android TV, webOS (LG) или Tizen (Samsung):

  • 📺 На Samsung: удерживайте Volume Down + Power 2 секунды (скриншот сохраняется в Галерея).
  • 🎵 На LG: нажмите и удерживайте Home, затем выберите Capture Screen.
  • 🤖 На Android TV: используйте Google Assistant с командой «Сделай скриншот».

Для внешних мониторов, подключённых к ПК:

  • 🖥️ Если монитор дублирует экран, используйте стандартные способы (см. раздел 1).
  • 🔄 Если монитор расширяет экран, в Windows нажмите Win + Shift + S и выберите нужный дисплей.
  • 🎮 Для игровых мониторов с высокой частотой (144 Гц+) используйте NVIDIA Ansel или AMD ReLive — они фиксируют кадр без артефактов.

Если телевизор или монитор не поддерживает скриншоты, альтернативные способы:

  • 📷 Сфотографируйте экран на смартфон (отключите вспышку, чтобы избежать бликов).
  • 🎥 Подключите захват видео (например, Elgato Cam Link) и сделайте снимок через OBS.
  • 🖨️ Используйте HDMI-сплиттер с функцией записи (например, Magewell Pro Capture).
⚠️ Внимание: На телевизорах с DRM-защитой (например, при просмотре Netflix или Disney+) скриншоты могут получиться чёрными или с водяными знаками. Это ограничение на уровне прошивки.
💡

Для мониторов с разрешением 4K и выше используйте форматы .png или .tiff — они сохраняют качество без сжатия. .jpg может добавить артефакты при высоком разрешении.

6. Продвинутые техники: скриншоты с прокруткой, 3D и VR

Для специфических задач — захвата длинных веб-страниц, 3D-моделей или виртуальной реальности — потребуются специализированные инструменты.

Захват прокручиваемых страниц:

  • 📜 Full Page Screen Capture (расширение для Chrome) — сохраняет страницу целиком в PNG.
  • 🌐 FireShot — поддерживает захват с анимацией (GIF).
  • 📄 Paparazzi! (для macOS) — рендерит страницу как браузер WebKit.

Скриншоты в 3D-приложениях:

  • 🎮 NVIDIA Ansel — фиксирует кадр в играх с поддержкой технологии (например, The Witcher 3, Cyberpunk 2077).
  • 🖥️ Blender или 3ds Max — встроенный рендер в Viewport (клавиша F12).
  • 🔍 Fraps — захват кадров в OpenGL/DirectX-приложениях.

Виртуальная реальность (VR):

  • 🕶️ Oculus Debug Tool — сохраняет снимок из шлема Meta Quest по команде oculus-diagnostics --screenshot.
  • 🎮 SteamVR — нажмите System Button + Trigger на контроллере Valve Index.
  • 🖼️ OpenComposite — альтернативный рендер для Oculus Rift.

Для терминалов и консоли:

  • 🐧 В Linux используйте script + cat для сохранения текстового вывода:
script output.txt

ls -l


exit


Затем конвертируйте в изображение через:


text2image output.txt -o screenshot.png

  • 🖥️ В Windows (CMD/PowerShell) используйте ConEmu или Windows Terminal с функцией Export Text as Image.

7. Оптимизация и обработка скриншотов

Сделанный скриншот часто требует доработки: обрезки, цензуры конфиденциальных данных или сжатия для отправки. Рассмотрим лучшие инструменты для постобработки.

Редактирование:

  • 🎨 GIMP — бесплатный аналог Photoshop с поддержкой слоёв и масок.
  • ✏️ Paint.NET — лёгкий редактор для Windows с плагинами.
  • 🖌️ Pixelmator Pro — оптимизирован для macOS и M1/M2.

Сжатие без потерь:

  • 📦 TinyPNG — уменьшает вес .png на 50–70% без видимых артефактов.
  • 🖼️ ImageOptim — пакетная оптимизация для macOS/Windows.
  • 🌐 Squoosh (от Google) — онлайн-компрессор с настройками качества.

Цензура конфиденциальных данных:

  • 🔒 Blurring Tool в ShareX — автоматическое размытие выделенных областей.
  • 🖱️ Pixelate в GIMP — пикселизация чувствительной информации.
  • 📝 Redact (расширение для Chrome) — скрывает текст на веб-страницах перед сохранением.

Конвертация форматов:

  • 🔄 XnConvert — пакетное преобразование .png → .webp или .avif.
  • 🖼️ FFmpeg — конвертация через командную строку: 
    ffmpeg -i input.png -q:v 2 output.jpg
⚠️ Внимание: При сжатии скриншотов с текстом (например, код или документы) избегайте форматов с потерями (.jpg). Используйте .png или .webp в режиме lossless.
💡

Чтобы быстро удалить фон со скриншота, используйте Remove.bg или инструмент Background Eraser в Canva. Это полезно для создания презентаций или мемов.

FAQ: Частые вопросы о сохранении картинок с монитора

Можно ли сделать скриншот с блокированного сайта (например, Netflix)?

Да, но не стандартными способами. Используйте:

  1. Второй монитор: подключите дополнительный дисплей и сделайте скриншот с него.
  2. Запись экрана: OBS Studio или Camtasia могут обойти блокировку.
  3. Фотоаппарат: сфотографируйте экран (качество будет хуже).

Обратите внимание: распределение контента с Netflix, Disney+ и других стриминговых сервисов может нарушать пользовательское соглашение.

Как сохранить скриншот в формате .svg (векторный)?

Стандартные инструменты не поддерживают .svg, но есть обходные пути:

  • Используйте Inkscape для трассировки растра в вектор.
  • В браузере нажмите F12 (Инструменты разработчика), найдите элемент и скопируйте его как SVG.
  • Для диаграмм или графиков экспортируйте данные в Excel и сохраняйте как .svg.
Почему скриншоты получаются размытыми на мониторе 4K?

Это связано с масштабированием (DPI). Решения:

  • В Windows отключите масштабирование для приложения, в котором делаете скриншот (правый клик по .exe → Свойства → Совместимость → Отключить масштабирование).
  • Используйте Win + Shift + S (в Windows 11 поддерживает HiDPI).
  • В macOS установите разрешение экрана в Настройки → Мониторы → Дополнительно на Low Resolution (временное решение).
Как автоматизировать создание скриншотов (например, каждый час)?

Можно ли восстановить случайно удалённый скриншот?

Да, если файл не был перезаписан:

  • На macOS попробуйте Time Machine
