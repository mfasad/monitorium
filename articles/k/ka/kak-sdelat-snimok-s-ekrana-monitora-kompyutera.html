Как сделать снимок с экрана монитора компьютера: все способы

Создание скриншота экрана монитора на компьютере требуется в 80% случаев для фиксации ошибок, сохранения важной информации или демонстрации интерфейса. Если при нажатии PrtSc ничего не происходит, проблема может крыться в отключенной функции буфера обмена, конфликте клавиатурных раскладок или отсутствии прав на сохранение файлов. На Windows 10/11 и macOS алгоритмы отличаются — ниже разберём все рабочие методы с учётом нюансов конкретных ОС и железных конфигураций.

Первое, что нужно проверить: работает ли клавиша Print Screen (может быть подписана как PrtScn или PrtSc). На некоторых ноутбуках (Lenovo, HP, Dell) её функционал дублируется с Fn, поэтому требуется одновременное нажатие. Если клавиатура механическая (например, Razer BlackWidow или Logitech G Pro), убедитесь, что драйверы клавиш установлены корректно — иначе система просто не распознаёт команду.

1. Стандартные горячие клавиши для скриншотов

Самый быстрый способ сделать снимок экрана — использовать комбинации клавиш. Они работают без установки дополнительного ПО, но результат зависит от версии Windows и настроек системы. На macOS алгоритм иной: здесь за скриншоты отвечает отдельное меню, активируемое клавишами.

На Windows 10/11 доступны следующие комбинации:

  • 🖥️ PrtSc — копирует весь экран в буфер обмена (требуется вставить в Paint или Word для сохранения).
  • 🖼️ Win + PrtSc — автоматически сохраняет скриншот в папку C:\Users\Имя_пользователя\Pictures\Screenshots.
  • 🎯 Alt + PrtSc — фиксирует только активное окно (например, браузер или игру).
  • 🖱️ Win + Shift + S — вызывает инструмент "Ножницы" для выделения области экрана.

На macOS (включая Ventura и Sonoma):

  • 🍎 Command + Shift + 3 — сохраняет скриншот всего экрана на рабочий стол.
  • 📱 Command + Shift + 4 — позволяет выбрать область для снимка.
  • 🖲️ Command + Shift + 4 + Пробел — фиксирует конкретное окно (появляется иконка камеры).
⚠️ Внимание: На некоторых ультрабуках (MacBook Air M1/M2, ASUS ZenBook) клавиша PrtSc может отсутствовать. В этом случае используйте экранную клавиатуру (Win + Ctrl + O на Windows) или назначьте сочетание клавиш вручную через Параметры → Устройства → Клавиатура.

2. Встроенный инструмент "Ножницы" (Windows)

Утилита "Ножницы" (Snipping Tool) входит в стандартный пакет Windows и позволяет делать скриншоты с задержкой, выделять произвольные области и добавлять разметку. Чтобы открыть её:

  1. Нажмите Win + S и введите Ножницы.
  2. Выберите режим: Произвольная форма, Прямоугольник, Окно или Весь экран.
  3. Для отложенного снимка нажмите "Задержка" (до 5 секунд).
  4. Сохраните файл в формате PNG, JPEG или GIF.

В Windows 11 инструмент обновлён: теперь он поддерживает запись экрана (до 30 секунд) и интеграцию с OneDrive для автоматической загрузки. Если утилита не открывается, проверьте её наличие через Параметры → Приложения → Дополнительные компоненты и установите при необходимости.

3. Скриншоты в играх и полноэкранных приложениях

Если вы пытаетесь сделать снимок экрана в игре (CS2, GTA V, Cyberpunk 2077) или полноэкранном приложении (Photoshop, AutoCAD), стандартные методы могут не сработать. Причины:

  • 🎮 Игровой движок блокирует системные комбинации (например, Unreal Engine 5 или Source 2).
  • 🖥️ Полноэкранный режим DirectX/Vulkan перехватывает клавиатурные события.
  • 🔒 Античит (EAC, BattleEye) запрещает скриншоты для защиты от читерства.

Решения:

  • 📷 Используйте встроенные игровые функции: в Steam — F12, в Origin — F1, в Epic Games — Alt + F1.
  • 🎛️ Переключитесь в оконный режим (Alt + Enter) и используйте PrtSc.
  • 🖥️ Для NVIDIA пользователей: активируйте Instant Replay в GeForce Experience (Alt + F10 для сохранения последних 5 минут геймплея).
⚠️ Внимание: Скриншоты из игр с DRM-защитой (Denuvo) могут сохраняться в зашифрованном виде. Чтобы их открыть, используйте официальные инструменты издателя (например, Ubisoft Connect для игр Ubisoft).

4. Программы для расширенных скриншотов

Если встроенных инструментов недостаточно, установите специализированное ПО. Ниже — сравнение популярных программ с учётом их возможностей и системных требований:

Программа Поддержка ОС Особенности Минусы
Lightshot Windows, macOS Быстрое выделение области, редактор, загрузка в облако Реклама в бесплатной версии
ShareX Windows Захват прокручиваемых страниц, OCR, более 50 инструментов Сложный интерфейс для новичков
Snagit Windows, macOS Запись видео, шаблоны аннотаций, интеграция с Slack Платная (от 50$)
Greenshot Windows Поддержка плагинов, экспорт в PDF/Excel Нет официальной поддержки macOS

Для профессиональной работы (например, создание мануалов или обучающих материалов) рекомендуется Snagit — он позволяет добавлять стрелки, текст и даже записывать видео с экрана. Если нужна бесплатная альтернатива, ShareX покрывает 90% задач, включая захват прокручиваемых веб-страниц (полезно для фиксации длинных чатов или документов).

Как сделать скриншот прокручиваемой страницы в ShareX

1. Установите программу с официального сайта.

2. Выберите режим "Capture → Scrolling Capture".

3. Наведите курсор на окно браузера и нажмите левую кнопку мыши.

4. Дождитесь автоматической прокрутки и сохранения изображения в PNG.

5. Скриншоты на Linux и других ОС

На дистрибутивах Linux (Ubuntu, Fedora, Arch) алгоритм создания снимков экрана зависит от графической оболочки (GNOME, KDE, XFCE). Общие методы:

  • 🐧 PrtSc — сохраняет скриншот в ~/Pictures (для GNOME).
  • 🖼️ Shift + PrtSc — выделение области (аналог Windows).
  • 📌 Alt + PrtSc — фиксирует активное окно.

Для расширенных возможностей установите Flameshot:

sudo apt install flameshot  # Для Debian/Ubuntu

sudo dnf install flameshot  # Для Fedora

После установки запустите из терминала командой flameshot gui — появится панель инструментов для выделения области и редактирования.

На ChromeOS (ноутбуки Chromebook) используйте:

  • 🔲 Ctrl + []]] (где ]]] — клавиша переключения окон) — скриншот всего экрана.
  • 🔳 Ctrl + Shift + []]] — выделение области.

6. Как редактировать и оптимизировать скриншоты

Сделанный снимок экрана часто требует доработки: обрезка, добавление текста или сжатие для отправки по почте. Инструменты для постобработки:

  • 🎨 Paint.NET — бесплатный аналог Photoshop с поддержкой слоёв.
  • 📏 GIMP — открытое ПО для профессионального редактирования.
  • 🗜️ TinyPNG — онлайн-сервис для сжатия PNG/JPEG без потери качества.
  • 🔍 IrfanView — лёгкая утилита для пакетной обработки (изменение размера, конвертация форматов).

Для быстрой обрезки:

  1. Откройте скриншот в Paint (Win + R → mspaint).
  2. Выберите инструмент "Выделение" и обведите нужную область.
  3. Нажмите "Обрезать" (или Ctrl + Shift + X).
  4. Сохраните как PNG (для прозрачности) или JPEG (для меньшего веса).
💡

Чтобы убрать лишние элементы (например, курсор мыши) с готового скриншота, используйте инструмент "Штамп" в Photoshop или "Исправление" в GIMP. Для автоматизации подойдёт плагин Topaz Gigapixel AI (увеличивает разрешение без потерь).

7. Автоматизация скриншотов: расписание и горячие клавиши

Если вам нужно регулярно фиксировать экран (например, для мониторинга сервера или записи процесса работы), настройте автоматизацию:

  • 🕒 По расписанию: Используйте Task Scheduler (Windows) или cron (Linux) для запуска скрипта на Python с библиотекой Pillow.
  • 🔧 Кастомные горячие клавиши: В AutoHotkey пропишите скрипт:
#IfWinActive ahk_exe chrome.exe  ; Только для окна Chrome

F9::  ; Назначить клавишу F9


{


Send {PrintScreen}


Sleep 500


Run "mspaint"


WinWait "Безымянный - Paint"


Send ^v  ; Вставить


Sleep 1000


Send !fs  ; Сохранить как...


WinWait "Сохранить как"


Send "screenshot_{%A_YYYY%}{%A_MM%}{%A_DD%}_{%A_Hour%}{%A_Min%}{%A_Sec%}.png{Enter}"


}

Для macOS настройте автоматическое сохранение через Automator:

  1. Откройте Automator и создайте новый "Быстрое действие".
  2. Добавьте блок "Сделать снимок экрана".
  3. Назначьте сочетание клавиш в Системные настройки → Клавиатура → Сочетания клавиш.
⚠️ Внимание: При автоматизации скриншотов в фоновом режиме (например, для удалённого рабочего стола) может фиксироваться чёрный экран. Чтобы избежать этого, используйте параметр /silent в утилитах вроде NirCmd или настройте виртуальный дисплей через Virtual Display Driver.
💡

Для профессиональной работы с скриншотами (например, для блога или документации) комбинируйте инструменты: ShareX для захвата + GIMP для редактирования + TinyPNG для оптимизации. Это сократит время обработки на 40% по сравнению со стандартными средствами Windows.

FAQ: Частые вопросы о скриншотах

Почему скриншот сохраняется чёрным?

Это типичная проблема при использовании удаленного рабочего стола (RDP) или виртуальных машин (VirtualBox, VMware). Решения:

  • Отключите аппаратное ускорение в настройках RDP (Подключение → Локальные ресурсы → Параметры производительности).
  • Используйте альтернативные инструменты вроде PicPick, которые корректно работают с виртуальными дисплеями.
  • Для VMware установите VMware Tools и включите опцию "Enable 3D acceleration".
Как сделать скриншот с задержкой?

Задержка нужна, чтобы успеть открыть меню или подготовить экран. Способы:

  • В "Ножницах" (Windows) нажмите "Задержка" и выберите время (до 5 секунд).
  • В macOS используйте Terminal:
screencapture -T 10 screenshot.png  # Задержка 10 секунд
Можно ли сделать скриншот с нескольких мониторов?

Да, но метод зависит от конфигурации:

  • На Windows комбинация PrtSc фиксирует все подключённые экраны как одно изображение.
  • На macOS по умолчанию сохраняется только основной дисплей. Чтобы захватить все, используйте:
screencapture -x -R$x1,y1,x2,y2 multi_monitors.png

Где x1,y1,x2,y2 — координаты области (узнать можно через Rectangle в Utilities).

Как отключить звук затвора при скриншоте на Mac?

Звук отключается через Terminal:

defaults write com.apple.screencapture disable_shadow -bool true

defaults write com.apple.screencapture type png


killall SystemUIServer

После выполнения перезагрузите компьютер. Чтобы вернуть звук, замените true на false.

Где хранятся скриншоты, сделанные через Steam?

По умолчанию они сохраняются в:

C:\Program Files (x86)\Steam\userdata\<Ваш_ID>\760\remote\<ID_игры>\screenshots

Чтобы найти папку:

  1. Откройте Steam и перейдите в "Просмотр → Скриншоты".
  2. Нажмите "Показать на диске" — откроется папка с файлами в формате JPEG.
