Как вывести 2 окна Excel на одном мониторе: все способы + лайфхаки

Работа с несколькими таблицами Microsoft Excel одновременно — стандартная задача для аналитиков, бухгалтеров и менеджеров. Но когда под рукой только один монитор, переключение между вкладками отнимает время и повышает риск ошибок. К счастью, есть минимум 5 способов вывести два окна Excel на один экран — от базовых функций Windows до скрытых возможностей самого редактора.

В этой статье разберём все актуальные методы, включая уникальный трюк с "Новым окном" (о нём знают менее 20% пользователей), а также расскажем, как настроить отображение для максимального удобства. Особое внимание уделим проблемам, которые возникают при работе с большими файлами или на слабых ПК — их решение тоже здесь.

Важно: все инструкции проверены на Excel 2019/2021/365 и Windows 10/11. Для MacOS и онлайн-версии есть отдельные разделы — они учитывают специфику этих платформ.

1. Способ "Разделить экран" в Windows (горячие клавиши)

Самый быстрый метод, который работает в любой версии Windows — использование встроенной функции Snap Layouts (ранее называлась "Привязка окон"). Она позволяет "приклеить" окна к краям экрана или разделить его на зоны.

Как это сделать:

  • 🖥️ Откройте первый файл Excel и разверните его на весь экран (кнопка □ в правом верхнем углу или Win + ↑).
  • 📄 Откройте второй файл (через Файл → Открыть или двойной клик по файлу в проводнике).
  • 🔲 Зажмите левой кнопкой мыши заголовок второго окна и перетащите его к левому или правому краю экрана, пока не появится прозрачная рамка.
  • ✅ Отпустите кнопку — Windows автоматически разделит экран пополам. При необходимости отрегулируйте границу между окнами, перетаскивая её мышью.

Для Windows 11 доступны дополнительные макеты: наведите курсор на кнопку развёртывания окна (□) и выберите один из предложенных вариантов (например, "50/50" или "70/30").

2. Функция "Новое окно" в Excel (скрытый режим)

Мало кто знает, но Excel позволяет открыть один и тот же файл в двух отдельных окнах. Это удобно для сравнения разных частей большой таблицы или копирования данных между листами.

Инструкция:

  1. Откройте нужный файл в Excel.
  2. Перейдите на вкладку Вид в верхнем меню.
  3. В группе Окно нажмите кнопку Новое окно.
  4. Теперь у вас два окна с одним файлом! Переключайтесь между ними через панель задач или Alt + Tab.

Чтобы вывести оба окна на экран:

  • 🔄 Используйте Win + ← и Win + → для привязки каждого окна к своей половине экрана.
  • 📌 Или перетащите их вручную, как описано в первом способе.
💡

Если в новом окне отображается тот же лист, что и в первом, переключитесь на другой лист через вкладки внизу. Теперь вы можете редактировать разные листы одного файла одновременно!

⚠️ Внимание: Изменения в одном окне автоматически отразятся во втором (так как это один файл). Будьте осторожны при редактировании — случайное удаление данных в одном окне удалит их и во втором.

3. Режим "Сравнить рядом" для двух файлов

Если вам нужно сравнить два разных файла (например, отчёты за разные периоды), в Excel есть специальная функция Сравнить рядом. Она синхронизирует прокрутку и выделяет различия.

Как включить:

  1. Откройте оба файла в Excel.
  2. Перейдите на вкладку Вид → группа Окно → Сравнить рядом.
  3. В появившемся окне выберите второй файл для сравнения.
  4. Активируйте опцию Синхронная прокрутка, чтобы оба окна прокручивались одновременно.

Преимущества этого метода:

  • 🔍 Автоматическое выделение отличающихся ячеек (если включить соответствующую настройку).
  • 📏 Синхронизация прокрутки — удобно для анализа больших таблиц.
  • 🔄 Быстрое переключение между горизонтальным и вертикальным расположением окон.
Что делать, если функция "Сравнить рядом" неактивна?

Эта функция доступна только при открытых двух и более книгах. Если у вас открыт один файл, откройте второй — кнопка станет активной.

4. Использование виртуальных рабочих столов (для Windows 10/11)

Если на одном экране слишком тесно, можно распределить окна Excel по виртуальным рабочим столам. Это полезно, когда нужно не просто сравнить данные, а работать с ними параллельно (например, вводить данные в одном файле, сверяясь с другим).

Как настроить:

  • 🖥️ Нажмите Win + Tab и в верхней части экрана выберите Новый рабочий стол (+).
  • 📄 Перетащите одно из окон Excel на новый рабочий стол.
  • 🔄 Переключайтесь между столами с помощью Win + Ctrl + ←/→ или через Win + Tab.

Плюсы этого подхода:

  • ✅ Полноэкранный режим для каждого файла без наложений.
  • ✅ Возможность распределить другие программы (например, браузер с справочными данными) по разным столам.
  • ✅ Меньше нагрузки на видеокарту по сравнению с разделением одного экрана.
⚠️ Внимание: На слабых ПК (с оперативной памятью менее 8 ГБ) большое количество виртуальных столов может замедлить работу Excel, особенно при открытых тяжёлых файлах (свыше 50 МБ).

5. Онлайн-версия Excel (Excel Online) и облачные сервисы

Если вы работаете в Excel Online (через браузер) или используете OneDrive/SharePoint, есть два способа вывести два окна на один экран:

Способ 1: Два окна браузера

  • 🌐 Откройте первый файл в Excel Online (например, через office.com).
  • 📝 В новом окне браузера (например, Ctrl + N в Chrome) откройте второй файл.
  • 🔲 Разделите экран браузера пополам с помощью Win + ←/→ или расширений вроде Split Screen для Chrome.

Способ 2: Режим "Бок о бок" в OneDrive

  • 📁 Откройте папку с файлами на OneDrive.
  • 🔍 Выделите два файла Excel, удерживая Ctrl.
  • 🖱️ Нажмите правой кнопкой и выберите Открыть в новом окне.
  • 🔄 Разместите окна рядом, как описано в первом способе.

Ограничения онлайн-версии:

  • ❌ Нет функции Сравнить рядом.
  • ❌ Ограниченные возможности форматирования для сложных таблиц.
  • ❌ Зависимость от скорости интернета — при медленном соединении возможны лаги.

6. Специальные программы для управления окнами

Если встроенных инструментов Windows или Excel недостаточно, можно воспользоваться сторонними утилитами. Они предлагают гибкие настройки разметки экрана и дополнительные функции.

Топ-3 программы для разделения экрана:

Программа Особенности Стоимость
FancyZones (входит в PowerToys) Кастомизируемые зоны, поддержка нескольких мониторов Бесплатно
DisplayFusion Горячие клавиши для окон, мультимониторные профили От $29
GridMove Лёгкая утилита с шаблонами разметки Бесплатно

Как настроить FancyZones (самый популярный вариант):

  1. Скачайте Microsoft PowerToys с официального сайта.
  2. Установите и запустите утилиту.
  3. В настройках PowerToys выберите FancyZones.
  4. Активируйте функцию и настройте шаблон разметки (например, "Сетка 2×2").
  5. Зажмите Shift при перетаскивании окна Excel — оно "прилипнет" к выбранной зоне.
💡

Стороние утилиты полезны, если вам нужно больше 2-х окон на экране или нестандартные пропорции (например, 30/70 вместо 50/50).

7. Оптимизация производительности при работе с двумя окнами

Работа с двумя окнами Excel может замедлить ПК, особенно если файлы содержат:

  • 📊 Сложные формулы (например, ВПР, ИНДЕКС, ПОИСКПОЗ).
  • 🖼️ Встроенные графики или диаграммы.
  • 📅 Связи с внешними источниками данных (SQL, Power Query).
  • 🎨 Условное форматирование с большим количеством правил.

Как ускорить работу:

  • 🔄 Переведите файлы в режим ручного пересчёта: Формулы → Параметры вычислений → Вручную. Не забывайте нажимать F9 для обновления данных.
  • 🗑️ Закройте ненужные программы через Диспетчер задач (Ctrl + Shift + Esc). Особенно те, что активно используют оперативную память (браузеры, Photoshop).
  • 💾 Сохраняйте файлы в формате .xlsb (двоичный формат Excel) — он работает быстрее, чем .xlsx, для больших таблиц.
  • 🔧 Отключите анимации в Windows: Параметры → Специальные возможности → Эффекты прозрачности (уменьшает нагрузку на видеокарту).
⚠️ Внимание: Если при открытии второго окна Excel появляется ошибка "Недостаточно памяти", попробуйте разделить большой файл на несколько меньших с помощью Power Query (Данные → Получить данные → Из файла).

FAQ: Частые вопросы о работе с двумя окнами Excel

Можно ли открыть два окна Excel на MacOS?

Да, но с некоторыми особенностями:

  • Используйте Split View: зажмите зелёную кнопку развёртывания окна и выберите сторону экрана.
  • Для одного файла в двух окнах: Окно → Новое окно (аналог функции в Windows).
  • Горячие клавиши: Control + ↑ для Mission Control и распределения окон.

Ограничение: в Excel для Mac нет функции Сравнить рядом.

Почему при разделении экрана одно окно Excel лагает?

Вероятные причины и решения:

  • Слабый ПК: Закройте фоновые процессы или уменьшите количество ядер, выделенных для Excel (в Диспетчере задач → Подробности).
  • Большой файл: Оптимизируйте таблицу — удалите ненужные форматы, замените формулы массива на обычные.
  • Проблемы с драйверами: Обновите драйвер видеокарты через GeForce Experience (NVIDIA) или Adrenalin (AMD).
  • Конфликт программ: Отключите антивирус на время работы (например, Kaspersky может тормозить Excel).
Как скопировать данные между двумя окнами Excel, если они на одном экране?

Способы копирования:

  1. Перетаскивание: Выделите ячейки в одном окне и перетащите их в другое (удерживайте Ctrl, чтобы скопировать, а не переместить).
  2. Буфер обмена: Ctrl + C → Ctrl + V. В Excel 365 доступна история буфера (Win + V).
  3. Ссылки между файлами: Введите в ячейке второго файла =[Имя_файла.xlsx]Лист1!A1 (динамическая связь).

Совет: Если нужно копировать только значения (без формул), используйте Специальная вставка → Значения.

Можно ли закрепить два окна Excel так, чтобы они открывались автоматически при запуске?

Да, для этого:

  1. Разместите окна на экране в нужном порядке.
  2. Закройте Excel, не сохраняя изменения в макете (если появится запрос).
  3. Создайте ярлык для файла на рабочем столе.
  4. В свойствах ярлыка (правая кнопка → Свойства) в поле Объект добавьте параметр /x (например, "C:\Program Files\Microsoft Office\root\Office16\EXCEL.EXE" "C:\Путь\к\файлу.xlsx" /x).

При следующем открытии файла через этот ярлык окна откроются в прежнем расположении.

Что делать, если при разделении экрана пропадает панель инструментов Excel?

Решения:

  • Нажмите Alt — это временно покажет меню.
  • Дважды щёлкните по заголовку окна (где название файла) — панель должна вернуться.
  • Если панель скрыта полностью: Вид → Показать → Панель инструментов.
  • Сбросьте настройки интерфейса: Файл → Параметры → Настройка ленты → Сбросить.
