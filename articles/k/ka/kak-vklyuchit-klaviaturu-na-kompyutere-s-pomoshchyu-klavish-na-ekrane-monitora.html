Экранная клавиатура на компьютере: как запустить с монитора за 30 секунд

Физическая клавиатура перестала реагировать на нажатия, а подключить запасную нет возможности — из-за отсутствия свободных USB-портов, поломки разъёма или неисправности самого устройства? В этом случае не обязательно прерывать работу: в Windows, macOS и Linux предусмотрена виртуальная клавиатура на экране, которую можно открыть через мышь, тачпад или сенсорный дисплей (при его наличии). Такой инструмент поможет временно заменить аппаратную клавиатуру, пока не будет устранена причина сбоя — будь то программный конфликт, механическое повреждение или ошибка подключения.

Эта статья поможет разобраться, как включить виртуальную клавиатуру на экране монитора в Windows 10/11, macOS и Linux, даже если у вас нет доступа к физическим клавишам. Мы рассмотрим не только стандартные методы через меню «Пуск», но и альтернативные способы — от горячих клавиш (если часть кнопок ещё работает) до запуска через командную строку. Отдельное внимание уделим нюансам для сенсорных мониторов и устройств с Touch Bar (как у некоторых моделей MacBook Pro).

Важно: если ваш монитор не поддерживает сенсорный ввод, вам потребуется рабочая мышь или тачпад. В противном случае для активации экранной клавиатуры может понадобиться подключение второго устройства (например, планшета в режиме USB-мыши).

1. Стандартный способ включения экранной клавиатуры в Windows

В большинстве случаев виртуальную клавиатуру в Windows 10/11 можно запустить через графический интерфейс. Этот метод подходит, если у вас работает мышь или сенсорный экран монитора. Вот пошаговая инструкция:

  • 🖱️ Нажмите на кнопку Пуск (или клавишу Win, если она работает) и выберите Параметры (значок шестерёнки).
  • 📱 Перейдите в раздел Специальные возможности → Клавиатура.
  • 🔧 Включите переключатель Использовать экранную клавиатуру. Система автоматически откроет её в нижней части экрана.
  • 🔄 Если клавиатура не появилась, проверьте раздел Панель задач — иногда она сворачивается в трей.

Альтернативный путь (если Параметры недоступны):

  1. Откройте Панель управления через поиск в меню Пуск.
  2. Выберите Специальные возможности → Центр специальных возможностей.
  3. Нажмите Включить экранную клавиатуру.

Примечание для Windows 11: В последних сборках путь немного изменился. Теперь экранную клавиатуру можно найти через Пуск → Все приложения → Специальные возможности → Экранная клавиатура. Также её можно закрепить на панели задач для быстрого доступа.

2. Горячие клавиши для быстрого запуска (если часть кнопок работает)

Если у вас зависла только часть клавиш (например, буквенная зона), но работают функциональные клавиши или комбинации с Win, можно воспользоваться горячими клавишами:

  • 🔑 Win + Ctrl + O — включает/выключает экранную клавиатуру в Windows 10/11 (работает не на всех сборках).
  • 🔑 Win + R → введите osk → нажмите Enter. Это универсальный способ для всех версий Windows.
  • 🔑 Win + U → откроется Центр специальных возможностей, где можно включить виртуальную клавиатуру.

Для macOS (если у вас Mac с подключённым монитором):

  • 🍎 Нажмите Option (Alt) + Command + F5 → выберите Клавиатура → включите Экранная клавиатура.
  • 🖥️ В Linux (Ubuntu/GNOME) используйте Super (Win) + Alt + S или установите пакет onboard через терминал.
💡

Если комбинация Win + R не срабатывает, попробуйте запустить командную строку через меню Выполнить (доступно по клику правой кнопкой на Пуск) и ввести osk вручную.

3. Запуск через командную строку или PowerShell

Если графический интерфейс недоступен (например, из-за сбоя explorer.exe), экранную клавиатуру можно активировать через консоль. Этот метод подходит для опытных пользователей:

  1. Откройте Командную строку или PowerShell:
    • Через меню Выполнить (Win + R → введите cmd).
    • Или через Диспетчер задач (Ctrl + Shift + Esc → Файл → Запустить новую задачу → введите cmd).
  2. Введите команду: 
    osk.exe

    и нажмите Enter.

    3. Для Linux (дистрибутивы на базе GNOME/KDE):

    sudo apt install onboard  # Установка (для Debian/Ubuntu)

    onboard                    # Запуск

    Если командная строка тоже недоступна, попробуйте запустить её через Безопасный режим с поддержкой командной строки (выбирается при загрузке Windows после нескольких принудительных перезагрузок).

    Что делать, если cmd не открывается?

    Если ни один из способов не срабатывает, попробуйте подключить компьютер к другому устройству по RDP (Удалённый рабочий стол) с другого ПК или планшета. Для этого нужно, чтобы на целевом компьютере был включён доступ по RDP (проверьте в Параметры → Система → Удалённый рабочий стол).

    4. Экранная клавиатура на сенсорных мониторах

    Если ваш монитор поддерживает сенсорный ввод (например, модели Dell P2418HT, HP EliteDisplay E243 или Microsoft Surface Studio), процесс активации клавиатуры упрощается. В этом случае:

    • 👆 Коснитесь экрана в области панели задач (внизу) и удерживайте палец 2 секунды — появится контекстное меню.
    • 📝 Выберите Показать экранную клавиатуру (в Windows 10/11 этот пункт появляется автоматически при обнаружении сенсора).
    • 🔄 Если клавиатура не появляется, проверьте настройки сенсорного ввода в Параметры → Устройства → Сенсорная панель.

    Для macOS на мониторах с сенсорной панелью (например, Apple Pro Display XDR):

    • 🍏 Откройте Системные настройки → Универсальный доступ → Клавиатура.
    • 🔘 Активируйте опцию Включить клавиатуру для сенсорных устройств.

    Важно: На некоторых сенсорных мониторах под Windows может потребоваться установка дополнительных драйверов от производителя (например, Dell Touch Driver или HP Display Assistant). Без них система не распознаёт касания как ввод с клавиатуры.

    5. Альтернативные программы для виртуальной клавиатуры

    Если встроенная экранная клавиатура не устраивает (например, из-за отсутствия поддержки раскладок или медленной работы), можно использовать сторонние утилиты:

    Программа ОС Особенности Ссылка (поиск)
    On-Screen Keyboard Portable Windows Портативная версия, не требует установки, поддержка макросов osk-portable
    Click-N-Type Windows Оптимизирована для людей с ограниченными возможностями, голосовой ввод click-n-type
    Florence Linux Лёгкая, с настраиваемым интерфейсом, поддержка мультитача florence virtual keyboard
    KeyStrokes macOS Интеграция с Touch Bar, жесты мыши keystrokes mac

    Преимущества сторонних решений:

    • 🎨 Настраиваемый дизайн (размер кнопок, цвета, прозрачность).
    • 🌍 Поддержка редких языковых раскладок (например, азербайджанская или грузинская).
    • 🔄 Автоматические замены текста и горячие клавиши для часто используемых фраз.
    💡

    Стороние клавиатуры могут конфликтовать со встроенными инструментами специальных возможностей. Перед установкой создайте точку восстановления системы (Win + R → rstrui).

    6. Решение проблем: клавиатура не включается

    Если экранная клавиатура не запускается, причины могут быть следующими:

    ⚠️ Внимание: На некоторых корпоративных ПК доступ к экранной клавиатуре может быть заблокирован через групповую политику (gpedit.msc). В этом случае обратитесь к администратору сети.
    • 🔌 Отсутствуют драйверы: Проверьте Диспетчер устройств на наличие восклицательных знаков в разделе Клавиатуры или Устройства HID.
    • 🛡️ Блокировка антивирусом: Временное отключите защиту (например, Kaspersky или Avast) и попробуйте снова.
    • 🔄 Сбой системы: Запустите проверку целостности файлов: 
      sfc /scannow
    • 🖥️ Проблемы с монитором: Если у вас многомониторная система, клавиатура может открываться на другом экране. Перетащите её мышью.

    Для Linux (если onboard не запускается):

    • 🐧 Проверьте зависимости: 
      sudo apt --fix-broken install
    • 🔧 Переустановите пакет: 
      sudo apt purge onboard && sudo apt install onboard

    7. Использование экранной клавиатуры для специальных задач

    Виртуальная клавиатура полезна не только при поломке физической. Её можно использовать для:

    • 🔐 Ввода паролей на публичных ПК (защита от кейлоггеров).
    • 🎮 Управления играми с поддержкой клавиатуры через сенсорный экран.
    • 📱 Тестирования мобильных версий сайтов в браузере (эмуляция тач-ввода).
    • 🖌️ Рисования символами в графических редакторах (например, в ASCII-арте).

Пример: в Photoshop можно использовать экранную клавиатуру для быстрого ввода значений параметров (например, размера кисти) без отрыва руки от графического планшета.

Для разработчиков полезно знать, что виртуальная клавиатура может конфликтовать с отладчиками (например, Visual Studio). В этом случае её лучше отключать через:

taskkill /f /im osk.exe

Часто задаваемые вопросы

Можно ли настроить автозапуск экранной клавиатуры при старте Windows?

Да. Для этого:

  1. Нажмите Win + R → введите shell:startup.
  2. Создайте ярлык для файла osk.exe (находится в C:\Windows\System32\).

Теперь клавиатура будет запускаться автоматически при входе в систему.

Почему экранная клавиатура тормозит?

Причины:

  • Нехватка оперативной памяти (закройте лишние программы).
  • Высокое разрешение экрана (уменьшите масштаб в настройках клавиатуры).
  • Конфликт с драйверами видеокарты (обновите их через GeForce Experience или AMD Adrenalin).
Как включить экранную клавиатуру на планшете с Windows?

На устройствах вроде Microsoft Surface она активируется автоматически при отключении физической клавиатуры. Если этого не произошло:

  1. Проведите пальцем от правого края экрана → Параметры → Изменение параметров компьютера.
  2. Выберите Специальные возможности → Клавиатура.
Работает ли экранная клавиатура в BIOS?

Нет. BIOS/UEFI не поддерживает виртуальные устройства ввода. Для навигации по меню BIOS при поломке клавиатуры потребуется:

  • Подключить USB-клавиатуру к другому порту.
  • Использовать PS/2-клавиатуру (если есть разъём).
  • Обновить BIOS с поддержкой USB-legacy (если проблема в этом).
⚠️ Внимание: Некоторые материнские платы (например, ASUS ROG или MSI MEG) позволяют управлять BIOS через мышь, но это зависит от версии прошивки.
Можно ли использовать экранную клавиатуру для управления играми?

Технически да, но с оговорками:

  • ✅ Поддерживаются игры с управлением через клавиши (например, Minecraft, Terraria).
  • ❌ Не подойдёт для шутеров (требуется быстрое нажатие клавиш).
  • 🎮 Для эмуляции джойстика лучше использовать vJoy + AntiMicro.
