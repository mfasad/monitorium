Как сделать скриншот только одного монитора: все способы для Windows, macOS и Linux

Работа с несколькими мониторами упрощает многозадачность, но стандартные инструменты создания скриншотов часто захватывают все экраны сразу. Это создаёт неудобства, когда нужно сохранить только рабочую область на основном дисплее или отправить коллеге снимок с дополнительного монитора без лишней информации. К счастью, существуют как встроенные решения в операционных системах, так и специализированные утилиты для точного контроля над областью захвата.

В этой статье мы разберём 7 проверенных способов, как сделать скриншот только одного монитора — от стандартных горячих клавиш до профессиональных программ с расширенными настройками. Вы узнаете, как обойти ограничения Windows 10/11, macOS и Linux, а также какие инструменты помогут автоматизировать процесс для регулярного использования. Особое внимание уделим скрытым функциям, о которых не пишут в официальной документации, но которые экономят минуты ежедневной работы.

Стандартные горячие клавиши: почему они не работают с несколькими мониторами

Большинство пользователей привыкли использовать комбинации вроде PrtScn (Print Screen) или Win + PrtScn для создания скриншотов. Однако в многомониторной конфигурации эти клавиши по умолчанию захватывают всю виртуальную область экрана, объединяя все дисплеи в один длинный снимок. Это происходит потому, что операционная система воспринимает несколько мониторов как единое рабочее пространство с расширенным разрешением.

Например, если у вас подключены два монитора с разрешением 1920×1080 каждый, то стандартный скриншот будет иметь размер 3840×1080 пикселей — даже если вам нужен только левый экран. Такое поведение заложено в архитектуре графических подсистем Windows и macOS, где приоритетом является сохранение контекста всех активных дисплеев.

  • 🖥️ PrtScn — копирует в буфер обмена изображение всех мониторов как единого целого.
  • 📋 Alt + PrtScn — захватывает только активное окно, но не отдельный монитор.
  • 💾 Win + PrtScn — сохраняет скриншот всех экранов в папку Снимки экрана, но опять же без разделения.
⚠️ Внимание: В некоторых сборках Windows 11 после обновления 2023 года появился баг, при котором Win + Shift + S захватывает только основной монитор, игнорируя дополнительные. Это не документированное поведение — проверьте его на своей системе перед использованием.

Единственное исключение — инструмент Ножницы (Snipping Tool), который позволяет вручную выделить область на конкретном экране. Однако это требует точности и не подходит для автоматизации. Далее мы рассмотрим более эффективные методы.

Способ 1: Использование Snipping Tool (Ножницы) в Windows

Встроенная утилита Snipping Tool (или Ножницы в русскоязычной версии) — самый простой способ сделать скриншот одного монитора без сторонних программ. Она доступна во всех версиях Windows, начиная с Vista, и позволяет выделять произвольные области экрана.

Чтобы воспользоваться ею:

  1. Откройте меню Пуск и введите «Ножницы» (или Snipping Tool).
  2. Запустите приложение и нажмите кнопку Создать.
  3. В выпадающем меню выберите режим Прямоугольная область.
  4. Кликните по любому месту на целевом мониторе и растяните рамку на весь экран (или выделите нужную область).
  5. Сохраните результат через меню Файл → Сохранить как.

Преимущество этого метода — точный контроль над областью захвата. Например, если на втором мониторе открыт браузер с конфиденциальными данными, вы можете выделить только рабочую область на первом экране, не захватывая лишнего. Однако у Snipping Tool есть и недостатки:

  • ⏱️ Требует ручного выделения каждый раз — не подходит для серии скриншотов.
  • 🎨 Нет возможности настроить горячие клавиши для быстрого вызова.
  • 🖼️ Качество снимка зависит от разрешения экрана (при масштабировании возможны артефакты).

Способ 2: Горячие клавиши Windows + Shift + S (обновлённый инструмент)

Начиная с Windows 10 версии 1809 (октябрь 2018 года), Microsoft добавила удобную комбинацию клавиш Win + Shift + S, которая вызывает улучшенную версию инструмента захвата. В отличие от классических Ножниц, этот метод позволяет:

  • 📸 Выбирать между прямоугольной, произвольной и полноэкранной областью захвата.
  • 🔄 Сохранять скриншоты в буфер обмена без промежуточных окон.
  • 📂 Автоматически открывать редактор для аннотирования (если нужно).

Алгоритм действий:

  1. Нажмите Win + Shift + S — экран затемнится, а вверху появится панель инструментов.
  2. Выберите режим Прямоугольник (вторая иконка слева).
  3. Кликните на целевом мониторе и растяните рамку на весь экран (или выделите нужную область).
  4. Снимок будет скопирован в буфер обмена — вставьте его в Paint, Word или другой редактор для сохранения.

Важная деталь: если у вас включён режим Ночной свет (Night Light), цвета на скриншоте могут искажаться. Отключите его перед захватом, если критична точность цветопередачи.

Если встроенные инструменты Windows вас не устраивают, обратите внимание на сторонние утилиты. Они предлагают расширенные функции, такие как:

  • 🎯 Автоматический выбор монитора по номеру (1, 2, 3...).
  • ⏱️ Отложенный захват с таймером.

  • 🖌️ Редактор со стрелками, текстом и размытием.

Способ 4: Скрипты на AutoHotkey для автоматического выбора монитора

Если вы предпочитаете автоматизацию и готовы потратить 10 минут на настройку, AutoHotkey (AHK) позволит создать пользовательские скрипты для захвата конкретного монитора по одной клавише. Это особенно удобно для профессионалов, которые делают десятки скриншотов в день.

Пример скрипта для захвата второго монитора по нажатию F1:

#NoEnv

SendMode Input



; Горячая клавиша F1 для захвата второго монитора


F1::


{


; Получаем разрешение второго монитора (индекс 2)


SysGet, monitorCount, MonitorCount


if (monitorCount < 2)


{


MsgBox, Второй монитор не найден!


return


}



; Координаты второго монитора (пример для разрешения 1920x1080)


X := 1920  ; Начало второго монитора (ширина первого)


Y := 0


W := 1920  ; Ширина второго монитора


H := 1080  ; Высота



; Захватываем область и сохраняем в файл


FileName := "C:\Screenshots\Monitor2_" . A_YYYYMMDD "_" . A_HHmmss . ".png"


Run, nircmd.exe savescreenshot "%FileName%" %X% %Y% %W% %H%


}


return

Для работы этого скрипта потребуется:

  1. Установить AutoHotkey с официального сайта.
  2. Скачать утилиту NirCmd (бесплатная) и положить nircmd.exe в папку со скриптом.
  3. Откорректировать координаты X, Y, W, H под разрешение ваших мониторов.
⚠️ Внимание: Координаты мониторов в Windows зависят от их физического расположения в настройках дисплея. Если второй монитор находится слева от основного, его координата X будет отрицательной (например, -1920). Проверьте расположение в Параметры → Система → Дисплей.

Преимущество этого метода — мгновенный захват без лишних кликов. Однако он требует начальной настройки и базовых знаний скриптинга. Для новичков проще использовать программы вроде ShareX.

Способ 5: macOS — как сделать скриншот одного экрана на Mac

На компьютерах Apple с несколькими мониторами процесс немного отличается от Windows. В macOS есть встроенные комбинации для захвата конкретного дисплея:

  • 📸 Command + Shift + 4, затем Пробел — выбирает окно (не монитор!).
  • 🖥️ Command + Shift + 4, затем нажмите Option — позволяет выделить конкретный монитор (рамка изменит цвет).
  • 💾 Command + Shift + 3 — сохраняет скриншоты всех экранов на рабочий стол.

Чтобы захватить только один монитор:

  1. Нажмите Command + Shift + 4 — курсор превратится в прицел.
  2. Удерживая Option, кликните на целевом мониторе — рамка выделения появится только вокруг него.
  3. Отпустите кнопки — снимок сохранится на рабочем столе в формате PNG.

В macOS Ventura и новее скриншоты по умолчанию сохраняются с тенью, что может быть неудобно для дальнейшей обработки. Чтобы отключить тень:

  1. Откройте Терминал (Applications → Utilities → Terminal).
  2. Введите команду: 
    defaults write com.apple.screencapture disable-shadow -bool true

    killall SystemUIServer

Для возврата теней выполните ту же команду, заменив true на false.

Как изменить формат скриншотов по умолчанию в macOS?

По умолчанию macOS сохраняет скриншоты в формате PNG. Чтобы изменить формат на JPG, JPEG, TIFF или PDF, используйте команду в Терминале:

defaults write com.apple.screencapture type jpg

killall SystemUIServer

Доступные форматы: png, jpg, tiff, pdf.

Способ 6: Linux — утилиты для многомониторных систем

В дистрибутивах Linux выбор инструмента для скриншотов зависит от используемой графической оболочки (GNOME, KDE, XFCE и др.). Большинство современных окружений имеют встроенные утилиты с поддержкой нескольких мониторов.

Для GNOME (Ubuntu, Fedora):

  • 📷 PrtScn — снимок всего экрана (всех мониторов).
  • 🖥️ Shift + PrtScn — выбор области (можно выделить один монитор).
  • 🪟 Alt + PrtScn — активное окно.

Для захвата конкретного монитора через терминал используйте утилиту scrot:

scrot -u -b -m 1 'monitor1_%Y-%m-%d.png'

Где:

  • -u — захват текущего окна (не нужно для монитора).
  • -b — включить рамку окна.
  • -m 1 — номер монитора (начиная с 0).

Для KDE Plasma (Kubuntu):

  1. Откройте Spectacle (стандартная утилита для скриншотов).
  2. Выберите режим Область экрана.
  3. Удерживая Ctrl, кликните на целевом мониторе — программа автоматически выделит его границы.
⚠️ Внимание: В некоторых дистрибутивах Linux координаты мониторов могут сбрасываться после переподключения или изменения разрешения. Перед использованием scrot или аналогичных утилит проверьте актуальные параметры командой xrandr.

Способ 7: Онлайн-сервисы для скриншотов (если нельзя установить ПО)

Если у вас нет прав администратора для установки программ или вы работаете на чужих компьютерах, можно использовать веб-сервисы для создания скриншотов. Они работают через браузер и позволяют выделять области экрана, в том числе на конкретном мониторе.

Популярные сервисы:

  • 🌐 Lightshot Online — расширение для Chrome/Firefox с ручным выделением области.
  • 📁 Screenshot.guru — загрузка скриншота с аннотациями.
  • 🔗 ImgBB — загрузка снимков с генерацией прямой ссылки.

Как пользоваться Lightshot Online:

  1. Установите расширение для вашего браузера.
  2. Нажмите на иконку расширения или используйте горячую клавишу (PrtScn по умолчанию).
  3. Выделите область на нужном мониторе — рамка будет видна только на нём.
  4. Сохраните снимок на компьютер или загрузите в облако.

Ограничения онлайн-сервисов:

  • ⚠️ Требуют подключения к интернету для загрузки.
  • 🔒 Некоторые сервисы сохраняют скриншоты на своих серверах (проверьте политику конфиденциальности).
  • ⏳ Могут быть медленнее локальных программ.

Этот способ подходит для разовых задач, но не заменяет специализированное ПО для регулярного использования.

💡

Онлайн-сервисы удобны для быстрой отправки скриншотов коллегам, но не подходят для работы с конфиденциальными данными. Всегда проверяйте, что снимок не сохраняется на сторонних серверах без вашего согласия.

FAQ: Частые вопросы о скриншотах на нескольких мониторах

Можно ли сделать скриншот одного монитора на ноутбуке с подключённым внешним дисплеем?

Да, все описанные методы работают и для ноутбуков. Главное — правильно определить номер монитора в настройках системы. В Windows проверьте порядок дисплеев в Параметры → Система → Дисплей (перетащите иконки мониторов, чтобы основной экран был слева). В macOS внешний монитор обычно имеет индекс 1, если он подключён справа от встроенного экрана.

Почему скриншот получается размытым, если на мониторе включено масштабирование?

Это происходит из-за того, что Windows рендерит изображение с учётом масштаба (например, 125% или 150%), а затем уменьшает его до исходного разрешения. Чтобы избежать размытости:

  1. Верните масштаб на 100% в настройках экрана.
  2. Используйте программы вроде ShareX, которые захватывают пиксели напрямую, без масштабирования.
  3. В macOS отключите опцию Масштабировать разрешение в настройках дисплея.
Как сделать скриншот только одного монитора в игре?

Большинство игр блокируют стандартные инструменты захвата экрана (например, PrtScn или Snipping Tool). В этом случае:

  • Используйте встроенные функции игры (обычно клавиша F12 для Steam или PrtScn в Origin/EA App).
  • Настройте ShareX на захват DirectX/OpenGL-окна (в настройках Capture → Game recording).
  • Для NVIDIA используйте GeForce Experience (Alt + F1), выбрав монитор в настройках захвата.

Обратите внимание: некоторые античиты (например, в Valorant или Fortnite) блокируют все сторонние утилиты захвата.

Можно ли автоматизировать создание скриншотов по расписанию?

Да, для этого подойдут:

  • ShareX с задачей Screen recorder → Timeline (захват по таймеру).
  • AutoHotkey со скриптом, который запускает захват по расписанию через Sleep.
  • В Linux — cron + scrot (пример команды для захвата каждого часа: 
    0     scrot -m 1 /path/to/screenshots/monitor1_%H-%M.png
Как удалить ненужные скриншоты, которые автоматически сохраняются в папке?

В Windows 10/11 скриншоты, сделанные через Win + PrtScn, сохраняются в папке C:\Users\Имя_пользователя\Pictures\Screenshots. Чтобы очистить её:

  1. Откройте папку и выделите все файлы (Ctrl + A).
  2. Удерживая Shift, нажмите Delete — это полностью удалит файлы, минуя корзину.
  3. Чтобы отключить автосохранение, измените параметры в реестре: 
    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer

    Значение: "ScreenshotIndex" (удалите или установите в 1)

В macOS скриншоты сохраняются на рабочем столе. Чтобы изменить папку по умолчанию, используйте команду: 

defaults write com.apple.screencapture location /Новая/папка/

killall SystemUIServer
