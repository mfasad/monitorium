Как убрать второй монитор со скриншота: полное руководство с картинками

Сделали скриншот, но на нём отобразился второй монитор, который портит композицию? Или нужно отправить коллеге только часть рабочего стола, а на снимке видно сразу два экрана? Эта проблема знакома многим, кто работает с многомониторными системами. К счастью, удалить лишний дисплей со скриншота можно за несколько минут — без специальных навыков и сложных программ.

В этой статье мы разберём 5 проверенных способов, как обрезать или скрыть второй монитор на скриншоте в зависимости от вашей операционной системы и доступных инструментов. Вы узнаете, как сделать это в стандартных приложениях Windows 10/11, macOS и Linux, а также с помощью бесплатных графических редакторов. Отдельно рассмотрим нюансы для игровых мониторов с высоким разрешением и случаи, когда второй экран отображается как чёрная область.

Все методы подходят как для PNG-скриншотов, так и для JPEG — главное, чтобы исходное изображение было хорошего качества. Если вы часто сталкиваетесь с этой проблемой, сохраните статью в закладки: здесь есть решения для любых ситуаций, от быстрой обрезки до профессиональной ретуши.

Почему второй монитор попадает на скриншот?

Прежде чем переходить к способам удаления, важно понять, почему так происходит. Причин несколько, и они зависят от того, как именно вы делаете снимок экрана:

  • 🖥️ Стандартные сочетания клавиш (PrtScn, Win + PrtScn, Alt + PrtScn) в Windows по умолчанию захватывают все подключённые дисплеи, если они настроены как расширение экрана.
  • 🎮 Игровые оверлеи (например, Steam Screenshot или NVIDIA ShadowPlay) часто сохраняют изображение со всех мониторов, даже если игра запущена только на одном.
  • 🖼️ Программы для скриншотов (типа Lightshot или Snipping Tool) могут автоматически захватывать виртуальный канвас, объединяющий оба экрана.
  • 🔄 Режимы дублирования/расширения: если мониторы настроены как зеркало (Duplicate), скриншот будет одинаковым на обоих, но при расширении (Extend) захватится вся область.

Интересно, что в macOS и Linux поведение по умолчанию может отличаться. Например, в Ubuntu сочетание Shift + PrtScn позволяет выбрать область вручную, а в Mac комбинация Cmd + Shift + 4 сразу предлагает обвести нужный фрагмент. Тем не менее, даже здесь второй монитор может "просочиться" на снимок, если неаккуратно выбрать зону.

Ещё один нюанс: если второй монитор отключён физически, но в системе остаётся его "призрак" (например, в настройках отображается как Дисплей 2: Отключён), некоторые программы могут всё равно резервировать под него место на скриншоте. Это чаще всего происходит после обновления драйверов NVIDIA или AMD.

Способ 1: Обрезка скриншота стандартными средствами ОС

Самый быстрый способ убрать второй монитор — просто обрезать изображение, оставив только нужную часть. Все современные операционные системы имеют встроенные инструменты для этого.

В Windows 10/11:

  1. Откройте скриншот в приложении "Фотографии" (кликните по файлу дважды).
  2. Нажмите на иконку "Обрезка и поворот" (ножницы в верхней панели).
  3. Выберите пропорции "Произвольное соотношение" или задайте свои (например, 16:9 для монитора).
  4. Перетащите углы рамки так, чтобы второй экран остался за её пределами.
  5. Сохраните результат через Файл → Сохранить как.

Альтернатива: используйте Snipping Tool (в Windows 11 — "Ножницы"):

  • 📱 Откройте программу через поиск в меню Пуск.
  • 🖼️ Выберите режим "Прямоугольное выделение".
  • 🎯 Обведите мышью только ту часть экрана, где находится основной монитор.
  • 💾 Сохраните новый скриншот через кнопку "Сохранить как".

В macOS:

На Mac обрезка ещё проще:

  1. Откройте скриншот в приложении "Просмотр" (Preview).
  2. Нажмите на иконку "Показать панель разметки" (карандаш в кружке).
  3. Выберите инструмент "Выделение" (прямоугольник с пунктиром).
  4. Обведите нужную область и нажмите Cmd + K (или выберите "Обрезать" в меню "Инструменты").

В Linux (Ubuntu, Mint и др.):

В большинстве дистрибутивов по умолчанию установлен GIMP или Shotwell. Для быстрой обрезки:

sudo apt install gimp  # если GIMP не установлен

gimp ваш_скриншот.png

В GIMP:

  1. Откройте изображение.
  2. Выберите инструмент "Обрезка" (иконка ножниц).
  3. Обведите область основного монитора и нажмите Enter.
  4. Экспортируйте результат через Файл → Экспорт как.

Также будьте осторожны при сохранении: некоторые форматы (например, JPEG) сжимают изображение, что может ухудшить качество текста на скриншоте.

Способ 2: Использование специализированных программ для скриншотов

Если вам часто приходится делать снимки экрана без второго монитора, имеет смысл установить программу, которая изначально захватывает только активный дисплей. Вот лучшие бесплатные решения:

Программа Платформа Особенности Как исключить второй монитор
Lightshot Windows, macOS Лёгкий редактор, загрузка в облако, горячие клавиши При захвате области (PrtScn) вручную обведите только основной экран
ShareX Windows Расширенные настройки, запись видео, OCR В настройках захвата выберите "Активный монитор" вместо "Весь экран"
Flameshot Linux, Windows, macOS Открытый код, редактор с эффектами При запуске (flameshot gui) выделите мышью только нужный дисплей
Snagit Windows, macOS Профессиональный инструмент с записью видео В настройках региона захвата укажите "Один монитор" и выберите основной

Пример настройки в ShareX:

  1. Установите программу с официального сайта и запустите её.
  2. Перейдите в Настройки → Захват → Регион захвата.
  3. В выпадающем меню "Регион захвата по умолчанию" выберите "Активный монитор".
  4. Теперь при нажатии PrtScn будет захвачен только тот экран, где находится курсор.

В Flameshot (для Linux) процесс ещё проще:

sudo apt install flameshot

flameshot gui

После запуска интерфейса вы увидите полупрозрачный экран — просто обведите мышью границы основного монитора и нажмите Enter. Программа автоматически сохранит только выделенную область.

💡

В ShareX можно настроить отдельные горячие клавиши для захвата активного монитора и всей области. Например, назначьте Ctrl + PrtScn для одного экрана, а Alt + PrtScn — для всех.

⚠️ Внимание: Некоторые программы (например, Lightshot) могут конфликтовать со встроенными инструментами Windows. Если после установки перестали работать стандартные сочетания клавиш, проверьте настройки программы или отключите её автозагрузку.

Способ 3: Настройка системы для захвата только одного монитора

Если вы не хотите каждый раз обрезать скриншоты, можно изменить настройки системы, чтобы по умолчанию захватывался только основной дисплей. Это особенно удобно для пользователей, которые часто делятся снимками экрана.

В Windows 10/11:

К сожалению, в Windows нет встроенной опции для захвата только одного монитора по умолчанию. Однако можно использовать PowerShell или AutoHotkey для создания пользовательского сочетания клавиш.

Пример скрипта для AutoHotkey (сохраните как ScreenshotSingleMonitor.ahk):

#IfWinActive ahk_exe explorer.exe  ; работает только на рабочем столе

^!p::  ; Ctrl + Alt + P


{


Sysget, Mon, MonitorCount


If (Mon > 1) {


Sysget, Mon, Monitor, 1  ; захватываем первый монитор


Left := MonLeft, Top := MonTop, Right := MonRight, Bottom := MonBottom


Width := Right - Left, Height := Bottom - Top


Run, nircmd.exe savescreenshot "%A_Desktop%\Screenshot_1.png" %Left% %Top% %Width% %Height%


}


}


return

Для работы скрипта понадобится утилита NirCmd (скачайте с официального сайта и поместите nircmd.exe в папку C:\Windows). Теперь при нажатии Ctrl + Alt + P будет сохранён скриншот только первого монитора.

В macOS:

На Mac есть встроенная возможность выбора монитора:

  1. Откройте "Терминал" и введите: 
    defaults write com.apple.screencapture disable-shadow -bool true

    killall SystemUIServer

    (это отключит тени на скриншотах для чистоты изображения).

  2. Теперь при нажатии Cmd + Shift + 4 зажмите клавишу Space — курсор превратится в камеру.
  3. Наведите курсор на нужный монитор (он подсветится синим) и кликните мышью.

В Linux (с использованием scrot):

Утилита scrot позволяет гибко настраивать захват экрана. Установите её:

sudo apt install scrot

Чтобы сделать скриншот только первого монитора (обычно :0.0), используйте команду:

scrot -u -b -d 1 'Screenshot_%Y-%m-%d_%H:%M:%S.png' -e 'mv $f ~/Pictures/'

Флаги:

  • -u — захватывает только активное окно (если нужно именно оно).
  • -b — включает рамку окна.
  • -d 1 — задержка в 1 секунду (чтобы успеть переключиться на нужный монитор).
💡

Настройка системы для захвата одного монитора сэкономит время, если вам приходится делать такие скриншоты регулярно. Однако для разовых задач проще использовать обрезку или специализированные программы.

⚠️ Внимание: В многомониторных системах с разным разрешением (например, 4K + Full HD) координаты экранов могут рассчитываться некорректно. Если скрипт или команда захватывает не тот монитор, проверьте их порядок в настройках дисплеев (Параметры → Система → Дисплей в Windows).

Способ 4: Ретушь скриншота (если второй монитор виден как чёрная область)

Иногда второй монитор на скриншоте отображается как чёрный прямоугольник — это происходит, если он был отключён или находился в спящем режиме. В таких случаях обрезка не поможет, так как чёрная область может накладываться на важные элементы. Здесь понадобится ретушь.

В Paint 3D (Windows 10/11):

  1. Откройте скриншот в Paint 3D (клик правой кнопкой → "Открыть с помощью").
  2. Выберите инструмент "Выделение" и обведите чёрную область.
  3. Нажмите Удалить (или Del).
  4. Если остались артефакты, используйте инструмент "Заполнить", чтобы подогнать фон под цвет основного монитора.

В GIMP (бесплатный аналог Photoshop):

Для более точной ретуши:

  1. Откройте изображение в GIMP.
  2. Выберите инструмент "Выделение прямоугольником" (R) и обведите чёрную область.
  3. Нажмите Edit → Fill with BG Color (или Shift + B), выбрав цвет, близкий к фону основного монитора.
  4. Для сглаживания краёв используйте "Размытие/Резкость" (Фильтры → Размытие → Размытие по Гауссу) с радиусом 2-3 px.

В Photoshop:

Если у вас есть Photoshop, воспользуйтесь инструментом "Заплатка" (J):

  1. Выделите чёрную область инструментом "Лассо" (L).
  2. Перейдите на слой с фоном и выберите "Редактирование → Выполнить заливку" с параметром "С учётом содержимого".
  3. Если нужно, дорисуйте недостающие элементы инструментом "Штамп" (S).

Для автоматизации процесса в Photoshop можно записать экшен:

  1. Откройте панель "Операции" (Alt + F9).
  2. Создайте новую операцию и запишите шаги: выделение → заливка → размытие.
  3. Сохраните и применяйте к другим скриншотам одним кликом.
Как подобрать цвет для заливки?

Используйте инструмент "Пипетка" (I), чтобы взять образец цвета с фона основного монитора. Если фон градиентный, берите цвет из области, ближайшей к чёрному прямоугольнику.

⚠️ Внимание: При ретуши скриншотов с текстом или интерфейсами (например, кодом или таблицами) заливка чёрной области может выглядеть неестественно. В таких случаях лучше использовать обрезку или переделать скриншот с правильными настройками захвата.

Способ 5: Изменение настроек многомониторной системы перед скриншотом

Если вам нужно сделать идеальный скриншот без лишних элементов, проще временно отключить второй монитор или изменить его режим отображения. Это гарантирует, что на снимке будет только то, что вам нужно.

Отключение второго монитора в Windows:

  1. Нажмите Win + P и выберите "Только экран компьютера" (если второй монитор — внешний).
  2. Или перейдите в Параметры → Система → Дисплей.
  3. Выберите второй монитор и нажмите "Отключить".
  4. Сделайте скриншот стандартным способом (PrtScn или Win + PrtScn).
  5. Включите монитор обратно через Win + P → Расширить.

В macOS:

  1. Откройте "Системные настройки → Мониторы".
  2. Удерживая Option (⌥), нажмите на кнопку "Обнаружить мониторы".
  3. Снимите галочку с ненужного дисплея или перетащите его в положение "Зеркалировать" (если нужно дублирование).

В Linux (с использованием xrandr):

Чтобы временно отключить второй монитор (например, HDMI-1):

xrandr --output HDMI-1 --off

Чтобы вернуть его обратно:

xrandr --output HDMI-1 --auto --right-of eDP-1

Замените HDMI-1 и eDP-1 на актуальные имена ваших дисплеев (узнать их можно командой xrandr -q).

Если второй монитор физически отключён, но в системе остаётся его "призрак", сбросьте настройки:

xrandr --output HDMI-1 --off

xrandr --auto

⚠️ Внимание: На некоторых ноутбуках (например, Dell XPS или MacBook Pro) при отключении внешнего монитора может сбрасываться масштабирование экрана или цветовой профиль. После повторного подключения проверьте настройки в Панель управления NVIDIA/AMD или Системные настройки → Мониторы.

Частые ошибки и как их избежать

Даже опытные пользователи иногда сталкиваются с проблемами при удалении второго монитора со скриншота. Вот топ-5 ошибок и их решения:

  1. Чёрный прямоугольник вместо второго монитора

    Причина: Монитор был отключён или находился в спящем режиме во время захвата.
    Решение: Используйте ретушь (Способ 4) или сделайте скриншот заново с включённым вторым экраном.

  2. Скриншот получается размытым после обрезки

    Причина: Сохранение в JPEG с высокой степенью сжатия или изменение размера изображения.
    Решение: Сохраняйте в PNG и избегайте масштабирования.

  3. Не получается обрезать скриншот в "Фотографиях" Windows

    Причина: Файл открыт в другой программе или имеет несовместимый формат (например, WEBP).
    Решение: Конвертируйте изображение в PNG через Paint или онлайн-конвертер.

  4. Программы для скриншотов не захватывают активное окно

    Причина: Конфликт горячих клавиш или отсутствие прав администратора.
    Решение: Перезапустите программу от имени администратора или переназначьте клавиши в её настройках.

  5. После отключения второго монитора сбиваются обои или ярлыки

    Причина: Windows запоминает положение иконок относительно виртуального рабочего стола, объединяющего оба экрана.
    Решение: Используйте утилиту DisplayFusion для фиксации позиций ярлыков или вручную перетащите их после возврата второго монитора.

Если вы работаете с игровыми мониторами (например, ASUS ROG Swift или Acer Predator), учтите, что их высокое разрешение (1440p, 4K) может создавать дополнительные сложности:

  • 🎮 Скриншоты будут весить больше (до 10-20 МБ в PNG).
  • 🖥️ Некоторые программы (например, Lightshot) могут некорректно захватывать области с масштабированием 125%-150%.
  • 🔍 При обрезке проверяйте итоговое разрешение — оно должно быть кратным 8 (например, 1920×1080, а не 1921×1079), иначе могут быть проблемы с отображением.

FAQ: Ответы на популярные вопросы

Можно ли удалить второй монитор со скриншота на телефоне?

Если вы сделали скриншот с компьютера и отправили его на телефон, используйте мобильные редакторы (например, Snapseed, PicsArt или стандартный "Google Фото") для обрезки. На Android также можно использовать режим "Прокрутка экрана" в некоторых приложениях для скриншотов, но он не поможет с многомониторными системами.

Почему после обрезки скриншот стал весить больше?

Это происходит, если вы сохранили его в формате PNG без сжатия. PNG поддерживает прозрачность и не имеет потерь качества, но может занимать много места. Чтобы уменьшить вес:

  • Используйте JPEG с качеством 80-90% (подходит для фото, но не для текста).
  • Возьмите утилиту PNGGauntlet или TinyPNG для оптимизации без потерь.
Как сделать скриншот только активного окна, игнорируя второй монитор?

В Windows используйте Alt + PrtScn — это сочетание захватывает только активное окно, даже если оно растянуто на два экрана. В macOS нажмите Cmd + Shift + 4, затем Space и кликните на нужное окно. В Linux с scrot используйте флаг -u:

scrot -u 'active_window_%Y-%m-%d.png'
Можно ли автоматизировать удаление второго монитора со всех скриншотов?

Да, для этого подойдёт скрипт на Python с использованием библиотеки Pillow. Пример:

from PIL import Image


def crop_screenshot(input_path, output_path, monitor_width):


img = Image.open(input_path)


width, height = img.size


left = 0


right = monitor_width


cropped = img.crop((left, 0, right, height))


cropped.save(output_path)



crop_screenshot("screenshot.png", "cropped.png", 1920)  # 1920 - ширина основного монитора

Сохраните скрипт как crop.py и запускайте его для пакетной обработки:

python crop.py
Почему на скриншоте второй монитор отображается перевёрнутым?

Это происходит, если в настройках дисплея для второго монитора установлен поворот на 90° или 270°. Чтобы исправить:

  1. В Windows: Параметры → Система → Дисплей → Дополнительные параметры дисплея → Свойства видеоадаптера → Поворот.
  2. В macOS: "Системные настройки → Мониторы → Поворот".
  3. В Linux: 
    xrandr --output HDMI-1 --rotate normal

После исправления повтора сделайте скриншот заново.
