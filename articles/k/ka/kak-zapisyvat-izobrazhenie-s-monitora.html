Запись изображения с монитора: подробное руководство для новичков и профессионалов

Запись изображения с экрана монитора — задача, с которой сталкиваются и геймеры, и преподаватели, и разработчики ПО. Кто-то хочет сохранить геймплей для стрима, кому-то нужно создать обучающий видеоурок, а третьи фиксируют баги в программах для отчёта технической поддержке. Способы записи экрана различаются по сложности, качеству итогового видео и доступным функциям: от встроенных утилит операционной системы до мощных профессиональных инструментов с поддержкой 4K-разрешения и 120 FPS.

В этой статье мы разберём все актуальные методы — от стандартных средств Windows 11/10 и macOS до специализированного софта вроде OBS Studio или Camtasia. Особое внимание уделим настройке параметров записи (разрешение, битрейт, кодек), чтобы избежать распространённых проблем: размытого изображения, подтормаживания или гигантского размера файла. Также вы узнаете, как записать звук с микрофона и системные звуки одновременно, и какие форматы видео лучше выбрать для дальнейшего монтажа.

Встроенные средства записи экрана в Windows

Операционные системы Windows 10 и Windows 11 имеют встроенный инструмент для записи экрана — Xbox Game Bar. Несмотря на название, он подходит не только для геймплея, но и для фиксации любых действий на мониторе. Главное ограничение: программа записывает только активное окно (не весь рабочий стол) и не поддерживает запись File Explorer или Проводника.

Чтобы начать запись:

  • 🎮 Нажмите комбинацию Win + Alt + R — запись начнётся сразу (по умолчанию видео сохраняется в Видео → Захват).
  • ⚙️ Для настроек параметров (разрешение, FPS, включение микрофона) откройте Параметры → Игры → Панель игр Xbox.
  • 🔊 Чтобы записать звук с микрофона, включите опцию "Записывать звук" → "Микрофон" в настройках панели.
  • ⏱️ Максимальная длительность записи — 4 часа (после автоматически создаётся новый файл).
⚠️ Внимание: Xbox Game Bar не записывает рабочий стол в Windows 10 Home — только игры и отдельные приложения (например, браузер). Для полноэкранной записи используйте сторонние программы.

Преимущества метода:

  • ✅ Не требует установки дополнительного ПО.
  • ✅ Минимальная нагрузка на систему (подходит для слабых ПК).
  • ✅ Быстрый доступ через горячие клавиши.

Недостатки:

  • ❌ Нет возможности выбирать кодек или битрейт.
  • ❌ Запись ограничена одним окном (не весь экран).
  • ❌ Нет инструментов для редактирования видео.

Запись экрана через PowerPoint (скрытая функция)

Мало кто знает, но Microsoft PowerPoint (начиная с версии 2013) имеет встроенный инструмент для записи экрана. Эта функция полезна, если нужно быстро создать презентацию с демонстрацией действий на мониторе или записать видеоурок с голосовым сопровождением.

Инструкция:

  1. Откройте PowerPoint и перейдите на вкладку Вставка → Запись экрана.
  2. Выделите область записи (можно захватить весь экран или его часть).
  3. Нажмите Записать — начнётся отсчёт 3 секунды.
  4. По завершении нажмите Остановить, а затем Сохранить (видео вставится как объект в слайд).
  5. Чтобы экспортировать видео отдельным файлом, кликните правой кнопкой по вставленному видео и выберите Сохранить медиафайл как....

Особенности метода:

  • 📹 Видео сохраняется в формате .mp4 с разрешением до 1920×1080.
  • 🎤 Можно записывать голос с микрофона прямо во время захвата экрана.
  • ⏱️ Нет ограничений по длительности записи.
⚠️ Внимание: В PowerPoint 2016 и старше функция записи экрана может отсутствовать в бесплатной онлайн-версии. Проверьте наличие вкладки Вставка → Запись экрана перед началом работы.
💡

Если вам нужно записать презентацию с анимациями, сначала сохраните её как видео через Файл → Экспорт → Создать видео, а затем добавьте голосовое сопровождение в любом видео редакторе.

Профессиональная запись с OBS Studio (бесплатно)

OBS Studio — это бесплатная программа с открытым исходным кодом, которая считается золотым стандартом для записи экрана и стриминга. Она поддерживает несколько источников видео (экран, веб-камера, захват игры), настройку битрейта, аппараное ускорение (через NVIDIA NVENC или AMD AMF) и продвинутые аудиофильтры.

Пошаговая настройка для записи экрана:

  1. Скачайте OBS Studio с официального сайта и установите.
  2. В главном окне нажмите + в блоке Источники и выберите Захват экрана (Screen Capture).
  3. Создайте новый источник, укажите монитор для записи (если их несколько) и настройте разрешение.
  4. Перейдите в Настройки → Вывод и выберите:
    • 📁 Путь для записи (например, D:\Записи).
    • 🎞️ Формат записи — лучше mp4 или mkv.
    • ⚙️ Качество записи — Неограниченное (для максимального качества).
    • 🔧 Кодировщик — NVIDIA NVENC H.264 (если есть видеокарта NVIDIA) или x264.
  5. В блоке Аудио настройте источники звука (системный звук, микрофон).
  6. Нажмите Начать запись (или используйте горячие клавиши).

    7. Рекомендуемые настройки для качественной записи:

    Параметр Рекомендуемое значение Пояснение
    Разрешение 1920×1080 или 2560×1440 Совпадает с разрешением монитора для избежания масштабирования.
    FPS 60 Оптимально для геймплея; для урока хватит 30 FPS.
    Битрейт 15 000–25 000 Кбит/с Выше — лучше качество, но больше вес файла.
    Кодировщик NVENC (NVIDIA) или AMF (AMD) Аппаратное кодирование снижает нагрузку на CPU.
    Формат MP4 Универсальный формат, поддерживаемый всеми плеерами.

    Преимущества OBS Studio:

    • 🎯 Полный контроль над параметрами записи (от разрешения до аудиомикшера).
    • 🖥️ Поддержка нескольких мониторов и источников видео.
    • 📊 Встроенные инструменты для стриминга на Twitch, YouTube.
    ⚠️ Внимание: При записи в MKV файл может не воспроизводиться до полного завершения записи. Используйте MP4, если нужно проверять видео по ходу.

    Запись экрана на macOS: QuickTime Player и альтернативы

    На компьютерах Mac запись экрана осуществляется через стандартное приложение QuickTime Player. Оно поддерживает запись всего экрана, его части или отдельного окна, а также захват звука с микрофона (но не системного аудио без дополнительных утилит).

    Инструкция для QuickTime Player:

    1. Откройте QuickTime Player (находится в Программы → Утилиты).
    2. В меню выберите Файл → Новая запись экрана.
    3. Нажмите на стрелочку рядом с кнопкой записи и выберите:
      • 🎤 Микрофон (если нужен голос).
      • 🖥️ Показывать клики мыши (опционально).
    4. Нажмите Записать и выделите область экрана (или кликните в любое место для записи всего монитора).
    5. Для остановки записи используйте панель в меню или комбинацию Command + Control + Esc.

      6. Ограничения QuickTime Player:

      • ❌ Не записывает системный звук (только микрофон).
      • ❌ Нет настроек битрейта или FPS.
      • ❌ Видео сохраняется в формате .mov, который может не поддерживаться некоторыми плеерами.

    Для записи системного звука на Mac потребуется дополнительная утилита, например, Soundflower или BlackHole. Эти программы создают виртуальный аудиоустройство, перенаправляющее системный звук в QuickTime Player.

    Как записать системный звук на Mac?

    1. Установите BlackHole (бесплатно) с официального сайта.

    2. Откройте Аудио-MIDI-настройка (в Программы → Утилиты).

    3. Создайте Мультивыходное устройство и добавьте в него BlackHole 2ch и Встроенный вывод.

    4. В QuickTime Player выберите BlackHole 2ch как источник звука.

    5. Запустите запись — теперь будет захватываться системный звук.

    Специализированные программы для записи экрана

    Если встроенные инструменты не подходят по функционалу, рассмотрите специализированные программы. Они предлагают расширенные возможности: запись с веб-камеры, аннотации в реальном времени, автоматическое выделение кликов мыши и экспорт в облако.

    Топ-5 программ для записи экрана:

    Программа Платформа Ключевые функции Цена
    Camtasia Windows, macOS Встроенный видео редактор, эффекты, шаблоны для обучающих видео. ~$300 (однократно)
    ScreenFlow macOS Мультитрековая запись, анимации, экспорт в 4K. ~$150
    Bandicam Windows Запись игр с высоким FPS (до 480 FPS), поддержка NVIDIA NVENC. ~$40
    Snagit Windows, macOS Захват скриншотов и видео, быстрая обрезка, добавление аннотаций. ~$60
    ShareX Windows Бесплатная, открытый код, запись GIF, поддержка плагинов. Бесплатно

    Для большинства задач достаточно бесплатных решений (OBS Studio или ShareX), но если вам нужны продвинутые инструменты монтажа или запись в 4K@120FPS, обратите внимание на платные программы. Например, Bandicam оптимизирован для записи игр и поддерживает аппаратное ускорение от NVIDIA/AMD, что критично для мощных AAA-проектов.

    ⚠️ Внимание: Некоторые программы (например, Fraps) устарели и не поддерживают современные разрешения или кодек H.265. Перед покупкой проверьте дату последнего обновления на официальном сайте.

    Как записать экран на Android и iOS

    Запись экрана на мобильных устройствах востребована для создания обзоров приложений, стримов мобильных игр или фиксации багов. На Android и iOS есть как встроенные инструменты, так и сторонние приложения с расширенным функционалом.

    На iPhone/iPad (iOS 11 и новее):

    1. Откройте Настройки → Пункт управления → Настройка элементов управления.
    2. Добавьте Запись экрана в список включённых элементов.
    3. Откройте Пункт управления (свайп вниз с правого верхнего угла) и нажмите на иконку записи (🎥).
    4. Чтобы записать звук с микрофона, удерживайте иконку записи и включите опцию Микрофон.

    На Android (версия 11 и новее):

    • 📱 Используйте встроенную функцию Screen Recorder (доступна в меню быстрых настроек).
    • 🎤 Для записи звука может потребоваться разрешение на доступ к микрофону.
    • ⚠️ На некоторых моделях (например, Xiaomi) запись экрана блокируется в банковских приложениях.

    Сторонние приложения для мобильных устройств:

    • 📱 AZ Screen Recorder (Android) — бесплатно, без водяных знаков, поддержка 1080p@60FPS.
    • 🍎 TechSmith Capture (iOS) — запись с веб-камеры и экрана одновременно.
    • 🎮 Mobizen — стриминг на YouTube/Twitch прямо с телефона.

    Ограничения мобильной записи:

    • ❌ На iOS невозможно записать системный звук без джейлбрейка.
    • ❌ На Android некоторые приложения (например, Netflix) блокируют запись экрана.
    • ❌ Качество видео зависит от производительности устройства (на слабых смартфонах возможны лаги).

    Оптимизация видео после записи: форматы и сжатие

    Записанное видео часто занимает много места на диске, особенно если вы использовали высокий битрейт или разрешение 4K. Чтобы уменьшить размер файла без критичной потери качества, воспользуйтесь инструментами сжатия или конвертации.

    Рекомендуемые программы для оптимизации видео:

    • 🎞️ HandBrake — бесплатный конвертер с поддержкой H.265 (HEVC) для максимального сжатия.
    • 📦 Shutter Encoder — пакетная обработка файлов, поддержка ProRes и DNxHD.
    • ☁️ CloudConvert — онлайн-сервис для конвертации без установки ПО.

    Оптимальные настройки для сжатия:

    • 📌 Формат: MP4 (универсальный) или MKV (для высокого качества).
    • 📌 Кодек: H.265 (HEVC) — сжимает в 2 раза эффективнее H.264 при том же качестве.
    • 📌 Битрейт: 8–12 Мбит/с для 1080p, 15–20 Мбит/с для 4K.
    • 📌 FPS: Оставляйте исходное значение (не уменьшайте 60 FPS до 30).

    Пример команды для сжатия через FFmpeg (для опытных пользователей):

    ffmpeg -i input.mp4 -c:v libx265 -crf 28 -preset fast -c:a aac -b:a 192k output.mp4

    Где:

    • -crf 28 — качество (ниже значение = лучше качество, но больше размер).
    • libx265 — кодек H.265.
    • -preset fast — баланс между скоростью кодирования и сжатием.
    ⚠️ Внимание: При конвертации в H.265 уделите внимание поддержке форматов вашим устройством. Некоторые старые телевизоры или медиаплееры не воспроизводят HEVC.
    💡

    Для максимального качества при минимальном размере файла используйте кодек H.265 с битрейтом 15–20 Мбит/с для 4K-видео.

    Частые проблемы и их решения

    При записи экрана пользователи сталкиваются с типичными проблемами: заикание видео, отсутствие звука, чёрный экран или слишком большой размер файла. Разберём причины и способы их устранения.

    Проблема 1: Видео записывается рывками или с низким FPS

    • 🔹 Причина: Недостаточная производительность CPU/GPU или высокий битрейт.
    • 🔹 Решение:
      • Уменьшите разрешение записи (например, с 4K до 1080p).
      • Используйте аппаратное кодирование (NVENC/AMF в OBS Studio).
      • Закройте фоновые программы (особенно браузер с множеством вкладок).

    Проблема 2: Нет звука в записи

    • 🔹 Причина: Неправильно выбран источник звука или отсутствуют драйверы.
    • 🔹 Решение:
      • В OBS Studio проверьте микшер аудио: включите Захват аудио рабочего стола.
      • На Windows обновите драйверы звуковой карты через Диспетчер устройств.
      • На Mac используйте BlackHole для захвата системного звука.

    Проблема 3: Чёрный экран при записи игр

    • 🔹 Причина: Защита от записи в играх с DRM (например, Denuvo) или конфликт с Fullscreen Optimizations.
    • 🔹 Решение:
      • Запустите игру в оконном режиме (Windowed или Borderless).
      • В OBS Studio используйте захват Игровой захват вместо Захват экрана.
      • Отключите Fullscreen Optimizations в свойствах исполняемого файла игры.

    Проблема 4: Видео занимает слишком много места

    • 🔹 Причина: Высокий битрейт или неоптимальный кодек (H.264 вместо H.265).
    • 🔹 Решение:
      • Сожмите видео через HandBrake (см. раздел выше).
      • В OBS Studio уменьшите битрейт до 10 000–15 000 Кбит/с.
      • Используйте формат MKV для записи, а затем конвертируйте в MP4.

    FAQ: Ответы на популярные вопросы

    Можно ли записать экран с звуком из браузера (например, музыку из YouTube)?

    На Windows для этого нужно включить захват системного звука в настройках программы (в OBS Studio это опция Захват аудио рабочего стола). На Mac потребуется дополнительная утилита вроде BlackHole или Soundflower, так как QuickTime Player не захватывает звук из браузера.

    Обратите внимание: запись аудио с YouTube, Spotify и других сервисов может нарушать их правила использования. Такие видео нельзя публиковать на общедоступных платформах.

    Как записать экран на ноутбуке с двумя мониторами?

    В большинстве программ (например, OBS Studio или Bandicam) можно выбрать, какой монитор записывать. Для этого:

    1. В источниках захвата выберите Захват экрана.
    2. Укажите номер монитора (обычно Монитор 1, Монитор 2 и т. д.).
    3. Настройте разрешение каждого монитора отдельно (если они разные).

    Если нужно записать оба экрана одновременно, добавьте два источника захвата и расположите их рядом в превью.

    Какое разрешение выбрать для записи, если мой монитор 4K, но видео будет смотреться на Full HD экранах?

    Оптимальный вариант — записывать в исходном разрешении (3840×2160), а затем уменьшать до 1920×1080 при монтаже или экспорте. Это сохранит детализацию, если видео когда-нибудь будет просматриваться на 4K-экране.

    Если место на диске ограничено, можно сразу записывать в 1080p, но тогда при масштабировании на больших экранах изображение будет менее чётким.

    Как убрать водяной знак из записанного видео?

    Если водяной знак добавлен программой (например, в бесплатной версии Bandicam), его можно убрать только покупкой лицензии или использованием другой программы. Альтернативные способы (обрезка, размытие) ухудшат качество видео.

    В бесплатных программах вроде OBS Studio или ShareX водяных знаков нет.

    Можно ли записать экран на Linux?

    Да, на Linux доступны следующие инструменты:

    • SimpleScreenRecorder — лёгкая утилита с поддержкой OpenGL и Vulkan.
    • OBS Studio — работает аналогично Windows-версии.
    • ffmpeg — консольная команда для записи: 
      ffmpeg -video_size 1920x1080 -framerate 30 -f x11grab -i :0.0+100,100 output.mp4

      где :0.0+100,100 — координаты и размер области захвата.

    Для записи звука может потребоваться настройка PulseAudio.
