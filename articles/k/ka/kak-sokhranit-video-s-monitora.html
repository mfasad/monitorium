Как записать видео с экрана монитора: полное руководство для ПК и ноутбуков

Запись видео с экрана монитора нужна для создания обучающих роликов, стримов, сохранения геймплея или фиксации ошибок в работе программ. Но не все знают, что качество итогового видео зависит не только от выбранного софта, но и от настроек системы, разрешений экрана и даже от типа подключения монитора (HDMI/DisplayPort могут давать разный результат при захвата). В этой статье разберём все актуальные способы — от встроенных инструментов ОС до профессиональных программ с поддержкой 4K@120Hz.

Особое внимание уделим проблемам, с которыми сталкиваются пользователи: почему запись тормозит при высоком разрешении, как убрать артефакты при захвате игр, и почему некоторые программы не видят второй монитор. Также сравним бесплатные и платные решения — вы поймёте, когда достаточно стандартных средств, а когда без OBS Studio или Bandicam не обойтись.

1. Встроенные средства Windows: Xbox Game Bar и PowerPoint

Если вам нужно быстро сохранить видео с экрана без установки дополнительного ПО, в Windows 10/11 есть два встроенных инструмента. Первый — Xbox Game Bar, изначально предназначенный для записи геймплея, но работающий с любыми окнами. Второй — малоизвестная функция записи экрана в Microsoft PowerPoint, которая подходит для создания презентаций с голосовым сопровождением.

Чтобы запустить запись через Xbox Game Bar, нажмите Win + Alt + R. Видео сохранится в формате .mp4 в папке C:\Users\[Ваше_имя]\Videos\Captures. Максимальное разрешение — 1080p при 30 FPS, что недостаточно для динамичных сцен. Зато инструмент не нагружает систему и работает даже на слабых ноутбуках.

  • 🎮 Плюсы: нет нужды скачивать программы, минимальная нагрузка на CPU, поддержка микрофона.
  • ⚠️ Минусы: нет записи с веб-камеры, ограничение по FPS, не захватывает рабочий стол (только окна).
  • 🔧 Как включить: Параметры → Игры → Xbox Game Bar → Включить запись игровых клипов.

В PowerPoint запись экрана спрятана в меню Вставка → Запись экрана. Здесь можно выбрать область захвата и добавить голосовое сопровождение. Видео вставится прямо в слайд, но его можно экспортировать отдельным файлом. Эта функция полезна для создания обучающих материалов, где важно сочетать демонстрацию экрана с пояснениями.

2. macOS: QuickTime Player и Terminal-команды

На Mac запись экрана реализована проще, чем на Windows. Встроенный QuickTime Player позволяет захватывать как весь экран, так и его часть, с возможностью включения микрофона. Чтобы начать запись, откройте программу, выберите Файл → Новая запись экрана и нажмите красную кнопку. Видео сохранится в .mov с кодеком H.264.

Для продвинутых пользователей есть альтернатива — запись через Terminal. Команда ffmpeg позволяет гибко настраивать параметры захвата, например, записывать только определённую область с высоким битрейтом. Пример команды для записи экрана с разрешением 1920×1080 и частотой 60 FPS:

ffmpeg -f avfoundation -i "1" -r 60 -vf "scale=1920:1080" -c:v libx264 -preset ultrafast -crf 18 -pix_fmt yuv420p output.mp4
  • 🍎 Плюсы QuickTime: простой интерфейс, поддержка Retina-дисплеев, экспорт в 4K.
  • 💻 Плюсы Terminal: полный контроль над настройками, возможность стриминга на сервер.
  • ⚠️ Общий минус: нет захвата системного звука без дополнительных утилит (нужен BlackHole или Soundflower).
💡

Если QuickTime не записывает звук, проверьте разрешения в Системные настройки → Защита и безопасность → Конфиденциальность → Микрофон.

3. Linux: простые и продвинутые способы

В Linux запись экрана зависит от окружения рабочего стола. Для GNOME (Ubuntu, Fedora) подойдёт встроенная утилита gnome-shell-screenshot, но она сохраняет только скриншоты. Для видео нужен SimpleScreenRecorder или FFmpeg. Первый предлагает графический интерфейс и поддерживает запись с OpenGL-ускорением, второй — максимальную гибкость через командную строку.

Пример команды для записи экрана в Ubuntu 22.04+ с звуком через PulseAudio:

ffmpeg -video_size $(xdpyinfo | grep dimensions | awk '{print $2}') -framerate 30 -f x11grab -i :0.0+0,0 -f pulse -i default -c:v libx264 -c:a aac -strict experimental output.mkv
Инструмент Поддерживаемые форматы Макс. FPS Захват звука
SimpleScreenRecorder MP4, MKV, AVI 120 Да (PulseAudio)
FFmpeg Любой (зависит от кодеков) 240+ Да (ALSA/PulseAudio)
VokoscreenNG MP4, WebM 60 Да
OBS Studio MP4, FLV, MKV Неограничено Да (с плагинами)

Для KDE Plasma есть встроенный Spectacle, но он тоже ограничен скриншотами. Для видео рекомендуется VokoscreenNG — лёгкая программа с поддержкой нескольких мониторов и настраиваемым битрейтом. Учтите, что в Linux захват Wayland-сессий (вместо X11) может требовать дополнительных настроек или использования PipeWire.

Как записать экран в Wayland?

В Wayland стандартные методы захвата (например, ffmpeg -f x11grab) не работают. Используйте:

  • wf-recorder для Sway/Hyprland.
  • OBS Studio с плагином pipewire (нужно установить obs-studio и pipewire из репозиториев).
  • Команду: ffmpeg -f kmsgrab -i - -c:v libx264 output.mp4 (требует права root).

4. Профессиональные программы: OBS Studio, Bandicam, Camtasia

Если встроенных средств недостаточно, обратите внимание на специализированный софт. OBS Studio — бесплатная программа с открытым кодом, которая подходит для стриминга и записи в высоком качестве. Она поддерживает многопоточное кодирование, захват нескольких источников (экран + камера + микрофон) и настройку битрейта до 100 Mbps.

Bandicam — платное решение, оптимизированное для записи игр. Его ключевое преимущество — технология Hardware Acceleration, которая снижает нагрузку на CPU за счёт использования GPU (NVIDIA NVENC/AMD AMF/Intel QSV). В бесплатной версии на видео накладывается watermark, а длительность записи ограничена 10 минутами.

  • 🎥 OBS Studio: лучший выбор для стримеров, поддерживает плагины, но требует настройки.
  • 🎮 Bandicam: идеален для геймплея, минимальные лаги, но платный.
  • 📚 Camtasia: для создания обучающих видео с редактором титров и эффектов.
  • ⚙️ ShareX: бесплатный, с открытым кодом, но без поддержки 4K@60FPS.

Важно: при записи игр в 4K с высоким FPS даже мощные ПК могут испытывать просадки производительности. Чтобы избежать лагов, уменьшите разрешение захвата до 1440p или используйте аппаратное кодирование (NVENC).

5. Захват видео с внешнего монитора (HDMI/DisplayPort)

Если вам нужно записать видео с внешнего монитора (например, приставки, второго ПК или телевизора), обычные программы не подойдут — они захватывают только основной экран. Здесь понадобится захватная карта (например, Elgato HD60 S+ или AVerMedia Live Gamer Ultra) или специализированное устройство вроде Blackmagic Design Intensity Shuttle.

Процесс записи через захватную карту:

  1. Подключите HDMI-кабель от источника (ПК, консоль) к входу карты.
  2. Соедините карту с компьютером via USB 3.0 или Thunderbolt.
  3. Установите драйверы и ПО (например, Elgato 4K Capture Utility).
  4. Запустите запись в поддерживаемом разрешении (до 4K@60Hz для топовых моделей).

Бюджетная альтернатива — использование HDMI-сплиттера с функцией записи (например, Magewell Pro Capture). Такие устройства дублируют сигнал на монитор и ПК одновременно. Однако они не захватывают звук через HDMI — для этого нужен отдельный аудиоэкстрактор.

💡

Захватная карта обязательна для записи с игровых консолей (PS5, Xbox Series X) или защищённых источников (Netflix, Disney+). Обычные программы не могут обойти HDCP!

6. Оптимизация качества: разрешение, FPS, битрейт

Качество итогового видео зависит от трёх ключевых параметров: разрешение, частота кадров (FPS) и битрейт. Например, запись в 4K@60FPS с битрейтом 20 Mbps будет выглядеть хуже, чем 1080p@60FPS с битрейтом 50 Mbps из-за артефактов сжатия. Оптимальные настройки зависят от задачи:

  • 📺 Для обучающих видео: 1080p@30FPS, битрейт 8–12 Mbps, кодек H.264.
  • 🎮 Для геймплея: 1440p@60FPS, битрейт 30–50 Mbps, кодек H.265 (HEVC).
  • 🖥️ Для записи рабочего стола: 1080p@15FPS, битрейт 5 Mbps (достаточно для статичных сцен).

Чтобы уменьшить размер файла без потери качества:

  • Используйте переменный битрейт (VBR) вместо постоянного (CBR).
  • Применяйте двухпроходное кодирование (в FFmpeg флаг -pass 2).
  • Для длинных записей разбейте видео на сегменты по 10–15 минут.

Если видео получается рваным или с зелёными артефактами, проблема может быть в:

  • Нехватке ресурсов CPU/GPU (проверьте загрузку в Диспетчере задач).
  • Конфликте кодеков (попробуйте сменить H.264 на H.265).
  • Несовместимости разрешений (например, запись 4K на мониторе с 1080p).
💡

Для проверки качества записи используйте тестовые сцены: быстрое движение мыши по экрану (проверка FPS) и градиенты цветов (проверка артефактов сжатия).

7. Решение распространённых проблем

Даже с правильными настройками запись экрана может идти не так, как задумано. Вот типичные проблемы и их решения:

Проблема Возможная причина Решение
Чёрный экран в записи Аппаратное ускорение отключено Включить в настройках программы (например, NVENC в OBS)
Звук не записывается Не выбран правильный аудиоустройство Проверьте настройки микрофона/системного звука в ПО
Видео тормозит Высокий битрейт или разрешение Уменьшите до 1080p@30FPS или используйте аппаратное кодирование
Нет второго монитора в списке источников Программа не поддерживает Multi-Monitor Используйте OBS Studio или захватную карту
Большой размер файла Слишком высокий битрейт Используйте HEVC (H.265) или уменьшите битрейт до 15–20 Mbps

Если проблема не решена, проверьте:

  • 🔌 Драйверы: обновите драйверы видеокарты и звуковой карты.
  • 🛠️ Конфликты ПО: закройте другие программы, использующие камеру/микрофон (Skype, Discord).
  • 🖥️ Производительность: запись 4K требует SSD (HDD может не справиться с потоком данных).
⚠️ Внимание: некоторые программы (например, Netflix, Amazon Prime Video) блокируют запись экрана из-за защиты HDCP. Обойти это можно только с помощью захватной карты с функцией HDCP stripping (но это может нарушать условия использования сервиса).

FAQ: Частые вопросы о записи видео с монитора

Можно ли записать видео с экрана на слабом ноутбуке?

Да, но придётся снизить настройки. Используйте:

  • Разрешение 720p вместо 1080p.
  • FPS 15–20 вместо 60.
  • Кодек H.264 с низким битрейтом (3–5 Mbps).
  • Программы с аппаратным ускорением (OBS с NVENC или Bandicam).

Если и это не помогает, попробуйте записать отдельные окна вместо всего экрана.

Как записать видео с экрана без watermark?

Большинство бесплатных программ (например, Bandicam, Apowersoft) накладывают watermark в демо-версии. Чтобы его убрать:

  • Используйте OBS Studio (полностью бесплатна).
  • В ShareX watermark отсутствует, но нет поддержки высоких FPS.
  • Для Bandicam купите лицензию (стоимость ~$40) или найдите акцию (иногда раздают ключи бесплатно).

Осторожно: некоторые "крякнутые" версии программ могут содержать вирусы.

Почему запись тормозит, а игра — нет?

Это происходит из-за того, что запись экрана — ресурсоёмкий процесс. Возможные причины:

  • Программа использует программное кодирование (CPU) вместо аппаратного (GPU).
  • Битрейт слишком высок для вашего жесткого диска (попробуйте записать на SSD).
  • Конфликт с антивирусом (добавьте программу для записи в исключения).

Решение: в настройках OBS/Bandicam выберите кодек NVENC (для NVIDIA) или AMF (для AMD) и уменьшите разрешение захвата.

Можно ли записать видео с экрана на Android/iOS?

На мобильных устройствах запись экрана реализована по-другому:

  • 📱 Android: используйте встроенную функцию (в Панель уведомлений → Запись экрана) или приложения вроде AZ Screen Recorder.
  • 🍎 iOS: добавьте кнопку записи в Пункт управления (Настройки → Пункт управления → Настройка элементов → Запись экрана).

Ограничения: на iOS нельзя записывать некоторые приложения (например, Netflix), а на Android может не захватываться звук из игр.

Как записать видео с экрана в 4K без потери качества?

Для записи в 4K без артефактов:

  1. Используйте OBS Studio с настройками: 
    Разрешение: 3840×2160

    FPS: 60
    

    Кодек: NVENC HEVC (для NVIDIA) или x264 с CRF 12–15
    

    Битрейт: 80–100 Mbps
    

    Формат: MKV (менее подвержен ошибкам)

  2. Записывайте на SSD NVMe (HDD не справится с потоком данных).
  3. Отключите фоновые процессы (особенно браузер с открытыми вкладками).
  4. Для игр уменьшите настройки графики (например, отключите сглаживание).

Если видео всё равно "рвётся", попробуйте записать в 1440p с последующим апскейлингом до 4K в редакторе.
