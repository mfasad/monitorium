Как сделать скриншот экрана компьютера: все способы для Windows, Mac и Linux

Создание скриншота (снимка экрана) — базовая задача, с которой сталкивается каждый пользователь ПК. Независимо от того, нужно ли сохранить важную информацию, поделиться ошибкой в программе или зафиксировать результат работы, умение быстро «скринить» монитор экономит время и упрощает общение. Однако методы создания скриншотов различаются в зависимости от операционной системы, версии ОС и даже модели ноутбука.

В этой статье мы разберём все актуальные способы сделать скриншот экрана: от стандартных горячих клавиш до специализированных программ и онлайн-сервисов. Особое внимание уделим нюансам для Windows 10/11, macOS и Linux, а также рассмотрим, как обрезать скриншот сразу после создания и где найти сохранённые файлы. Если вы ищете способ сделать снимок конкретного окна, выделенной области или даже прокручиваемой веб-страницы — здесь найдёте решение.

Стандартные горячие клавиши для скриншотов в Windows

В операционных системах Windows 10 и Windows 11 предусмотрено несколько комбинаций клавиш для создания снимков экрана. Они работают без установки дополнительного ПО и подходят для большинства задач.

Самый простой способ — нажать клавишу PrtScn (Print Screen). Она находится в верхнем ряду клавиатуры, часто рядом с F12. При нажатии PrtScn изображение всего экрана копируется в буфер обмена, и его можно вставить (Ctrl + V) в любой графический редактор (Paint, Photoshop) или документ. Однако этот метод не сохраняет файл автоматически — его придётся экспортировать вручную.

  • 🖥️ PrtScn — скриншот всего экрана в буфер обмена.
  • 🪟 Alt + PrtScn — снимок активного окна (только того, которое открыто сверху).
  • 💾 Win + PrtScn — сохраняет скриншот всего экрана в папку Изображения → Снимки экрана (файл в формате .png).
  • 🎯 Win + Shift + S — вызов инструмента «Ножницы» для выделения области экрана (доступно с Windows 10 Creators Update).

Последняя комбинация (Win + Shift + S) особенно удобна: она позволяет выбрать прямоугольную область, свободную форму или захватить конкретное окно. После выделения скриншот копируется в буфер обмена, а в правом нижнем углу появляется уведомление с предложением редактировать снимок в приложении «Ножницы».

Где Windows сохраняет скриншоты по умолчанию?

Если вы использовали комбинацию Win + PrtScn, система автоматически сохраняет файл в папке C:\Users\[Ваше_имя_пользователя]\Pictures\Screenshots. Имя файла формируется по шаблону Снимок экрана (N).png, где N — порядковый номер. Например: Снимок экрана (1).png, Снимок экрана (2).png и так далее.

Для других методов (например, PrtScn или Alt + PrtScn) файл не сохраняется автоматически — его нужно вставить в редактор и экспортировать вручную. Чтобы не терять скриншоты, рекомендуем сразу же сохранять их в удобную папку или облачное хранилище.

⚠️ Внимание: На некоторых ноутбуках (например, Lenovo, HP) клавиша PrtScn может работать только в комбинации с Fn. Попробуйте нажать Fn + PrtScn, если стандартный способ не срабатывает.

Как сделать скриншот на Mac (macOS)

На компьютерах Apple процесс создания скриншотов отличается от Windows. Здесь используется три основные комбинации клавиш, каждая из которых сохраняет файл на рабочий стол в формате .png:

  • 🖥️ Command (⌘) + Shift + 3 — скриншот всего экрана.
  • 🪟 Command (⌘) + Shift + 4 — выбор области экрана (после нажатия курсор превращается в прицел).
  • 📄 Command (⌘) + Shift + 4 → Пробел — снимок конкретного окна (после нажатия пробела окна подсвечиваются синим).

По умолчанию файлы сохраняются на рабочем столе с именами вида Снимок экрана [дата] в [время].png. Чтобы изменить место сохранения, откройте Терминал и введите команду:

defaults write com.apple.screencapture location /путь/к/папке

killall SystemUIServer

Например, для сохранения в папку Документы используйте:

defaults write com.apple.screencapture location ~/Documents
⚠️ Внимание: На macOS Mojave и новее после создания скриншота в правом нижнем углу появляется миниатюра. Если её проигнорировать, файл сохранится автоматически. Если кликнуть на миниатюру, откроется редактор для обрезки и разметки.

Скриншоты в Linux: инструменты и команды

В дистрибутивах Linux (например, Ubuntu, Fedora, Mint) нет универсальных горячих клавиш — всё зависит от используемой графической оболочки (GNOME, KDE, XFCE). Однако большинство систем поддерживают комбинацию PrtScn для снимка всего экрана и Alt + PrtScn для активного окна.

Для расширенных возможностей (задержка, выделение области) удобно использовать утилиту «Flameshot». Установить её можно через терминал:

  • 🐧 Для Ubuntu/Debian: 
    sudo apt install flameshot
  • 📦 Для Fedora: 
    sudo dnf install flameshot
  • ⚙️ Для Arch Linux: 
    sudo pacman -S flameshot

После установки Flameshot можно запустить из меню приложений или назначить собственную горячую клавишу. Утилита позволяет:

  • 🔍 Выделять область экрана с pixel-точностью.
  • 🎨 Добавлять стрелочки, текст и размытие.
  • 💾 Сохранять в PNG, JPG или копировать в буфер.
💡

В GNOME (например, в Ubuntu) можно использовать встроенную утилиту gnome-screenshot. Запустите её из терминала с параметрами, например: 

gnome-screenshot -a -f ~/screenshot.png
где -a — выделение области, -f — путь для сохранения.

Программы для скриншотов: топ-5 утилит с расширенными функциями

Стандартные средства ОС подходят для базовых задач, но если вам нужны скриншоты с задержкой, запись видео с экрана, аннотации или захват прокручиваемых страниц, стоит обратить внимание на специализированное ПО. Вот пятерка лучших программ:

  1. ShareX (Windows) — бесплатная утилита с открытым кодом. Поддерживает захват областей, GIF-анимацию, OCR-распознавание текста и загрузку на облачные сервисы. Минус: сложный интерфейс для новичков.
  2. Snagit (Windows/macOS) — платное решение с продвинутым редактором. Позволяет создавать пошаговые инструкции, добавлять эффекты и экспортировать в разные форматы. Плюс: интеграция с Microsoft Office.
  3. Lightshot (Windows/macOS) — простая программа для быстрых снимков с возможностью поиска похожих изображений в интернете. Особенность: можно сразу загрузить скриншот на сервер и получить ссылку.
  4. Greenshot (Windows) — бесплатный аналог Snagit с открытым кодом. Поддерживает плагины для расширения функционала. Минус: нет версии для macOS.
  5. Monosnap (Windows/macOS) — кроссплатформенная утилита с облачным хранилищем. Удобна для командной работы. Плюс: можно рисовать на скриншоте до его сохранения.

Для большинства пользователей хватит возможностей ShareX или Lightshot. Если же вам нужны профессиональные инструменты (например, для создания туториалов), обратите внимание на Snagit.

Как сделать скриншот прокручиваемой страницы?

Для захвата длинных веб-страниц (например, ленты в соцсетях) подойдут расширения для браузеров:

- GoFullPage (Chrome/Firefox) — сохраняет страницу в PDF или PNG.

- FireShot (Chrome) — позволяет захватить видимую часть или всю страницу.

- Awesome Screenshot — поддерживает аннотации и размытие.

В ShareX также есть функция Capture Scrolling Window, но она работает не со всеми сайтами.

Скриншоты онлайн: когда нет доступа к ПК

Если вам нужно сделать снимок экрана на чужом компьютере (где нельзя устанавливать программы) или с мобильного устройства, на помощь придут онлайн-сервисы. Они работают прямо в браузере и не требуют регистрации. Популярные варианты:

  • 🌐 Screenshot.net — захват видимой области браузера или загрузка файла для разметки.
  • 🖼️ ImgBB — загрузка скриншота с компьютера и получение ссылки для Sharing.
  • 📱 WebCapture — создание снимков веб-страниц по URL (полезно для сохранения главной страницы сайта).

Основной недостаток онлайн-сервисов — они не могут захватить системные окна или игры в полноэкранном режиме, так как работают внутри браузера. Зато они идеальны для создания снимков веб-страниц или редактирования уже готовых изображений.

⚠️ Внимание: При использовании онлайн-сервисов для конфиденциальных данных (например, скриншоты банковских операций) убедитесь, что сайт использует HTTPS и не сохраняет файлы на своих серверах дольше необходимого.

Как скринить экран в играх и полноэкранных приложениях

Многие игры и программы (например, видеоплееры) блокируют стандартные комбинации клавиш для скриншотов. В таких случаях помогают:

  • 🎮 Встроенные инструменты игры. Например, в Steam скриншот делается клавишей F12, а файлы сохраняются в Steam\userdata\[ID]\760\remote\[ID_игры]\screenshots.
  • 🖥️ Режим окна. Переключите игру в оконный режим (Alt + Enter) и используйте стандартные методы (например, Win + Shift + S).
  • 📹 Программы для записи экрана. OBS Studio или NVIDIA ShadowPlay могут делать снимки даже в полноэкранных приложениях.

Для владельцев видеокарт NVIDIA доступна функция «Мгновенная фотография» через панель GeForce Experience. Чтобы её активировать:

  1. Откройте GeForce Experience (Alt + Z).
  2. Перейдите в Настройки → Сочетания клавиш.
  3. Назначьте клавишу для Мгновенная фотография (по умолчанию Alt + F1).
  4. Снимки сохранятся в Видео → Снимки экрана.
💡

Для скриншотов в играх лучше использовать специализированные инструменты (Steam, GeForce Experience, OBS), так как стандартные методы часто блокируются античитами или DirectX.

FAQ: Частые вопросы о скриншотах

Можно ли сделать скриншот с задержкой?

Да, для этого подойдут программы вроде ShareX или Snagit, где можно настроить таймер (например, 5 или 10 секунд). В Windows 10/11 задержку поддерживает встроенное приложение «Ножницы» (откройте его через поиск и выберите Задержка в меню).

Как сделать скриншот только одного монитора в многомониторной системе?

В Windows при нажатии Win + PrtScn сохраняются снимки всех мониторов. Чтобы захватить только один экран, используйте Win + Shift + S и вручную выделите нужную область. В macOS при выборе области (Command + Shift + 4) можно тянуть курсор только по одному дисплею.

Почему скриншоты сохраняются в низком качестве?

Качество зависит от разрешения экрана и формата файла. Если снимки получаются размытыми, проверьте:

  • Разрешение монитора (должно быть родным, например, 1920×1080 для Full HD).
  • Формат сохранения ( PNG лучше JPG для текста и графики).
  • Масштабирование в системе (в Windows проверьте в Параметры → Система → Экран, чтобы масштаб был 100%).
Как сделать скриншот с меню (например, контекстного меню правой кнопки мыши)?summary>

Стандартные методы не захватывают контекстные меню, так как они исчезают при нажатии клавиш. Решения:

  • Используйте ShareX с функцией Capture Last Region (позволяет сделать снимок после закрытия меню).
  • В Windows 10/11 откройте «Ножницы», выберите Задержка → 5 секунд, затем откройте меню.
  • На macOS используйте Command + Shift + 4, затем нажмите Пробел и кликните по меню.
Можно ли автоматизировать создание скриншотов (например, каждые 10 минут)?

Да, для этого подойдут:

  • ShareX — поддерживает задачи по расписанию (Tools → Task Scheduler).
  • AutoHotkey — скрипт для автоматического нажатия PrtScn с заданным интервалом.
  • Cron (Linux/macOS) — можно настроить запуск scrot или gnome-screenshot по времени.
