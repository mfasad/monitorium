Как сделать снимок экрана монитора: все способы от Print Screen до продвинутых утилит

  • 2. Встроенный инструмент "Ножницы" (Snipping Tool) в Windows 10/11

Сохранение изображения с монитора — задача, с которой сталкивается каждый пользователь ПК. Будь то фиксация ошибки для техподдержки, сохранение важного фрагмента видео или создание обучающего материала, скриншоты экономят время и упрощают коммуникацию. Однако не все знают, что способов сделать снимок экрана существует больше десятка: от стандартных горячих клавиш до специализированных программ с расширенными функциями.

В этой статье мы разберём не только базовые методы вроде PrtScn, но и малоизвестные трюки для Windows 11/10, macOS и Linux. Вы узнаете, как сохранить полноэкранный снимок, выделенную область или даже прокручиваемую веб-страницу в одно касание. А для профессионалов — обзор утилит с поддержкой 4K, анимации и облачного хранения. Начнём с самого простого!

1. Стандартные горячие клавиши в Windows: от Print Screen до Win+Shift+S

Большинство пользователей привыкли к клавише PrtScn (Print Screen), но в современных версиях Windows её возможности расширились. Теперь не нужно вручную вставлять снимок в Paint — система сама сохраняет файлы в папку Снимки экрана.

Вот полный список комбинаций:

  • 🖥️ PrtScn — копирует весь экран в буфер обмена (нужно вставить в графический редактор).
  • 🖼️ Win + PrtScn — сохраняет снимок всего экрана в C:\Users\[Имя_пользователя]\Pictures\Screenshots.
  • 🪟 Alt + PrtScn — копирует в буфер только активное окно.
  • ✂️ Win + Shift + S — открывает инструмент "Ножницы" для выделения произвольной области (результат копируется в буфер).

Последний способ особенно удобен для захвата фрагментов: после нажатия Win+Shift+S экран затемняется, и вы можете выбрать прямоугольник, свободную форму или даже конкретное окно. Снимок автоматически копируется в буфер — остаётся только вставить его в документ или редактор.

2. Встроенный инструмент "Ножницы" (Snipping Tool) в Windows 10/11

Утилита "Ножницы" (Snipping Tool) — это продвинутая альтернатива горячим клавишам. Она позволяет не только выделять области, но и добавлять аннотации, сохранять в разных форматах (.png, .jpg, .gif) и даже записывать видео с экрана (в Windows 11).

Как открыть и использовать:

  1. Нажмите Win + S и введите "Ножницы".
  2. Выберите режим захвата:
    • 📐 Прямоугольник — стандартный выбор области.
    • 🎨 Произвольная форма — рисуете границу мышкой.
    • 🪟 Окно — захват конкретного приложения.
    • 🖥️ Весь экран — аналог PrtScn.
  • После захвата откроется редактор, где можно обрезать изображение, добавить текст или стрелки.
  • Сохраните файл через Ctrl + S или кнопку "Сохранить как".

    • В Windows 11 инструмент обновили: теперь он поддерживает задержку захвата до 10 секунд (полезно для фиксации контекстных меню) и интеграцию с OneDrive для автоматической загрузки в облако.

    3. Скриншоты на macOS: Command+Shift+3/4 и предварительный просмотр

    На компьютерах Apple процесс немного отличается, но не менее удобен. Основные комбинации:

    • 🖥️ Command + Shift + 3 — сохраняет весь экран на рабочий стол в формате .png.
    • 📐 Command + Shift + 4 — позволяет выбрать область мышкой.
    • 🪟 Command + Shift + 4 + Пробел — захват конкретного окна (появится подсветка).
    • 🎥 Command + Shift + 5 — открывает панель с расширенными опциями (включая запись видео).

    По умолчанию снимки сохраняются на рабочий стол с именем Снимок экрана [дата] в [время].png. Чтобы изменить папку сохранения, используйте команду в Терминале:

    defaults write com.apple.screencapture location /путь/к/папке

    killall SystemUIServer

    Для редактирования скриншота сразу после создания удерживайте клавишу Control — снимок откроется в приложении Предварительный просмотр (Preview), где можно обрезать изображение, добавить подписи или конвертировать в другой формат.

    ⚠️ Внимание: На macOS Ventura и новее при использовании Command+Shift+5 снимки по умолчанию сохраняются в формате .heic (более компактном, но не все программы его поддерживают). Чтобы вернуть .png, измените формат в настройках панели скриншотов.

    4. Продвинутые программы для скриншотов: Lightshot, ShareX, Snagit

    Если встроенных инструментов недостаточно, обратите внимание на сторонние утилиты. Они предлагают:

    • 🔍 Захват прокручиваемых страниц (например, длинных статей).
    • ☁️ Автоматическую загрузку в облако с генерацией ссылки.
    • 🎨 Расширенный редактор (размытие, фигуры, эффекты).
    • 📊 История снимков для быстрого доступа к прошлым скриншотам.

    Сравнение популярных программ:

    Программа Бесплатная версия Захват прокрутки Облачное хранилище Поддержка 4K
    Lightshot Да Нет Да (prnt.sc) Да
    ShareX Да Да (плагин) Да (много сервисов) Да
    Snagit Нет (30-дневный триал) Да Да (TechSmith) Да
    Greenshot Да Нет Нет Да

    ShareX — лидер среди бесплатных решений: поддерживает OCR (распознавание текста на скриншотах), запись GIF и интеграцию с Google Drive, Dropbox. Для профессионалов подойдёт Snagit — он умеет снимать видео с веб-камеры одновременно со скриншотом и имеет библиотеку шаблонов для аннотаций.

    💡

    В ShareX можно настроить автоматическое добавление водяного знака на все снимки. Это полезно для защиты авторских прав при публикации скриншотов в интернете.

    5. Скриншоты в Linux: от Print Screen до Flameshot

    В дистрибутивах Linux нет единого стандарта для скриншотов, но большинство окружений рабочего стола (GNOME, KDE, XFCE) имеют встроенные инструменты. Базовые комбинации:

    • 🖥️ PrtScn — сохраняет весь экран в ~/Pictures.
    • 📐 Shift + PrtScn — выбор области.
    • 🪟 Alt + PrtScn — активное окно.

    Для расширенных возможностей установите Flameshot:

    sudo apt install flameshot  # Для Debian/Ubuntu

    sudo dnf install flameshot  # Для Fedora

    Flameshot позволяет:

    • 🎨 Добавлять стрелки, текст и размытие до сохранения.
    • 🔗 Копировать снимок сразу в буфер или сохранить в файл.
    • 📁 Настраивать папку сохранения и формат (PNG, JPG, BMP).

    Чтобы назначить Flameshot на клавишу PrtScn, перейдите в Настройки → Клавиатура → Сочетания клавиш и создайте новое сочетание с командой flameshot gui.

    6. Как сделать скриншот с высоким разрешением (4K, Retina)

    При работе с мониторами 4K или Retina-дисплеями стандартные методы могут сохранять снимки в пониженном разрешении. Чтобы зафиксировать реальное качество:

    В Windows:

    • Используйте ShareX с настройкой "Capture → Screen recording → Use screen resolution".
    • В Snagit выберите "Image → Capture → Full Screen" и отключите опцию "Reduce colors".

    На macOS:

    • По умолчанию снимки сохраняются в разрешении экрана. Если изображение получается размытым, проверьте настройки дисплея в Системные настройки → Мониторы (должно быть выбрано По умолчанию для дисплея).

    В Linux:

    • В Flameshot перейдите в Configuration → Save и установите "Save quality" на 100%.

    Критическая деталь: На ноутбуках с гибридной графикой (например, NVIDIA Optimus) скриншоты могут сохраняться с артефактами. В этом случае временно переключитесь на дискретную видеокарту или используйте внешний монитор.

    7. Автоматизация: скриншоты по таймеру и через командную строку

    Иногда нужно сделать снимок в строго определённое время или по событию. Для этого подойдут:

    В Windows:

    • 🕒 Snipping Tool с задержкой: в Windows 11 выберите "Задержка" (до 10 секунд) перед захватом.
    • 📜 PowerShell: 
      Add-Type -AssemblyName System.Windows.Forms

      $screen = [System.Windows.Forms.Screen]::PrimaryScreen.Bounds
      

      $bitmap = New-Object System.Drawing.Bitmap($screen.Width, $screen.Height)
      

      $graphics = [System.Drawing.Graphics]::FromImage($bitmap)
      

      $graphics.CopyFromScreen($screen.Location, [System.Drawing.Point]::Empty, $screen.Size)
      

      $bitmap.Save("C:\screenshot.png", [System.Drawing.Imaging.ImageFormat]::Png)

    На macOS:

    • ⏱️ Используйте screencapture в Терминале: 
      screencapture -T 5 capture.png  # Задержка 5 секунд

    В Linux:

    • 🐧 Утилита scrot (установите через sudo apt install scrot): 
      scrot -d 10 'screenshot_%Y-%m-%d.png'  # Задержка 10 секунд

    Для регулярных задач (например, мониторинг изменения данных на экране) можно написать простой скрипт на Python с библиотекой Pillow:

    from PIL import ImageGrab

    import time
    


    time.sleep(5)  # Задержка 5 секунд
    

    screenshot = ImageGrab.grab()
    

    screenshot.save("autosave.png")

    💡

    Командная строка и скрипты полезны для автоматизации, но требуют прав администратора для захвата некоторых защищённых окон (например, видеоплееров с DRM).

    Частые проблемы и их решения

    Даже с простыми инструментами иногда возникают трудности. Рассмотрим типичные случаи:

    1. Скриншот получается чёрным:

    • 🎮 Видеоигры или приложения с DRM (например, Netflix, Disney+) блокируют захват экрана. Решение: используйте внешнюю камеру или специализированное ПО вроде OBS Studio.
    • 🖥️ На некоторых ноутбуках с гибридной графикой помогает отключение дискретной видеокарты в панели управления NVIDIA/AMD.

    2. Некорректные цвета:

    • 🎨 Проверьте цветовой профиль монитора в настройках Windows (Параметры → Система → Экран → Дополнительные параметры экрана). Для профессиональной работы используйте калибровку через DisplayCAL.

    3. Не работает Print Screen:

    • 🔧 На некоторых клавиатурах (Microsoft Surface, некоторые модели Acer) требуется нажать Fn + PrtScn.
    • 🔄 Перезагрузите клавиатуру через Диспетчер устройств или обновите драйверы.
    ⚠️ Внимание: В виртуальных машинах (VirtualBox, VMware) горячие клавиши хостовой системы могут конфликтовать с гостевой ОС. Используйте встроенные инструменты виртуальной машины или настройте "гостевые дополнения" (Guest Additions).
    Почему скриншоты в играх получаются зелёными или фиолетовыми?

    Это связано с особенностями рендеринга в DirectX 12 или Vulkan. Решение:

    1. Отключите полноэкранную оптимизацию в свойствах ярлыка игры.

    2. Используйте встроенный в игру фотомод (например, NVIDIA Ansel).

    3. Сделайте снимок через Steam Overlay (F12).

    FAQ: Ответы на популярные вопросы

    Можно ли сделать скриншот с айфона и передать его на компьютер?

    Да, для этого:

    1. Нажмите Боковая кнопка + Увеличение громкости (на iPhone без кнопки Home) или Кнопка Home + Кнопка питания (на старых моделях).
    2. Откройте фотографию в приложении "Фото", нажмите "Поделиться" и выберите AirDrop (для Mac) или отправьте на почту.
    3. Альтернатива: используйте iCloud — снимки автоматически синхронизируются с компьютером.
    Как сделать скриншот длинной веб-страницы целиком?

    Стандартные инструменты не поддерживают прокрутку, но есть обходные пути:

    • 🌐 Расширение GoFullPage для Chrome/Firefox — сохраняет страницу в PNG или PDF.
    • 📱 В ShareX выберите "Capture → Scrolling capture" (требует настройки).
    • 💻 На macOS используйте Papercut или CleanShot X.
    Где хранятся скриншоты, сделанные через Steam?

    Файлы сохраняются в папке:

    C:\Program Files (x86)\Steam\userdata\[Ваш_ID]\760\remote\[ID_игры]\screenshots

    Чтобы найти свой [Ваш_ID], откройте Steam, перейдите в Настройки → Интерфейс и включите отображение вашего профиля в адресной строке.

    Можно ли сделать скриншот с видео в YouTube?

    Да, но:

    • 📺 Стандартные методы (PrtScn, "Ножницы") сработают, но качество будет низким из-за сжатия видео.
    • 🎬 Для кадра в высоком разрешении поставьте видео на паузу, разверните на весь экран и используйте ShareX с настройкой "Capture → Screen recording → High quality".
    • ⚠️ Некоторые видео (с DRM) блокируют захват — в этом случае сфотографируйте экран на телефон.
    Как отредактировать скриншот перед сохранением?

    В большинстве программ есть встроенный редактор:

    • 🖌️ "Ножницы" (Windows): инструменты для обрезки, ручки, маркера и ластика.
    • 🎨 Preview (macOS): поддержка слоёв, коррекция цвета, добавление фигур.
    • ✏️ ShareX: более 20 инструментов, включая размытие, пикселизацию и добавление водяных знаков.

    Для профессионального редактирования подойдёт Adobe Photoshop или бесплатный GIMP.
