Как сделать скриншот экрана на компьютере: полное руководство для Windows и macOS

Создание скриншотов (снимков экрана) — одна из самых востребованных операций при работе за компьютером. Без этого не обойтись ни при составлении инструкций, ни при фиксации ошибок, ни при сохранении важной информации с веб-страниц. Однако даже опытные пользователи не всегда знают все способы создания скриншотов и их нюансы. В этой статье мы разберём все актуальные методы для Windows 10/11 и macOS, включая горячие клавиши, встроенные утилиты и специализированное ПО.

Особое внимание уделим скрытым возможностям системных инструментов — например, как сделать скриншот активного окна без фона или захватить прокручиваемую веб-страницу целиком. А для тех, кто работает с графикой профессионально, приведём сравнение популярных программ для расширенного скриншотинга с поддержкой аннотаций и облачного сохранения.

1. Горячие клавиши для скриншотов в Windows

Самый быстрый способ сделать снимок экрана — использовать комбинации клавиш. В Windows их несколько, и каждая имеет своё предназначение. Основное преимущество этого метода — мгновенный результат без необходимости открывать дополнительные программы.

Самая универсальная комбинация — PrtScn (Print Screen). При её нажатии снимок всего экрана копируется в буфер обмена, откуда его можно вставить в любой графический редактор (Paint, Photoshop) или документ. Однако у этого метода есть недостаток: если у вас несколько мониторов, в буфер попадёт изображение со всех экранов одновременно.

  • 🖥️ PrtScn — скриншот всего экрана (всех мониторов) в буфер обмена
  • 🪟 Alt + PrtScn — скриншот только активного окна в буфер обмена
  • 💾 Win + PrtScn — скриншот всего экрана с автоматическим сохранением в папку Снимки экрана
  • 🎮 Win + Alt + PrtScn — скриншот активной игры или приложения (только для Windows 10/11 с поддержкой Xbox Game Bar)

Обратите внимание на комбинацию Win + Shift + S — она вызывает инструмент "Фрагмент и ручка", о котором мы подробно расскажем в следующем разделе. Этот метод позволяет выбирать произвольную область экрана для захвата, что особенно удобно при работе с документами или веб-страницами.

💡

Если клавиша PrtScn не работает, проверьте, не отключена ли она в BIOS/UEFI или не конфликтует ли с драйверами клавиатуры. На некоторых ноутбуках (например, Lenovo или HP) для активации Print Screen требуется нажать Fn.

2. Инструмент"Ножницы" (Snipping Tool) и"Фрагмент" в Windows 10/11

Встроенная утилита "Ножницы" (в Windows 11 переименована в "Фрагмент") предлагает больше возможностей, чем простые горячие клавиши. С её помощью можно:

  • 📐 Выбирать произвольную область экрана (включая непрямоугольные формы в Windows 11)
  • 🖼️ Делать скриншот всей страницы с прокруткой (только в браузере Microsoft Edge)
  • ⏱️ Устанавливать задержку перед захватом (от 3 до 10 секунд)
  • 🎨 Добавлять аннотации (текст, стрелки, выделение) сразу после создания скриншота

Чтобы открыть инструмент, нажмите Win + Shift + S или найдите его через поиск по системе. В Windows 11 интерфейс стал более современным: после захвата области появится уведомление с предложением редактировать снимок. В Windows 10 скриншот сразу копируется в буфер обмена, а для редактирования нужно открыть полную версию утилиты.

В Windows 11 инструмент"Фрагмент" умеет распознавать текст на скриншотах (OCR) и копировать его в буфер обмена — это уникальная функция среди встроенных утилит. Однако для полноценной работы OCR требуется подключение к интернету (данные отправляются на серверы Microsoft для обработки).

3. Скриншоты на macOS: комбинации клавиш и утилита"Снимок экрана"

На компьютерах Apple процесс создания скриншотов организован иначе, чем в Windows. Здесь нет клавиши PrtScn, но зато есть несколько специализированных комбинаций с использованием клавиши Command (⌘):

  • 🖥️ ⌘ + Shift + 3 — скриншот всего экрана (сохраняется на рабочий стол)
  • 🪟 ⌘ + Shift + 4 — выбор области экрана для захвата
  • 📄 ⌘ + Shift + 4 → Пробел — скриншот конкретного окна (появится камера при наведении)
  • 💾 ⌘ + Ctrl + Shift + 3/4 — скриншот в буфер обмена (без сохранения файла)

По умолчанию снимки сохраняются на рабочий стол в формате .png. Изменить формат (на .jpg, .pdf и др.) или папку для сохранения можно через терминал, введя команду:

defaults write com.apple.screencapture type jpg && killall SystemUIServer

Для расширенных возможностей (например, запись видео с экрана или захват с задержкой) используйте встроенную утилиту "Снимок экрана". Чтобы открыть её, нажмите ⌘ + Shift + 5. Здесь доступны:

  • 🎥 Запись всего экрана или выбранной области
  • ⏱️ Таймер задержки (5 или 10 секунд)
  • 📍 Показ/скрытие указателя мыши на скриншоте

4. Специализированные программы для скриншотов

Если встроенных инструментов недостаточно, на помощь приходят сторонние программы. Они предлагают расширенный функционал: захват прокручиваемых страниц, запись видео с экрана, облачное сохранение и редактирование. Вот самые популярные решения:

Программа Ключевые возможности Бесплатная версия Поддерживаемые ОС
Lightshot Быстрый захват области, редактирование, загрузка в облако, поиск похожих изображений Да Windows, macOS
ShareX Захват прокручиваемых страниц, запись GIF, OCR, более 50 инструментов редактирования Да Только Windows
Snagit Профессиональное редактирование, шаблоны аннотаций, интеграция с Microsoft Office Нет (30-дневный триал) Windows, macOS
Greenshot Поддержка плагинов, экспорт в Confluence/Jira, настройка горячих клавиш Да Только Windows

Для большинства пользователей оптимальным выбором станет ShareX — это бесплатная программа с открытым исходным кодом, которая поддерживает захват прокручиваемых страниц (например, длинных чатов или веб-сайтов). Её единственный недостаток — отсутствие версии для macOS.

Если вам нужна кроссплатформенность, обратите внимание на Lightshot или Snagit. Последний особенно популярен среди блогеров и создателей обучающих материалов благодаря встроенным шаблонам для аннотаций (стрелки, выноски, размытие).

Как сделать скриншот прокручиваемой страницы в ShareX?

1. Установите и запустите ShareX.

2. Выберите в меню"Захват" →"Захват прокручиваемой области".

3. Обведите рамкой область браузера с прокручиваемым контентом.

4. Дождитесь автоматической прокрутки и склейки скриншота.

5. Сохраните результат в нужном формате (PNG, JPG или PDF).

5. Скриншоты в играх: Steam, Xbox Game Bar и NVIDIA ShadowPlay

Геймеры часто сталкиваются с необходимостью зафиксировать игровой момент. Для этого не обязательно использовать стандартные инструменты системы — многие платформы и видеокарты имеют встроенные решения:

  • 🎮 Steam: нажмите F12 — снимок сохранится в облако и на ваш ПК (папка Steam\userdata\<ID>\760\remote)
  • 🖥️ Xbox Game Bar (Windows 10/11): Win + Alt + PrtScn — сохраняет скриншот в Видео\Захват
  • 📊 NVIDIA ShadowPlay (для видеокарт NVIDIA): настройте горячие клавиши в GeForce Experience (по умолчанию Alt + F1)
  • 🎯 AMD ReLive (для видеокарт AMD): включите в AMD Software и используйте Ctrl + Shift + S

Важно: скриншоты, сделанные через Xbox Game Bar, могут иметь ниже качество, чем при использовании специализированных утилит. Это связано с оптимизацией для записи игрового процесса. Для максимального качества лучше использовать NVIDIA ShadowPlay или AMD ReLive, где можно настроить разрешение и битрейт.

💡

Для стримеров и профессиональных геймеров лучший выбор — NVIDIA ShadowPlay или AMD ReLive, так как они позволяют записывать не только скриншоты, но и видео с минимальными потерями FPS.

Если вы играете на macOS, то стандартные комбинации клавиш (⌘ + Shift + 3/4) сработают и в полноэкранном режиме большинства игр. Однако некоторые titles (особенно с античитами) могут блокировать создание скриншотов — в этом случае поможет внешняя камера или запись экрана через QuickTime Player.

6. Проблемы при создании скриншотов и их решения

Иногда пользователи сталкиваются с трудностями при попытке сделать снимок экрана. Рассмотримные проблемы и способы их устранения:

⚠️ Внимание: Если скриншоты делаются чёрными или с артефактами, проверьте драйверы видеокарты. Обновите их через официальный сайт производителя (NVIDIA, AMD, Intel) или используйте утилиту Display Driver Uninstaller (DDU) для полной переустановки.
  • 🖼️ Скриншот не сохраняется: проверьте права доступа к папке Снимки экрана (в Windows) или рабочему столу (в macOS). Попробуйте сохранить в другую директорию.
  • 🔒 Заблокирован захват в игре/приложении: некоторые программы (например, Netflix или игры с Denuvo) блокируют скриншоты. Используйте внешнюю камеру или запись экрана.
  • 🖥️ На скриншоте отображается только один монитор: в Windows комбинация PrtScn захватывает все экраны, но некоторые программы (например, Lightshot) могут требовать дополнительных настроек для мультимониторных систем.
  • 📱 Не работают горячие клавиши на ноутбуке: на некоторых моделях (например, ASUS или Acer) требуется нажимать Fn + PrtScn. Проверьте настройки BIOS/UEFI.

Если вы используете виртуальную машину (например, VirtualBox или VMware), то стандартные комбинации клавиш могут не работать. В этом случае:

  1. Установите Дополнения гостевой ОС (Guest Additions) для VirtualBox или VMware Tools.
  2. Используйте хост-комбинации: для VirtualBox это Host + E (по умолчанию Правая Ctrl + E).
  3. Либо делайте скриншоты с хост-машины (например, фотографируйте экран виртуальной машины).

7. Оптимизация и обработка скриншотов

Создание скриншота — только половина дела. Часто требуется его оптимизировать (уменьшить размер файла) или отредактировать (добавить стрелки, размыть конфиденциальную информацию). Вот несколько советов:

  • 📉 Уменьшение размера: используйте TinyPNG или Squoosh для сжатия без потери качества. Для пакетной обработки подойдёт XnConvert.
  • 🖌️ Редактирование: в Windows 11 встроенный инструмент"Фрагмент" позволяет добавлять текст и стрелки. Для более сложных задач используйте GIMP (бесплатно) или Adobe Photoshop.
  • 🔍 Распознавание текста (OCR): если на скриншоте есть текст, который нужно редактировать, используйте Microsoft OneNote (встроенный OCR) или Adobe Acrobat.
  • 🌐 Облачное сохранение: сервисы вроде Imgur, Lightshot или Google Drive позволяют быстро поделиться скриншотом по ссылке.

Для профессиональной работы с скриншотами (например, создания мануалов или презентаций) рекомендуем использовать Snagit. Эта программа позволяет:

  • Автоматически нумеровать скриншоты в серии.
  • Применять шаблоны оформления (например, для обозначения кнопок или полей ввода).
  • Экспортировать в PowerPoint или Word с сохранением редактируемых элементов.

Если вам нужно часто делать скриншоты веб-страниц, обратите внимание на расширения для браузеров:

  • GoFullPage (для Chrome и Firefox) — захватывает всю страницу с прокруткой.
  • Awesome Screenshot — позволяет редактировать и комментировать скриншоты прямо в браузере.
  • FireShot — сохраняет в PDF или отправляет на печать.

FAQ: Частые вопросы о скриншотах

🔹 Как сделать скриншот только одного монитора в многомониторной системе?

В Windows стандартные горячие клавиши (PrtScn) захватывают все экраны. Чтобы сделать снимок только одного монитора:

  1. Используйте Win + Shift + S и вручную выделите нужный экран.
  2. Или воспользуйтесь программой Lightshot/ShareX, где можно выбрать конкретный монитор в настройках захвата.

В macOS при нажатии ⌘ + Shift + 4 затем Пробел вы можете выбрать конкретный дисплей для захвата.

🔹 Почему скриншоты в играх получаются чёрными?

Это типичная проблема для игр с защитой от читов (например, Denuvo или Easy Anti-Cheat). Решения:

  • Используйте встроенные инструменты игры (например, F12 в Steam).
  • Запустите игру в оконном режиме и сделайте скриншот стандартными средствами ОС.
  • Для NVIDIA/AMD используйте ShadowPlay/ReLive — они обходят некоторые ограничения.

Если ничего не помогает, сфотографируйте экран внешней камерой (например, смартфоном).

🔹 Как сделать скриншот с задержкой?

Задержка полезна, когда нужно подготовить интерфейс (например, открыть выпадающее меню). Способы:

  • В Windows: инструмент"Фрагмент" (Win + Shift + S) → нажмите на"⏱️ Задержка" и выберите 3, 5 или 10 секунд.
  • В macOS: утилита"Снимок экрана" (⌘ + Shift + 5) → кнопка"Параметры" → укажите задержку.
  • В ShareX: в настройках захвата выберите"Задержка перед захватом" (до 60 секунд).
🔹 Куда сохраняются скриншоты по умолчанию?

Зависит от ОС и метода захвата:

  • Windows:
    • PrtScn → только в буфер обмена (не сохраняется в файл).
    • Win + PrtScn → папка Этот компьютер → Изображения → Снимки экрана.
    • "Ножницы"/"Фрагмент" → можно выбрать папку при сохранении.
  • macOS: рабочий стол (файлы называются Снимок экрана [дата] в [время].png).
  • Steam: Steam\userdata\<ваш_ID>\760\remote.

Папку сохранения можно изменить в настройках большинства программ для скриншотов.

🔹 Можно ли сделать скриншот с touchbar на MacBook Pro?

Да, но это требует специальной комбинации:

  1. Нажмите ⌘ + Shift + 6 — это сделает снимок содержимого Touch Bar.
  2. Файл сохранится на рабочем столе в формате .png.

Обратите внимание: этот метод работает только на моделях MacBook Pro с Touch Bar (2016 года и новее).
