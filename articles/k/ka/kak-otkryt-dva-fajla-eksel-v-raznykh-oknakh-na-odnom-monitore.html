Как открыть два файла Excel в разных окнах на одном мониторе: пошаговые инструкции для всех версий

Работа с несколькими таблицами Microsoft Excel одновременно — стандартная задача для аналитиков, бухгалтеров и менеджеров. Однако по умолчанию программа открывает новые файлы в одном окне на разных вкладках, что усложняет сравнение данных или копирование формул. Если у вас только один монитор, решение всё равно есть: можно принудительно разделить окна Excel по рабочему пространству, используя встроенные функции Windows, горячие клавиши или настройки самой программы.

В этой статье мы разберём 5 проверенных способов открыть два файла Excel в разных окнах на одном экране — от базовых (для новичков) до продвинутых (для опытных пользователей). Также вы узнаете, как закрепить окна рядом для удобного перетаскивания данных, почему иногда методы не работают, и что делать, если Excel упорно открывает всё в одной вкладке. Инструкции актуальны для Windows 10/11, macOS и даже для онлайн-версии Excel 365.

Прежде чем переходить к способам, убедитесь, что у вас установлена последняя версия Microsoft Office (или хотя бы не старше 2016 года). В старых версиях (2010 и ранее) функционал многозадачности ограничен, и некоторые методы могут не сработать. Также проверьте, не включён ли режим Открывать все документы в одном окне — его нужно отключить (об этом подробнее в разделе про настройки Excel).

1. Способ: Горячие клавиши Windows для разделения окон

Самый быстрый метод — использовать стандартные сочетания клавиш Windows для управления окнами. Он работает в Windows 10/11 и не требует изменений в настройках Excel. Вот как это сделать:

  1. Откройте первый файл Excel (двойной клик по документу или через меню Файл → Открыть).
  2. Зажмите Ctrl + N, чтобы создать новое окно Excel (или откройте второй файл через Файл → Открыть в этом окне).
  3. Выделите первое окно (кликните по нему на панели задач).
  4. Нажмите Win + Стрелка влево — окно займёт левую половину экрана.
  5. Выделите второе окно и нажмите Win + Стрелка вправо — оно зафиксируется справа.

Теперь оба файла будут открыты в отдельных окнах, прикреплённых к краям монитора. Чтобы изменить пропорции, перетащите разделительную линию мышью. Этот метод работает и для трёх окон: используйте Win + Стрелка вверх/вниз для размещения в углу экрана.

⚠️ Внимание: Если после нажатия Win + Стрелка окно не фиксируется, проверьте, не отключена ли функция Закрепление окон в настройках Windows. Для этого перейдите в Параметры → Система → Многозадачность и убедитесь, что опция Закрепление окон включена.

2. Способ: Настройки Excel для открытия в отдельных окнах

По умолчанию Excel может открывать все файлы в одном окне на разных вкладках. Чтобы это исправить, нужно изменить параметры программы:

  1. Откройте Excel и перейдите в Файл → Параметры (или Excel → Настройки на Mac).
  2. Выберите раздел Дополнительно.
  3. Прокрутите до блока Экран.
  4. Снимите галочку с пункта Показывать все окна на панели задач (в некоторых версиях — Отображать все окна в одном).
  5. Также убедитесь, что отключён параметр Игнорировать другие приложения, использующие DDE (если он есть).
  6. Нажмите OK и перезапустите Excel.

После этих изменений каждый новый файл будет открываться в отдельном окне, которое можно расположить на экране независимо от других. Если опция отсутствует в вашей версии, попробуйте следующий способ.

Excel 2021/365 Файл → Параметры → Экран Отображать все окна в одном

Если после изменений настройки не применяются, закройте все окна Excel, затем откройте файлы зажав клавишу Shift — это сбросит кэш программы.

3. Способ: Запуск Excel в отдельном процессе (для опытных)

Этот метод подходит, если предыдущие не сработали. Он заставляет Excel открывать каждый файл в отдельном процессе системы, что гарантированно разделит окна. Инструкция для Windows:

  1. Закройте все окна Excel.
  2. Нажмите Win + R, введите excel.exe /x и нажмите Enter. Это сбросит настройки программы.
  3. Теперь откройте первый файл через проводник (двойной клик).
  4. Затем зажмите Shift и откройте второй файл — он откроется в новом окне.

Альтернативный вариант — запустить Excel дважды через командную строку с разными процессами:

start excel.exe "C:\Путь\к\файлу1.xlsx"

start excel.exe "C:\Путь\к\файлу2.xlsx"

Этот способ единственный, который работает в 100% случаев, если файлы открываются в одном окне из-за сбоя настроек Excel. Однако после такого запуска программы могут потребовать больше оперативной памяти.

⚠️ Внимание: Использование ключа /x сбросит некоторые персональные настройки Excel (например, последние используемые папки). Рекомендуем предварительно сохранить важные данные.

4. Способ: Использование режима "Новое окно" в Excel

В Excel 365 и Excel 2019/2021 есть встроенная функция для создания нового окна с тем же файлом. Её можно адаптировать для работы с двумя разными документами:

  1. Откройте первый файл в Excel.
  2. Перейдите на вкладку Вид и нажмите Новое окно (или New Window на английской версии).
  3. В новом окне откройте второй файл через Файл → Открыть.
  4. Теперь у вас два независимых окна, которые можно расположить на экране.

Этот метод полезен, если нужно сравнивать один и тот же файл в разных частях (например, листы с начала и конца документа). Для двух разных файлов лучше комбинировать его с горячими клавишами Windows (способ 1).

Что делать, если кнопка "Новое окно" неактивна?

Кнопка Новое окно может быть заблокирована, если файл открыт в режиме Защищённый просмотр или если документ сохранён в формате .xls (старый формат Excel). Сохраните файл как .xlsx и повторите попытку. Также проверьте, не включён ли режим Режим совместимости в свойствах ярлыка Excel (правый клик → Свойства → Совместимость).

5. Способ: Онлайн-версия Excel 365 и браузерные окна

Если вы работаете с Excel Online (бесплатная веб-версия), разделить окна ещё проще:

  1. Откройте первый файл в браузере (например, через office.com).
  2. Щёлкните правой кнопкой по вкладке браузера и выберите Дублировать (в Chrome) или Клонировать вкладку (в Firefox).
  3. В новом окне откройте второй файл.
  4. Разместите окна браузера на экране с помощью Win + Стрелка (как в способе 1).

Преимущество этого метода — не нужно настраивать Excel, а файлы автоматически сохраняются в OneDrive. Однако в онлайн-версии отсутствуют некоторые функции (например, надстройки или сложные формулы).

Для удобства можно использовать расширения браузера вроде Tab Resize (для Chrome), которые позволяют делить экран на части одним кликом.

Частые проблемы и решения

Даже после выполнения всех инструкций окна Excel могут вести себя непредсказуемо. Вот типичные ошибки и их решения:

  • 🔹 Окна сливаются при перетаскивании: Отключите функцию Группировка окон в настройках Windows (Параметры → Система → Многозадачность).
  • 🔹 Excel открывает файлы в одном окне несмотря на настройки: Удалите файл Excel.exe.config в папке %APPDATA%\Microsoft\Excel (он пересоздастся автоматически).
  • 🔹 Не работают горячие клавиши Win + Стрелка: Обновите драйверы видеокарты или проверьте, не конфликтует ли клавиатура с программным обеспечением (например, AutoHotkey).
  • 🔹 Второму окну не хватает места: Используйте комбинацию Win + Стрелка вверх/вниз, чтобы расположить окна в квадрантах экрана.

Если ни один из способов не помог, попробуйте переустановить Microsoft Office с официального сайта. Перед этим экспортируйте важные данные, так как процесс может удалить пользовательские шаблоны.

⚠️ Внимание: В корпоративных версиях Excel (например, в составе Microsoft 365 для бизнеса) администратор может ограничивать многозадачность через групповую политику. В этом случае обратитесь в IT-отдел вашей компании.

FAQ: Ответы на популярные вопросы

Можно ли открыть два окна Excel на одном мониторе в версиях старше 2010 года?

В Excel 2007/2010 функционал ограничен, но есть обходной путь:

  1. Откройте первый файл.
  2. Запустите второй экземпляр Excel через Пуск → Все программы → Microsoft Office → Excel (не через файл!).
  3. В новом окне откройте второй файл.

Минус — высокое потребление памяти, так как запускаются два процесса EXCEL.EXE.

Почему при копировании данных между окнами Excel появляется ошибка?

Это происходит из-за защиты между окнами. Решения:

  • Используйте буфер обмена Windows (Win + V).
  • Скопируйте данные в Блокнот, а затем вставьте в целевой файл.
  • Отключите надстройку DDE в настройках Excel (раздел Дополнительно → Общие).
Как зафиксировать окна Excel так, чтобы они не перекрывали друг друга?

В Windows 10/11 используйте функцию Виртуальные рабочие столы:

  1. Нажмите Win + Tab и создайте новый рабочий стол.
  2. Перетащите одно окно Excel на первый стол, второе — на второй.
  3. Переключайтесь между столами через Win + Ctrl + Стрелка влево/вправо.

Alternatively, используйте сторонние утилиты вроде FancyZones (входит в PowerToys от Microsoft).

Можно ли открыть два файла Excel в разных окнах на Mac?

Да, но алгоритм отличается:

  1. Откройте первый файл.
  2. В Dock кликните правой кнопкой по иконке Excel и выберите Новое окно.
  3. В новом окне откройте второй файл.
  4. Используйте Control + Command + F для полноэкранного режима или перетаскивайте окна вручную.

Если окна сливаются, отключите опцию Объединять окна приложений в Системные настройки → Dock.

Как сравнить два файла Excel бок о бок с подсветкой различий?

Для этого:

  1. Откройте оба файла в отдельных окнах (любым из описанных способов).
  2. Перейдите на вкладку Вид → Рядом (View → View Side by Side).
  3. Активируйте опцию Синхронная прокрутка (Synchronous Scrolling).
  4. Для подсветки различий используйте надстройку Inquire (доступна в Excel 2013+ через Файл → Параметры → Надстройки).
