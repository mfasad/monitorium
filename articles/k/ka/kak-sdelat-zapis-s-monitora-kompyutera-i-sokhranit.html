Запись экрана компьютера с сохранением: полное руководство для новичков и профессионалов

Запись видео с экрана монитора требуется для создания обучающих роликов, стримов, демонстрации ошибок техподдержке или фиксации геймплея. Если при попытке сохранить записанный файл вы получаете ошибку 0x80070005 (отказано в доступе) или видео сохраняется без звука — проблема кроется в неверных настройках кодека или прав доступа к папке. На Windows 10/11 встроенный инструмент Xbox Game Bar автоматически сохраняет записи в C:\Users\[Имя]\Videos\Captures, но ограничен по функционалу: нет возможности выбрать битрейт или формат вывода.

Современные мониторы с разрешением 4K и частотой 144 Гц требуют особого подхода: стандартные программы вроде OBS Studio могут "проседать" по FPS при записи, если не отключить аппаратное ускорение в настройках. А на macOS запись через QuickTime Player по умолчанию сохраняет видео в формате .mov, который не поддерживается большинством видеоредакторов без конвертации. Далее разберём рабочие способы записи экрана с учётом этих нюансов — от базовых до профессиональных.

Встроенные средства Windows для записи экрана

Начиная с Windows 10, в системе предустановлены два инструмента для скринкастинга: Xbox Game Bar и Средство захвата Steps Recorder. Первый подходит для записи игр и приложений, второй — для создания пошаговых инструкций с аннотациями. Главный минус обоих: они не умеют захватывать рабочий стол (проводник, панель задач) из-за ограничений безопасности.

Чтобы запустить запись через Xbox Game Bar, нажмите Win + Alt + R. Видео сохранится автоматически по пути C:\Users\[Имя]\Videos\Captures в формате MP4 с битрейтом ~12 Мбит/с. Если кнопка записи неактивна, проверьте:

  • 🔹 Настройки конфиденциальности: Параметры → Конфиденциальность → Журнал активности → включите "Разрешить Xbox записывать игровые клипы".
  • 🔹 Совместимость приложения: Game Bar работает только с DirectX 11/12. Для старых игр (DirectX 9) используйте OBS.
  • 🔹 Звук: По умолчанию записывается только звук из приложения. Чтобы захватить микрофон, нажмите Win + Alt + M во время записи.
⚠️ Внимание: Если после записи файл не сохраняется, проверьте свободное место на системном диске. Windows резервирует до 4 ГБ под буфер записи, и при нехватке места процесс прервётся без уведомления.
💡

Для записи рабочего стола в Windows без ограничений используйте комбинацию Win + G, затем вручную выберите область захвата в настройках Game Bar (значок шестерёнки).

Запись экрана на macOS: QuickTime и альтернативы

На Mac запись экрана реализована через стандартное приложение QuickTime Player. Оно поддерживает захват как всего экрана, так и выбранной области, но имеет ключевые ограничения:

  • 🍎 Формат вывода — .mov (кодек H.264), который занимает на 30% больше места, чем MP4 при том же качестве.
  • 🍎 Нет возможности записывать системный звук и микрофон одновременно без дополнительных утилит вроде Soundflower.
  • 🍎 Максимальное разрешение записи ограничено разрешением монитора (на Retina-дисплеях видео сохраняется в "родном" разрешении).

Пошаговая инструкция для QuickTime:

  1. Откройте QuickTime Player → Файл → Новая запись экрана.
  2. Нажмите на стрелку рядом с кнопкой записи, чтобы выбрать источник звука (по умолчанию — без звука).
  3. Захватите область экрана кликом или перетаскиванием.
  4. Для остановки записи используйте панель в менюбар или Command + Control + Esc.

Для профессиональной работы на Mac лучше использовать ScreenFlow ($129) — программа поддерживает многокамерную запись, редактирование таймлайна и экспорт в 4K с настройкой битрейта до 50 Мбит/с. Бесплатная альтернатива — OBS Studio, но её настройка на macOS требует ручной установки виртуального аудиодрайвера.

Программы для записи экрана: сравнение функций

Сторонние программы предлагают гибкие настройки качества, поддержку нескольких источников звука и дополнительные функции вроде рисования на экране или веб-камеры в картине. Ниже сравнение популярных решений:

Программа Поддержка 4K Битрейт (Мбит/с) Форматы вывода Особенности
OBS Studio Да (до 4K60) 1–100 MP4, MKV, FLV Бесплатная, поддержка стриминга, плагины
Camtasia Да (до 4K30) 5–30 MP4, AVI, MOV Встроенный редактор, эффекты, $299

Критичный нюанс: При записи игр с высокой частотой кадров (144+ Гц) в OBS или Bandicam обязательно отключите вертикальную синхронизацию (V-Sync) в настройках игры. Иначе запись будет "рваться" из-за конфликта частот захвата и рендеринга.

Запись экрана в Linux: простые и продвинутые методы

На Linux запись экрана осложняется отсутствием универсальных инструментов: каждый дистрибутив использует свой стэк для работы с видео. Базовые решения:

  • 🐧 ffmpeg: Консольная утилита для записи с настройкой кодека, разрешения и FPS. Пример команды: 
    ffmpeg -video_size 1920x1080 -framerate 30 -f x11grab -i :0.0+100,100 output.mp4

    Здесь :0.0 — номер экрана, +100,100 — смещение области захвата.

  • 🐧 SimpleScreenRecorder: Графический интерфейс с поддержкой OpenGL-ускорения. Подходит для записи игр под Wine.
  • 🐧 Kazam: Простой инструмент для Ubuntu/Debian с возможностью записи звука через PulseAudio.

Для записи звука вместе с видео в ffmpeg добавьте параметры:

ffmpeg -f pulse -i default -f x11grab -i :0.0 -c:v libx264 -c:a aac output.mkv
⚠️ Внимание: На дистрибутивах с Wayland (Fedora 25+, Ubuntu 22.04+) стандартные методы записи экрана не работают. Используйте wf-recorder или переключитесь на Xorg через экран входа.

Как сохранить запись без потери качества

Частая проблема при сохранении — видео "пикселится" или имеет низкий битрейт. Это происходит из-за:

  • 🔧 Автоматического сжатия встроенными инструментами (например, Xbox Game Bar сжимает видео до 8 Мбит/с).
  • 🔧 Неправильного выбора кодека: H.264 подходит для большинства задач, но для профессионального монтажа лучше использовать ProRes (macOS) или DNxHD (Windows/Linux).
  • 🔧 Ограничений файловой системы: FAT32 не поддерживает файлы >4 ГБ, поэтому для длинных записей используйте NTFS или exFAT.

Оптимальные настройки для сохранения:

  • 📁 Формат: MP4 (совместимость) или MKV (без потерь при редактировании).
  • 🎞️ Кодек: H.264 (bitrate 20–50 Мбит/с для Full HD) или H.265 (HEVC) для 4K.
  • 🔊 Аудио: AAC 192–320 кбит/с. Для профессиональных задач — PCM (WAV).

Если видео не сохраняется из-за ошибки "Недостаточно памяти", попробуйте:

  1. Разбить запись на части (например, в OBS включите опцию "Авторазделение файла" каждые 2 ГБ).
  2. Использовать внешний SSD-накопитель с интерфейсом USB 3.0+ (HDD не подходит из-за низкой скорости записи).
  3. Отключить фоновые процессы, потребляющие дисковое пространство (антивирус, индексирование поиска).
Как проверить битрейт сохранённого видео?

Откройте файл в MediaInfo (бесплатная утилита) или выполните команду в терминале:

ffprobe -v error -select_streams v:0 -show_entries stream=bit_rate -of default=noprint_wrappers=1:nokey=1 input.mp4

Распространённые ошибки и их решения

Ошибки при записи экрана делятся на три категории: проблемы с сохранением файла, артефакты на видео и отсутствие звука. Рассмотрим типичные случаи:

Ошибка Причина Решение
Видео сохраняется без звука Не выбран источник звука или конфликт драйверов В OBS: добавьте источник "Захват аудиоустройства" → выберите "Стереомикшер". На Windows обновите драйверы Realtek Audio.
Чёрный экран в записи Аппаратное ускорение или защита DRM (Netflix, игры с Denuvo) Отключите аппаратное кодирование в настройках программы или используйте DuRec для обхода DRM.
Файл повреждён при сохранении Нехватка места на диске или сбой файловой системы Проверьте диск на ошибки: chkdsk C: /f (Windows) или fsck (Linux). Сохраняйте на SSD.
Запись лагает Высокая нагрузка на CPU/GPU Уменьшите разрешение записи до 720p или снизьте FPS до 30. В OBS включите "Использовать многопоточное кодирование".

Если при записи через Xbox Game Bar появляется ошибка 0x80070005, выполните:

  1. Запустите командную строку от имени администратора.
  2. Введите: 
    icacls "C:\Users\[Имя]\Videos" /grant [Имя]:F /T

    Замените [Имя] на ваше имя пользователя.

  3. Перезагрузите ПК.
💡

90% проблем с сохранением видео решаются проверкой прав доступа к папке и свободного места на диске.

Дополнительные устройства для записи экрана

Для профессиональной записи (например, стримов с консоли или захвата сигнала с внешнего монитора) используют внешние захватывающие устройства. Они подключаются между источником сигнала (ПК, приставка) и монитором, записывая "чистый" видеопоток без нагрузки на систему.

Популярные модели:

  • 🎮 Elgato HD60 S+: Захват 4K60 HDR с нулевой задержкой, подключение по USB-C. Цена ~$200.
  • 🎮 AVerMedia Live Gamer 4K: Поддержка HDMI 2.0, запись на microSD. Цена ~$250.
  • 🎮 Magewell Pro Capture: PCIe-карта для студийной записи, 4K120. Цена ~$1000.

Преимущества внешних захватов:

  • 🔹 Нет нагрузки на CPU/GPU основного ПК.
  • 🔹 Поддержка HDMI/DisplayPort с защитой HDCP (можно записывать Netflix, Blu-ray).
  • 🔹 Аппаратное кодирование в H.264/H.265 без потерь качества.
⚠️ Внимание: При использовании захватывающего устройства с монитором 144 Гц убедитесь, что устройство поддерживает пропускную способность HDMI 2.0 (18 Гбит/с). Иначе частота кадров в записи упадёт до 60 Гц.

FAQ: Ответы на частые вопросы

Можно ли записать экран на ноутбуке без установки программ?

Да, на Windows используйте Xbox Game Bar (Win + Alt + R), на macOS — QuickTime Player. На Linux потребуется установка ffmpeg или SimpleScreenRecorder через менеджер пакетов.

Почему записанное видео занимает слишком много места?

Причины:

  • Используется кодек без сжатия (например, Ut Video или Lagith).
  • Высокий битрейт (например, 100 Мбит/с для Full HD).
  • Формат вывода — AVI или MOV (менее эффективны, чем MP4).

Решение: В настройках программы выберите кодек H.264 с битрейтом 15–25 Мбит/с для Full HD.

Как записать экран с веб-камерой в одном окне?

В OBS Studio:

  1. Добавьте источник "Захват экрана".
  2. Добавьте источник "Устройство захвата видео" → выберите веб-камеру.
  3. Расположите и масштабируйте окно камеры поверх экрана (используйте "Трансформирование").

В Camtasia или Bandicam аналогичная функция называется "Picture-in-Picture" (PiP).

Можно ли записывать экран на ПК с двумя мониторами?

Да, но есть нюансы:

  • 🖥️ OBS Studio позволяет выбрать конкретный монитор в настройках источника "Захват экрана".
  • 🖥️ В Bandicam нажмите F12 и выберите область захвата вручную.
  • 🖥️ Встроенные средства (Xbox Game Bar, QuickTime) записывают только основной монитор.
Как убрать водяной знак из записанного видео?

Если водяной знак добавлен программой (например, бесплатной версией Bandicam), удалить его можно только:

  • Покупкой лицензии.
  • Обрезкой краёв видео в редакторе (если знак по углам).
  • Использованием альтернативных программ без водяных знаков (OBS, ShareX).

Для автоматического удаления логотипов существуют нейросетевые инструменты вроде Topaz Video AI, но они требуют мощного ПК и оставляют артефакты.
