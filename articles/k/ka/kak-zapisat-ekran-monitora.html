Как записать экран монитора: подробное руководство для всех ОС

Запись экрана монитора — необходимый навык для создания обучающих видео, стримов, демонстрации программного обеспечения или фиксации ошибок. Независимо от того, нужна ли вам простая скринкаст-сессия или профессиональная запись с высоким разрешением, современные операционные системы предлагают множество встроенных и сторонних решений. В этой статье мы разберём все актуальные методы записи экрана на Windows 10/11, macOS и дистрибутивах Linux, а также рассмотрим специализированные программы для продвинутых пользователей.

Особое внимание уделим настройкам качества записи, выбору формата выходного файла и оптимизации процесса для разных задач — от записи игрового процесса до создания презентаций. Если вы никогда не записывали экран раньше, наша пошаговая инструкция поможет избежать распространённых ошибок, таких как низкое качество видео или проблемы со звуком.

Встроенные средства записи экрана в Windows 10 и 11

Microsoft интегрировала инструменты для записи экрана ещё в Windows 10, а в Windows 11 их функциональность была расширена. Самый простой способ начать запись — использовать Xbox Game Bar, который изначально создавался для геймеров, но подходит и для обычных задач.

Чтобы активировать запись:

  • 🎮 Нажмите комбинацию клавиш Win + Alt + R — это универсальный хоткей для начала/остановки записи.
  • 🔧 Если хоткей не работает, откройте Параметры → Игры → Панель игр Xbox и включите опцию"Записывать игровые клипы и скриншоты".
  • 📁 Видеофайлы сохраняются в папке C:\Users\[Ваше_имя]\Videos\Captures в формате .mp4.

Ограничения Game Bar:

  • ❌ Не записывает рабочий стол (только окна приложений).
  • ❌ Максимальное разрешение — 1920×1080 при 30 FPS.
  • ❌ Нет возможности выбирать аудиоисточник (записывается только звук из приложения).

Запись экрана на macOS: QuickTime Player и Screen Recording

Пользователи Mac имеют в своём распоряжении два встроенных инструмента: QuickTime Player и утилиту Screen Recording, доступную через Launchpad → Другие. Первый подходит для записи с микрофоном, второй — для быстрого захвата экрана без дополнительных настроек.

Инструкция для QuickTime Player:

  1. Откройте приложение через Spotlight (нажмите Cmd + Пробел и введите"QuickTime").
  2. Выберите Файл → Новая запись экрана.
  3. Нажмите на стрелку рядом с кнопкой записи, чтобы выбрать источник звука (микрофон или системный аудио).
  4. Нажмите"Записать" и выделите область экрана или выберите запись всего дисплея.

Для утилиты Screen Recording:

  • 🖥️ Нажмите Cmd + Shift + 5 — откроется панель инструментов.
  • 🎤 Выберите"Запись всего экрана" или"Запись выбранной части".
  • 🔊 Нажмите"Параметры", чтобы включить микрофон или показать клики мыши.
⚠️ Внимание: На macOS запись системного звука без дополнительного ПО возможна только в macOS Mojave 10.14.4 и новее. В более старых версиях потребуется установка виртуального аудиодрайвера, например, Soundflower.

Лучшие программы для записи экрана на ПК

Если встроенных инструментов недостаточно, стоит обратить внимание на специализированное ПО. Ниже — сравнение популярных программ с учётом их возможностей и ограничений:

Программа Бесплатная версия 4K/60FPS Запись веб-камеры Стриминг
OBS Studio ✅ Да ✅ Да ✅ Да ✅ Да
Camtasia ❌ Нет (триал 30 дней) ✅ Да ✅ Да ❌ Нет
Bandicam ✅ Да (водяной знак) ✅ Да ❌ Нет ❌ Нет
ShareX ✅ Да ❌ Нет (макс. Full HD) ❌ Нет ❌ Нет

OBS Studio — лидер среди бесплатных решений. Программа поддерживает:

  • 🎥 Многопоточную запись с нескольких источников (экран + камера + микрофон).
  • 🎛️ Гибкие настройки битрейта, кодеков (H264, HEVC) и разрешений.
  • 🌐 Стриминг на Twitch, YouTube или Facebook Gaming без дополнительного ПО.

Для установки:

  1. Скачайте дистрибутив с официального сайта.
  2. При первом запуске выберите"Оптимизировать для записи".
  3. Добавьте источник"Захват экрана" и настройте аудиоустройства в меню"Микшер".
Как убрать лаги при записи в OBS?

Если видео записывается рывками, попробуйте:

1. Уменьшить разрешение до 1280×720.

2. Снизить FPS до 30.

3. В настройках выставить кодировщик NVENC (для видеокарт NVIDIA) или AMF (для AMD).

4. Закрыть фоновые процессы через Диспетчер задач.

Запись экрана в Linux: простые и продвинутые способы

В дистрибутивах Linux выбор инструментов зависит от используемого окружения рабочего стола (GNOME, KDE, XFCE). Большинство современных дистрибутивов (например, Ubuntu 22.04+) включают утилиту GNOME Screen Recorder, но для расширенных возможностей потребуется установка дополнительного ПО.

Базовый способ через ffmpeg (работает в любом дистрибутиве):

ffmpeg -video_size $(xdpyinfo | grep dimensions | awk'{print $2}') -framerate 30 -f x11grab -i:0.0+100,100 -f pulse -i default -c:v libx264 -c:a aac -b:a 192k output.mp4

Эта команда записывает:

  • 📹 Область экрана начиная с координат X=100, Y=100.
  • 🎤 Звук с микрофона по умолчанию (pulseaudio).
  • 📦 Выходной файл output.mp4 с битрейтом аудио 192 kbps.

Для пользователей KDE Plasma доступен Kazam — простой записчик с интерфейсом на Python/GTK. Установить его можно командой:

sudo apt install kazam # Для Debian/Ubuntu

sudo dnf install kazam # Для Fedora

⚠️ Внимание: В некоторых дистрибутивах (например, Arch Linux) для записи звука через ffmpeg потребуется установить пакет pulseaudio-alsa и настроить права доступа к устройству /dev/snd.

Как записать экран с высоким FPS для игр и динамичного контента

Для записи игрового процесса или быстроменяющихся сцен (например, анимации в Blender) стандартные инструменты не подходят из-за ограничений по FPS и битрейту. Здесь на помощь приходят специализированные программы:

  • 🎮 NVIDIA ShadowPlay (только для видеокарт NVIDIA GTX/RTX) — записывает в 4K@60FPS с минимальной нагрузкой на CPU.
  • 🖥️ AMD ReLive (для видеокарт Radeon) — поддерживает H.265 (HEVC) для экономии места.
  • 📹 Mirillis Action! — платное решение с поддержкой записи в 4K@120FPS.

Настройка NVIDIA ShadowPlay:

  1. Откройте GeForce Experience и перейдите в Настройки → Запись.
  2. Включите"Запись рабочего стола" (требуется драйвер версии 451.48+).
  3. Выберите качество"4K 60 FPS" и формат MP4.
  4. Назначьте хоткеи для начала/остановки записи (по умолчанию Alt + F9).
💡

Для максимального качества в играх отключите вертикальную синхронизацию (V-Sync) в настройках графики. Это уменьшит задержки при записи, но может вызвать разрывы кадров (screen tearing).

Решение распространённых проблем при записи экрана

Даже с правильными настройками пользователи сталкиваются с типичными ошибками. Вот как их исправить:

Проблема Возможная причина Решение
Видео записывается рывками Нехватка ресурсов CPU/GPU Снизьте разрешение до 1280×720 или используйте аппаратное кодирование (NVENC/AMF)
Нет звука в записи Неверный источник аудио В настройках программы выберите правильное устройство (например,"Стереомикшер" в Windows)
Чёрный экран в записи Защита DRM (Netflix, игры) Используйте OBS с источником"Захват окна" вместо"Захват экрана"
Большой размер файла Высокий битрейт Установите битрейт 8000–12000 Kbps для Full HD или используйте кодеки HEVC (H.265)

Если проблема связана с DRM-защитой (например, при записи Netflix или Disney+), обойти её легально невозможно. Большинство стриминговых сервисов блокируют захват экрана на уровне ОС. В таких случаях единственный выход — использование внешнего устройства захвата (например, Elgato HD60 S+).

💡

Для записи защищённого контента (DRM) потребуется аппаратный захват видео через HDMI. Программные методы в 99% случаев будут блокироваться системой защиты.

Оптимизация записанного видео: обрезка, сжатие, добавление субтитров

После записи часто требуется постобработка: удаление лишних фрагментов, улучшение качества или добавление аннотаций. Для этих задач подойдут:

  • 🎬 Shotcut — бесплатный кроссплатформенный редактор с поддержкой 4K.
  • 📊 HandBrake — для перекодировки видео в более компактные форматы.
  • 🔤 Aegisub — специализированный инструмент для создания субтитров.

Пример команды для сжатия видео через ffmpeg:

ffmpeg -i input.mp4 -vcodec libx265 -crf 28 -preset fast -acodec aac -b:a 128k output_compressed.mp4

Параметры:

  • -crf 28 — качество сжатия (чем выше значение, тем сильнее сжатие, но хуже качество).
  • -preset fast — баланс между скоростью кодирования и размером файла.

FAQ: Ответы на частые вопросы

Можно ли записать экран на ноутбуке без установки программ?

Да, в Windows 10/11 используйте Xbox Game Bar (Win + Alt + R), на macOS — QuickTime Player или Cmd + Shift + 5. В Linux (GNOME) нажмите Ctrl + Alt + Shift + R для начала записи.

Как записать экран с звуком из браузера (например, вебинар)?

В Windows включите запись системного звука через"Стереомикшер" (откройте Параметры звука → Свойства устройства → Дополнительные свойства и активируйте опцию). На macOS в QuickTime Player выберите"Внутренний микрофон (встроенный)" как источник.

Почему запись получается с зелёным экраном или артефактами?

Этоная проблема при использовании аппаратного ускорения. Попробуйте:

  1. Отключить аппаратное кодирование в настройках программы (выберите Software (x264)).
  2. Обновить драйверы видеокарты.
  3. Снизить разрешение записи до 1280×720.
Как записать экран на двух мониторах?

В OBS Studio добавьте два источника"Захват экрана" и настройте их позиции. В Windows через Game Bar запись нескольких мониторов невозможна — используйте сторонние программы (Bandicam, Mirillis Action!).

Сколько места на диске занимает 1 час записи в Full HD?

Примерный расчёт:

  • H.264, 30 FPS, битрейт 8000 Kbps → ~3.5 ГБ/час.
  • H.265 (HEVC), те же настройки → ~1.8 ГБ/час.
  • Without compression (RAW) → до 100 ГБ/час.
