Создание скриншотов — одна из самых востребованных функций при работе с компьютером. Без них сложно обойтись, когда нужно сохранить важную информацию с экрана, продемонстрировать ошибку в программе или поделиться интересным контентом. Но как именно сделать снимок экрана, если под рукой нет смартфона? Ответ прост: для этого предусмотрены специальные клавиши на клавиатуре и встроенные инструменты операционной системы.

Многие пользователи знают о существовании кнопки Print Screen, но не все понимают, как ею правильно пользоваться. Более того, в зависимости от операционной системы — Windows, macOS или Linux — комбинации клавиш и механизмы сохранения скриншотов отличаются. В этой статье мы разберём все возможные способы, включая горячие клавиши, системные утилиты и сторонние программы, а также расскажем, как редактировать и сохранять полученные снимки.

Особое внимание уделим типичным ошибкам, из-за которых скриншоты могут не сохраняться или получаться неполными. Например, почему иногда кнопка PrtScn не работает, как сделать снимок только активного окна, и что делать, если скриншот не копируется в буфер обмена. Также вы узнаете, как настроить автоматическое сохранение снимков в нужную папку и какие программы помогут расширить функционал.

Клавиша Print Screen (PrtScn) — основной инструмент для скриншотов в Windows

На большинстве клавиатур для ПК и ноутбуков есть специальная кнопка Print Screen (иногда сокращённо PrtScn, PrntScrn или PrtSc). Она расположена в правом верхнем углу, рядом с клавишами Scroll Lock и Pause/Break. Эта кнопка — самый простой способ сделать снимок всего экрана.

При нажатии на Print Screen изображение с монитора копируется в буфер обмена. Чтобы сохранить его как файл, нужно вставить скриншот в любой графический редактор (например, Paint, Photoshop или даже Word) и сохранить документ. Однако этот метод имеет недостаток: если вы забудете вставить снимок до следующей операции копирования, он будет утерян.

  • 🖥️ Весь экран: одно нажатие PrtScn — копирует всё, что видно на мониторе.
  • 🪟 Активное окно: комбинация Alt + PrtScn — снимок только того окна, которое сейчас открыто.
  • 💾 Автоматическое сохранение: в Windows 8/10/11 добавлена функция сохранения скриншотов в папку при нажатии Win + PrtScn.
⚠️ Внимание: На некоторых ноутбуках (например, Lenovo, HP) кнопка PrtScn работает только в комбинации с Fn. Если снимок не создаётся, попробуйте нажать Fn + PrtScn.

В Windows 11 и Windows 10 скриншоты, сделанные через Win + PrtScn, автоматически сохраняются в папку Этот компьютер → Изображения → Снимки экрана. Файлы именуются как Снимок экрана (1).png, Снимок экрана (2).png и так далее. Если папка отсутствует, система создаст её при первом использовании комбинации.

Горячие клавиши для скриншотов в Windows: полный список

Помимо стандартной кнопки Print Screen, в Windows есть несколько других комбинаций, которые позволяют гибко управлять процессом создания снимков. Вот все доступные варианты:

Комбинация клавиш Действие Где сохраняется
PrtScn Снимок всего экрана Буфер обмена
Alt + PrtScn Снимок активного окна Буфер обмена
Win + PrtScn Снимок всего экрана Папка "Снимки экрана"
Win + Shift + S Выбор области для снимка (инструмент "Ножницы") Буфер обмена
Win + G (в игре) Снимок экрана через Xbox Game Bar Папка "Видео → Захваты"

Самая удобная комбинация для большинства пользователей — Win + Shift + S. Она вызывает инструмент "Вырезка и скетч" (ранее "Ножницы"), который позволяет выбрать произвольную область экрана. После выделения фрагмента снимок копируется в буфер обмена, и вы можете вставить его в документ или редактор.

Если вам нужно сделать скриншот меню "Пуск" или другого системного окна, стандартные методы могут не сработать. В этом случае поможет комбинация Ctrl + PrtScn (в некоторых версиях Windows) или сторонние программы вроде ShareX.

💡

Чтобы быстро найти все сохранённые скриншоты, введите в поиске Windows запрос kind:=screenshot — система покажет все файлы снимков экрана.

Как сделать скриншот на Mac: клавиши Command + Shift

На компьютерах Apple процесс создания снимков экрана организован иначе. Здесь нет кнопки Print Screen, но есть несколько специализированных комбинаций с клавишей Command (⌘). Все они сохраняют скриншоты непосредственно в файлы на рабочем столе в формате .png.

  • 🖥️ Весь экран: Command + Shift + 3 — снимок всего дисплея.
  • 📝 Выбранная область: Command + Shift + 4, затем выделите часть экрана мышью.
  • 🪟 Одно окно: Command + Shift + 4, затем нажмите Пробел и кликните на нужное окно.
  • 📋 Буфер обмена: добавьте Control к любой комбинации (например, Command + Control + Shift + 4), чтобы скриншот не сохранялся как файл, а копировался в буфер.

На macOS Mojave и новее после нажатия комбинаций появляется миниатюра снимка в правом нижнем углу экрана. Если кликнуть на неё, откроется окно предварительного просмотра с возможностью обрезки, разметки и отправки. Если проигнорировать миниатюру, скриншот автоматически сохранится на рабочем столе.

По умолчанию файлы скриншотов называются как Снимок экрана [дата] в [время].png. Изменить папку сохранения можно через терминал, введя команду:

defaults write com.apple.screencapture location /путь/к/папке

killall SystemUIServer

⚠️ Внимание: На ноутбуках MacBook с сенсорной панелью Touch Bar комбинации работают так же, но для удобства можно добавить кнопку скриншота в панель через Системные настройки → Клавиатура → Настройка Touch Bar.

Скриншоты в Linux: инструменты и клавиши для разных дистрибутивов

В системах на базе Linux (например, Ubuntu, Fedora, Mint) нет универсальной клавиши для скриншотов, но большинство дистрибутивов поддерживают комбинацию PrtScn по умолчанию. Однако поведение этой кнопки зависит от используемой графической оболочки (GNOME, KDE, XFCE и др.).

Вот основные способы:

  • 🖥️ Весь экран: PrtScn — сохраняет снимок в папку ~/Pictures/Screenshots (в GNOME).
  • 📝 Выбранная область: Shift + PrtScn — позволяет выделить часть экрана.
  • 🪟 Активное окно: Alt + PrtScn — снимок текущего окна.
  • 📋 Буфер обмена: Ctrl + PrtScn — копирует снимок в буфер (в некоторых дистрибутивах).

Если стандартные комбинации не работают, проверьте настройки в меню Параметры системы → Клавиатура → Сочетания клавиш. Там можно переназначить горячие клавиши для скриншотов или отключить конфликтующие комбинации.

Для расширенных возможностей (например, задержка снимка, добавление эффектов) в Linux часто используют утилиты:

  • 📸 Shutter — позволяет делать снимки с задержкой, редактировать их и добавлять аннотации.
  • 🖼️ Flameshot — мощный инструмент для выделения областей с возможностью разметки.
  • 🔧 GNOME Screenshot — стандартное приложение для GNOME.

Проблемы со скриншотами: почему не работает Print Screen и как это исправить

Иногда кнопка Print Screen перестаёт работать, и снимки экрана не создаются. Причины могут быть разными: от отключённой функции в BIOS до конфликта программ. Вот наиболее распространённые проблемы и их решения:

  • 🔄 Клавиша заблокирована: на некоторых клавиатурах PrtScn работает как SysRq. Чтобы вернуть стандартное поведение, нажмите Fn + PrtScn или проверьте настройки в BIOS.
  • 🖱️ Конфликт с драйверами: обновление или переустановка драйверов клавиатуры может решить проблему. Используйте Диспетчер устройств (devmgmt.msc).
  • 📁 Нет прав на сохранение: если скриншоты не сохраняются в папку, проверьте разрешения для пользователя. Например, в Windows папка Снимки экрана должна быть доступна для записи.
  • 🛠️ Проблемы с буфером обмена: если снимок не вставляется, перезапустите процесс rdpclip.exe через Диспетчер задач.

В ноутбуках Dell и Acer иногда требуется включить функцию Print Screen в BIOS (раздел Advanced → Function Key Behavior). Если после этого проблема остаётся, попробуйте использовать альтернативные методы, например, инструмент "Ножницы" (Win + Shift + S).

Если скриншоты делаются, но получаются чёрными (например, при попытке снять видео или игру), это связано с защитой контента (DRM). В этом случае поможет:

  • Использование Xbox Game Bar (Win + G) для игр.
  • Захват экрана через OBS Studio или ShareX.
  • Отключение аппаратного ускорения в настройках браузера (если проблема с видео в Chrome или Firefox).
Почему скриншоты в играх получаются чёрными?

Это связано с технологией защиты контента (например, HDCP или Denuvo), которая блокирует захват экрана. Решение — использовать специализированные программы вроде Fraps или NVIDIA ShadowPlay, если у вас видеокарта NVIDIA.

Сторонние программы для скриншотов: когда стандартных средств недостаточно

Встроенные инструменты операционных систем подходят для большинства задач, но если вам нужны расширенные функции — задержка снимка, запись видео с экрана, добавление аннотаций или загрузка в облако — стоит обратить внимание на сторонние программы. Вот самые популярные:

Программа Особенности Платформа
ShareX Захват областей, GIF-запись, загрузка на сервер, OCR Windows
Lightshot Быстрое выделение областей, редактирование, поиск похожих изображений Windows, macOS
Snagit Профессиональный редактор скриншотов, запись видео, шаблоны Windows, macOS
Flameshot Открытый исходный код, разметка, настройка горячих клавиш Linux, Windows, macOS
Greenshot Легковесная утилита с поддержкой плагинов Windows

ShareX — одна из самых функциональных бесплатных программ. Она позволяет не только делать снимки экрана, но и записывать видео, распознавать текст на изображениях (OCR), автоматически загружать файлы на Imgur, Google Drive и другие сервисы. Например, вы можете настроить автоматическую загрузку скриншота в облако и получение ссылки для быстрой отправки коллегам.

Если вам нужно сделать скриншот с задержкой (например, чтобы успеть открыть выпадающее меню), в ShareX или Snagit есть функция таймера. Выставляете задержку в 3–10 секунд, запускаете захват, и программа сделает снимок через указанное время.

💡

Для профессиональной работы с скриншотами (например, для создания инструкций или презентаций) лучше использовать Snagit или ShareX — они экономят время за счёт автоматических действий и интеграции с облачными сервисами.

Как редактировать и аннотировать скриншоты: инструменты и советы

Часто скриншот нужно не просто сохранить, но и отредактировать: обрезать, добавить стрелки, текст или размыть конфиденциальную информацию. Для этого подойдут как встроенные инструменты, так и сторонние редакторы.

В Windows 10/11 после нажатия Win + Shift + S появляется панель инструментов для разметки. Она позволяет:

  • 🖋️ Рисунок пером или маркером.
  • 📏 Добавление прямых линий и фигур.
  • 🔤 Вставка текста.
  • 🧽 Ластик для удаления пометок.
  • 🖼️ Обрезка краёв снимка.

Для более серьёзного редактирования подойдут:

  • 🎨 Paint.NET — бесплатный редактор с поддержкой слоёв.
  • 🖌️ GIMP — полноценная замена Photoshop.
  • 📱 Canva — онлайн-сервис для создания коллажей и аннотаций.

Если вам нужно размыть часть изображения (например, скрыть личные данные), в Paint.NET есть инструмент Эффекты → Размытие → Размытие по Гауссу. В GIMP для этого используйте фильтр Фильтры → Размытие → Размытие по Гауссу.

⚠️ Внимание: При отправке скриншотов с конфиденциальной информацией (пароли, номера документов) всегда проверяйте, что все чувствительные данные размыты или удалены. Даже если вы отправляете снимок trusted лицу, лучше перестраховаться.

FAQ: Частые вопросы о скриншотах

🔹 Почему скриншот не сохраняется в папку, а только копируется в буфер?

В Windows по умолчанию кнопка PrtScn копирует снимок в буфер обмена. Чтобы он автоматически сохранялся как файл, используйте комбинацию Win + PrtScn (для всего экрана) или настройте параметры в инструменте "Ножницы" (Параметры → Сохранять снимки).

🔹 Как сделать скриншот только одного монитора в многомониторной системе?

В Windows при нажатии PrtScn снимаются все подключённые экраны. Чтобы захватить только один монитор, используйте Win + Shift + S, затем выделите нужную область. В macOS при выборе области (Command + Shift + 4) также можно ограничиться одним дисплеем.

🔹 Можно ли сделать скриншот с помощью мыши, без клавиатуры?

Да, в Windows для этого есть инструмент "Ножницы" (Пуск → Стандартные → Ножницы). В macOS можно использовать утилиту Снимок экрана (Launchpad → Утилиты). Также многие программы (например, Lightshot) позволяют делать снимки через контекстное меню мыши.

🔹 Куда сохраняются скриншоты на Mac?

По умолчанию снимки экрана сохраняются на рабочем столе в формате .png. Изменить папку сохранения можно через терминал командой defaults write com.apple.screencapture location /путь/к/папке, после чего нужно перезапустить SystemUIServer.

🔹 Как сделать скриншот на планшете или смартфоне?

Эта статья посвящена скриншотам на компьютерах, но для смартфонов обычно используется комбинация кнопки питания + уменьшения громкости (на Android) или кнопка Home + питание (на старых iPhone). На iPhone X и новее — кнопка питания + увеличение громкости.
