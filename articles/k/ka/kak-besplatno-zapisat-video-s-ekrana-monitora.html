Как записать видео с экрана монитора бесплатно: полное руководство 2026

Запись видео с экрана монитора стала неотъемлемой частью работы, обучения и творчества.hether вы создаёте обучающие ролики, фиксируете баги в программах, стримите геймплей или сохраняете вебинары — качественный скринкаст упрощает коммуникацию и экономит время. К счастью, для этого не обязательно покупать дорогостоящее ПО: современные операционные системы и бесплатные утилиты предлагают все необходимые инструменты.

В этой статье мы разберём 7 способов записи экрана без водяных знаков и ограничений, включая встроенные средства Windows 11/10, macOS и Linux, а также лучшие бесплатные программы с открытым исходным кодом. Вы узнаете, как захватить весь экран или его часть, добавить звук с микрофона, оптимизировать качество видео и избежать распространённых ошибок. Особое внимание уделим настройкам для игровых мониторов с высокой частотой обновления (144+ Гц) и многомониторных конфигураций.

Встроенные инструменты Windows: Xbox Game Bar и PowerPoint

Если вы используете Windows 10 или Windows 11, система уже содержит два мощных инструмента для записи экрана — Xbox Game Bar и скрытую функцию в Microsoft PowerPoint. Первый оптимизирован для геймеров, но подходит и для обычных задач, второй позволяет записывать презентации с комментариями.

Чтобы запустить запись через Xbox Game Bar, нажмите комбинацию Win + Alt + R. Инструмент захватит активное окно (идеально для игр или программ в полноэкранном режиме). Для ручного управления используйте панель, которая открывается по Win + G. Здесь можно:

  • 🎮 Включить/выключить запись микрофона (Win + Alt + M)
  • 📼 Сохранить последние 30 секунд (Win + Alt + G — полезно для фиксации багов)
  • 🔊 Захватить системный звук (в настройках Параметры → Игра → Xbox Game Bar)

PowerPoint предлагает более гибкие настройки: запись можно начать с любой слайда, добавляя пояснения голосом. Для этого откройте презентацию, перейдите на вкладку Вставка → Запись экрана. Здесь доступны:

  • 🖥️ Выбор области экрана (не только окно PowerPoint)
  • 🎤 Одновременная запись с веб-камеры
  • ⏱️ Таймер задержки (3 секунды для подготовки)
💡

Если Xbox Game Bar не запускается, проверьте службу XboxGipSvc в Диспетчере задач (Ctrl+Shift+Esc). Иногда она отключается после обновлений Windows.

⚠️ Внимание: Xbox Game Bar не поддерживает запись рабочего стола и проводника Windows. Для этих целей используйте PowerPoint или сторонние программы.

macOS: QuickTime Player и встроенная запись экрана

Пользователи Mac имеют одно из самых удобных решений для скринкастинга — QuickTime Player. Этот простой инструмент позволяет записывать экран с микрофоном, сохраняя видео в высоком качестве без водяных знаков. Чтобы начать:

  1. Откройте QuickTime Player (находится в папке Программы).
  2. Выберите Файл → Новая запись экрана.
  3. Нажмите на стрелку рядом с кнопкой записи, чтобы выбрать источник звука (внутренний микрофон или системный аудио через BlackHole).
  4. Захватите весь экран или выделенную область, нажав Запись.

Для записи системного звука (например, музыки из браузера) потребуется установить виртуальный аудиодрайвер BlackHole (бесплатно на официальном сайте). После установки выберите BlackHole как входное устройство в настройках QuickTime.

Альтернативный способ — использовать горячие клавиши macOS:

  • 🎥 Shift + Command + 5 — открывает панель записи экрана с расширенными опциями (запись всего экрана, выделенной области или окна).
  • 📱 Здесь же можно включить отображение кликов мыши (Параметры → Показывать клики мыши).
Как записать экран на Mac с звуком из браузера?

1. Установите BlackHole (2ch или 16ch версию).

2. В Настройки → Звук выберите BlackHole как устройство вывода.

3. В QuickTime выберите BlackHole как источник звука для записи.

4. Запустите запись — теперь будет захвачен звук из браузера или других программ.

⚠️ Внимание: При записи через Shift+Command+5 видео сохраняется в формате .mov (кодеки H.264 или ProRes). Для совместимости с Windows конвертируйте файлы в MP4 через HandBrake.

Linux: простые команды для записи экрана

В мире Linux запись экрана часто осуществляется через терминал с помощью утилит ffmpeg и SimpleScreenRecorder. Первый вариант требует знания команд, но даёт максимальный контроль над параметрами, второй предлагает графический интерфейс.

Для записи через ffmpeg используйте команду:

ffmpeg -video_size $(xdpyinfo | grep dimensions | awk '{print $2}') -framerate 30 -f x11grab -i :0.0+100,100 -f pulse -i default -c:v libx264 -crf 23 -preset veryfast -c:a aac -b:a 192k output.mp4

Расшифровка параметров:

  • video_size — автоматически определяет разрешение экрана.
  • framerate 30 — частота кадров (увеличьте до 60 для геймплея).
  • :0.0+100,100 — смещение от верхнего левого угла (здесь 100px вправо и вниз).
  • -f pulse -i default — захват звука с микрофона по умолчанию.

Для упрощения процесса установите SimpleScreenRecorder:

sudo apt install simplescreenrecorder  # Для Debian/Ubuntu

sudo dnf install simplescreenrecorder  # Для Fedora

Программа поддерживает:

  • 🎞️ Запись выделенной области или окна.
  • 🔊 Выбор аудиоисточника (ALSA или PulseAudio).
  • ⚙️ Настройку битрейта и кодеков (H.264, VP8, VP9).
⚠️ Внимание: При записи через ffmpeg в Wayland (вместо X11) используйте -f kmsgrab вместо -f x11grab. Подробности смотрите в документации вашего дистрибутива.

Бесплатные программы для записи экрана без водяных знаков

Если встроенных инструментов недостаточно, рассмотрите специализированные программы с открытым исходным кодом. Они предлагают расширенные функции: наложение веб-камеры, рисование во время записи, стриминг в реальном времени.

Топ-3 бесплатных решения:

  1. OBS Studio — стандарт де-факто для стримеров и профессионалов.
    • 🎮 Поддержка нескольких сцен (переключение между экраном, камерой, презентацией).
    • 📡 Стриминг на Twitch, YouTube, Facebook без дополнительного ПО.
    • 🎛️ Продвинутые настройки кодеков (NVENC для NVIDIA, AMF для AMD).
  • ShareX — лёгкая утилита для Windows с открытым кодом.
    • 🖼️ Захват GIF-анимаций (полезно для коротких демо).
    • 🔗 Автоматическая загрузка на Imgur, Google Drive.
    • 📝 Встроенный редактор скриншотов.
    • Kazam — простой инструмент для Linux.
      • 🎥 Запись с веб-камеры в отдельном окне.
      • ⏱️ Таймер отсрочки записи.
      • 📁 Сохранение в MP4, WebM или AVI.

    OBS Studio поддерживает запись с мониторов с частотой обновления 144 Гц и выше, но для этого необходимо вручную указать FPS в настройках вывода (Настройки → Вывод → Режим вывода: Продвинутый → Частота кадров).

    Как записать экран с высоким разрешением (4K, 144 Гц)

    Запись видео в 4K или с частотой 144+ Гц предъявляет высокие требования к железу и настройкам кодека. Даже мощные ПК могут испытывать лаги при неправильных параметрах. Вот ключевые рекомендации:

    Для Windows/macOS:

    • 🖥️ В OBS Studio выберите кодировщик NVENC HEVC (для NVIDIA RTX 20/30/40 серии) или AMF (для AMD Radeon RX 5000+).
    • 📊 Установите битрейт:
      • 1080p60: 12–18 Mbps
      • 1440p144: 25–35 Mbps
      • 4K60: 40–60 Mbps
    • 🔧 В настройках видео укажите базовое разрешение, равное разрешению монитора, и масштабирование до целевого (например, с 3840×2160 до 1920×1080).

    Для Linux:

    • 🐧 Используйте ffmpeg с аппаратным ускорением: 
      ffmpeg -f kmsgrab -i - -c:v h264_nvenc -preset slow -b:v 30M -f pulse -i default -c:a aac -b:a 192k output.mp4
    • 🔌 Убедитесь, что драйверы NVIDIA или AMD поддерживают VA-API/Vulkan.
    ⚠️ Внимание: При записи 4K видео на HDD (не SSD) могут возникать артефакты из-за недостаточной скорости записи. Используйте диск с скоростью чтения/записи от 100 MB/s.
    💡

    Для игровых мониторов с G-Sync/FreeSync отключите синхронизацию в настройках драйвера перед записью — это уменьшит рывки в итоговом видео.

    Запись экрана на ноутбуке: экономия заряда батареи

    Запись видео с экрана на ноутбуке быстро разряжает батарею из-за высокой нагрузки на CPU/GPU. Чтобы продлить автономность:

    Оптимизация в Windows:

    • 🔋 Переключитесь на схему питания Высокая производительность (несмотря на название, это снизит просадки FPS).
    • 📉 Ограничьте FPS записи до 30 кадров (если не требуется плавность).
    • 🔊 Отключите запись микрофона, если он не нужен — обработка аудио потребляет ~5-10% CPU.

    Оптимизация в macOS:

    • 🍏 Включите режим Экономия энергии в Системные настройки → Аккумулятор.
    • 🌡️ Закройте фоновые приложения (особенно Chrome и Slack — они активно используют CPU).
    • 🔄 Используйте QuickTime Player вместо OBS — он менее ресурсоёмкий.

    Для Linux:

    • 🐧 Установите TLP для управления питанием: 
      sudo apt install tlp tlp-rdw

      sudo systemctl enable tlp

    • 📊 Запускайте ffmpeg с параметром -thread_queue_size 2048 для снижения нагрузки на CPU.
    💡

    Если ноутбук перегревается во время записи, поднимите его на подставку для улучшения циркуляции воздуха или используйте охлаждающую подставку с USB-питанием.

    Решение распространённых проблем при записи экрана

    Даже с правильными настройками могут возникать ошибки. Вот как их устранить:

    Проблема 1: Видео записывается рывками

    • 🖥️ Проверьте загрузку CPU/GPU в Диспетчере задач (Windows) или htop (Linux/macOS). Если загрузка близка к 100%, уменьшите разрешение или FPS.
    • 🔧 В OBS переключитесь с x264 на аппаратный кодировщик (NVENC/AMF).
    • 💾 Записывайте на SSD вместо HDD.

    Проблема 2: Нет звука в записи

    • 🔊 В Windows проверьте, что в Xbox Game Bar включён захват звука (Win + G → Настройки → Захват).
    • 🎧 На Mac установите BlackHole для захвата системного звука.
    • 🔌 В OBS добавьте источник Захват аудиовыхода (для Windows 11).

    Проблема 3: Видео сохраняется без расширения или не открывается

    • 📁 Убедитесь, что на диске достаточно места (для 10 минут 1080p60 требуется ~1–2 GB).
    • 🔄 Перекодируйте файл через HandBrake или ffmpeg: 
      ffmpeg -i input.mkv -c:v copy -c:a aac output.mp4
    ⚠️ Внимание: Если используете OBS на Windows 10 с включённым Game Mode, отключите его в Параметры → Игра → Режим игры — он может конфликтовать с записью.

    FAQ: Частые вопросы о записи экрана

    Можно ли записать экран на Windows 7 без программ?

    В Windows 7 нет встроенных инструментов для записи экрана. Используйте бесплатные программы вроде CamStudio (устаревшая, но работает) или ShareX (поддерживает Windows 7 с обновлением KB3033929). Альтернатива — ffmpeg с параметром -f gdigrab (вместо x11grab).

    Как записать экран с звуком из браузера на Mac?

    Установите виртуальный аудиодрайвер BlackHole, затем в QuickTime Player выберите его как источник звука. В браузере (Chrome/Safari) включите вывод звука на BlackHole через Настройки → Звук → Выход.

    Почему OBS Studio не видит монитор с разрешением 4K?

    Проверьте, что в настройках Видео установлено базовое разрешение 3840×2160. Если монитор подключён по DisplayPort, попробуйте переподключить его или обновить драйверы GPU. В редких случаях помогает отключение HDR в настройках Windows (Параметры → Система → Дисплей).

    Как уменьшить размер итогового видео без потери качества?

    Используйте HandBrake с пресетом H.265 (HEVC) и настройками:

    • Кодек: H.265 (x265)
    • Качество: RF 20–24 (ниже = лучше качество, но больший размер)
    • Аудио: AAC 128 kbps

    Для 10-минутного видео 1080p это сократит размер с 1.5 GB до 300–500 MB.

    Можно ли записать экран на Android-смартфоне и передать видео на монитор?

    Да, используйте встроенную функцию записи экрана (начиная с Android 11) или приложение AZ Screen Recorder. Для передачи на монитор подключите смартфон по USB-C to HDMI или используйте Miracast/Google Cast для беспроводного отображения. Качество будет ограничено разрешением экрана телефона.
