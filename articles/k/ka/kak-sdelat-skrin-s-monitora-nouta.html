Как сделать скриншот с экрана ноутбука: все способы за 2 минуты

Сколько раз вы пытались сохранить важную информацию с экрана, но теряли время на поиски кнопки PrtSc или запутанные комбинации? Создание скриншота на ноутбуке кажется простой задачей — пока не сталкиваешься с нюансами: где сохраняется файл, как обрезать ненужные части или запечатлеть только активное окно. Эта статья не просто перечислит способы, а поможет выбрать оптимальный метод в зависимости от вашей операционной системы, модели ноутбука и задачи — будь то сохранение чека из онлайн-банка или создание обучающего гида с аннотациями.

Мы разберём не только стандартные инструменты Windows 10/11 и macOS, но и альтернативные программы с расширенными функциями (например, запись видео с экрана или распознавание текста на скриншоте). Особое внимание уделим распространённым ошибкам: почему скриншот не сохраняется в буфер обмена, как избежать артефактов при съёмке видеоигр или полноэкранных приложений. В конце вас ждёт сравнительная таблица всех методов — чтобы больше никогда не гуглить «как сделать скрин на ноутбуке Lenovo/HP/Asus».

1. Стандартные комбинации клавиш для Windows

На большинстве ноутбуков под управлением Windows скриншот делается за считанные секунды — без установки дополнительного ПО. Главное — запомнить правильные комбинации и понять, куда сохраняется результат.

Базовая кнопка PrtSc (или Print Screen) копирует весь экран в буфер обмена. Но есть нюансы:

  • 🖥️ Весь экран в буфер: нажмите PrtSc. Чтобы вставить скриншот, откройте Paint, Word или любой графический редактор и используйте Ctrl + V.
  • 🪟 Только активное окно: зажмите Alt + PrtSc. Это сработает, даже если окно свернуто (но должно быть активно!).
  • 💾 Сохранение в файл: комбинация Win + PrtSc автоматически сохраняет скриншот в папку Этот компьютер → Изображения → Скриншоты. Файл будет в формате .png с названием типа Скриншот (1).png.
  • 🎮 Для геймеров: в Windows 10/11 зажмите Win + Alt + PrtSc — скриншот игры сохранится в Видео → Записи (работает только для игр из Microsoft Store или запущенных в полноэкранном режиме).

На некоторых ноутбуках (например, HP Pavilion или Dell Inspiron) кнопка PrtSc может быть совмещена с другой функцией (например, уменьшением яркости). В этом случае придётся нажимать Fn + PrtSc. Чтобы проверить, посмотрите на клавиатуру: если PrtSc выделен другим цветом (обычно синим или оранжевым), значит, требуется Fn.

2. Инструмент «Ножницы» и его скрытые функции

Встроенная утилита «Ножницы» (или Snipping Tool в английской версии) позволяет делать скриншоты с задержкой, обрезать фрагменты и даже рисовать поверх изображения. Она есть во всех версиях Windows, начиная с Vista, но в Windows 11 получила значительные обновления.

Как открыть:

  • 🔍 Через поиск: введите «Ножницы» в строке поиска рядом с кнопкой Пуск.
  • ⚙️ Через меню: Пуск → Все программы → Стандартные → Ножницы.
  • 🖱️ Горячие клавиши: Win + Shift + S (открывает режим вырезки сразу, без запуска программы).

После запуска выберите один из режимов:

РежимОписаниеКогда использовать
Произвольная формаРисование области мышьюДля скриншотов нестандартных объектов (например, округлых кнопок)
ПрямоугольникВыделение прямоугольной областиДля большинства задач (самый универсальный)
ОкноСкриншот активного окнаЕсли нужно захватить только одно окно из нескольких открытых

В Windows 11 «Ножницы» научились распознавать текст на скриншотах (OCR). После создания снимка нажмите на иконку Текстовые действия (значок с буквой A) — программа выделит весь текст, который можно скопировать или отредактировать. Это удобно для сохранения информации из изображений (например, номеров заказов или адресов).

3. Скриншоты на ноутбуках с macOS

На MacBook и других ноутбуках Apple процесс создания скриншотов отличается от Windows, но не менее удобен. Здесь есть три основные комбинации:

Базовые комбинации:

  • 📸 Command + Shift + 3 — скриншот всего экрана. Файл сохранится на Рабочий стол в формате .png.
  • 📱 Command + Shift + 4 — выделение области мышью. После нажатия курсор превратится в прицел, которым можно обвести нужный фрагмент.
  • 🪟 Command + Shift + 4 → Пробел — скриншот конкретного окна. После нажатия Пробела окна подсветятся синим — кликните на нужное.

В macOS Mojave и новее появился расширенный инструмент скриншотов. Чтобы его открыть, используйте комбинацию Command + Shift + 5. Внизу экрана появится панель с опциями:

  • 🎥 Запись всего экрана или выбранной области.
  • ⏱️ Таймер задержки (5 или 10 секунд).
  • 📁 Выбор папки для сохранения (по умолчанию — Рабочий стол, но можно выбрать Документы, Буфер обмена или даже Почту).
  • 🖼️ Показать миниатюру после создания (полезно для быстрого редактирования).

Если скриншоты сохраняются в неудобном месте, измените папку по умолчанию через Терминал. Откройте его (Программы → Утилиты → Терминал) и введите команду:

defaults write com.apple.screencapture location ~/Downloads/ && killall SystemUIServer

Эта команда перенаправит все скриншоты в папку Загрузки. Замените Downloads на любой другой путь, если нужно.

⚠️ Внимание: На ноутбуках MacBook Pro с Touch Bar комбинации клавиш работают так же, но кнопка Command может быть скрыта за функциональными иконками. Чтобы её увидеть, нажмите Fn.

5. Онлайн-сервисы для скриншотов

Если нельзя установить программу (например, на рабочем ноутбуке с ограничениями), используйте онлайн-сервисы. Они работают прямо в браузере и не требуют скачивания. Главный плюс — доступ с любого устройства, даже с Chromebook.

Популярные сервисы:

  • 🌐 Screenshot.net — захват видимой части экрана или всей страницы (включая прокрутку).
  • 📱 Lightshot (веб-версия) — простой редактор с возможностью поделиться ссылкой.
  • 🎨 Nimbus Screenshot — расширение для Chrome с записью видео и аннотациями.
  • 🔍 Awesome Screenshot — захват длинных страниц (например, ленты Facebook или результатов поиска).

Как пользоваться:

  1. Откройте сервис в браузере.
  2. Нажмите кнопку Сделать скриншот (или Capture).
  3. Выделите область мышью или выберите весь экран.
  4. Отредактируйте (добавьте текст, стрелки, размытие) и сохраните на компьютер или поделитесь ссылкой.

Важно: Онлайн-сервисы могут сохранять ваши скриншоты на своих серверах. Если на изображении есть конфиденциальные данные (пароли, номера карт), используйте офлайн-программы или удаляйте скриншоты после загрузки.

⚠️ Внимание: Некоторые сервисы (например, Lightshot) добавляют водяные знаки на бесплатных аккаунтах. Чтобы их убрать, зарегистрируйтесь или используйте альтернативы вроде ShareX.

6. Скриншоты в играх и полноэкранных приложениях

Захват изображения из игры или видео часто вызывает проблемы: чёрный экран, артефакты или ошибки. Это связано с тем, что многие игры используют эксклюзивный полноэкранный режим (DirectX или OpenGL), который блокирует стандартные методы съёмки.

Решения:

  • 🎮 Steam: нажмите F12 — скриншот сохранится в облако Steam и на ваш компьютер (папка Steam → userdata → [ваш ID] → 760 → remote).
  • 🖥️ NVIDIA/AMD: если у вас видеокарта NVIDIA, используйте GeForce Experience (Alt + F1). Для AMD — Radeon Software (Alt + R).
  • 📹 OBS Studio: если нужно записать видео, это лучший вариант. Бесплатная программа с поддержкой захвата игры без потери FPS.
  • 🔄 Windowed Mode: переключите игру в оконный режим (Alt + Enter) — тогда сработают стандартные PrtSc или «Ножницы».

Для Minecraft, Roblox и других игр на Java стандартные методы обычно работают. Но если игра использует Anti-Cheat (например, Easy Anti-Cheat или BattlEye), некоторые программы для скриншотов могут быть заблокированы.

Почему скриншоты в играх получаются чёрными?

Чёрный экран на скриншоте — это защита от читерства. Некоторые античиты (например, в Valorant или Fortnite) блокируют доступ к буферу кадра. Решения:

1. Используйте встроенные инструменты игры (обычно есть кнопка для скриншотов в настройках).

2. Отключите античит (если это возможно в одиночной игре).

3. Сделайте фото экрана на телефон — радикальный, но рабочий способ.

7. Редактирование и аннотирование скриншотов

Скриншот готов, но на нём нужно выделить важные детали? Вот инструменты для редактирования:

Быстрые решения (без установки):

  • 🎨 Paint 3D (встроен в Windows 10/11) — 3D-эффекты, обрезка, текст.
  • 📝 Microsoft Word/PowerPoint — добавление стрелок, выносок, обводка.
  • 🌐 Pixlr — онлайн-фоторедактор с слоями (аналог Photoshop).

Продвинутые инструменты:

  • 🖌️ GIMP — бесплатный аналог Photoshop с поддержкой плагинов.
  • 📏 Figma или Canva — для создания инфографики на основе скриншотов.
  • 🔍 IrfanView — лёгкая программа для пакетной обработки (например, изменить размер 100 скриншотов за раз).

Если нужно размыть личные данные (например, email или номер телефона), используйте инструмент Размытие в Paint 3D или плагин Blur в GIMP. Для macOS подойдёт встроенное приложение Просмотр (Command + О для открытия скриншота, затем инструмент Размытие в панели редактирования).

💡

Для профессиональных аннотаций (например, для технической документации) используйте Snagit или Markup Hero. Они позволяют добавлять пошаговые инструкции, выноски и даже голосовые комментарии.

8. Автоматизация и горячие клавиши

Если вы делаете скриншоты ежедневно (например, для работы с клиентами или тестирования ПО), настройте горячие клавиши и автоматизацию:

В Windows:

  • Создайте ярлык для «Ножниц» и назначьте ему комбинацию (например, Ctrl + Alt + S). Для этого:
    1. Найдите программу «Ножницы» в меню Пуск.
    2. ПКМ → Дополнительно → Создать ярлык.
    3. ПКМ по ярлыку → Свойства → Ярлык → Назначить сочетание клавиш.
  • Используйте AutoHotkey для создания скриптов. Например, этот код сделает скриншот активного окна и сохранит его в Downloads:
    • #IfWinActive ahk_exe explorer.exe  ; Работает только в Проводнике

    F9::
    

    WinGetActiveTitle, Title
    

    Run, powershell -command "(New-Object -ComObject {1370D558-B5D5-4357-A7FF-996D421E4E4D}).CaptureActiveWindowToFile('C:\Users\%USERNAME%\Downloads\%Title%.png')"
    

    return

    В macOS:

    • Измените папку сохранения по умолчанию (см. раздел про macOS выше).
    • Настройте Automator для автоматической обработки скриншотов (например, конвертация в .jpg или отправка по email).
    • Используйте Alfred с плагином Screenshot для быстрого доступа к функциям съёмки.

    Для плановой съёмки (например, мониторинг изменения цен на сайте) подойдёт Screencast-O-Matic или Camtasia с функцией таймера. Эти программы могут делать скриншоты по расписанию и отправлять их на email.

    ⚠️ Внимание: Горячие клавиши в Windows могут конфликтовать с сочетаниями в играх или специализированном ПО (например, AutoCAD или Photoshop). Перед настройкой проверьте, не используются ли выбранные комбинации в других программах.

    FAQ: Частые вопросы о скриншотах на ноутбуке

    Мой ноутбук не реагирует на PrtSc. Что делать?

    Возможные причины:

    • Кнопка PrtSc работает только с Fn (попробуйте Fn + PrtSc).
    • Клавиша отключена в BIOS (редко, но бывает на некоторых Lenovo или Acer).
    • Драйвер клавиатуры не установлен (обновите через Диспетчер устройств).
    • Используйте альтернативу: Win + Shift + S (инструмент вырезки).
    Как сделать скриншот на ноутбуке с сенсорным экраном?

    На устройствах с Windows (например, Microsoft Surface или HP Spectre):

    • Нажмите и удерживайте кнопку Windows (внизу экрана) + кнопку Уменьшить громкость.
    • Или используйте стилус: в Windows Ink есть функция Экранная вырезка (открывается через центр уведомлений).

    На macOS сенсорный экран не поддерживается (кроме iPad в режиме Sidecar).

    Можно ли сделать скриншот с задержкой?

    Да, вот способы:

    • В «Ножницах» (Windows): нажмите Отложить и выберите время (до 5 секунд).
    • В macOS: используйте Command + Shift + 5 → Параметры → Таймер.
    • В ShareX: в настройках захвата установите задержку до 60 секунд.

    Это полезно, если нужно захватить выпадающее меню или контекстное окно.

    Куда сохраняются скриншоты на ноутбуке с Linux?

    Зависит от метода:

    • PrtSc: обычно в /home/имя_пользователя/Изображения или /Pictures.
    • Shift + PrtSc: выделение области, сохраняется в ту же папку.
    • Alt + PrtSc: скриншот активного окна.
    • В GNOME путь можно изменить через dconf-editor (раздел org.gnome.gnome-screenshot).
    Как сделать скриншот на ноутбуке с двумя мониторами?

    Способы в зависимости от ОС:

    • Windows:
      • PrtSc — захватит оба экрана как один.
      • Win + PrtSc — сохранит оба экрана в файл.
      • «Ножницы» (Win + Shift + S) — выделите область мышью (можно тянуть через оба монитора).
      • macOS:
        • Command + Shift + 3 — оба экрана.
        • Command + Shift + 4 → перетащите мышь с одного монитора на другой.

        Если мониторы имеют разное разрешение, скриншот будет соединяться по верхнему левому углу основного дисплея.
