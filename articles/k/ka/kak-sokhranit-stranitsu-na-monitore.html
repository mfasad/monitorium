Как сохранить страницу с монитора: от скриншота до записи видео

Вы когда-нибудь сталкивались с ситуацией, когда нужно сохранить то, что отображается на экране монитора? Будь то важная информация с веб-страницы, кадр из видео, настройки системы или просто красивый фон рабочего стола — вариантов масса. Но не всегда очевидно, какой способ выбрать: достаточно ли простого скриншота, нужна ли запись экрана с звуком, или требуется сохранить страницу в формате PDF для дальнейшей печати.

В этой статье мы разберём все актуальные методы сохранения экрана — от базовых комбинаций клавиш до специализированных программ для продвинутых пользователей. Вы узнаете, как сделать скриншот на Windows 10/11, macOS и Linux, как записать видео с экрана без потери качества, и даже как сохранить веб-страницу целиком со всеми элементами. А ещё — нюансы работы с несколькими мониторами, сохранение изображений в высоком разрешении и обход ограничений некоторых сайтов.

Неважно, новичок вы или опытный пользователь: здесь найдётся решение для любой задачи. Начнём с самого простого — и постепенно перейдём к более сложным техникам.

1. Базовый способ: скриншот всего экрана или активного окна

Самый быстрый метод — использовать встроенные инструменты операционной системы. Они не требуют установки дополнительного софта и работают в 99% случаев.

На Windows для этого предусмотрены горячие клавиши:

  • 🖥️ PrtScn (Print Screen) — копирует весь экран в буфер обмена. Затем вставьте изображение в Paint, Word или любой графический редактор (Ctrl + V).
  • 🪟 Alt + PrtScn — сохраняет только активное окно (например, браузер или программу).
  • 💾 Win + PrtScn — автоматически сохраняет скриншот в папку Этот компьютер → Изображения → Скриншоты (формат PNG).

На macOS комбинации другие:

  • 🍎 Command + Shift + 3 — скриншот всего экрана (сохраняется на рабочий стол).
  • 🖼️ Command + Shift + 4 — выделение области экрана мышью. После нажатия курсор превратится в прицел.
  • 📝 Command + Shift + 4 + Пробел — скриншот конкретного окна (окно подсветится синим).

Для Linux (например, Ubuntu или KDE Plasma) стандартные сочетания могут отличаться, но обычно работают:

  • 🐧 PrtScn — скриншот всего экрана (сохраняется в ~/Pictures).
  • 🖱️ Alt + PrtScn — активное окно.
  • 📐 Shift + PrtScn — выбор области.

Если горячие клавиши не работают, проверьте:

  1. Не отключены ли они в настройках системы (например, в Параметры → Специальные возможности → Клавиатура на Windows).
  2. Не конфликтуют ли они с софтом для игровых клавиатур (например, Razer Synapse или Logitech G HUB).
  3. На некоторых ноутбуках (например, Lenovo или HP) может потребоваться удерживать Fn (например, Fn + PrtScn).

💡

На Windows 11 скриншоты по Win + PrtScn сохраняются не только в папку, но и автоматически копируются в буфер обмена. Это удобно для быстрой вставки в мессенджеры или документы.

2. Продвинутые инструменты: «Ножницы» и встроенные редакторы

Если нужна не просто картинка, а скриншот с разметкой (стрелочки, текст, обрезка), используйте встроенные утилиты:

На Windows 10/11 это приложение «Ножницы»** (Snipping Tool):

  • 🔍 Запустите через поиск в меню Пуск или нажмите Win + Shift + S (режим вырезки).
  • 📏 Выберите тип скриншота: произвольная форма, прямоугольник, окно или весь экран.
  • ✏️ После захвата откроется редактор, где можно добавить аннотации (маркер, ластик, текст).
  • 💾 Сохраните файл в PNG, JPEG или GIF.

На macOS встроенный редактор появляется после создания скриншота (миниатюра в правом нижнем углу). Кликните по ней, чтобы:

  • 🖌️ Обрезать края.
  • 📝 Добавить текст или подпись.
  • 🔗 Поделиться напрямую в Messages или Mail.

Для Linux популярны утилиты вроде Flameshot или Shutter (устанавливаются через терминал): 

sudo apt install flameshot  # Для Debian/Ubuntu

sudo pacman -S flameshot    # Для Arch Linux

Flameshot позволяет:
  • 🎨 Размывать части изображения.
  • 🔍 Подсвечивать области.
  • 📌 Закреплять скриншоты на рабочем столе.

Если вам часто приходится делать скриншоты с пометками, рассмотрите специализированные программы:

  • 🖥️ ShareX (Windows) — бесплатная утилита с поддержкой OCR, записью GIF и загрузкой на облачные сервисы.
  • 🍎 CleanShot X (macOS) — платный инструмент с расширенными функциями (например, скрытие значков рабочего стола).
  • 🐧 Ksnip (Linux/Windows/macOS) — кроссплатформенный редактор с поддержкой аннотаций и пикселизации.

⚠️ Внимание: При использовании сторонних программ для скриншотов проверьте их разрешения. Некоторые утилиты (например, Lightshot) загружают изображения на свои серверы по умолчанию, что может быть небезопасно для конфиденциальных данных.

3. Сохранение веб-страницы целиком: PDF и другие форматы

Скриншот экрана фиксирует только видимую область, но что делать, если нужно сохранить всю страницу — например, длинную статью или таблицу? Здесь помогут другие методы:

Способ 1: Печать в PDF (работает в любом браузере):

  1. Откройте нужную страницу в Chrome, Firefox или Edge.
  2. Нажмите Ctrl + P (или Command + P на macOS).
  3. В поле «Назначение»** выберите Сохранить как PDF.
  4. Уберите галочку Фоновые графики, если они не нужны (уменьшит размер файла).
  5. Нажмите Сохранить.

Способ 2: Расширения для браузера:

  • 🌐 Full Page Screen Capture (Chrome) — сохраняет страницу в PNG или PDF одним кликом.
  • 🦊 FireShot (Firefox/Chrome) — позволяет редактировать скриншот перед сохранением.
  • 📱 GoFullPage — захватывает даже лениво загружаемый контент (например, бесконечную ленту в соцсетях).

Способ 3: Консольные инструменты (для продвинутых пользователей): 

wkhtmltopdf https://example.com output.pdf  # Сохранение страницы в PDF

puppeteer screenshot --fullpage https://example.com screenshot.png  # Node.js

Метод Плюсы Минусы Подходит для

Консольные утилиты Автоматизация, высокое качество Требует технических знаний Массовое сохранение страниц

Если страница содержит динамический контент (например, анимацию или видео), ни один из этих методов не сохранит его в исходном виде. В таком случае поможет только запись экрана (см. следующий раздел).

⚠️ Внимание: Некоторые сайты (например, Netflix или YouTube) блокируют возможность сохранения страницы через Ctrl + P или расширения. Это связано с защитой авторских прав. Обход таких ограничений может нарушать пользовательское соглашение.

4. Запись видео с экрана: когда скриншота недостаточно

Если нужно сохранить не статичное изображение, а процесс (например, баг в программе, геймплей или вебинар), потребуется запись экрана. Вот основные способы:

Встроенные средства Windows 10/11:

  • 🎮 Игровая панель Xbox (Win + G):
    • Откройте нужное окно.
    • Нажмите Win + G, затем кликните на кнопку записи (или Win + Alt + R).
    • Видео сохранится в Видео → Захват (формат MP4).
  • ⚙️ Ограничения:
    • Записывает только активное окно (не весь экран).
    • Максимальное качество — 1080p при 30 FPS.

macOS:

  • 🎥 Встроенная запись экрана (Command + Shift + 5):
    • Выберите область записи или весь экран.
    • Нажмите Запись (можно включить микрофон).
    • Видео сохранится в MOV на рабочий стол.

Специализированные программы:

  • 🖥️ OBS Studio (бесплатно, Windows/macOS/Linux) — профессиональный инструмент для стримов и записи с настройкой битрейта, источников звука и т.д.
  • 🎬 Camtasia (платный) — удобный редактор с эффектами и аннотациями.
  • 📱 ScreenFlow (macOS) — оптимизирован для создания туториалов.

Для записи игрового процесса или приложений с высокими требованиями к FPS (например, 3D-моделирование), используйте:

  • 🎮 NVIDIA ShadowPlay (для видеокарт NVIDIA) — минимальная нагрузка на систему.
  • 🖼️ AMD ReLive (для видеокарт AMD) — поддерживает запись в 4K.

Как записать экран с звуком из системы и микрофона?

Чтобы записать и звук из колонок, и голос с микрофона одновременно:

1. В OBS Studio добавьте источники: Захват аудио вывода (для звука из ПК) и Захват аудио входа (для микрофона).

2. В настройках звука выберите правильные устройства (например, Стерео Микшер для системного звука).

3. Перед записью проверьте уровни громкости в микшере (чтобы не было перегруза).

Если вам нужно записать вебинар или онлайн-встречу (например, в Zoom или Google Meet), учтите:

  • 🔇 Некоторые платформы блокируют запись экрана через стандартные средства (показывают чёрный экран).
  • 🎤 Для захвата звука конференции может потребоваться виртуальный аудиодрайвер (например, VB-Cable).
  • 📹 В Zoom есть встроенная функция записи (разрешение хозяина встречи).

5. Работа с несколькими мониторами: как сохранить нужный экран

Если у вас многомониторная система (например, ноутбук + внешний монитор), стандартные методы могут захватывать не тот экран. Вот как управлять процессом:

На Windows:

  • 🖥️ PrtScn — скриншот всех мониторов как одно изображение.
  • 🪟 Win + Shift + S (Snipping Tool) — позволяет выбрать область на любом экране.
  • 🎮 Win + Alt + G — запись последних 30 секунд только активного окна (даже если оно на втором мониторе).

На macOS:

  • 🍎 Command + Shift + 4 + Пробел — наведите курсор на нужный монитор и кликните по окну.
  • 🖼️ В утилите «Скриншоты»** (macOS Mojave и новее) можно выбрать конкретный дисплей в меню.

Проблемы и решения:

  • 🔄 Разные разрешения мониторов: Если экраны с разным DPI (например, 4K и Full HD), скриншот может получиться искажённым. Решение — сделать скриншоты каждого монитора отдельно.
  • 🖱️ Курсор перескакивает между экранами: В OBS Studio настройте МULTI-MONITOR CAPTURE в источниках.
  • 🔊 Звук записывается только с основного монитора: Проверьте настройки аудиоустройств в панели управления.

💡

При записи видео с нескольких мониторов в OBS Studio добавьте отдельный источник Захват экрана для каждого дисплея. Это позволит контролировать компоновку в конечном видео.

Если вам нужно сохранить расширенный рабочий стол (например, для презентации), используйте:

  • 📏 ShareX → Capture → Multi-monitor.
  • 🖼️ Snagit (платный) — поддерживает панорамные скриншоты.

6. Сохранение экрана в высоком разрешении: 4K, HDR и другие нюансы

Если вы работаете с профессиональной графикой, 4K-мониторами или HDR-контентом, стандартные методы могут не подойти. Вот как сохранить экран без потери качества:

Проблема 1: Скриншот в низком разрешении

  • 🔍 Причина: многие программы по умолчанию сохраняют скриншоты в 96 DPI (даже если монитор 4K).
  • 🛠️ Решение:
    • В Windows измените настройки масштабирования: Параметры → Система → Дисплей → Масштаб (установите 100%).
    • В macOS используйте Terminal: 
      defaults write com.apple.screencapture disable-shadow -bool true

      defaults write com.apple.screencapture type png

Проблема 2: HDR-контент отображается тускло

  • 🌈 Причина: большинство форматов (PNG, JPEG) не поддерживают HDR.
  • 🛠️ Решение:
    • Используйте Windows HD Color + NVIDIA Ansel (для игр).
    • Сохраняйте в TIFF или EXR (поддерживают 10+ бит цветности).

Проблема 3: Артефакты при скриншотах в играх

  • 🎮 Причина: многие игры используют Fullscreen Exclusive Mode, который блокирует стандартные методы захвата.
  • 🛠️ Решение:
    • Переключите игру в оконный режим (Alt + Enter).
    • Используйте Steam Overlay (F12) или NVIDIA ShadowPlay.
    • Для DirectX 12 попробуйте Windows Game Bar (Win + Alt + PrtScn).

Формат Поддержка HDR Макс. битность Сжатие Рекомендации
PNG ❌ Нет 8 бит Без потерь Для скриншотов текста/интерфейсов
JPEG ❌ Нет 8 бит С потерями Для фотографий (не для HDR)
TIFF ⚠️ Частично 16+ бит Без потерь Для профессиональной печати
EXR ✅ Да 32 бит Без потерь Для 3D-рендера и HDR

Важно: При сохранении скриншотов с мониторов с разрешением выше 4K (например, 5K или 8K) используйте форматы без сжатия (TIFF, BMP), чтобы избежать артефактов на мелких деталях.

7. Автоматизация и массовое сохранение экрана

Если вам нужно регулярно сохранять экран (например, для мониторинга, создания документации или тестирования ПО), ручные методы будут неэффективны. Здесь помогут скрипты и специализированные инструменты.

Способ 1: Планировщик задач (Windows)

  • 📅 Откройте Планировщик задач (taskschd.msc).
  • 🔄 Создайте задачу с триггером (например, каждые 5 минут).
  • 📝 В действии укажите команду: 
    powershell -command "(New-Object -ComObject WScript.Shell).SendKeys('{PRTSC}')"
  • 💾 Добавьте второй шаг для сохранения из буфера (например, через Paint или IrfanView).

Способ 2: Скрипты на Python 

import pyautogui

import time



while True:


screenshot = pyautogui.screenshot()


screenshot.save(f"screenshot_{time.strftime('%Y%m%d_%H%M%S')}.png")


time.sleep(300)  # Пауза 5 минут

Установите библиотеку: pip install pyautogui

Способ 3: Программы для автоматического захвата

  • 🖥️ AutoScreenCap — делает скриншоты по расписанию с настраиваемым именем файла.
  • 📊 ScreenRec — записывает экран и загружает видео на облако (полезно для удалённой работы).
  • 🤖 Robotask — автоматизирует скриншоты + обработку (например, отправку по email).

Примеры применения:

  • 📈 Мониторинг биржевых графиков: скриншоты каждые 10 минут для анализа трендов.
  • 🐞 Тестирование ПО: автоматическая фиксация багов с привязкой ко времени.
  • 📚 Создание документации: последовательные скриншоты шагов настройки программы.

⚠️ Внимание: При автоматизированном сохранении экрана учитывайте нагрузку на систему. Постоянная запись или скриншоты каждую секунду могут привести к лагам, особенно на слабых ПК. Оптимальный интервал — не чаще 1 скриншота в 30 секунд.

8. Сохранение экрана на мобильных устройствах и планшетах

Если вам нужно сохранить экран не на ПК, а на смартфоне или планшете, методы будут другими. Вот актуальные способы для популярных устройств:

Android:

  • 📱 Стандартный способ: одновременно нажмите Питание + Уменьшение громкости (на большинстве устройств).
  • 🤖 Жесты: на некоторых прошивках (например, MIUI или One UI) можно сделать скриншот проведением трёх пальцев по экрану.
  • 📲 Расширенные функции:
    • Длинный скриншот (прокрутка): после стандартного скриншота нажмите Прокрутить (доступно на Samsung, Xiaomi, Huawei).
    • Запись экрана: в панели уведомлений найдите кнопку Запись экрана (или установите приложение AZ Screen Recorder).

iPhone/iPad:

  • 🍏 Скриншот: нажмите Боковая кнопка + Увеличение громкости (на моделях с Face ID).
  • 🎥 Запись экрана:
    1. Добавьте кнопку записи в Пункт управления (Настройки → Пункт управления → Настройка элементов).
