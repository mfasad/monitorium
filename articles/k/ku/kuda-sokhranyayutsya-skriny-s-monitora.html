Куда сохраняются скриншоты с монитора: пути хранения в Windows, macOS и Linux

Скриншоты — незаменимый инструмент для фиксации информации с экрана, но многие пользователи сталкиваются с проблемой: куда же деваются сделанные снимки? По умолчанию операционные системы сохраняют их в стандартные папки, которые не всегда очевидны. В Windows это может быть Изображения\Скриншоты, в macOS — рабочий стол, а в Linux пути зависят от дистрибутива и настроек окружения. Если вы не можете найти скриншот или хотите изменить место сохранения, эта статья поможет разобраться во всех нюансах.

Особенно актуальна тема для тех, кто активно использует горячие клавиши (PrtScn, Win+PrtScn, Cmd+Shift+3/4) или специализированные утилиты вроде Snipping Tool, Lightshot или Flameshot. Мы рассмотрим не только стандартные пути хранения, но и способы их перенастройки, а также решения проблем с пропавшими снимками. Отдельное внимание уделим особенностям многомониторных систем и виртуальных машин, где логика сохранения может отличаться.

Стандартные пути сохранения скриншотов в Windows

В операционных системах семейства Windows (от Windows 7 до Windows 11) скриншоты сохраняются в разных папках в зависимости от способа их создания. Рассмотрим основные варианты:

  • 🖼️ Клавиша PrtScn (Print Screen): Копирует изображение всего экрана в буфер обмена. Файл не сохраняется автоматически — его нужно вставить в графический редактор (например, Paint или Photoshop) и сохранить вручную.
  • 📁 Комбинация Win + PrtScn: Сохраняет снимок всего экрана в папку C:\Users\[Имя_пользователя]\Pictures\Screenshots (в Windows 10/11). В Windows 7/8 эта комбинация может не работать без дополнительных настроек.
  • ✂️ Утилита Snipping Tool (Shift + Win + S): По умолчанию предлагает сохранить снимок вручную, но последнюю версию можно настроить на автоматическое сохранение в Документы\Snipping Tool.
  • 🔍 Игровой режим (Win + Alt + PrtScn): Сохраняет скриншоты из игр в Видео\Захват (только для приложений, поддерживающих Xbox Game Bar).

Если папка Screenshots отсутствует, она будет создана автоматически при первом использовании комбинации Win + PrtScn. В Windows 11 путь можно изменить через реестр или групповую политику, но это требует прав администратора.

⚠️ Внимание: В виртуальных машинах (например, VirtualBox или VMware) комбинация Win + PrtScn может конфликтовать с хост-системой. Чтобы сделать снимок внутри ВМ, используйте Host Key + PrtScn (по умолчанию PrtScn захватит экран хоста).

Где хранятся скриншоты в macOS: особенности системы

В macOS логика сохранения скриншотов более предсказуема, но имеет свои нюансы. По умолчанию все снимки экрана сохраняются на рабочий стол в формате ПНГ с именем вида Снимок экрана [дата] в [время].png. Однако путь можно легко изменить через терминал или настройки.

Основные комбинации и их поведение:

Комбинация клавиш Что сохраняется Путь по умолчанию
Cmd + Shift + 3 Весь экран Рабочий стол
Cmd + Shift + 4 Выделенная область Рабочий стол
Cmd + Shift + 4 + Пробел Активное окно Рабочий стол
Cmd + Ctrl + Shift + 3/4 Копирование в буфер (без сохранения файла) —

Чтобы изменить папку сохранения, откройте Терминал и введите команду:

defaults write com.apple.screencapture location /путь/к/папке

killall SystemUIServer

Например, для сохранения в Документы/Screenshots:

defaults write com.apple.screencapture location ~/Documents/Screenshots
defaults write com.apple.screencapture type jpg

После этого перезагрузите SystemUIServer.

Linux: где ищутся скриншоты в разных дистрибутивах

В Linux пути сохранения скриншотов зависят от используемого окружения рабочего стола (GNOME, KDE Plasma, XFCE) и настроек пользователя. Большинство дистрибутивов по умолчанию сохраняют снимки в домашнюю папку (~/) или в ~/Pictures.

Распространённые варианты:

  • 🐧 GNOME (Ubuntu, Fedora): Комбинация PrtScn сохраняет в ~/Pictures/Screenshots. Если папка отсутствует, она создаётся автоматически.
  • 💻 KDE Plasma (Kubuntu, KDE Neon): Использует утилиту Spectacle, которая по умолчанию сохраняет в ~/Pictures или предлагает выбрать путь.
  • 🖥️ XFCE (Xubuntu, Linux Mint XFCE): Скриншоты сохраняются в ~/ (домашняя папка) с именем Screenshot_[дата].png.
  • 📌 Терминальные утилиты: Команды вроде scrot или gnome-screenshot --file=~/custom_path/image.png позволяют явно указать путь.

Чтобы изменить стандартную папку в GNOME, используйте dconf Editor или команду:

gsettings set org.gnome.gnome-screenshot auto-save-directory "file:///путь/к/папке"
⚠️ Внимание: В некоторых дистрибутивах (например, Arch Linux) утилиты для скриншотов могут отсутствовать по умолчанию. Установите их через пакетный менеджер: 
sudo pacman -S scrot  # для Arch/Manjaro

sudo apt install scrot # для Debian/Ubuntu

Как изменить папку сохранения скриншотов в Windows

Если вас не устраивает стандартная папка Pictures\Screenshots, её можно перенастроить несколькими способами. Рассмотрим два основных метода: через реестр и с помощью символических ссылок.

Метод 1: Редактирование реестра

  1. Нажмите Win + R, введите regedit и подтвердите.
  2. Перейдите по пути: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer
  3. Создайте новый строковый параметр (String Value) с именем ScreenshotIndex.
  4. В качестве значения укажите полный путь к новой папке, например: D:\MyScreenshots
  5. Перезагрузите компьютер.

Метод 2: Символическая ссылка (для продвинутых)

Если вы не хотите редактировать реестр, можно создать символическую ссылку, перенаправляющую стандартную папку в другое место:

mklink /D "C:\Users\[Имя_пользователя]\Pictures\Screenshots" "D:\Новая_папка"

Эта команда заменит стандартную папку ссылкой на новое расположение. Важно: оригинальную папку Screenshots перед этим нужно удалить или переименовать.

Поиск и восстановление удалённых скриншотов

Если скриншот был удалён или вы не можете его найти, воспользуйтесь следующими методами:

  • 🔍 Поиск по системе: Введите в строке поиска Windows или Spotlight (macOS) запрос вида тип:изображение дата:сегодня или kind:png created:today.
  • 🗑️ Корзина: Проверьте корзину — возможно, файл был удалён случайно. В macOS корзина называется Trash.
  • 🛠️ Программы восстановления: Утилиты вроде Recuva (Windows), Disk Drill (macOS/Linux) или PhotoRec (кроссплатформенная) могут восстановить удалённые файлы, если они не были перезаписаны.
  • ☁️ Облачные резервные копии: Если включена синхронизация с OneDrive, Google Drive или iCloud, проверьте папки Скриншоты в облаке.

Для восстановления через Recuva:

  1. Скачайте и установите программу с официального сайта.
  2. Выберите тип файлов Графика.
  3. Укажите диск, на котором хранились скриншоты (обычно C:).
  4. Запустите глубокое сканирование.
  5. Найдите файлы с расширением .png или .jpg и восстановите их.
⚠️ Внимание: Если скриншот был сделан в виртуальной машине и не сохранился на хост-систему, ищите его внутри гостевой ОС. Для этого подключите общую папку между хостом и ВМ или используйте функцию drag-and-drop (если поддерживается).

Специфика скриншотов в многомониторных системах

При работе с несколькими мониторами логика сохранения скриншотов может отличаться. Вот ключевые моменты:

  • 🖥️🖥️ Windows: Комбинация Win + PrtScn сохраняет все экраны в один файл. Если нужно сфотографировать только активный монитор, используйте Snipping Tool или Alt + PrtScn (скопирует только активное окно).
  • 🍎 macOS: Комбинация Cmd + Shift + 3 по умолчанию захватит все подключённые дисплеи. Чтобы сделать снимок только основного экрана, временно отключите дополнительные мониторы или используйте Cmd + Shift + 4 для выделения области.
  • 🐧 Linux: В GNOME и KDE утилиты скриншотов обычно предлагают выбор — захватывать все экраны или только активный. Например, в Flameshot можно настроить это в параметрах запуска.

Если скриншот многомониторной системы сохраняется в низком разрешении, проверьте настройки масштабирования в параметрах дисплея. В Windows это можно сделать в Параметры → Система → Дисплей → Масштаб и разметка.

Как сделать скриншот только одного монитора в Windows?

Используйте комбинацию Win + Shift + S (вырезка области), затем вручную выделите нужный экран. Или настройте Snipping Tool на захват окна: откройте утилиту, нажмите Отложить, затем выберите Окно и кликните по нужному монитору (если он отображается как отдельное окно в менеджере задач).

Проблемы со скриншотами: почему они не сохраняются

Иногда скриншоты не сохраняются или пропадают. Распространённые причины и решения:

Проблема Возможная причина Решение
Скриншоты не сохраняются при нажатии Win + PrtScn Отключена служба Clipboard или конфликт с драйверами клавиатуры Перезапустите explorer.exe через диспетчер задач или обновите драйверы клавиатуры
Файлы сохраняются в неверной папке Изменены настройки реестра или символическая ссылка сломана Проверьте путь в реестре или пересоздайте символическую ссылку
Скриншоты черные или пустые Конфликт с графическими драйверами или защитой от скриншотов (например, в Netflix или банковских приложениях) Обновите драйвер видеокарты или используйте сторонние утилиты вроде ShareX
Не работают горячие клавиши в Linux Отсутствует утилита для скриншотов или конфликт с менеджером окон Установите scrot или gnome-screenshot и настройте сочетания клавиш в параметрах системы

Если проблема сохраняется, попробуйте использовать альтернативные программы:

  • 🪟 Windows: ShareX, Lightshot, Greenshot.
  • 🍎 macOS: CleanShot X, Snagit.
  • 🐧 Linux: Flameshot, Shutter.
💡

Если скриншоты не сохраняются в играх, попробуйте отключить полноэкранный режим (перейдите в оконный) или используйте встроенные инструменты игры (например, F12 в Steam).

FAQ: Частые вопросы о скриншотах

Можно ли изменить формат сохранения скриншотов с PNG на JPG?

Да, в Windows для этого потребуется стороннее ПО (например, ShareX позволяет выбрать формат). В macOS используйте команду в терминале:

defaults write com.apple.screencapture type jpg

В Linux (например, в scrot) укажите формат при запуске:

scrot screenshot.jpg
Куда сохраняются скриншоты с телефона, если я подключил его как второй монитор?

Если вы используете телефон как второй экран через Duet Display, Spacedesk или аналогичные программы, скриншоты делаются на основном ПК и сохраняются в его стандартную папку. Сами приложения для дублирования экрана обычно не поддерживают создание скриншотов с мобильного устройства.

Как сделать скриншот в BIOS или при загрузке Windows?

В BIOS/UEFI или на экране загрузки горячие клавиши не работают. Используйте:

  • 📱 Фотоаппарат или телефон — сфотографируйте экран.
  • 💾 Специальные утилиты вроде BlueScreenView (для экранов BSOD) или HWiNFO (для логов загрузки).
Почему скриншоты в играх получаются чёрными?

Это связано с защитой от читерства (например, Easy Anti-Cheat или Denuvo). Решения:

  • Используйте встроенные инструменты игры (обычно клавиша F12 или PrtScn в Steam).
  • Отключите полноэкранный режим (перейдите в оконный).
  • Используйте NVIDIA ShadowPlay или AMD ReLive для записи экрана.
Можно ли автоматически загружать скриншоты в облако?

Да, для этого настройте:

  • 🪟 Windows: OneDrive (включите автоматическую загрузку папки Screenshots) или ShareX с интеграцией Google Drive.
  • 🍎 macOS: Включите iCloud Drive и сохраняйте скриншоты в папку iCloud.
  • 🐧 Linux: Настройте rclone для синхронизации с Yandex Диском или Dropbox.
