Как узнать герцовку монитора в Windows 10: 5 способов с пояснениями

Частота обновления экрана (герцовка) — это параметр, от которого зависит плавность картинки на мониторе. В играх, при просмотре видео или работе с графикой даже разница между 60 Гц и 144 Гц может быть заметна невооружённым глазом. Но как проверить, какая герцовка установлена на вашем мониторе в Windows 10? Особенно если система автоматически снижает частоту из-за проблем с драйверами или кабелем.

В этой статье мы разберём все способы — от стандартных настроек до специализированных утилит. Вы узнаете не только где посмотреть текущую герцовку, но и как её изменить, если монитор поддерживает более высокие значения. А ещё объясним, почему иногда Windows 10 «не видит» максимальную частоту обновления и что с этим делать.

Способ 1: через параметры экрана Windows 10

Самый простой метод — использовать встроенные настройки системы. Он подходит для большинства мониторов, подключённых по HDMI, DisplayPort или DVI.

Откройте меню Пуск → Параметры (шестерёнка) → Система → ДISPLAY. Прокрутите вниз до раздела «Дополнительные параметры экрана» и кликните по ссылке Свойства графического адаптера. В открывшемся окне перейдите на вкладку Монитор — там в выпадающем списке Частота обновления экрана будет указано текущее значение в герцах.

  • ✅ Плюсы: не требует установки дополнительного ПО.
  • ⚠️ Минусы: иногда список доступных частот ограничен из-за устаревших драйверов.
  • 🔧 Если нет нужной герцовки: обновите драйвер видеокарты (см. раздел про драйверы).
⚠️ Внимание: Если монитор подключён через VGA или адаптер (например, HDMI→DVI), максимальная частота может быть искусственно ограничена до 60 Гц даже при поддержке более высоких значений.

Способ 2: через панель управления NVIDIA/AMD/Intel

Производители видеокарт предлагают собственные утилиты для тонкой настройки дисплеев. Они часто показывают больше информации, чем стандартные настройки Windows.

Для карт NVIDIA:

  1. Кликните правой кнопкой на рабочем столе → Панель управления NVIDIA.
  2. Перейдите в ДISPLAY → Изменение разрешения.
  3. В разделе Частота обновления выберите нужное значение из списка.

Для AMD Radeon:

  1. Откройте AMD Radeon Software (иконка в трее или через правый клик на рабочем столе).
  2. Перейдите в Display → выберите монитор → Custom Resolutions.

💡

Если в списке нет нужной герцовки, попробуйте создать кастомное разрешение в панелях NVIDIA/AMD. Для этого потребуется знать максимальные параметры вашего монитора (см. инструкцию к модели).

Производитель Путь к настройкам герцовки Особенности
NVIDIA Панель управления → ДISPLAY → Изменение разрешения Поддерживает G-Sync и кастомные разрешения
AMD Radeon Software → Display → Custom Resolutions Требует перезагрузку для применения нестандартных частот
Intel Панель управления графикой → ДISPLAY → Общие настройки Ограниченные опции для встроенной графики

Способ 3: через командную строку (CMD)

Для продвинутых пользователей есть метод проверки герцовки через Command Prompt. Он полезен, если графический интерфейс недоступен (например, при удалённом подключении).

Откройте CMD от имени администратора и выполните команду: 

wmic desktopmonitor get screenheight, screenwidth, refreshrate

В ответе будет строка с текущим разрешением и частотой обновления (например, 1920 1080 144 означает 144 Гц).

  • 📌 Альтернативная команда: powercfg /query — показывает текущий план электропитания, который может ограничивать герцовку.
  • ⚠️ Ограничение: команда не покажет доступные частоты, только текущую.

Способ 4: сторонние утилиты (AIDA64, HWInfo, TestUFO)

Если системные инструменты не показывают реальную герцовку (например, из-за багов драйверов), поможет специализированное ПО. Вот топ-3 программы:

  1. AIDA64:
    • Перейдите в ДISPLAY → Монитор.
    • Посмотрите строку Refresh Rate.
  2. HWInfo:
    • Запустите программу → Monitor → Current Refresh Rate.
  • TestUFO (онлайн-тест):
    • Откройте сайт testufo.com в браузере.
    • Прокрутите до раздела Frame Rate — там отобразится текущая частота.

      • TestUFO особенно полезен для проверки реальной герцовки в играх или браузере, так как показывает фактическое значение, а не то, что «думает» Windows.

      Скачать и установить AIDA64|Запустить программу|Перейти в раздел DISPLAY → Монитор|Найти строку Refresh Rate|Сверить значение с паспортом монитора-->

      Способ 5: обновление драйверов для разблокировки герцовки

      Частая проблема: монитор поддерживает 144 Гц, но в настройках Windows 10 доступны только 60 Гц. Виновник — устаревшие или неправильно установленные драйверы.

      Что делать:

      1. Обновите драйвер видеокарты:
        • Для NVIDIA: скачайте с сайта nvidia.com.
        • Для AMD: используйте AMD Adrenalin.
        • Для Intel: обновите через Intel Driver & Support Assistant.
    • Удалите старые драйверы с помощью Display Driver Uninstaller (DDU) (перед установкой новых).
    • Проверьте кабель: DisplayPort 1.4 поддерживает до 240 Гц, а HDMI 2.0 — только до 144 Гц при 1080p.

    ⚠️ Внимание: После обновления драйверов может потребоваться сброс настроек монитора (через меню самого дисплея) или перезагрузка ПК в Безопасном режиме.
    Почему после обновления драйверов герцовка не изменилась?

    Если монитор по-прежнему работает на 60 Гц, проверьте:

    1. Тип кабеля — например, HDMI 1.4 не потянет 144 Гц на 4K.

    2. Настройки BIOS/UEFI — некоторые материнские платы ограничивают частоту для встроенной графики.

    3. Профиль цветов — попробуйте сбросить его в настройках Windows (Параметры → Система → ДISPLAY → Дополнительные параметры экрана → Сбросить).

    Как проверить максимальную герцовку монитора

    Чтобы узнать, какую частоту поддерживает ваш монитор, не полагайтесь только на настройки Windows. Вот надёжные источники:

    • 📄 Документация: найдите модель монитора на сайте производителя (например, ASUS ROG Swift PG279Q поддерживает до 165 Гц).
    • 🔍 Меню монитора: зажмите кнопку Menu на корпусе → найдите раздел Information или Display.
    • 🛠️ Тесты: используйте Blurry Busters UFO Test (testufo.com) для визуальной проверки.

    Если монитор поддерживает 120+ Гц, но Windows 10 не даёт выбрать эту частоту, проблема может быть в:

    • Кабеле (например, HDMI 1.4 вместо DisplayPort).
    • Адаптере (некоторые конвертеры HDMI→DVI блокируют высокие частоты).
    • Драйвере (особенно актуально для ноутбуков с гибридной графикой).
    💡

    Если монитор поддерживает 144 Гц, но в Windows доступны только 60 Гц, в 90% случаев виноват кабель или драйвер. Начните проверку с них!

    Частые ошибки и как их исправить

    Даже после правильной настройки герцовки могут возникать проблемы. Вот самые распространённые:

    Проблема Причина Решение
    Черный экран после смены герцовки Монитор не поддерживает выбранную частоту Подождите 15 секунд — Windows вернёт старые настройки. Если нет — перезагрузите ПК в Безопасном режиме.
    Мерцание экрана на высокой герцовке Плохой кабель или помехи Попробуйте другой кабель или уменьшите частоту до 120 Гц.
    Герцовка сбрасывается после сна Баг драйвера или плана электропитания Отключите Быстрый запуск в настройках электропитания.

    Если ни один из способов не помог, проверьте монитор на другом ПК или ноутбуке. Возможно, проблема в самом дисплее (например, повреждён разъём DisplayPort).

    ⚠️ Внимание: На некоторых мониторах (особенно UltraWide или 4K) максимальная герцовка доступна только при снижении разрешения. Например, LG 27GN950 выдаёт 144 Гц на 3840×2160, но 160 Гц — только на 3440×1440.

    FAQ: ответы на популярные вопросы

    Можно ли повысить герцовку выше заводской (например, с 60 Гц до 75 Гц)?

    Технически да, но это не рекомендуется. Разгон герцовки (overclocking) может привести к:

    • Перегреву матрицы.
    • Артефактам (мерцанию, полосам).
    • Сокращению срока службы монитора.

    Если всё же хотите попробовать, используйте Custom Resolution Utility (CRU) + NVIDIA/AMD Panel для создания нестандартного профиля. Но сначала проверьте, нет ли аналогичных отзывов о вашей модели на форумах (например, Overclockers.ru или Reddit).

    Почему в играх FPS выше, чем герцовка монитора?

    Это нормально! Если ваша видеокарта выдаёт 200 FPS, а монитор работает на 144 Гц, лишние кадры либо игнорируются (при включённом V-Sync), либо отображаются частично (без V-Sync), что может вызывать теаринг (разрывы изображения).

    Решения:

    • Включите V-Sync в настройках игры (ограничит FPS до герцовки монитора).
    • Используйте G-Sync (для NVIDIA) или FreeSync (для AMD) для плавной синхронизации.
    • Ограничьте FPS через RivaTuner или NVIDIA Refle.

    Как узнать герцовку на ноутбуке с двумя видеокартами (Intel + NVIDIA)?

    На ноутбуках с гибридной графикой (Optimus) герцовка может зависеть от того, какая видеокарта активна. Чтобы проверить:

    1. Подключите внешний монитор к дискретной видеокарте (обычно разъём HDMI/DP на задней панели).
    2. В Панели управления NVIDIA выберите Управление параметрами 3D → Предпочтительный графический процессор → Высокопроизводительный процессор NVIDIA.
    3. Проверьте герцовку через Параметры Windows или AIDA64.

    Если герцовка всё равно ограничена, обновите драйверы обеих видеокарт и проверьте настройки BIOS (иногда там есть опция Switchable Graphics).

    Влияет ли герцовка на производительность в играх?

    Да, но косвенно. Сама герцовка не нагружает видеокарту, но:

    • Чем выше Гц, тем больше FPS нужно для плавности. Например, для 144 Гц желательно иметь стабильные 144 FPS.
    • На высоких герцовках становятся заметны микрозадержки (input lag), поэтому важно оптимизировать настройки игры.
    • Мониторы с 1 Гц = 1 мс (например, 240 Гц) дают преимущество в шутерах (CS2, Valorant), но требуют топовой видеокарты.

    Для офисной работы или кино 60 Гц достаточно, а для киберспорта приоритет — 144+ Гц.

    Можно ли включить 144 Гц на мониторе через USB-C?

    Да, но с оговорками:

    • USB-C с DisplayPort Alt Mode поддерживает до 240 Гц (например, на MacBook Pro или современных ультрабуках).
    • USB-C без Alt Mode (обычный USB 3.0) не передаёт видео — максимальная герцовка будет ограничена 30 Гц.
    • Проверьте спецификацию порта в документации ноутбука или монитора.

    Если подключение через USB-C даёт только 60 Гц, попробуйте:

    • Использовать кабель USB-C → DisplayPort.
    • Обновить драйвер Thunderbolt (если есть).
