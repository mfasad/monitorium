Как записать видео с экрана монитора со звуком: 8 проверенных способов

Запись видео с экрана монитора со звуком требует не только захвата изображения, но и синхронной фиксации системного аудио, микрофона или обоих источников одновременно. Если вы пытались сохранить геймплей, вебинар или стрим через стандартные средства Windows 10/11 (например, Win + Alt + R), то столкнулись с ограничением: встроенный Xbox Game Bar не захватывает звук из браузера или большинства приложений. Проблема усугубляется, если нужно записать защищённый контент (например, фильмы с Netflix или платформы с DRM) — многие программы просто откажутся работать или выдадут чёрный экран.

Решение зависит от задачи: для простых скринкастов подойдут бесплатные утилиты вроде OBS Studio или ShareX, а для профессиональной записи с постобработкой потребуются плагины типа NVIDIA ShadowPlay (для видеокарт GeForce) или AMD ReLive (для Radeon). Ниже разберём все варианты — от консольных команд до специализированного софта, — с учётом поддержки 4K-разрешения, частоты кадров и многоканального аудио.

1. Стандартные средства Windows: когда хватит встроенных инструментов

Windows 10 и 11 включают два инструмента для записи экрана: Xbox Game Bar и Средство захвата шагов (для отладки). Первый подходит для записи игр и некоторых приложений, второй — для создания пошаговых видеоинструкций с аннотациями. Оба варианта бесплатны, но имеют критические ограничения:

  • 🎮 Xbox Game Bar: захватывает только игры и приложения из Microsoft Store (не работает с браузерами). Звук записывается только с микрофона — системное аудио игнорируется. Максимальное разрешение: 1080p при 30 FPS.
  • 📝 Средство захвата шагов: сохраняет скриншоты с комментариями в формате .mht (не видео!). Полезно для создания мануалов, но не для динамичного контента.

Чтобы запустить запись через Xbox Game Bar:

  1. Нажмите Win + G, подтвердите, что это игра (галочка «Да, это игра»).
  2. Кликните на иконку записи (или используйте Win + Alt + R).
  3. Видео сохранится в C:\Users\[Ваше_имя]\Videos\Captures.
⚠️ Внимание: Если после нажатия Win + G появляется сообщение «Не удалось найти игру», значит приложение не поддерживается. Решение — использовать сторонний софт.
💡

Для записи системного звука через Xbox Game Bar придётся использовать обходной путь: включить в настройках Windows «Стереомикшер» (правый клик по иконке динамика → «Открыть параметры звука» → «Панель управления звуком» → вкладка «Запись» → включить «Стереомикшер»).

2. OBS Studio: лучший бесплатный инструмент для профессионалов

OBS Studio (Open Broadcaster Software) — золотой стандарт для стримеров и создателей контента. Программа поддерживает запись с экрана со звуком в разрешении до 4K@60FPS, позволяет накладывать несколько аудиодорожек (микрофон + системный звук), добавлять оверлеи и транслировать одновременно на Twitch, YouTube или Facebook Gaming. Главный плюс — отсутствие водяных знаков и ограничений по времени записи.

Чтобы настроить запись с звуком:

  1. Скачайте OBS Studio с официального сайта (избегайте репаков с вирусами!).
  2. В разделе «Источники» добавьте «Захват экрана» (Display Capture) или «Захват окна» (Window Capture).
  3. В «Микшере звука» выберите устройства:
    • 🎙️ Микрофон: ваше устройство записи (например, Yeti Blue).
    • 🔊 Рабочий стол: системное аудио (для этого в настройках звука Windows включите «Стереомикшер», как описано выше).
  • В «Настройках» → «Выход» установите формат .mp4 или .mkv, битрейт не ниже 15 000 Кбит/с для Full HD.

    3. ShareX: лёгкий скринкастер с расширенными функциями

    ShareX — бесплатная утилита с открытым исходным кодом, которая умеет не только делать скриншоты, но и записывать видео с экрана со звуком. В отличие от OBS, она проще в настройке и занимает меньше ресурсов, но не подходит для записи игр (нет поддержки DirectX/OpenGL). Идеальна для создания обучающих видео, записей вебинаров или демонстрации работы ПО.

    Особенности ShareX:

    • 🎥 Запись выбранной области экрана или всего рабочего стола.
    • 🔊 Поддержка микрофона и системного звука (требуется настройка «Стереомикшера»).
    • ⚡ Быстрый экспорт в .mp4 или .gif (для коротких клипов).
    • 📁 Автоматическая загрузка на YouTube, Google Drive или FTP.

    Инструкция по записи:

    1. Установите ShareX с официального сайта.
    2. Откройте меню «Захват» → «Запись экрана».
    3. Выберите область записи и нажмите «Запись».
    4. После завершения видео сохранится в папку по умолчанию или откроется в редакторе.
    Как включить запись системного звука в ShareX

    1. Откройте «Настройки» → «Захват» → «Запись экрана».

    2. В поле «Аудио устройство» выберите «Стереомикшер» (если он отображается).

    3. Если «Стереомикшер» отсутствует, включите его через «Панель управления звуком» Windows (как описано в разделе про OBS).

    4. Перезапустите ShareX.

    4. NVIDIA ShadowPlay и AMD ReLive: запись для геймеров

    Если ваш ПК оснащён видеокартой NVIDIA GeForce (серии GTX 10/16/20/30/40) или AMD Radeon (серии RX 400 и новее), вы можете использовать встроенные технологии записи: NVIDIA ShadowPlay (часть пакета GeForce Experience) или AMD ReLive. Эти инструменты оптимизированы для захвата игр с минимальными потерями FPS и поддерживают запись системного звука без дополнительных настроек.

    Сравнение функций:

    Функция NVIDIA ShadowPlay AMD ReLive
    Макс. разрешение 8K (на RTX 30/40) 4K
    FPS До 240 До 120
    Запись микрофона Да Да
    Поддержка DRM Нет (чёрный экран) Нет (чёрный экран)

    Как включить NVIDIA ShadowPlay:

    1. Откройте GeForce Experience (установите, если не установлен).
    2. Перейдите в «Настройки» → «Запись».
    3. Включите «Вручную» или «Автоматически» (для записей последних 5–20 минут).
    4. Нажмите Alt + F9 для начала/окончания записи.
    ⚠️ Внимание: ShadowPlay не записывает защищённый контент (например, игры с Denuvo или стриминговые сервисы). Для таких случаев используйте OBS с плагином Game Capture HD.

    5. Bandicam: платное решение для высококачественной записи

    Bandicam — проприетарная программа, популярная среди геймеров и профессионалов за поддержку записи в 4K@120FPS с минимальной нагрузкой на систему. В отличие от бесплатных аналогов, она умеет захватывать видео с внешних устройств (например, Elgato или AVerMedia) и предлагает встроенный редактор для обрезки клипов. Минус — водяной знак в бесплатной версии и стоимость лицензии (~$40).

    Ключевые фичи Bandicam:

    • 🎮 Режим игры: оптимизирован для DirectX/OpenGL/Vulkan (минимальные лаги).
    • 📹 Захват устройств: веб-камеры, IP-камеры, TV-тюнеры.
    • 🔧 Настройка кодеков: выбор между H.264, H.265 (HEVC), MPEG-1.
    • 📊 Планировщик: автоматическая запись по расписанию.

    Инструкция по настройке:

    1. Скачайте Bandicam с официального сайта.
    2. В главном окне выберите режим «Экран» или «Игра».
    3. Нажмите «Настройки» → «Звук» и выберите источники (микрофон + системный звук).
    4. В «Формат» установите .mp4 с кодеком H.264 для баланса качества и размера.
    5. Нажмите F12 для начала записи.
    💡

    Bandicam подходит для записи игр с высокими FPS, но не бесплатен. Для стриминга лучше комбинировать с OBS Studio.

    6. Запись через консоль: FFmpeg для продвинутых пользователей

    Если вам нужно автоматизировать запись экрана (например, для серверных логов или круглосуточного мониторинга), используйте FFmpeg — кроссплатформенный инструмент для работы с видео. Он позволяет записывать экран со звуком одной командой, сохраняя результат в любом формате. Минус — отсутствие графического интерфейса и необходимость знания командной строки.

    Пример команды для записи экрана и системного звука в output.mp4:

    ffmpeg -f gdigrab -framerate 30 -i desktop -f dshow -i audio="Стереомикшер (Realtek Audio)" -c:v libx264 -preset ultrafast -c:a aac -strict experimental output.mp4

    Расшифровка параметров:

    • -f gdigrab: захват экрана через GDI (Windows).
    • -framerate 30: частота кадров (увеличьте до 60 для плавности).
    • -i audio="...": источник звука (замените на имя вашего устройства).
    • -preset ultrafast: минимальная задержка (но большой размер файла).
    ⚠️ Внимание: Для работы FFmpeg требуется указать правильное имя аудиоустройства. Чтобы его узнать, выполните команду ffmpeg -list_devices true -f dshow -i dummy.

    7. Онлайн-сервисы: когда нельзя установить программу

    Если вы работаете на чужом ПК или не можете установить софт, используйте онлайн-сервисы для записи экрана. Они работают через браузер, но имеют ограничения: максимальная длительность записи (обычно 5–15 минут), водяные знаки и отсутствие поддержки системного звука (только микрофон). Популярные варианты:

    • 🌐 ScreenApp: запись до 10 минут, экспорт в .mp4 или .gif.
    • 🌐 ApowerSoft: поддержка веб-камеры, но требует установки расширения.
    • 🌐 VEED.IO: встроенный редактор, но ограничение 5 минут в бесплатной версии.

    Как записать экран через ScreenApp:

    1. Перейдите на сайт и нажмите «Start Recording».
    2. Разрешите доступ к микрофону и экрану в браузере.
    3. Выберите вкладку или весь экран для записи.
    4. По завершении скачайте файл или получите ссылку для-sharinga.
    💡

    Для записи системного звука через онлайн-сервисы используйте виртуальный аудиокабель (например, VB-Cable), направляя звук с ПК на вход микрофона в браузере.

    8. Запись защищённого контента: обход DRM

    Сервисы вроде Netflix, Disney+ или Amazon Prime Video используют технологию DRM (Digital Rights Management), которая блокирует запись экрана. Большинство программ (включая OBS и Bandicam) при попытке захвата такого контента покажут чёрный экран. Обойти ограничение можно двумя способами:

    1. Аппаратный захват: используйте внешнюю карту захвата (например, Elgato HD60 S+ или Magewell Pro Capture). Подключите ПК к карте через HDMI, а карту — к другому устройству для записи. DRM блокирует только программный захват, но не аппаратный.
    2. Виртуальная машина: запустите браузер с контентом в виртуальной машине (например, VirtualBox), а запись ведите с хост-системы. Минус — возможны лаги и потеря качества.
    ⚠️ Внимание: Обход DRM может нарушать пользовательское соглашение сервиса и местное законодательство. Используйте этот метод только для личного использования (например, сохранения лекций или вебинаров с разрешением автора).

    FAQ: Частые вопросы

    Можно ли записать экран со звуком без программ?

    Да, но с ограничениями. В Windows 10/11 используйте Xbox Game Bar (Win + Alt + R), но он не захватывает системный звук из браузера. Для записи звука включите «Стереомикшер» в настройках Windows и выберите его как источник микрофона.

    Почему в записанном видео нет звука?

    Причины:

    1. В программе не выбран источник звука (проверьте настройки микшера).
    2. Отсутствует устройство «Стереомикшер» в Windows (включите его в «Панели управления звуком»).
    3. DRM-защита контента (например, в Netflix или Spotify).
    4. Устаревшие драйвера звуковой карты (обновите через Диспетчер устройств).
    Какое разрешение выбрать для записи?

    Зависит от задачи:

    • 720p (1280×720): для вебинаров или презентаций (малый размер файла).
    • 1080p (1920×1080): универсальный вариант для YouTube или стримов.
    • 4K (3840×2160): для профессионального монтажа или геймплея (требует мощный ПК).

    Для записи текстового контента (например, программного кода) хватит 720p. Для динамичных сцен (игры, анимация) — минимум 1080p@60FPS.

    Как уменьшить размер записанного видео?

    Используйте эти методы:

    • Уменьшите битрейт в настройках программы (например, в OBS установите 8 000–12 000 Кбит/с вместо 20 000).
    • Конвертируйте видео в H.265 (HEVC) через HandBrake или FFmpeg (сжатие до 50% без потери качества).
    • Обрежьте ненужные фрагменты в редакторе (например, Shotcut или OpenShot).
    • Удалите аудиодорожку, если она не нужна (ffmpeg -i input.mp4 -an -vcodec copy output.mp4).
    Какая программа лучше для записи стримов?

    Для стриминга оптимальны:

    • OBS Studio: бесплатный, поддерживает трансляцию на Twitch/YouTube и локальную запись.
    • Streamlabs Desktop: надстройка над OBS с удобным интерфейсом и встроенными виджеты (донаты, алерты).
    • NVIDIA ShadowPlay: если стримите игры и нужна минимальная нагрузка на ПК.

    Для профессиональных стримов комбинируйте OBS с плагинами для удаления шумов (Noise Suppression) и наложения графики.
